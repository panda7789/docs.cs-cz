<span data-ttu-id="c3742-254">Izolované úložiště byste neměli používat v následujících situacích:</span><span class="sxs-lookup"><span data-stu-id="c3742-254">You should not use isolated storage in the following situations:</span></span>

- <span data-ttu-id="c3742-256">Pro uložení kódu.</span><span class="sxs-lookup"><span data-stu-id="c3742-256">To store code.</span></span>

- <span data-ttu-id="c3742-257">Pro uložení nastavení konfigurace a nasazení, které řídí správci.</span><span class="sxs-lookup"><span data-stu-id="c3742-257">To store configuration and deployment settings, which administrators control.</span></span> <span data-ttu-id="c3742-258">(Předvolby uživatele se nepovažují za nastavení konfigurace, protože je správci neovládají.)</span><span class="sxs-lookup"><span data-stu-id="c3742-258">(User preferences are not considered to be configuration settings because administrators do not control them.)</span></span>

<span data-ttu-id="c3742-259">Mnoho aplikací používá databáze k ukládání a izolaci dat. V tomto případě jeden nebo více řádků v databázi může představovat úložiště pro konkrétního uživatele.</span><span class="sxs-lookup"><span data-stu-id="c3742-259">Many applications use a database to store and isolate data, in which case one or more rows in a database might represent storage for a specific user.</span></span> <span data-ttu-id="c3742-260">Můžete se rozhodnout pro používání izolovaného úložiště namísto databáze, pokud je počet uživatelů nízký, pokud je režie používání databáze značná, nebo pokud neexistuje žádné zařízení pro databázi.</span><span class="sxs-lookup"><span data-stu-id="c3742-260">You might choose to use isolated storage instead of a database when the number of users is small, when the overhead of using a database is significant, or when no database facility exists.</span></span> <span data-ttu-id="c3742-261">Také pokud aplikace vyžaduje úložiště, které je pružnější a složitější než možnosti řádku v databázi. Izolované úložiště může poskytnout reálnou alternativu.</span><span class="sxs-lookup"><span data-stu-id="c3742-261">Also, when the application requires storage that is more flexible and complex than what a row in a database provides, isolated storage can provide a viable alternative.</span></span>
