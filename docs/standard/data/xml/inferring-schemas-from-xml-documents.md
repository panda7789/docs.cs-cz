## <a name="the-schema-inference-process"></a><span data-ttu-id="dffc4-104">Proces odvození schématu</span><span class="sxs-lookup"><span data-stu-id="dffc4-104">The Schema Inference Process</span></span>  
 <span data-ttu-id="dffc4-105"><xref:System.Xml.Schema.XmlSchemaInference> Třída <xref:System.Xml.Schema?displayProperty=nameWithType> oboru názvů se používá ke generování jednoho nebo více schémat XML Schema Definition Language (XSD) ze struktury dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="dffc4-105">The <xref:System.Xml.Schema.XmlSchemaInference> class of the <xref:System.Xml.Schema?displayProperty=nameWithType> namespace is used to generate one or more XML Schema definition language (XSD) schemas from the structure of an XML document.</span></span> <span data-ttu-id="dffc4-106">Vygenerovaná schémata lze použít k ověření původního dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="dffc4-106">The generated schemas may be used to validate the original XML document.</span></span>  
  
 <span data-ttu-id="dffc4-107">V případě, že je dokument XML zpracován <xref:System.Xml.Schema.XmlSchemaInference> třídou, <xref:System.Xml.Schema.XmlSchemaInference> vytvoří třída předpoklady o komponentách schématu, které popisují prvky a atributy v dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="dffc4-107">As an XML document is processed by the <xref:System.Xml.Schema.XmlSchemaInference> class, the <xref:System.Xml.Schema.XmlSchemaInference> class makes assumptions about the schema components that describe the elements and attributes in the XML document.</span></span> <span data-ttu-id="dffc4-108"><xref:System.Xml.Schema.XmlSchemaInference> Třída také odvodí komponenty schématu omezeného způsobu tím, že odvozuje nejvíc omezující typ pro určitý element nebo atribut.</span><span class="sxs-lookup"><span data-stu-id="dffc4-108">The <xref:System.Xml.Schema.XmlSchemaInference> class also infers schema components in a constrained way by inferring the most restrictive type for a particular element or attribute.</span></span> <span data-ttu-id="dffc4-109">Jak je shromažďováno více informací o dokumentu XML, tato omezení jsou vyzpůsobena odvozením méně omezujících typů.</span><span class="sxs-lookup"><span data-stu-id="dffc4-109">As more information about the XML document is gathered, these constraints are loosened by inferring less restrictive types.</span></span> <span data-ttu-id="dffc4-110">Nejméně omezující typ, který lze odvodit, je `xs:string`.</span><span class="sxs-lookup"><span data-stu-id="dffc4-110">The least restrictive type that can be inferred is `xs:string`.</span></span>

 <span data-ttu-id="dffc4-112">V předchozím příkladu, pokud je zjištěn `attribute1` atribut s hodnotou `6` <xref:System.Xml.Schema.XmlSchemaInference> procesu, předpokládá se, že je typu `xs:unsignedByte`.</span><span class="sxs-lookup"><span data-stu-id="dffc4-112">In the example above, when the `attribute1` attribute is encountered with a value of `6` by the <xref:System.Xml.Schema.XmlSchemaInference> process, it is assumed to be of type `xs:unsignedByte`.</span></span> <span data-ttu-id="dffc4-113">Při zjištění druhého `parent` elementu v <xref:System.Xml.Schema.XmlSchemaInference> procesu je omezení odpojeno změnou typu na `xs:string` , protože hodnota `attribute1` atributu je nyní. `A`</span><span class="sxs-lookup"><span data-stu-id="dffc4-113">When the second `parent` element is encountered by the <xref:System.Xml.Schema.XmlSchemaInference> process, the constraint is loosened by modifying the type to `xs:string` because the value of the `attribute1` attribute is now `A`.</span></span> <span data-ttu-id="dffc4-114">Podobně `minOccurs` atribut pro všechny `child` elementy odvozené ve schématu jsou rozvoditelné na `minOccurs="0"` , protože druhý nadřazený prvek nemá žádné podřízené prvky.</span><span class="sxs-lookup"><span data-stu-id="dffc4-114">Similarly, the `minOccurs` attribute for all the `child` elements inferred in the schema are loosened to `minOccurs="0"` because the second parent element has no child elements.</span></span>
