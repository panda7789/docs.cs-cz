 <span data-ttu-id="5e530-109">V prostředí s oborem názvů je volání DOM úrovně 2 `setAttribute("A:b", "123")` výsledkem atribut s předponou `A` a místním názvem `b`.</span><span class="sxs-lookup"><span data-stu-id="5e530-109">In a namespace-aware environment, the call to the DOM Level 2 `setAttribute("A:b", "123")` results in an attribute with a prefix of `A` and a local name of `b`.</span></span> <span data-ttu-id="5e530-110">Tímto způsobem funguje Microsoft .NET Framework DOM.</span><span class="sxs-lookup"><span data-stu-id="5e530-110">This is how the Microsoft .NET Framework DOM works.</span></span>
