# <a name="reading-xml-data-using-xpathdocument-and-xmldocument"></a><span data-ttu-id="46f4a-102">Načítání dat XML pomocí XPathDocument a XmlDocument</span><span class="sxs-lookup"><span data-stu-id="46f4a-102">Reading XML Data using XPathDocument and XmlDocument</span></span>

## <a name="reading-xml-documents-using-the-xpathdocument-class"></a><span data-ttu-id="46f4a-105">Čtení dokumentů XML pomocí třídy XPathDocument</span><span class="sxs-lookup"><span data-stu-id="46f4a-105">Reading XML Documents using the XPathDocument Class</span></span>  
 <span data-ttu-id="46f4a-106">Třída <xref:System.Xml.XPath.XPathDocument> poskytuje rychlé znázornění dokumentu XML, který je jen pro čtení a který je v paměti, pomocí datového modelu XPath.</span><span class="sxs-lookup"><span data-stu-id="46f4a-106">The <xref:System.Xml.XPath.XPathDocument> class provides a fast, read-only, in-memory representation of an XML document using the XPath data model.</span></span> <span data-ttu-id="46f4a-107">Instance <xref:System.Xml.XPath.XPathDocument> třídy jsou vytvořeny pomocí některého z šesti konstruktorů.</span><span class="sxs-lookup"><span data-stu-id="46f4a-107">Instances of the <xref:System.Xml.XPath.XPathDocument> class are created using one of its six constructors.</span></span> <span data-ttu-id="46f4a-108">Tyto konstruktory umožňují číst dokument XML pomocí <xref:System.IO.Stream>, <xref:System.IO.TextReader>nebo <xref:System.Xml.XmlReader> objektu a také `string` cestu k souboru XML.</span><span class="sxs-lookup"><span data-stu-id="46f4a-108">These constructors allow you to read an XML document using a <xref:System.IO.Stream>, <xref:System.IO.TextReader>, or <xref:System.Xml.XmlReader> object, as well as the `string` path to an XML file.</span></span>  
  
 <span data-ttu-id="46f4a-109">Následující příklad znázorňuje použití konstruktoru `string` třídy <xref:System.Xml.XPath.XPathDocument> ke čtení dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="46f4a-109">The following example illustrates using the <xref:System.Xml.XPath.XPathDocument> class's `string` constructor to read an XML document.</span></span>

```vb  
Dim document As XPathDocument = New XPathDocument("books.xml")  
```  
  
```csharp  
XPathDocument document = new XPathDocument("books.xml");  
```  
  
## <a name="reading-xml-documents-using-the-xmldocument-class"></a><span data-ttu-id="46f4a-110">Čtení dokumentů XML pomocí třídy XmlDocument</span><span class="sxs-lookup"><span data-stu-id="46f4a-110">Reading XML Documents using the XmlDocument Class</span></span>  
 <span data-ttu-id="46f4a-111">Třída <xref:System.Xml.XmlDocument> je upravitelná reprezentace v paměti dokumentu XML implementující XML úrovně 1 Core a Core DOM úrovně 2 (W3C model DOM (Document Object Model)).</span><span class="sxs-lookup"><span data-stu-id="46f4a-111">The <xref:System.Xml.XmlDocument> class is an editable in-memory representation of an XML document implementing W3C Document Object Model (DOM) Level 1 Core and Core DOM Level 2.</span></span> <span data-ttu-id="46f4a-112">Instance <xref:System.Xml.XmlDocument> třídy jsou vytvořeny pomocí některého z jeho tří konstruktorů.</span><span class="sxs-lookup"><span data-stu-id="46f4a-112">Instances of the <xref:System.Xml.XmlDocument> class are created using one of its three constructors.</span></span> <span data-ttu-id="46f4a-113">Můžete vytvořit nový prázdný objekt <xref:System.Xml.XmlDocument> voláním konstruktoru třídy <xref:System.Xml.XmlDocument> bez parametrů.</span><span class="sxs-lookup"><span data-stu-id="46f4a-113">You can create a new, empty <xref:System.Xml.XmlDocument> object by calling the <xref:System.Xml.XmlDocument> class constructor with no parameters.</span></span> <span data-ttu-id="46f4a-114">Po volání konstruktoru použijte metodu <xref:System.Xml.XmlDocument.Load%2A> k načtení dat XML do nového objektu <xref:System.Xml.XmlDocument> z objektu <xref:System.IO.Stream>, <xref:System.IO.TextReader>nebo <xref:System.Xml.XmlReader> a také `string` cestou k souboru XML.</span><span class="sxs-lookup"><span data-stu-id="46f4a-114">After calling the constructor, use the <xref:System.Xml.XmlDocument.Load%2A> method to load XML data into the new <xref:System.Xml.XmlDocument> object from a <xref:System.IO.Stream>, <xref:System.IO.TextReader>, or <xref:System.Xml.XmlReader> object, as well as the `string` path to an XML file.</span></span>  
  
 <span data-ttu-id="46f4a-115">Následující příklad znázorňuje použití konstruktoru třídy <xref:System.Xml.XmlDocument> bez parametrů a metody <xref:System.Xml.XmlDocument.Load%2A> pro čtení dokumentu XML.</span><span class="sxs-lookup"><span data-stu-id="46f4a-115">The following example illustrates using the <xref:System.Xml.XmlDocument> class constructor with no parameters and the <xref:System.Xml.XmlDocument.Load%2A> method to read an XML document.</span></span>

```vb  
document.Load("books.xml")  
```  
  
```csharp  
XmlDocument document = new XmlDocument();  
document.Load("books.xml");  
```
