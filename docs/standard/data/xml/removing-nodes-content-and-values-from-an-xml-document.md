# <a name="removing-nodes-content-and-values-from-an-xml-document"></a>Odebrání uzlů, obsahu a hodnot z dokumentu XML
Jakmile je XML model DOM (Document Object Model) (DOM) v paměti, můžete odebrat uzly ze stromu nebo odebrat obsah a hodnoty z určitých typů uzlů. Informace o tom, jak odebrat uzel typu list nebo celý podstrom uzlu z dokumentu, naleznete v tématu [Odebrání uzlů z modelu DOM](removing-nodes-from-the-dom.md). Informace o odebrání atributů prvku naleznete v tématu [Odebrání atributů z uzlu elementu v modelu DOM](removing-attributes-from-an-element-node-in-the-dom.md). Informace o odebrání obsahu uzlu, ale zrušení uzlu ve stromové struktuře, naleznete v tématu [Odebrání obsahu uzlu v modelu DOM](removing-node-content-in-the-dom.md).
