    <span data-ttu-id="ed8a4-113">Když *specifikátor přesnosti* ovládacích prvků zobrazuje počet zlomkových číslic ve výsledném řetězci, výsledný řetězec číslo, které se zaokrouhlí na reprezentovatelné výsledek nejblíže neomezeně přesné výsledky.</span><span class="sxs-lookup"><span data-stu-id="ed8a4-113">When *precision specifier* controls the number of fractional digits in the result string, the result string reflects a number that is rounded to a representable result nearest to the infinitely precise result.</span></span> <span data-ttu-id="ed8a4-114">Pokud existují dva stejně téměř reprezentovatelné výsledky:</span><span class="sxs-lookup"><span data-stu-id="ed8a4-114">If there are two equally near representable results:</span></span>
