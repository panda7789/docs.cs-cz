<span data-ttu-id="e2f81-105"><xref:System.TimeSpan>Formátovací řetězec definuje řetězcovou reprezentaci <xref:System.TimeSpan> hodnoty, která je výsledkem operace formátování.</span><span class="sxs-lookup"><span data-stu-id="e2f81-105">A <xref:System.TimeSpan> format string defines the string representation of a <xref:System.TimeSpan> value that results from a formatting operation.</span></span> <span data-ttu-id="e2f81-106">Vlastní formátovací řetězec se skládá z jednoho nebo více vlastních <xref:System.TimeSpan> specifikátorů formátu spolu s libovolným počtem literálních znaků.</span><span class="sxs-lookup"><span data-stu-id="e2f81-106">A custom format string consists of one or more custom <xref:System.TimeSpan> format specifiers along with any number of literal characters.</span></span> <span data-ttu-id="e2f81-107">Libovolný řetězec, který není [standardním řetězcem formátu TimeSpan](standard-timespan-format-strings.md) , je interpretován jako <xref:System.TimeSpan> řetězec vlastního formátu.</span><span class="sxs-lookup"><span data-stu-id="e2f81-107">Any string that isn't a [Standard TimeSpan format string](standard-timespan-format-strings.md) is interpreted as a custom <xref:System.TimeSpan> format string.</span></span>
