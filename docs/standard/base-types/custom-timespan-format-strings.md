> <span data-ttu-id="59c34-109">Vlastní specifikátory formátu <xref:System.TimeSpan> také nezahrnují symbol znaménka, který umožňuje odlišit záporné a kladné časové intervaly.</span><span class="sxs-lookup"><span data-stu-id="59c34-109">Custom <xref:System.TimeSpan> format specifiers also don't include a sign symbol that enables you to differentiate between negative and positive time intervals.</span></span> <span data-ttu-id="59c34-110">Chcete-li zahrnout symbol znaménka, je nutné sestavit řetězec formátu pomocí podmíněné logiky.</span><span class="sxs-lookup"><span data-stu-id="59c34-110">To include a sign symbol, you have to construct a format string by using conditional logic.</span></span> <span data-ttu-id="59c34-111">Část s [dalšími znaky](#other-characters) obsahuje příklad.</span><span class="sxs-lookup"><span data-stu-id="59c34-111">The [Other characters](#other-characters) section includes an example.</span></span>
