# <a name="get-started"></a>Začínáme

Existuje mnoho způsobů, jak začít pracovat s .NET.  Protože .NET je platforma pro masivní, existuje více článků v této dokumentaci, které ukazují, jak můžete začít s rozhraním .NET, každý z hlediska různých.

## <a name="get-started-using-net-languages"></a>Začínáme s využitím jazyků .NET

* [Jazyka C# Začínáme](../csharp/getting-started/index.md) články a [výukových programů jazyka C#](../csharp/tutorials/index.md) poskytují celou řadou způsobů, jak začít pracovat v jazyce C#-zaměřenou na způsob.

* [F # Začínáme](../fsharp/tutorials/getting-started/index.md) kurzech nejdete tři hlavní způsoby, které můžete použít F #: pomocí sady Visual Studio, Visual Studio Code nebo nástroje příkazového řádku.

* [Jazyka Visual Basic Začínáme](../visual-basic/getting-started/index.md) články poskytují pokyny pro používání jazyka Visual Basic v sadě Visual Studio.

## <a name="get-started-using-net-core"></a>Začínáme s využitím .NET Core

* [Začínáme s .NET Core](../core/get-started.md) poskytuje přehled o články, které ukazují, jak začít pracovat s .NET Core v různých operačních systémů a pomocí různých nástrojů.

* [Kurzy k .NET Core](../core/tutorials/index.md) podrobně popisují řadu způsobů, jak můžete začít s .NET Core pomocí operačního systému a nástrojů podle výběru.

## <a name="get-started-using-docker-on-net-framework"></a>Začínáme s využitím Dockeru na rozhraní .NET Framework

[Docker v rozhraní .NET Framework](../framework/docker/index.md) ukazuje, jak můžete pomocí rozhraní .NET Framework v kontejnerech Docker pro Windows.
