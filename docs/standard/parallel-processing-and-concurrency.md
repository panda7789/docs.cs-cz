# <a name="parallel-processing-concurrency-and-async-programming-in-net"></a>Paralelní zpracování, souběžnost a asynchronní programování v rozhraní .NET
Rozhraní .NET poskytuje několik způsobů, jak napsat asynchronní kód, aby vaše aplikace lépe reagovala na uživatele a napsala paralelní kód, který používá více vláken spuštění k maximalizaci výkonu počítače uživatele.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Asynchronní programování](../../docs/standard/async.md)  
 Popisuje mechanismy pro asynchronní programování poskytované rozhraním .NET.  
  
 [Paralelní programování](../../docs/standard/parallel-programming/index.md)  
 Popisuje programovací model založený na úlohách, který zjednodušuje paralelní vývoj a umožňuje psát efektivní, jemně odstupňovaný a škálovatelný paralelní kód v přirozeném idiomu, aniž byste museli pracovat přímo s vlákny nebo fondem vláken.  

 [Threading](../../docs/standard/threading/index.md)  
 Popisuje základní souběžnost a synchronizační mechanismy poskytované rozhraním .NET.
