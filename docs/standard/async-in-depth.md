```csharp

{
    // Execution is synchronous here
    var client = new HttpClient();
    
    // Execution of GetFirstCharactersCountAsync() is yielded to the caller here
    // GetStringAsync returns a Task<string>, which is *awaited*

    // becoming synchronous again.

    }

    }
```

## <a name="deeper-dive-into-task-and-tasklttgt-for-a-cpu-bound-operation"></a><span data-ttu-id="a7436-176">Dozvědět více o úkolu a úkolu&lt;T&gt; pro operace vázané na procesor</span><span class="sxs-lookup"><span data-stu-id="a7436-176">Deeper Dive into Task and Task&lt;T&gt; for a CPU-Bound Operation</span></span>

```csharp
}
```
