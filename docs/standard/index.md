# <a name="net-guide"></a>Průvodce technologií .NET

Průvodce rozhraním .NET poskytuje velké množství informací o .NET.  V závislosti na vaší znalosti s .NET je vhodné prozkoumat různé části tohoto průvodce a další části dokumentace k rozhraní .NET.

## <a name="new-to-net"></a>Novinka do .NET

Pokud chcete mít přehled o .NET, podívejte [se na to, co je .NET?](https://dotnet.microsoft.com/learn/dotnet/what-is-dotnet).

Pokud s .NET začínáte, přečtěte si článek [Začínáme](get-started.md) .

Pokud dáváte přednost průvodci hlavními funkcemi .NET, podívejte se na [prohlídku rozhraní .NET](tour.md).

Můžete si také přečíst o [komponentách architektury .NET](components.md) a získat přehled o různých "částech" rozhraní .NET a o tom, jak se vejdou do sebe.

## <a name="new-to-net-core"></a>Novinka pro .NET Core

Pokud s .NET Core začínáte, podívejte se na [Začínáme s .NET Core](../core/get-started.md).

## <a name="new-to-net-standard"></a>Novinka .NET Standard

Pokud .NET Standard, přečtěte si [.NET Standard](net-standard.md).

## <a name="porting-net-framework-code-to-net-core"></a>Přenos kódu .NET Framework do .NET Core

Pokud se chystáte portovat aplikaci, službu nebo některé součásti systému na .NET Core, přečtěte si část [přenos do .NET Core z .NET Framework](../core/porting/index.md).

## <a name="porting-a-nuget-package-from-net-framework-to-net-standard-or-net-core"></a>Přenos balíčku NuGet z .NET Framework do .NET Standard nebo .NET Core

Pokud chcete, aby balíček NuGet byl .NET Standard, přečtěte si [od .NET Framework přenos do .NET Core](../core/porting/index.md).  Nástroje pro .NET Standard a .NET Core jsou sdílené, takže obsah bude relevantní pro přenos do .NET Standard a také pro .NET Core.
