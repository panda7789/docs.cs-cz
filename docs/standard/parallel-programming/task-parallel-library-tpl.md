dynamicky, aby byly co nejefektivněji využity všechny procesory, které jsou k dispozici.</span><span class="sxs-lookup"><span data-stu-id="04893-105">The TPL scales the degree of concurrency you can maximize the performance of your code while focusing on the work that your program is designed to accomplish.</span></span>
