# <a name="for-further-reading-parallel-programming"></a>Pro další čtení (paralelní programování)

Následující zdroje obsahují další informace o paralelním programování v rozhraní .NET:

- [Vzory pro paralelní programování: Pochopení a použití paralelní chods s dokumentem .NET Framework 4](https://www.microsoft.com/download/details.aspx?id=19222) popisuje společné paralelní vzory a osvědčené postupy pro vývoj paralelních komponent s využitím těchto vzorů.

- [Paralelní programování s .NET](https://devblogs.microsoft.com/pfxteam/) blog obsahuje mnoho podrobných článků o paralelním programování v rozhraní .NET.

- [Ukázky pro paralelní programování se stránkou rozhraní .NET Framework](https://code.msdn.microsoft.com/ParExtSamples) obsahuje mnoho ukázek, které demonstrují zprostředkující a pokročilé techniky paralelního programování.
