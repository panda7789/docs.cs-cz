## <a name="examples-of-global-namespaces"></a><span data-ttu-id="a2fc6-115">Příklady globálních oborů názvů</span><span class="sxs-lookup"><span data-stu-id="a2fc6-115">Examples of Global Namespaces</span></span>  
 <span data-ttu-id="a2fc6-116">Následující příklad deklaruje výchozí globální obor názvů pomocí příkazu `Imports` a poté používá literál XML pro inicializaci objektu <xref:System.Xml.Linq.XElement> v tomto oboru názvů:</span><span class="sxs-lookup"><span data-stu-id="a2fc6-116">The following example declares a default global namespace by using the `Imports` statement, and then uses an XML literal to initialize an <xref:System.Xml.Linq.XElement> object in that namespace:</span></span>

## <a name="global-namespaces-and-embedded-expressions"></a><span data-ttu-id="a2fc6-120">Globální obory názvů a vložené výrazy</span><span class="sxs-lookup"><span data-stu-id="a2fc6-120">Global Namespaces and Embedded Expressions</span></span>

    </Root>

 <span data-ttu-id="a2fc6-126">Můžete znovu deklarovat obor názvů ve vloženém výrazu následujícím způsobem:</span><span class="sxs-lookup"><span data-stu-id="a2fc6-126">You could re-declare the namespace in the embedded expression, as follows:</span></span>

        <%= <Child xmlns="http://www.adventure-works.com"/> %>  
    </Root>  
Console.WriteLine(root)

 <span data-ttu-id="a2fc6-127">Tento příklad vytvoří následující výstup:</span><span class="sxs-lookup"><span data-stu-id="a2fc6-127">This example produces the following output:</span></span>

<Root xmlns="http://www.adventure-works.com">  
  <Child xmlns="http://www.adventure-works.com" />  
</Root>

 <span data-ttu-id="a2fc6-128">Je to ale více náročný na použití než globální výchozí obor názvů, což je lepší přístup.</span><span class="sxs-lookup"><span data-stu-id="a2fc6-128">However, this is more cumbersome to use than the global default namespace, which is a better approach.</span></span> <span data-ttu-id="a2fc6-129">S globálním výchozím oborem názvů můžete použít literály XML bez deklarování oborů názvů.</span><span class="sxs-lookup"><span data-stu-id="a2fc6-129">With the global default namespace, you can use XML literals without declaring namespaces.</span></span> <span data-ttu-id="a2fc6-130">Výsledný kód XML bude v globálně deklarovaném výchozím oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="a2fc6-130">The resulting XML will be in the globally-declared default namespace.</span></span>

    Sub Main()  
        Dim root As XElement = <Root>  
                                   <%= <Child/> %>  
                               </Root>

 <span data-ttu-id="a2fc6-131">Tento příklad vytvoří následující výstup:</span><span class="sxs-lookup"><span data-stu-id="a2fc6-131">This example produces the following output:</span></span>
