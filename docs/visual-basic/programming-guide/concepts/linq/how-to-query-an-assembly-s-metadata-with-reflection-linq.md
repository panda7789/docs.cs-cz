# <a name="how-to-query-an-assemblys-metadata-with-reflection-linq-visual-basic"></a><span data-ttu-id="f5dc6-102">Postupy: vytvoření dotazu na metadata sestavení s reflexí (LINQ) (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="f5dc6-102">How to: Query An Assembly's Metadata with Reflection (LINQ) (Visual Basic)</span></span>
<span data-ttu-id="f5dc6-103">Následující příklad ukazuje, jak lze pomocí technologie LINQ použít s reflexí k načtení konkrétních metadat pro metody, které odpovídají zadanému vyhledávacímu kritériu.</span><span class="sxs-lookup"><span data-stu-id="f5dc6-103">The following example shows how LINQ can be used with reflection to retrieve specific metadata about methods that match a specified search criterion.</span></span> <span data-ttu-id="f5dc6-104">V tomto případě dotaz vyhledá názvy všech metod v sestavení, které vracejí výčtové typy jako pole.</span><span class="sxs-lookup"><span data-stu-id="f5dc6-104">In this case, the query will find the names of all the methods in the assembly that return enumerable types such as arrays.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f5dc6-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="f5dc6-105">Example</span></span>  
  
```vb
Imports System.Linq
Imports System.Reflection  

Module Module1  
    Sub Main()  
        Dim asmbly As Assembly =
            Assembly.Load("System.Core, Version=3.5.0.0, Culture=neutral, PublicKeyToken= b77a5c561934e089")  
  
        Dim pubTypesQuery = From type In asmbly.GetTypes()
                            Where type.IsPublic
                            From method In type.GetMethods()
                            Where method.ReturnType.IsArray = True
                            Let name = method.ToString()
                            Let typeName = type.ToString()
                            Group name By typeName Into methodNames = Group  
  
        Console.WriteLine("Getting ready to iterate")  
        For Each item In pubTypesQuery  
            Console.WriteLine(item.methodNames)  
  
            For Each type In item.methodNames  
                Console.WriteLine(" " & type)  
            Next  
        Next  
        Console.WriteLine("Press any key to exit... ")  
        Console.ReadKey()  
    End Sub  
End Module  
```  
  
<span data-ttu-id="f5dc6-106">V příkladu se používá <xref:System.Reflection.Assembly.GetTypes%2A?displayProperty=nameWithType> Metoda k vrácení pole typů v zadaném sestavení.</span><span class="sxs-lookup"><span data-stu-id="f5dc6-106">The example uses the <xref:System.Reflection.Assembly.GetTypes%2A?displayProperty=nameWithType> method to return an array of types in the specified assembly.</span></span> <span data-ttu-id="f5dc6-107">Filtr [klauzule WHERE](../../../language-reference/queries/where-clause.md) je použit, aby byly vráceny pouze veřejné typy.</span><span class="sxs-lookup"><span data-stu-id="f5dc6-107">The [Where Clause](../../../language-reference/queries/where-clause.md) filter is applied so that only public types are returned.</span></span> <span data-ttu-id="f5dc6-108">Pro každý veřejný typ je poddotaz generován pomocí <xref:System.Reflection.MethodInfo> pole, které je vráceno <xref:System.Type.GetMethods%2A?displayProperty=nameWithType> voláním.</span><span class="sxs-lookup"><span data-stu-id="f5dc6-108">For each public type, a subquery is generated by using the <xref:System.Reflection.MethodInfo> array that is returned from the <xref:System.Type.GetMethods%2A?displayProperty=nameWithType> call.</span></span> <span data-ttu-id="f5dc6-109">Tyto výsledky jsou filtrovány tak, aby vracely pouze metody, jejichž návratový typ je pole nebo jiný typ, který implementuje <xref:System.Collections.Generic.IEnumerable%601> .</span><span class="sxs-lookup"><span data-stu-id="f5dc6-109">These results are filtered to return only those methods whose return type is an array or else a type that implements <xref:System.Collections.Generic.IEnumerable%601>.</span></span> <span data-ttu-id="f5dc6-110">Nakonec jsou tyto výsledky seskupeny pomocí názvu typu jako klíč.</span><span class="sxs-lookup"><span data-stu-id="f5dc6-110">Finally, these results are grouped by using the type name as a key.</span></span>
