## <a name="similarities"></a><span data-ttu-id="dc294-109">Podobnosti</span><span class="sxs-lookup"><span data-stu-id="dc294-109">Similarities</span></span>  
 <span data-ttu-id="dc294-110">Struktury a třídy jsou podobné v následujících ohledech:</span><span class="sxs-lookup"><span data-stu-id="dc294-110">Structures and classes are similar in the following respects:</span></span>  
  
- <span data-ttu-id="dc294-111">Oba jsou typy *kontejnerů* , což znamená, že obsahují jiné typy jako členy.</span><span class="sxs-lookup"><span data-stu-id="dc294-111">Both are *container* types, meaning that they contain other types as members.</span></span>  
  
- <span data-ttu-id="dc294-112">Oba mají členy, které mohou zahrnovat konstruktory, metody, vlastnosti, pole, konstanty, výčty, události a obslužné rutiny událostí.</span><span class="sxs-lookup"><span data-stu-id="dc294-112">Both have members, which can include constructors, methods, properties, fields, constants, enumerations, events, and event handlers.</span></span> <span data-ttu-id="dc294-113">Nepleťte si však tyto členy s deklarovanými *prvky* struktury.</span><span class="sxs-lookup"><span data-stu-id="dc294-113">However, do not confuse these members with the declared *elements* of a structure.</span></span>  
  
- <span data-ttu-id="dc294-114">Členové obou můžou mít individuální úrovně přístupu.</span><span class="sxs-lookup"><span data-stu-id="dc294-114">Members of both can have individualized access levels.</span></span> <span data-ttu-id="dc294-115">Například jeden člen může být deklarován `Public` a další `Private`.</span><span class="sxs-lookup"><span data-stu-id="dc294-115">For example, one member can be declared `Public` and another `Private`.</span></span>  
  
- <span data-ttu-id="dc294-116">Oba můžou implementovat rozhraní.</span><span class="sxs-lookup"><span data-stu-id="dc294-116">Both can implement interfaces.</span></span>  
  
- <span data-ttu-id="dc294-117">Oba můžou mít sdílené konstruktory s parametry nebo bez.</span><span class="sxs-lookup"><span data-stu-id="dc294-117">Both can have shared constructors, with or without parameters.</span></span>  
  
- <span data-ttu-id="dc294-118">Oba můžou vystavit *výchozí vlastnost*za předpokladu, že vlastnost přebírá aspoň jeden parametr.</span><span class="sxs-lookup"><span data-stu-id="dc294-118">Both can expose a *default property*, provided that property takes at least one parameter.</span></span>  
  
- <span data-ttu-id="dc294-119">Obě můžou deklarovat a vyvolat události a obě můžou deklarovat delegáty.</span><span class="sxs-lookup"><span data-stu-id="dc294-119">Both can declare and raise events, and both can declare delegates.</span></span>  
  
## <a name="differences"></a><span data-ttu-id="dc294-120">Rozdíly</span><span class="sxs-lookup"><span data-stu-id="dc294-120">Differences</span></span>  
 <span data-ttu-id="dc294-121">Struktury a třídy se liší v následujících údajích:</span><span class="sxs-lookup"><span data-stu-id="dc294-121">Structures and classes differ in the following particulars:</span></span>  
  
- <span data-ttu-id="dc294-122">Struktury jsou *typy hodnot*; třídy jsou *odkazové typy*.</span><span class="sxs-lookup"><span data-stu-id="dc294-122">Structures are *value types*; classes are *reference types*.</span></span> <span data-ttu-id="dc294-123">Proměnná typu struktury obsahuje data struktury místo toho, aby obsahovala odkaz na data jako typ třídy.</span><span class="sxs-lookup"><span data-stu-id="dc294-123">A variable of a structure type contains the structure's data, rather than containing a reference to the data as a class type does.</span></span>  
  
- <span data-ttu-id="dc294-124">Struktury používají alokaci zásobníku; třídy používají přidělení haldy.</span><span class="sxs-lookup"><span data-stu-id="dc294-124">Structures use stack allocation; classes use heap allocation.</span></span>  
  
- <span data-ttu-id="dc294-125">Všechny prvky struktury jsou ve výchozím nastavení `Public`; proměnné a konstanty třídy jsou `Private` ve výchozím nastavení, zatímco ostatní členové třídy jsou ve výchozím nastavení `Public`.</span><span class="sxs-lookup"><span data-stu-id="dc294-125">All structure elements are `Public` by default; class variables and constants are `Private` by default, while other class members are `Public` by default.</span></span> <span data-ttu-id="dc294-126">Toto chování pro členy třídy zajišťuje kompatibilitu s výchozím systémem Visual Basic 6,0.</span><span class="sxs-lookup"><span data-stu-id="dc294-126">This behavior for class members provides compatibility with the Visual Basic 6.0 system of defaults.</span></span>  
  
- <span data-ttu-id="dc294-127">Struktura musí mít alespoň jednu nesdílenou proměnnou nebo nevlastní nesdílenou, nevlastní prvek události; Třída může být zcela prázdná.</span><span class="sxs-lookup"><span data-stu-id="dc294-127">A structure must have at least one nonshared variable or nonshared, noncustom event element; a class can be completely empty.</span></span>  
  
- <span data-ttu-id="dc294-128">Prvky struktury nelze deklarovat jako `Protected`; Členové třídy mohou.</span><span class="sxs-lookup"><span data-stu-id="dc294-128">Structure elements cannot be declared as `Protected`; class members can.</span></span>  
  
- <span data-ttu-id="dc294-129">Postup struktury může zpracovávat události pouze v případě, že se jedná o [sdílený](../../../../visual-basic/language-reference/modifiers/shared.md)`Sub` proceduru a pouze pomocí [příkazu AddHandler](../../../../visual-basic/language-reference/statements/addhandler-statement.md); libovolná procedura třídy může zpracovávat události pomocí klíčového slova [Handles](../../../../visual-basic/language-reference/statements/handles-clause.md) nebo příkazu `AddHandler`.</span><span class="sxs-lookup"><span data-stu-id="dc294-129">A structure procedure can handle events only if it is a [Shared](../../../../visual-basic/language-reference/modifiers/shared.md)`Sub` procedure, and only by means of the [AddHandler Statement](../../../../visual-basic/language-reference/statements/addhandler-statement.md); any class procedure can handle events, using either the [Handles](../../../../visual-basic/language-reference/statements/handles-clause.md) keyword or the `AddHandler` statement.</span></span> <span data-ttu-id="dc294-130">Další informace najdete v tématu [události](../../../../visual-basic/programming-guide/language-features/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="dc294-130">For more information, see [Events](../../../../visual-basic/programming-guide/language-features/events/index.md).</span></span>  
  
- <span data-ttu-id="dc294-131">Deklarace proměnných struktury nemůžou určovat Inicializátory ani počáteční velikosti polí. deklarace proměnných třídy mohou.</span><span class="sxs-lookup"><span data-stu-id="dc294-131">Structure variable declarations cannot specify initializers or initial sizes for arrays; class variable declarations can.</span></span>  
  
- <span data-ttu-id="dc294-132">Struktury implicitně dědí z třídy <xref:System.ValueType?displayProperty=nameWithType> a nemůžou dědit z žádného jiného typu. třídy mohou dědit z jakékoliv třídy nebo jiných tříd než <xref:System.ValueType?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="dc294-132">Structures implicitly inherit from the <xref:System.ValueType?displayProperty=nameWithType> class and cannot inherit from any other type; classes can inherit from any class or classes other than <xref:System.ValueType?displayProperty=nameWithType>.</span></span>  
  
- <span data-ttu-id="dc294-133">Struktury nejsou dědičné; třídy jsou.</span><span class="sxs-lookup"><span data-stu-id="dc294-133">Structures are not inheritable; classes are.</span></span>  
  
- <span data-ttu-id="dc294-134">Struktury nejsou nikdy ukončeny, takže modul CLR (Common Language Runtime) nikdy nevolá metodu <xref:System.Object.Finalize%2A> v jakékoli struktuře; třídy jsou ukončeny systémem uvolňování paměti (GC), který volá <xref:System.Object.Finalize%2A> na třídu, když zjistí, že neexistují žádné aktivní odkazy.</span><span class="sxs-lookup"><span data-stu-id="dc294-134">Structures are never terminated, so the common language runtime (CLR) never calls the <xref:System.Object.Finalize%2A> method on any structure; classes are terminated by the garbage collector (GC), which calls <xref:System.Object.Finalize%2A> on a class when it detects there are no active references remaining.</span></span>  
  
- <span data-ttu-id="dc294-135">Struktura nevyžaduje konstruktor. Třída dělá.</span><span class="sxs-lookup"><span data-stu-id="dc294-135">A structure does not require a constructor; a class does.</span></span>  
  
- <span data-ttu-id="dc294-136">Struktury mohou mít nesdílené konstruktory pouze v případě, že převezmou parametry; třídy mohou mít s parametry nebo bez nich.</span><span class="sxs-lookup"><span data-stu-id="dc294-136">Structures can have nonshared constructors only if they take parameters; classes can have them with or without parameters.</span></span>  
  
 <span data-ttu-id="dc294-137">Každá struktura má implicitní veřejný konstruktor bez parametrů.</span><span class="sxs-lookup"><span data-stu-id="dc294-137">Every structure has an implicit public constructor without parameters.</span></span> <span data-ttu-id="dc294-138">Tento konstruktor inicializuje všechny prvky dat struktury na jejich výchozí hodnoty.</span><span class="sxs-lookup"><span data-stu-id="dc294-138">This constructor initializes all the structure's data elements to their default values.</span></span> <span data-ttu-id="dc294-139">Toto chování nelze předefinovat.</span><span class="sxs-lookup"><span data-stu-id="dc294-139">You cannot redefine this behavior.</span></span>  
  
## <a name="instances-and-variables"></a><span data-ttu-id="dc294-140">Instance a proměnné</span><span class="sxs-lookup"><span data-stu-id="dc294-140">Instances and Variables</span></span>  
 <span data-ttu-id="dc294-141">Vzhledem k tomu, že struktury jsou typy hodnot, každá proměnná struktury je trvale svázána s instancí jednotlivé struktury.</span><span class="sxs-lookup"><span data-stu-id="dc294-141">Because structures are value types, each structure variable is permanently bound to an individual structure instance.</span></span> <span data-ttu-id="dc294-142">Ale třídy jsou odkazové typy a proměnná objektu může odkazovat na různé instance třídy v různých časech.</span><span class="sxs-lookup"><span data-stu-id="dc294-142">But classes are reference types, and an object variable can refer to various class instances at different times.</span></span> <span data-ttu-id="dc294-143">Toto rozlišení má vliv na využití struktur a tříd následujícími způsoby:</span><span class="sxs-lookup"><span data-stu-id="dc294-143">This distinction affects your usage of structures and classes in the following ways:</span></span>  
  
- <span data-ttu-id="dc294-144">**Operace.**</span><span class="sxs-lookup"><span data-stu-id="dc294-144">**Initialization.**</span></span> <span data-ttu-id="dc294-145">Proměnná struktury implicitně obsahuje inicializaci elementů pomocí konstruktoru bez parametrů struktury.</span><span class="sxs-lookup"><span data-stu-id="dc294-145">A structure variable implicitly includes an initialization of the elements using the structure's parameterless constructor.</span></span> <span data-ttu-id="dc294-146">Proto je `Dim s As struct1` ekvivalentem `Dim s As struct1 = New struct1()`.</span><span class="sxs-lookup"><span data-stu-id="dc294-146">Therefore, `Dim s As struct1` is equivalent to `Dim s As struct1 = New struct1()`.</span></span>  
  
- <span data-ttu-id="dc294-147">**Přiřazují se proměnné.**</span><span class="sxs-lookup"><span data-stu-id="dc294-147">**Assigning Variables.**</span></span> <span data-ttu-id="dc294-148">Když přiřadíte jednu proměnnou struktury k druhé nebo předáte instanci struktury do argumentu procedury, aktuální hodnoty všech prvků proměnné se zkopírují do nové struktury.</span><span class="sxs-lookup"><span data-stu-id="dc294-148">When you assign one structure variable to another, or pass a structure instance to a procedure argument, the current values of all the variable elements are copied to the new structure.</span></span> <span data-ttu-id="dc294-149">Když přiřadíte jednu proměnnou objektu k druhé nebo předáte proměnnou objektu proceduře, je zkopírován pouze odkazový ukazatel.</span><span class="sxs-lookup"><span data-stu-id="dc294-149">When you assign one object variable to another, or pass an object variable to a procedure, only the reference pointer is copied.</span></span>  
  
- <span data-ttu-id="dc294-150">**Nepřiřazuje se nic.**</span><span class="sxs-lookup"><span data-stu-id="dc294-150">**Assigning Nothing.**</span></span> <span data-ttu-id="dc294-151">Proměnné struktury můžete přiřadit hodnotu [Nothing](../../../../visual-basic/language-reference/nothing.md) , ale instance bude nadále přidružena k proměnné.</span><span class="sxs-lookup"><span data-stu-id="dc294-151">You can assign the value [Nothing](../../../../visual-basic/language-reference/nothing.md) to a structure variable, but the instance continues to be associated with the variable.</span></span> <span data-ttu-id="dc294-152">Můžete i nadále volat své metody a přistupovat k jejím datovým prvkům, i když jsou proměnné prvky opětovně inicializovány přiřazením.</span><span class="sxs-lookup"><span data-stu-id="dc294-152">You can still call its methods and access its data elements, although variable elements are reinitialized by the assignment.</span></span>  
  
     <span data-ttu-id="dc294-153">Naproti tomu, pokud nastavíte proměnnou objektu na `Nothing`, oddělíte ji z jakékoli instance třídy a nebudete mít přístup k žádným členům přes proměnnou, dokud k ní nepřiřadíte jinou instanci.</span><span class="sxs-lookup"><span data-stu-id="dc294-153">In contrast, if you set an object variable to `Nothing`, you dissociate it from any class instance, and you cannot access any members through the variable until you assign another instance to it.</span></span>  
  
- <span data-ttu-id="dc294-154">**Více instancí.**</span><span class="sxs-lookup"><span data-stu-id="dc294-154">**Multiple Instances.**</span></span> <span data-ttu-id="dc294-155">K objektové proměnné může být přiřazena jiná instance třídy v různých časech a několik proměnných objektu může ve stejnou dobu odkazovat na stejnou instanci třídy.</span><span class="sxs-lookup"><span data-stu-id="dc294-155">An object variable can have different class instances assigned to it at different times, and several object variables can refer to the same class instance at the same time.</span></span> <span data-ttu-id="dc294-156">Změny, které provedete v hodnotách členů třídy, ovlivňují tyto členy při zpřístupnění prostřednictvím jiné proměnné odkazující na stejnou instanci.</span><span class="sxs-lookup"><span data-stu-id="dc294-156">Changes you make to the values of class members affect those members when accessed through another variable pointing to the same instance.</span></span>  
  
     <span data-ttu-id="dc294-157">Prvky struktury jsou však izolované v rámci své vlastní instance.</span><span class="sxs-lookup"><span data-stu-id="dc294-157">Structure elements, however, are isolated within their own instance.</span></span> <span data-ttu-id="dc294-158">Změny jejich hodnot se neprojeví v žádné jiné proměnné struktury, ani v jiných instancích stejné deklarace `Structure`.</span><span class="sxs-lookup"><span data-stu-id="dc294-158">Changes to their values are not reflected in any other structure variables, even in other instances of the same `Structure` declaration.</span></span>  
  
- <span data-ttu-id="dc294-159">**Porovnávání.**</span><span class="sxs-lookup"><span data-stu-id="dc294-159">**Equality.**</span></span> <span data-ttu-id="dc294-160">Testování rovnosti dvou struktur musí být provedeno pomocí testu elementu po prvku.</span><span class="sxs-lookup"><span data-stu-id="dc294-160">Equality testing of two structures must be performed with an element-by-element test.</span></span> <span data-ttu-id="dc294-161">Dvě objektové proměnné lze porovnat pomocí metody <xref:System.Object.Equals%2A>.</span><span class="sxs-lookup"><span data-stu-id="dc294-161">Two object variables can be compared using the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="dc294-162"><xref:System.Object.Equals%2A> určuje, zda dvě proměnné odkazují na stejnou instanci.</span><span class="sxs-lookup"><span data-stu-id="dc294-162"><xref:System.Object.Equals%2A> indicates whether the two variables point to the same instance.</span></span>
