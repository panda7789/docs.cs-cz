 <span data-ttu-id="dd0c2-180">Převod z *typu odkazu* kopíruje pouze ukazatel na hodnotu.</span><span class="sxs-lookup"><span data-stu-id="dd0c2-180">A conversion from a *reference type* copies only the pointer to the value.</span></span> <span data-ttu-id="dd0c2-181">Samotná hodnota není ani zkopírována ani měněna žádným způsobem.</span><span class="sxs-lookup"><span data-stu-id="dd0c2-181">The value itself is neither copied nor changed in any way.</span></span> <span data-ttu-id="dd0c2-182">Jediná věc, kterou lze změnit, je datový typ proměnné obsahující ukazatel.</span><span class="sxs-lookup"><span data-stu-id="dd0c2-182">The only thing that can change is the data type of the variable holding the pointer.</span></span> <span data-ttu-id="dd0c2-183">V následujícím příkladu je datový typ převeden z odvozené třídy na svou základní třídu, ale objekt, na který obě proměnné nyní odkazuje, zůstane beze změny.</span><span class="sxs-lookup"><span data-stu-id="dd0c2-183">In the following example, the data type is converted from the derived class to its base class, but the object that both variables now point to is unchanged.</span></span>

```vb  
' Assume class cSquare inherits from class cShape.  
Dim shape As cShape  
Dim square As cSquare = New cSquare

```
