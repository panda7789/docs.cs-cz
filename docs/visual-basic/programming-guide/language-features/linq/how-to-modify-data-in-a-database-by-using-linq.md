### <a name="to-add-code-to-modify-the-database-and-display-the-results"></a><span data-ttu-id="f1f20-128">Přidání kódu pro úpravu databáze a zobrazení výsledků</span><span class="sxs-lookup"><span data-stu-id="f1f20-128">To add code to modify the database and display the results</span></span>

     <span data-ttu-id="f1f20-137">Dvojitým kliknutím na formulář Windows, Form1 přidáte kód do události <xref:System.Windows.Forms.Form.Load> pro dotaz na tabulku Customers, která je vystavena jako vlastnost <xref:System.Data.Linq.DataContext>.</span><span class="sxs-lookup"><span data-stu-id="f1f20-137">Double-click the Windows Form, Form1, to add code to the <xref:System.Windows.Forms.Form.Load> event to query the Customers table that is exposed as a property of your <xref:System.Data.Linq.DataContext>.</span></span> <span data-ttu-id="f1f20-138">Přidejte následující kód:</span><span class="sxs-lookup"><span data-stu-id="f1f20-138">Add the following code:</span></span>

    ```vb
    Private db As northwindDataContext

    Private Sub Form1_Load(ByVal sender As System.Object,
                           ByVal e As System.EventArgs
                          ) Handles MyBase.Load
      db = New northwindDataContext()

      RefreshData()
    End Sub

    Private Sub RefreshData()
      Dim customers = From cust In db.Customers
                      Where cust.City(0) = "W"
                      Select cust

      DataGridView1.DataSource = customers
    End Sub
    ```

3. <span data-ttu-id="f1f20-139">Z **panelu nástrojů**přetáhněte do formuláře tři ovládací prvky <xref:System.Windows.Forms.Button>.</span><span class="sxs-lookup"><span data-stu-id="f1f20-139">From the **Toolbox**, drag three <xref:System.Windows.Forms.Button> controls onto the form.</span></span> <span data-ttu-id="f1f20-140">Vyberte první ovládací prvek `Button`.</span><span class="sxs-lookup"><span data-stu-id="f1f20-140">Select the first `Button` control.</span></span> <span data-ttu-id="f1f20-141">V okně **vlastnosti** nastavte `Name` ovládacího prvku `Button` na `AddButton` a `Text` na `Add`.</span><span class="sxs-lookup"><span data-stu-id="f1f20-141">In the **Properties** window, set the `Name` of the `Button` control to `AddButton` and the `Text` to `Add`.</span></span> <span data-ttu-id="f1f20-142">Vyberte druhé tlačítko a vlastnost `Name` nastavte na `UpdateButton` a vlastnost `Text` na `Update`.</span><span class="sxs-lookup"><span data-stu-id="f1f20-142">Select the second button and set the `Name` property to `UpdateButton` and the `Text` property to `Update`.</span></span> <span data-ttu-id="f1f20-143">Vyberte třetí tlačítko a vlastnost `Name` nastavte na `DeleteButton` a vlastnost `Text` na `Delete`.</span><span class="sxs-lookup"><span data-stu-id="f1f20-143">Select the third button and set the `Name` property to `DeleteButton` and the `Text` property to `Delete`.</span></span>

    ```vb

                                   ByVal e As System.EventArgs
                                  ) Handles UpdateButton.Click
      Dim updateCust = (From cust In db.Customers
                        Where cust.CustomerID = "JILLF").ToList()(0)

      updateCust.ContactName = "Jill Shrader"
      updateCust.Country = "Wales"
      updateCust.CompanyName = "Red Yonder Airlines"
      updateCust.City = "Cardiff"

      Try
        db.SubmitChanges()
      Catch
        ' Handle exception.
      End Try

      RefreshData()
    End Sub
    ```
