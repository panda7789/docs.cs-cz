 <span data-ttu-id="2ef6a-107">Předat argument *hodnotou* zadáním [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) – klíčové slovo pro odpovídající parametr v definici postupu.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-107">You pass an argument *by value* by specifying the [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="2ef6a-108">Pokud použijete tento mechanismus předávání, Visual Basic hodnotu základní programovací element zkopíruje do místní proměnné v postupu.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-108">When you use this passing mechanism, Visual Basic copies the value of the underlying programming element into a local variable in the procedure.</span></span> <span data-ttu-id="2ef6a-109">Kód postup nemá přístup k podkladové element v volající kód.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-109">The procedure code does not have any access to the underlying element in the calling code.</span></span>

 <span data-ttu-id="2ef6a-117">Ale mechanismus předávání a typ elementu spolu souvisejí.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-117">However, the passing mechanism and element type are interrelated.</span></span> <span data-ttu-id="2ef6a-118">Hodnota typu odkaz je ukazatel na data jinde v paměti.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-118">The value of a reference type is a pointer to the data elsewhere in memory.</span></span> <span data-ttu-id="2ef6a-119">To znamená, že při předání typu odkazu hodnotou, kód postupu má ukazatel na základní element data, i když nemá přístup k podkladové elementu samotného.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-119">This means that when you pass a reference type by value, the procedure code has a pointer to the underlying element's data, even though it cannot access the underlying element itself.</span></span> <span data-ttu-id="2ef6a-120">Pokud element je proměnná typu pole, například kód postup nemá přístup k proměnné sám sebe, ale má přístup k poli členy.</span><span class="sxs-lookup"><span data-stu-id="2ef6a-120">For example, if the element is an array variable, the procedure code does not have access to the variable itself, but it can access the array members.</span></span>
