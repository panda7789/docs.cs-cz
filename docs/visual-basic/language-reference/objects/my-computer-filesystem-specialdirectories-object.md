## <a name="remarks"></a>Poznámky  
 Informace o metodách a vlastnostech `My.Computer.FileSystem.SpecialDirectories` objektu naleznete v tématu <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories> .  
  
 Další informace najdete v tématu [Postup: načtení obsahu adresáře moje dokumenty](../../developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).
