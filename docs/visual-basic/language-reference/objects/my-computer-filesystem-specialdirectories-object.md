# <a name="mycomputerfilesystemspecialdirectories-object"></a>My.Computer.FileSystem.SpecialDirectories – objekt
Poskytuje vlastnosti pro přístup k často odkazovaným adresářům.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metodách a vlastnostech objektu `My.Computer.FileSystem.SpecialDirectories` naleznete v tématu <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Další informace najdete v tématu [Postup: načtení obsahu adresáře moje dokumenty](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).  
  
## <a name="requirements"></a>Požadavky  
 **Obor názvů:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Třída:** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> (poskytuje přístup k <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>)  
  
 **Sestavení:** Knihovna Visual Basic runtime (v souboru Microsoft. VisualBasic. dll)
