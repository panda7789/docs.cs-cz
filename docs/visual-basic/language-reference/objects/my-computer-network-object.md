## <a name="remarks"></a>Poznámky  
 Informace o metodách a vlastnostech `My.Computer.Network` objektu naleznete v tématu <xref:Microsoft.VisualBasic.Devices.Network> .  
  
 Další informace najdete v tématu [provádění síťových operací](../../developing-apps/programming/computer-resources/performing-network-operations.md).
