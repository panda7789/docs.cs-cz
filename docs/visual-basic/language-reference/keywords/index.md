# <a name="keywords-visual-basic"></a>Klíčová slova (Visual Basic)

Následující tabulky uvádějí všechna Visual Basic klíčová slova jazyka.

## <a name="reserved-keywords"></a>Vyhrazená klíčová slova

Následující klíčová slova jsou *vyhrazena*, což znamená, že je nelze použít jako názvy pro programovací prvky, jako jsou proměnné nebo procedury. Toto omezení můžete obejít uzavřením názvu do hranatých závorek ( `[]` ). Další informace naleznete v tématu "řídicí názvy" v [deklarovaných názvech elementů](../../programming-guide/language-features/declared-elements/declared-element-names.md).

> [!NOTE]
> Nedoporučujeme používat řídicí názvy, protože to může způsobit těžkou čitelnost kódu a může vést k drobným chybám, které může být obtížné najít.
