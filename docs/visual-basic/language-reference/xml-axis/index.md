# <a name="xml-axis-properties-visual-basic"></a>Vlastnosti osy XML (Visual Basic)
Témata v této části dokumentují syntaxi vlastností osy XML v Visual Basic. Vlastnosti osy XML usnadňují přístup k XML přímo ve vašem kódu.  
  
## <a name="in-this-section"></a>V tomto oddílu  
  
|Téma|Description|  
|-----------|-----------------|  
|[Vlastnost osy atributu XML](xml-attribute-axis-property.md)|Popisuje, jak získat přístup k atributům <xref:System.Xml.Linq.XElement> objektu.|  
|[Vlastnost osy podřízeného XML](xml-child-axis-property.md)|Popisuje, jak přistupovat k podřízeným <xref:System.Xml.Linq.XElement> objektům objektu.|  
|[Vlastnost osy nástupce XML](xml-descendant-axis-property.md)|Popisuje, jak přistupovat k následníkům <xref:System.Xml.Linq.XElement> objektu.|  
|[Vlastnost indexeru rozšíření](extension-indexer-property.md)|Popisuje, jak přistupovat k jednotlivým prvkům v kolekci <xref:System.Xml.Linq.XElement> <xref:System.Xml.Linq.XAttribute> objektů nebo.|  
|[Vlastnost hodnoty XML](xml-value-property.md)|Popisuje, jak přistupovat k hodnotě prvního prvku kolekce <xref:System.Xml.Linq.XElement> nebo <xref:System.Xml.Linq.XAttribute> objektů.|
