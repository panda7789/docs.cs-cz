# <a name="array-subscript-expression-missing"></a>Výraz dolního indexu pole chybí.
Inicializace pole opustí jeden nebo více dolních indexů, které definují meze pole. Například příkaz může obsahovat výraz `myArray (5,5,,10)` , který opustí třetí dolní index.  
  
 **ID chyby:** BC30306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte chybějící dolní index.
