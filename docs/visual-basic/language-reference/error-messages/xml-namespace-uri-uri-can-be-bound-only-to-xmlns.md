# <a name="xml-namespace-uri-httpwwww3orgxml1998namespace-can-be-bound-only-to-xmlns"></a>Identifikátor URI oboru názvů XML `http://www.w3.org/XML/1998/namespace` ; může být vázaný jenom na xmlns.
Identifikátor URI `http://www.w3.org/XML/1998/namespace` se používá v deklaraci oboru názvů XML. Tento identifikátor URI je vyhrazený obor názvů a nemůže být zahrnut v deklaraci oboru názvů XML.  
  
 **ID chyby:** BC31183  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
Odeberte deklaraci oboru názvů XML nebo nahraďte identifikátor URI `http://www.w3.org/XML/1998/namespace` platným identifikátorem URI oboru názvů.
