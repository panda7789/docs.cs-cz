# <a name="optional-expected"></a>Bylo očekáváno klíčové slovo 'Optional'.
Volitelný argument deklarace procedury je následován povinným argumentem. Každý argument následující po volitelném argumentu musí být také volitelný.  
  
 **ID chyby:** BC30202  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pokud má být argument požadován, přesuňte jej před první volitelný argument v seznamu argumentů.  
  
2. Pokud argument má být nepovinný, použijte `Optional` klíčové slovo.
