# <a name="typename-is-a-type-and-cannot-be-used-as-an-expression"></a>'\<typename> je typ a nelze jej použít jako výraz.
K názvu typu dojde v případě, že je vyžadován výraz. Výraz se musí skládat z některé kombinace proměnných, konstant, literálů, vlastností a `Function` volání procedur.  
  
 **ID chyby:** BC30108  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte název typu a vytvořte výraz pomocí platných prvků.
