# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Název souboru nebo třídy nebyl v průběhu operace Automation nalezen. (Visual Basic)
Název zadaný pro název souboru nebo třídu ve volání `GetObject` funkce nebyl nalezen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte názvy a zkuste to znovu. Ujistěte se, že název použitý pro `class` parametr odpovídá zaregistrovanému v systému.
