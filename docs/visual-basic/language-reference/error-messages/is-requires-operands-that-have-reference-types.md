# <a name="is-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-typename"></a>Klíčové slovo 'Is' vyžaduje operandy typu odkaz, ale tento operand má typ hodnoty '\<typename>'.
`Is`Operátor porovnání Určuje, zda dvě proměnné objektu odkazují na stejnou instanci. Toto porovnání není definováno pro typy hodnot.  
  
 **ID chyby:** BC30020  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- K porovnání dvou typů hodnot použijte příslušný operátor aritmetického porovnání nebo `Like` operátor.
