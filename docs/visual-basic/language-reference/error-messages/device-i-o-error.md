# <a name="device-io-error"></a>Vstupně-výstupní chyba zařízení
V době, kdy program používal zařízení, jako je například tiskárna nebo disková jednotka, došlo k chybě vstupu nebo výstupu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zajistěte, aby zařízení správně fungovalo, a potom operaci opakujte.
