# <a name="xml-axis-properties-do-not-support-late-binding"></a>Vlastnosti osy XML nepodporují pozdní vazbu.
Na vlastnost osy XML se odkazuje pro Netypový objekt.  
  
 **ID chyby:** BC31168  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- <xref:System.Xml.Linq.XElement>Před odkazem na vlastnost Axis XML se ujistěte, že objekt je objekt se silným typem.
