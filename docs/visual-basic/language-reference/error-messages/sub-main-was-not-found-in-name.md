# <a name="sub-main-was-not-found-in-name"></a>Procedura 'Sub Main' nebyla v '\<name>' nalezena.
`Sub Main`chybí nebo bylo pro něj zadáno nesprávné umístění.  
  
 **ID chyby:** BC30420  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Dodejte chybějící `Sub Main` příkaz nebo pokud existuje, přesuňte jej do příslušného umístění v kódu. Další informace o `Sub Main` najdete v tématu [hlavní postup v Visual Basic](../../programming-guide/program-structure/main-procedure.md).  
  
2. Zadejte umístění spouštěcího objektu projektu v poli **spouštěcí formulář** **Návrháře projektu**.
