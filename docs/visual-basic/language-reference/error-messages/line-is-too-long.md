# <a name="line-is-too-long"></a>Řádek je příliš dlouhý.
Zdrojové textové řádky nesmí být delší než 65535 znaků.  
  
 **ID chyby:** BC30494  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkraťte délku řádku na 65535 znaků nebo méně.
