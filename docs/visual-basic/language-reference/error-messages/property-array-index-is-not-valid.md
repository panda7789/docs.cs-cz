# <a name="property-array-index-is-not-valid"></a>Index pole vlastností není platný.
Zadaná hodnota není platná pro index pole vlastností.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte dokumentaci ke komponentě a ujistěte se, že je index v platném rozsahu pro zadanou vlastnost.
