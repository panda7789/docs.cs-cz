# <a name="expression-does-not-produce-a-value"></a>Výsledkem výrazu není hodnota.
Pokusili jste se použít výraz, který nevytváří hodnotu v kontextu produkující hodnoty, jako je například volání metody `Sub` v kontextu, kde `Function` je očekáváno klíčové slovo.  
  
 **ID chyby:** BC30491  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte výraz na jeden, který vytvoří hodnotu.
