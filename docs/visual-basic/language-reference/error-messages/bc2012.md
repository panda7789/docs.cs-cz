# <a name="cant-open-filename-for-writing"></a>Nejde otevřít\<filename > pro zápis.
Zadaný soubor nelze otevřít pro zápis, pravděpodobně protože již byl otevřen.  
  
 **ID chyby:** BC2012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zavřete soubor a znovu ho otevřete.  
  
2. Ověřte oprávnění souboru.
