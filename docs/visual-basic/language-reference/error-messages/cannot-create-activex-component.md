# <a name="cannot-create-activex-component"></a>Součást prvku ActiveX nelze vytvořit.
Pokusili jste se ovládací prvek ActiveX umístit do formuláře v době návrhu nebo přidat formulář do projektu s ovládacím prvkem ActiveX, ale přidružené informace v registru se nenašly.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Informace v registru byly pravděpodobně odstraněny nebo poškozeny. Přeinstalujte ovládací prvek ActiveX nebo se obraťte na dodavatele ovládacího prvku.
