# <a name="string-constants-must-end-with-a-double-quote"></a>Řetězcové konstanty musí končit dvojitými uvozovkami.
Řetězcové konstanty musí začínat a končit uvozovkami.  
  
 **ErrorID:** BC30648  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že řetězcový literál končí znakem uvozovky ("). Pokud vložíte hodnoty z jiných textových editorů, ujistěte se, že vložený znak je platným znakem uvozovek a nikoli jedním ze znaků, které jej připomínají, například "inteligentní" nebo "typografické" uvozovky ("") nebo dvě jednoduché uvozovky (' ').
