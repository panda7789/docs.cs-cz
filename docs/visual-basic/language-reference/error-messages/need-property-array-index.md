# <a name="need-property-array-index"></a>Je vyžadován index pole vlastností.
Hodnota této vlastnosti se skládá z pole, nikoli z jedné hodnoty. Nezadali jste index pole vlastností, ke kterému jste se pokusili získat přístup.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vyhledejte v dokumentaci ke komponentě rozsah indexů odpovídajících poli. Zadejte odpovídající index v příkazu pro přístup k vlastnosti.
