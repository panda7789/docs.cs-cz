# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Tento klíč již je přidružen k elementu této kolekce.
Zadaný klíč pro člena kolekce již identifikoval jiného člena kolekce. Klíč je řetězec určený v `Add` metodě, který jednoznačně identifikuje konkrétního člena kolekce.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pro tohoto člena použijte jiný klíč.
