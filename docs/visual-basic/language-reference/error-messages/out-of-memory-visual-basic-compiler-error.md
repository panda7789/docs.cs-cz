# <a name="out-of-memory-visual-basic-compiler-error"></a>Nedostatek paměti (chyba kompilátoru jazyka Visual Basic)
Byla požadována více paměti, než je k dispozici.  
  
 **ID chyby:** BC2004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zavřete nepotřebné aplikace, dokumenty a zdrojové soubory.  
  
- Eliminujte nepotřebné ovládací prvky a formuláře, aby se načetla méně najednou  
  
- Snižte počet `Public` proměnných.  
  
- Ověřte dostupné místo na disku.  
  
- Zvětšete dostupnou paměť RAM instalací další paměti nebo změnou přidělení paměti.  
  
- Ujistěte se, že je paměť uvolněna v případě, že již není potřebná.
