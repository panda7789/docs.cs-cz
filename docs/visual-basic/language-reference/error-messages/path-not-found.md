# <a name="path-not-found"></a>Cesta nebyla nalezena.
V průběhu operace přístupu k souboru nebo přístupu k disku se v operačním systému nepodařilo najít zadanou cestu. Cesta k souboru obsahuje specifikaci jednotky a adresáře a podadresáře, které je nutné procházet a vyhledat soubor. Cesta může být relativní nebo absolutní.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte a znovu zadejte cestu.
