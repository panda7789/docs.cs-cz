# <a name="identifier-expected"></a>Byl očekáván identifikátor.
Programový prvek, který není rozpoznatelným názvem elementu, nastane, pokud kontext vyžaduje název elementu. Jednou z možných příčin je, že atribut byl zadán jinde než na začátku příkazu.  
  
 **ID chyby:** BC30203  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda jsou všechny atributy v příkazu umístěny na začátek.  
  
- Ověřte, zda jsou všechny názvy elementů v příkazu zadány správně.
