# <a name="cant-create-necessary-temporary-file"></a>Nezbytný dočasný soubor nelze vytvořit.
Jednotka je plná a obsahuje adresář určený proměnnou prostředí TEMP nebo proměnná prostředí TEMP určuje neplatnou jednotku nebo adresář jen pro čtení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pokud je to kompletní, odstraňte soubory z disku.  
  
2. V proměnné prostředí TEMP zadejte jinou jednotku.  
  
3. Zadejte platnou jednotku pro proměnnou prostředí TEMP.  
  
4. Odebere omezení jen pro čtení z aktuálně zadané jednotky nebo adresáře.
