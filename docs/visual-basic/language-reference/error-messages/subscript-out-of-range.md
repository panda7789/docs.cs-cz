# <a name="subscript-out-of-range-visual-basic"></a>Dolní index mimo rozsah (Visual Basic)
Dolní index pole není platný, protože spadá mimo povolený rozsah. Nejnižší hodnota dolního indexu pro dimenzi je vždy 0 a nejvyšší hodnota dolního indexu je vrácena `GetUpperBound` metodou pro tuto dimenzi.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte dolní index tak, aby ležel v platném rozsahu.
