# <a name="methodname-has-multiple-definitions-with-identical-signatures"></a>Metoda '\<methodname>' má několik definic s identickými podpisy.
`Function` `Sub` Deklarace procedury nebo používá stejný název procedury a seznam argumentů jako předchozí deklarace. Jednou z možných příčin je pokus o přetížení původní procedury. Přetížené procedury musí mít různé seznamy argumentů.  
  
 **ID chyby:** BC30269  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte název procedury nebo seznam argumentů nebo odeberte duplicitní deklaraci.
