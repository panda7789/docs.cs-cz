# <a name="expression-too-complex"></a>Výraz je příliš složitý.
Výraz s plovoucí desetinnou čárkou obsahuje příliš mnoho vnořených dílčích výrazů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Rozdělte výraz do tolika samostatných výrazů, jak je potřeba, aby se zabránilo výskytu chyby.
