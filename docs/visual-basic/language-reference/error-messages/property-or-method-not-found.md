# <a name="property-or-method-not-found"></a>Vlastnost nebo metoda nebyla nalezena.
Odkazovaná metoda objektu nebo vlastnost objektu není definována.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Je možné, že název objektu je špatně napsaný. Chcete-li zjistit, jaké vlastnosti a metody jsou definovány pro objekt, zobrazte Prohlížeč objektů. Vyberte příslušnou knihovnu objektů pro zobrazení seznamu dostupných vlastností a metod.
