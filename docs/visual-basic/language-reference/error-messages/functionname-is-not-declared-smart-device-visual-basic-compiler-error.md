# <a name="functionname-is-not-declared-smart-devicevisual-basic-compiler-error"></a>není \<functionname> deklarovaný (chyba inteligentního zařízení/Visual Basic kompilátoru).
<`functionname`> není deklarováno. Funkce vstupně-výstupní operace se soubory jsou v `Microsoft.VisualBasic` oboru názvů běžně dostupné, ale cílová verze prostředí .NET Compact Framework ji nepodporuje.  
  
 **ID chyby:** BC30766  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Provádět operace se soubory pomocí funkcí definovaných v `System.IO` oboru názvů.
