# <a name="automation-error"></a>Chyba automatizace

Při provádění metody nebo při získávání nebo nastavování vlastnosti proměnné objektu došlo k chybě. Aplikace ohlásila chybu, která vytvořila objekt.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte vlastnosti `Err` objektu a určete zdroj a povahu chyby.  
  
2. Použijte `On Error Resume Next` příkaz těsně před příkaz k přístupu a potom zkontrolujte chyby ihned po příkazu k přístupu.
