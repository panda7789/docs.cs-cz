# <a name="constant-expression-not-representable-in-type-typename"></a>Konstantní výraz nelze reprezentovat typem '\<typename>'.
Snažíte se vyhodnotit konstantu, která se nevejde do cílového typu, obvykle proto, že přetéká rozsah.  
  
 **ID chyby:** BC30439  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte cílový typ na jeden, který dokáže zpracovat konstantu.
