# <a name="file-access-with-visual-basic"></a>Přístup k souborům v jazyce Visual Basic

Objekt `My.Computer.FileSystem` poskytuje nástroje pro práci se soubory a složkami. Jeho vlastnosti, metody a události umožňují vytvářet, kopírovat, přesouvat, zkoumat a odstraňovat soubory a složky. `My.Computer.FileSystem`poskytuje lepší výkon než`FileOpen`starší `FileClose` `Input`funkce `InputString` `LineInput`( , , , , , , atd.), které jsou poskytovány v jazyce Visual Basic pro zpětnou kompatibilitu.
