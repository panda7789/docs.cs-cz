# <a name="file-access-with-visual-basic"></a>Přístup k souborům v jazyce Visual Basic

`My.Computer.FileSystem`Objekt poskytuje nástroje pro práci se soubory a složkami. Jeho vlastnosti, metody a události umožňují vytvořit, kopírovat, přesunout, prozkoumat a odstranit soubory a složky. `My.Computer.FileSystem`poskytuje lepší výkon než starší funkce ( `FileOpen` , `FileClose` ,, `Input` , `InputString` atd `LineInput` .), které jsou k dispozici Visual Basic z důvodu zpětné kompatibility.
