# <a name="division-by-zero-visual-basic-error"></a>Dělení nulou (chyba Visual Basic)
Hodnota výrazu použitá jako dělitele je nulová.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte pravopis proměnných ve výrazu. Nesprávně napsaná proměnná může implicitně vytvořit číselnou proměnnou inicializovaná na nulu.  
  
2. Zkontroluje předchozí operace s proměnnou ve výrazu, zejména ty, které byly předány do procedury jako argumenty z jiných postupů.
