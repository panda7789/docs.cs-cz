# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>Cílová verze prostředí .NET Compact Framework nepodporuje použití modifikátoru ANSI, auto ani Unicode.
Verze prostředí .NET Compact Framework, se kterou pracujete `Ansi` , nepodporuje `Auto` `Unicode` modifikátory, a.  
  
 **ID chyby:** BC30763  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte modifikátor.
