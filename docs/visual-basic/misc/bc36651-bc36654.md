<span data-ttu-id="e0069-103">Z těchto argumentů nelze odvodit datové typy parametrů typu v metodě, \<methodname> protože je přípustný více než jeden typ.</span><span class="sxs-lookup"><span data-stu-id="e0069-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="e0069-104">Je možné, že se chybu podaří odstranit explicitním zadáním datových typů.</span><span class="sxs-lookup"><span data-stu-id="e0069-104">Specifying the data type(s) explicitly might correct this error.</span></span>

> <span data-ttu-id="e0069-107">Pokud se zadáním argumentů nejedná o možnost (například pro operátory dotazu ve výrazech dotazu), zobrazí se chybová zpráva bez druhé věty.</span><span class="sxs-lookup"><span data-stu-id="e0069-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>
