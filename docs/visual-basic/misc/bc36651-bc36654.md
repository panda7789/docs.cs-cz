<span data-ttu-id="a2bb3-105">Došlo k pokusu o použití odvození typu k určení typu nebo typů parametrů typu nebo parametrů ve volání generické procedury.</span><span class="sxs-lookup"><span data-stu-id="a2bb3-105">An attempt has been made to use type inference to determine the type or types of the type parameter or parameters in a call to a generic procedure.</span></span> <span data-ttu-id="a2bb3-106">Kompilátor najde více než jeden možný datový typ pro jeden nebo více parametrů typu a oznámí tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="a2bb3-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>
