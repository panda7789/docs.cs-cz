# <a name="cannot-convert-start-value-of-type-type1-and-step-value-of-type-type2-to-a-common-type"></a>Počáteční hodnotu typu \<type1> a hodnotu kroku typu nelze převést \<type2> na společný typ.
Zadané `For...Next` hodnoty ovládacího prvku mají jiné typy, a proto nejsou platné, protože nemohou být převedeny na společný typ.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte typ jedné nebo více hodnot tak, aby všechny hodnoty mohly být převedeny na společný typ.
