# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>Komentář XML se nemůže vyskytovat v rámci metody nebo vlastnosti.
Komentář XML se nemůže vyskytovat v rámci metody nebo vlastnosti. Komentář XML bude ignorován. Bloky komentářů XML se nemůžou vyskytovat v rámci metod nebo vlastností.  
  
 **ID chyby:** BC42303  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte blok komentáře nebo ho přesuňte do vhodnějšího umístění.
