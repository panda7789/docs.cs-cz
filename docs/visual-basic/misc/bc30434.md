# <a name="events-in-a-module-cannot-be-declared-specifier"></a>Pro události v modulu nejde provést \<specifier> deklarace.
Použili jste specifikátor, který není platný pro událost v rámci `Module` příkazu. Pro moduly nelze nikdy vytvořit instanci, nepodporují dědičnost a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30434  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
