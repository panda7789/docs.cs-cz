# <a name="interface-can-inherit-only-from-another-interface"></a>Rozhraní může dědit jenom z jiného rozhraní.
Pokusili jste se zdědit z něčeho, co není rozhraní.  
  
 **ID chyby:** BC30354  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že základní rozhraní nebo rozhraní, ze kterých se pokoušíte dědit, jsou ve skutečnosti rozhraní.
