# <a name="parameter-parametername-already-has-a-matching-argument"></a>Parametr \<parametername> už má shodný argument.
Argument je předán pomocí názvu více než jednou během stejného volání procedury.  
  
 **ID chyby:** BC36567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, zda je v některých názvech argumentů uveden chybný název.  
  
- Odebere duplicitní argument ze seznamu argumentů.
