# <a name="property-parameters-cannot-have-the-name-value"></a>Parametry vlastnosti nemůžou mít název Value.
Parametr v `Property` příkazu má název `Value` . Tento název je vyhrazen v rámci definice vlastnosti pro novou hodnotu přiřazenou k vlastnosti v `Set` proceduře.  
  
 **ID chyby:** BC32201  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte název parametru v `Property` příkazu.
