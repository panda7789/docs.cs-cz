# <a name="argument-not-specified-for-parameter-parametername"></a>Není zadán argument pro parametr ' \<parametername> '
Nebyl zadán argument pro požadovaný parametr.  
  
 **ID chyby:** BC36569  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument pro zadaný parametr.
