# <a name="the-file-is-currently-open-for-writing"></a>Soubor je aktuálně otevřen pro zápis.
Byl proveden pokus o přístup k souboru, který je aktuálně otevřen pro zápis.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zavřete soubor a zkuste operaci zopakovat.
