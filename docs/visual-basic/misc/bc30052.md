# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Pole překračuje limit 32 dimenzí.
Pole nesmí mít více než 32 dimenzí.  
  
 **ID chyby:** BC30052  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Snižte počet rozměrů v deklaraci pole.
