# <a name="inheriting-from-systemclassname-is-not-valid"></a>Dědění z ' System. \<classname> ' není platné
Třída nemůže dědit z `System.Array` , `System.Delegate` , `System.Enum` nebo `System.ValueType` .  
  
 **ID chyby:** BC30015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Inherits` příkaz nebo ho změňte tak, aby dědil z platné základní třídy.
