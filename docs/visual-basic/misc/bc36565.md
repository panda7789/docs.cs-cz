# <a name="too-many-arguments"></a>Příliš mnoho argumentů
Počet argumentů dodaných ve volání procedury překračuje počet argumentů definovaných pro proceduru.  
  
 **ID chyby:** BC36565  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Prohlédněte si definici procedury a odeberte nadbytečné argumenty volání.
