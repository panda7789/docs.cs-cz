# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Výrazy dotazů nejsou platné v ladicích oknech.
V okně **kukátka** nebo v **příkazovém podokně** během relace ladění byl zadán dotaz LINQ. Dotazy nejsou k dispozici v ladicích oknech.  
  
 **ID chyby:** BC36609
