# <a name="set-parameter-cannot-be-declared-specifier"></a>Pro parametr set nelze provést deklaraci. \<specifier>
V proceduře vlastnosti byl zadán neplatný modifikátor s parametrem `Set` .  
  
 **ID chyby:** BC31065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte neplatné modifikátory, například `ByRef` , `ParamArray` a `Optional` .
