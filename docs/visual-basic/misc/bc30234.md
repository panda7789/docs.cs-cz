# <a name="specifier-is-not-valid-on-a-withevents-declaration"></a>' \<specifier> ' není platný v deklaraci WithEvents.
`Dim`Příkaz obsahuje klíčové slovo, `ReadOnly` které je neplatné v kombinaci s `WithEvents` klíčovým slovem.  
  
 **ID chyby:** BC30234  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte z příkazu neplatné klíčové slovo `Dim` .
