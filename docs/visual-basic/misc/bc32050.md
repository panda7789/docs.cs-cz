```vb  
    findLargest(testMatrix)

```  
  
 <span data-ttu-id="f00fc-110">V předchozím kódu dvě volání metody `findLargest` vytvoří tuto chybu, protože parametr typu `t` volá pro jednorozměrné pole, zatímco argumenty typu, které kompilátor odvodí z volání, jsou skalární ( `testValue` ) a dvourozměrné pole ( `testMatrix` ).</span><span class="sxs-lookup"><span data-stu-id="f00fc-110">In the preceding code, the two calls to `findLargest` both produce this error, because the type parameter `t` calls for a one-dimensional array, whereas the type arguments the compiler infers from the calls are a scalar (`testValue`) and a two-dimensional array (`testMatrix`).</span></span>
