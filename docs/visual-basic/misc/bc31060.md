# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>' \<propertyname> ' implicitně definuje ' \<membername> ', který je v konfliktu se členem stejného názvu v \<type> ' \<typename> '.
Název člena typu je v konfliktu s názvem členu implicitně vytvořeným pro vlastnost. Vlastnosti implicitně vytvoří několik implicitních proměnných, které mohou být v konfliktu s definovanými členy.  
  
 **ID chyby:** BC31060  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přejmenujte explicitně deklarovaný člen na odebrání konfliktu pojmenování.
