# <a name="catch-must-end-with-a-matching-end-try"></a>Klíčové slovo Catch musí být ukončené párovým klíčovým blokem End Try.
`Catch`Příkaz se zobrazí ve vašem kódu bez odpovídajícího `End Try` příkazu. `Catch`příkazy musí být uzavřeny `End Try` příkazem.  
  
 **ID chyby:** BC30441  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte `Catch` příkaz.  
  
2. Přidejte `End Try` příkaz pro uzavření bloku.
