# <a name="option-strict-can-be-followed-only-by-on-or-off"></a>Za možností Option Strict může následovat jenom klíčové slovo on nebo off.
`Option Strict`příkazy musí určovat buď `On` nebo `Off` . Všechny ostatní hodnoty jsou neplatné.  
  
 **ID chyby:** BC30620  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `On` nebo `Off` pro `Option Strict` příkaz.
