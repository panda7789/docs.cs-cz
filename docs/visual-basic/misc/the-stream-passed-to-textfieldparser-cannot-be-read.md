# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>Datový proud předaný parametru TextFieldParser nelze číst.
[Objekt TextFieldParser](../language-reference/objects/textfieldparser-object.md) nemůže číst datový proud, který byl předán do něj. Může to být výsledek pokusu o čtení souboru, který není textový soubor.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda je soubor textový soubor.
