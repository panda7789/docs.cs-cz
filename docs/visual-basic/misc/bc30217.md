# <a name="string-constant-expected"></a>Očekávala se řetězcová konstanta.
Znak jiný než dvojité uvozovky (") nastane, pokud kontext vyžaduje řetězcovou konstantu. Jednou z možných příčin je, že uvozovky v řetězci chybí.  
  
 **ID chyby:** BC30217  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zajistěte, aby byl řetězec správně uzavřen v uvozovkách.  
  
2. Zadejte řetězcovou konstantu.
