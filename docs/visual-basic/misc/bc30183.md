# <a name="keyword-is-not-valid-as-an-identifier"></a>Klíčové slovo není platné jako identifikátor.
Klíčové slovo nastane tam, kde je vyžadován název elementu. Kontext volá pro prvek, jako je například proměnná nebo procedura.  
  
 **ID chyby:** BC30183  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte klíčové slovo platným názvem elementu.
