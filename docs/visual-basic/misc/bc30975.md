# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Řídicí proměnná cyklu for se už používá v nadřazeném cyklu for.
Vnořené `For` smyčky musí používat jedinečné proměnné ovládacího prvku.  
  
 **ID chyby:** BC30975  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte jeden z názvů proměnných ovládacího prvku.
