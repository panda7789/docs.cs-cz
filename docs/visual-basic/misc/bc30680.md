# <a name="end-region-must-be-preceded-by-a-matching-region"></a>Klíčovému #End oblasti musí předcházet párové klíčové slovo #Region.
`#Region`Můžete zadat blok kódu pro rozbalení nebo sbalení při použití funkce sbalení editoru Visual Studio Code. Začátek a konec `#Region` příkazu musí být ve stejném bloku kódu.  
  
 **ID chyby:** BC30680  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vložte `#Region` na příslušné místo před odpovídající `#End Region` příkaz.
