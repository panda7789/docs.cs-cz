<span data-ttu-id="16459-103">Procedura nebo vlastnost jsou deklarovány jako `MustOverride` v rámci třídy, která je definována v několika částečných deklaracích, ale jedna z částečných definic je určena `NotInheritable` pro třídu.</span><span class="sxs-lookup"><span data-stu-id="16459-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="16459-104">Pokud rozdělíte definici třídy mezi několik částečných deklarací, kompilátor považuje třídu za sjednocení všech jejích částečných deklarací.</span><span class="sxs-lookup"><span data-stu-id="16459-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="16459-105">To platí nejen pro členy, ale také k implementaci, dědičnosti a úrovni přístupu.</span><span class="sxs-lookup"><span data-stu-id="16459-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>
