# <a name="attribute-value-is-not-valid-expecting-value"></a>Hodnota atributu není platná. očekává se hodnota Value.
Byla zadána neplatná hodnota pro atribut v literálu dokumentu XML. Platné hodnoty atributu pro literál dokumentu XML jsou zadány ve specifikaci XML 1,0.  
  
 **ID chyby:** BC31181  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Aktualizujte hodnotu atributu na platnou hodnotu, jak je uvedeno ve specifikaci XML 1,0.
