# <a name="character-is-not-valid"></a>Znak není platný.
Aktuální kontext nepovoluje tento znak. Například vlnovka ( `~` ) nebo svislá čára ( `|` ) nemůže být součástí deklarovaného názvu elementu.  
  
 **ID chyby:** BC30037  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, zda omylem byl vložen nadbytečný znak do programovacího prvku.  
  
2. Zkontrolujte, zda je programový prvek zadán nebo vytvořen pomocí pouze znaků, které jsou platné pro tento druh prvku.
