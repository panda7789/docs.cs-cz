# <a name="xml-comment-tag-returns-is-not-permitted-on-a-declare-sub-language-element"></a>Značka komentáře XML Returns není povolená u elementu jazyka Declare Sub.
Značka komentáře XML Returns není povolená u elementu jazyka Declare Sub. Komentář XML bude ignorován.  
  
 Komentář XML pro `Declare Sub` deklaraci nemůže obsahovat `<` `>` značku vrácení.  
  
 **ID chyby:** BC42315  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebere nepodporovanou `<` `>` značku vrácení.
