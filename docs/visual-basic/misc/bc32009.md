# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Příkazy deklarace metod musí být první na logickém řádku.
`Function`Příkaz nebo `Sub` následuje dvojtečka (:) oddělovač příkazů `Function`nebo `Sub` musí být jediným příkazem na jeho řádku zdroje.  
  
 **ID chyby:** BC32009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
