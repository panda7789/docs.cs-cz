# <a name="argument-path-is-nothing-or-empty"></a>Argument path má hodnotu Nothing nebo je prázdný.
Název cesty není platný, protože je inicializovaný `Nothing` nebo je prázdný. Cesta musí mít konkrétní typ dat a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte platný název cesty.
