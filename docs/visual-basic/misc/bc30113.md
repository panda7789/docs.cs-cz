# <a name="namespace-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>' \<namespace> ' není platný název a nelze jej použít jako název kořenového oboru názvů.
Kvalifikační název oboru názvů není platný název elementu. Název musí obsahovat jenom abecední znaky, desítkové číslice a podtržítka a musí začínat znakem abecedy nebo podtržítkem ( `_` ).  
  
 **ID chyby:** BC30113  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pokud je obor názvů součástí sady vnořených oborů názvů, zkontrolujte, zda některý z vnějších názvů obsahuje neplatné znaky.
