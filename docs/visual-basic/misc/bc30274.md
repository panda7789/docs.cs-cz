# <a name="parameter-parametername-of-methodname-already-has-a-matching-argument"></a>Parametr \<parametername> pro \<methodname> už má shodný argument.
Argument je předán pomocí názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC30274  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, jestli je u některého z názvů argumentů nesprávně napsaný název.  
  
- Odebere duplicitní argument ze seznamu argumentů.
