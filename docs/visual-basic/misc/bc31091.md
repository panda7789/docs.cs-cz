# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>\<typename>Nepovedlo se importovat typ ze sestavení nebo modulu \<name> .
Ujistěte se, že jsou všechny odkazované knihovny vzájemně kompatibilní.  
  
 **ID chyby:** BC31091  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda nejsou poškozeny soubory obsahující odkazovaný typ.
