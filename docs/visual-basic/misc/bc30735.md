# <a name="type-in-a-module-cannot-be-declared-specifier"></a>Pro typ v modulu nejde provést \<specifier> deklarace.
V rámci modulu byl deklarován typ s neplatnými modifikátory.  
  
 **ID chyby:** BC30735  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nepoužívejte `Protected` ani `Protected Friend` modifikátory u typů deklarovaných v rámci modulu.
