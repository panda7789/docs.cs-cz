# <a name="local-variables-within-generic-methods-cannot-be-declared-static"></a>Místní proměnné v rámci obecných metod nejde deklarovat jako static.
Deklarace místní proměnné v rámci obecné procedury Určuje `Static` .  
  
 Visual Basic a .NET Framework v současné době nepodporuje žádnou kombinaci statických proměnných a obecných postupů.  
  
 **ID chyby:** BC32068  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Static` klíčové slovo z deklarace proměnné.
