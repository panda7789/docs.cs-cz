# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Specifikátory jsou platné jenom na začátku deklarace.
Klíčová slova určující přístupnost a jiná označení se musí vyskytovat před klíčovým slovem deklarace, například `Dim` nebo `Function` .  
  
 **ID chyby:** BC30181  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte všechny specifikátory mezi seznamem atributů (pokud existuje) a klíčovým slovem deklarace.
