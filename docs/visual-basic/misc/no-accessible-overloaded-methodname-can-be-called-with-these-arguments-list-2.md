# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-narrowing-conversion-list"></a>\<methodname>S těmito argumenty bez zužujícího převodu se dají volat žádné přetížené přetížené:\<list>
Byla volána přetížená metoda, ale metodu nelze spárovat se seznamem poskytnutých argumentů bez zužujícího převodu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte `Option Strict Off`.
  
2. Změňte argumenty tak, aby odpovídaly podpisu přetížené metody.
