# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion-ltlistgt"></a>Žádná dostupná přetížení &#39; &lt;methodname&gt; &#39; nelze volat s těmito argumenty bez zužujícího převodu: &lt;seznamu&gt;
Byla volána přetížená metoda, ale metoda nejde spárovat se seznamem zadané argumenty bez zužujícího převodu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte `Option Strict Off`.
  
2.  Změna argumentů tak, aby odpovídaly podpis přetížené metody.
