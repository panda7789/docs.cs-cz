# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>Název prostředku se \<resourcename> nedá použít víc než jednou.
Název zadaného prostředku v dynamicky propojené knihovně nebo spustitelném souboru byl použit více než jednou.  
  
 **ID chyby:** BC31502  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Jedinečné názvy prostředků.
