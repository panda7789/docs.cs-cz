# <a name="expected-closing--for-embedded-expression"></a>Očekávalo se ukončení vloženého výrazu% >.
V rámci vloženého výrazu v literálu XML chybí požadované uzavírací `%>` znaky.  
  
 **ID chyby:** BC31159  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte požadované uzavírací `%>` znaky na konec vloženého výrazu.
