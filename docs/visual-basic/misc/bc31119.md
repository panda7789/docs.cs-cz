# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>Příkaz end RemoveHandler musí být prvním příkazem na řádku.
`End RemoveHandler`Příkaz sleduje dvojtečku (:) oddělovač příkazů `End RemoveHandler`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC31119  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
