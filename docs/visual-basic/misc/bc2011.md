# <a name="unable-to-open-response-file-filename"></a>Nedá \<filename> se otevřít soubor odpovědí.
Soubor, ke kterému se pokoušíte získat přístup, není k dispozici.  
  
 **ID chyby:** BC2011  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že je připojení k síti stále aktivní.  
  
- Ověřte oprávnění k souboru.
