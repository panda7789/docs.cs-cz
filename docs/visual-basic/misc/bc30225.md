# <a name="option-specifier-statement-can-only-appear-once-per-file"></a>Příkaz Option \<specifier> může být v každém souboru použit pouze jednou.
`Option Compare`Příkazy, `Option Explicit` a `Option Strict` lze použít pouze jednou ve stejném zdrojovém souboru.  
  
 **ID chyby:** BC30225  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte duplikovaný `Option` příkaz.
