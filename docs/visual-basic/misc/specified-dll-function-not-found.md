# <a name="specified-dll-function-not-found"></a>Zadaná funkce knihovny DLL nebyla nalezena.
Knihovna DLL (Dynamic-Link Library) v odkazu na knihovnu uživatelů byla nalezena, ale zadaná funkce knihovny DLL nebyla nalezena v rámci knihovny DLL.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. V deklaraci funkce zadejte platné ordinální číslo.  
  
2. Ujistěte se, že název a alias knihovny DLL jsou správné.
