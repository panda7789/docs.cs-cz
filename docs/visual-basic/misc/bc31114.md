# <a name="custom-event-must-end-with-a-matching-end-event"></a>Klíčové slovo Custom Event musí být ukončené párovým klíčovým řetězcem End Event.
`Custom Event`Deklarace musí končit `End Event` příkazem.  
  
 **ID chyby:** BC31114  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda `Custom Event` deklarace končí `End Event` příkazem.
