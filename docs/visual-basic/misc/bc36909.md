# <a name="cannot-infer-a-data-type-for-variablename-because-the-array-dimensions-do-not-match"></a>Nejde odvodit datový typ pro \<variablename> , protože se neshodují rozměry pole.
Pokud dimenze použité k inicializaci pole neodpovídají dimenzím v deklaraci, kompilátor nemůže odvodit datový typ pole. Například následující kód způsobuje tuto chybu.  
  
```vb  
' Valid. exampleArray1 is a one-dimensional array of integers.  
Dim exampleArray1() = New Integer() {1, 2, 3}  
' Not valid.  
'Dim exampleArray2(,) = New Integer() {1, 2, 3}  
'Dim exampleArray3(,) = New Integer() {}  
```  
  
 **ID chyby:** BC36909
