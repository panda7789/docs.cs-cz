# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Typ se \<typename> nedá použít jako atribut, protože není deklarovaný jako Public.
Atributy musí být veřejné.  
  
 **ID chyby:** BC31516  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte `Public` modifikátor do třídy atributů.
