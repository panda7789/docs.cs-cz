<span data-ttu-id="32563-112">Následující příklad používá `AttributeUsage` atribut k určení, které prvky programu může nová verze `ExtensionAttribute` použít na.</span><span class="sxs-lookup"><span data-stu-id="32563-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="32563-113">Příklad určuje tři členy `AttributeTargets` výčtu: `Assembly` , `Class` , a `Method` .</span><span class="sxs-lookup"><span data-stu-id="32563-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="32563-114">Tato chyba způsobí Vynechání některého z těchto prvků.</span><span class="sxs-lookup"><span data-stu-id="32563-114">The omission of any one of these elements will cause this error.</span></span>

    End Class
