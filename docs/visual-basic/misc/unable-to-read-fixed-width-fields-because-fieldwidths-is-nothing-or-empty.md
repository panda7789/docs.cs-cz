# <a name="unable-to-read-fixed-width-fields-because-fieldwidths-is-nothing-or-empty"></a>Pole s pevnou šířkou nelze číst, protože hodnoty parametru FieldWidths má hodnotu Nothing nebo není prázdné.
`TextFieldParser`Pole s pevnou šířkou nelze načíst, protože `FieldWidths` vlastnost je nastavena na hodnotu `Nothing` nebo je prázdná.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nastavte `FieldWidths` na platnou hodnotu.
