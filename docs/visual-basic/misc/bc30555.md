# <a name="default-member-membername-is-not-a-property"></a>Výchozí člen není \<membername> vlastností.
Pokusili jste se deklarovat člen `Default` . `Default`Klíčové slovo se však používá k identifikaci výchozích vlastností.  
  
 **ID chyby:** BC30555  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, zda jste identifikovali vlastnost pomocí `Default` , nebo jinak odeberte `Default` klíčové slovo.
