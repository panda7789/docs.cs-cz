# <a name="property-parameters-cannot-be-declared-byref"></a>\<Property>pro parametry nejde provést deklarace ByRef.
Parametry pro vlastnosti musí být deklarovány pomocí `ByVal` modifikátoru.  
  
 **ID chyby:** BC30651  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Definujte parametry vlastností jako `ByVal` .
