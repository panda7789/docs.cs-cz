# <a name="events-cannot-have-a-return-type"></a>Události nemůžou mít návratový typ.
Události mohou přijímat argumenty, ale nemohou přímo vracet hodnoty.  
  
 **ID chyby:** BC30032  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte návratový typ z `Event` příkazu.
