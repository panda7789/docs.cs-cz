# <a name="namespacename-is-ambiguous"></a>' \<namespacename> ' je dvojznačný.
Zadali jste název, který je nejednoznačný, a proto je v konfliktu s jiným názvem. Kompilátor Visual Basic nemá žádná pravidla pro řešení konfliktů; názvy je nutné jednoznačně odlišit.  
  
 **ID chyby:** BC30554  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Plně kvalifikované jméno.
