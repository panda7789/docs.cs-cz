# <a name="sub-main-is-declared-more-than-once-in-namespace-message"></a>Procedura Sub Main je v: deklarovaná víc než jednou \<namespace> :\<message>
`Sub Main`dá se deklarovat jenom jednou v rámci oboru názvů.  
  
 **ID chyby:** BC30738  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že projekt obsahuje pouze jednu `Sub Main` proceduru.
