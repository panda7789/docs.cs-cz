# <a name="bracketed-identifier-is-missing-closing-"></a>U identifikátoru v závorkách chybí uzavírací znak].
Hranaté závorky v názvu s řídicím znakem musí být v párové dvojici.  
  
 **ID chyby:** BC30034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Na konec řídicího názvu umístěte pravou hranatou závorku.
