<span data-ttu-id="5e17e-103">Obecné rozhraní dědí více než jednou z jiného obecného rozhraní a dvě dědičnost by mohly kolidovat pro určité hodnoty argumentů typu.</span><span class="sxs-lookup"><span data-stu-id="5e17e-103">A generic interface inherits more than once from another generic interface, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 `Public Interface interfaceA(Of u)`  
  
 `End Interface`  
  
 `Public Interface derivedInterface(Of t1, t2)`  
  
 `Inherits interfaceA(Of t1), interfaceA(Of t2)`  
  
 `End Interface`  
  
 <span data-ttu-id="5e17e-105">Pokud `derivedInterface` je vytvořena nebo implementováno poskytnutí stejného typu pro obojí `t1` a `t2` , musí dědit dvě verze `interfaceA` s identickými argumenty typu.</span><span class="sxs-lookup"><span data-stu-id="5e17e-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="5e17e-106">Tím by vznikla nejednoznačnost, o které verzi přistupuje.</span><span class="sxs-lookup"><span data-stu-id="5e17e-106">Doing so would produce an ambiguity about which version to access.</span></span>
