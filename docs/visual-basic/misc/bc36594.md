# <a name="definition-of-method-method-is-not-accessible-in-this-context"></a>Definice metody \<method> není v tomto kontextu dostupná.
V operátoru byla zahrnuta agregační funkce, která neexistuje `Into` .  
  
 **ID chyby:** BC36594  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte pravopis názvu agregované funkce a proveďte potřebné opravy.  
  
- Přidejte metodu jako vlastní agregační funkci (metoda rozšíření). Další informace o vytvoření vlastní agregační funkce naleznete v tématu [klauzule Aggregate](../language-reference/queries/aggregate-clause.md).
