# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>Příkazu End AddHandler musí předcházet párové prohlášení AddHandler.
`End AddHandler`Došlo k příkazu bez odpovídajícího `AddHandler` příkazu. `End AddHandler`musí předcházet odpovídající `AddHandler` příkaz.  
  
 **ID chyby:** BC31124  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že předchozí `AddHandler` příkaz je platný a správně napsaný.
