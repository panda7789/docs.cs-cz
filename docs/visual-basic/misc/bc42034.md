# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Název souboru je už deklarovaný s jiným identifikátorem GUID a hodnotou kontrolního součtu.
Byl proveden pokus o deklarování souboru, ale soubor již existuje s jiným identifikátorem GUID a kontrolní hodnotou.  
  
 **ID chyby:** BC42034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte název souboru a cestu a ujistěte se, že jsou všechny správné.  
  
- Odstraňte redundantní soubor, pokud není potřeba.
