# <a name="shared-sub-new-cannot-be-declared-specifier"></a>Pro sdílenou proceduru Sub New nejde deklarovat \<specifier> .
Došlo k pokusu o použití specifikátoru v `Shared Sub New` deklaraci, která není platná.  
  
 **ID chyby:** BC30480  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
