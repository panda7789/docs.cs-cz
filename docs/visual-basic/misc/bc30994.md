# <a name="object-initializer-syntax-cannot-be-used-to-initialize-an-instance-of-type-object"></a>K inicializaci instance typu Object nelze použít syntaxi inicializátoru objektu.
Nelze inicializovat instanci `Object` pomocí syntaxe inicializátoru objektu. Instance `Object` nemá žádné vlastnosti nebo pole k přiřazení hodnoty a syntaxe inicializátoru objektu vyžaduje alespoň jednu takovou vlastnost nebo pole.  
  
```vb  
' Not valid.  
' Dim obj1 = New Object With {}  
' Dim obj2 = New Object With {.ToString = <some value>}  
```  
  
 **ID chyby:** BC30994  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarovat instance typu `Object` bez použití seznamu inicializátorů:  
  
    ```vb  
    Dim obj3 as Object  
    Dim obj4 as New Object()  
    ```
