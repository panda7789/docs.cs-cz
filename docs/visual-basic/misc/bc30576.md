# <a name="statement-does-not-declare-a-get-or-set-method"></a>Příkaz nedeklaruje metodu GET nebo set.
Vašemu příkazu se nepovedlo dodat `Get` `Set` příkaz nebo prohlášení o `Property` postupu. Vlastnost je definována jako blok kódu uzavřený v rámci `Property` `End Property` příkazů a. V tomto bloku se každý `Property` postup zobrazí jako vnitřní blok uzavřený v příkazu deklarace a příkazu end.  
  
 **ID chyby:** BC30576  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte `Get` `Set` příkaz deklarace nebo.
