# <a name="finally-can-only-appear-once-in-a-try-statement"></a>Klíčové slovo finally může být v příkazu try uvedeno jenom jednou.
`Finally`slouží k dokončení `Try...Catch...Finally` bloku, takže se může na konci bloku vyskytovat pouze jednou.  
  
 **ID chyby:** BC30381  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Vyhledejte a odstraňte redundantní `Finally` .
