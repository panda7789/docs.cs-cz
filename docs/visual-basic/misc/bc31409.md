# <a name="method-methodname1-must-be-declared-overloads-because-another-methodname2-is-declared-overloads"></a>\<method>pro \<methodname1> musí být deklarované klíčové slovo Overloads, protože jiná \<methodname2> deklarace má deklaraci Overloads.
Jiná vlastnost nebo metoda se stejným názvem je deklarována s `Overloads` modifikátorem.  
  
 **ID chyby:** BC31409  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte `Overloads` modifikátor do vlastnosti nebo metody.  
  
2. Odeberte `Overloads` Modifikátor ze všech vlastností nebo metod se stejným názvem.
