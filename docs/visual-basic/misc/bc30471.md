# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>Výraz není pole ani metoda, a proto nemůže mít seznam argumentů.
Zadali jste seznam argumentů pro výraz, který není polem nebo metodou. Seznamy argumentů mohou mít pouze pole nebo metody.  
  
 **ID chyby:** BC30471  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte seznam argumentů.
