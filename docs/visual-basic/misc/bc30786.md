# <a name="declare-statements-in-a-module-cannot-be-declared-specifier"></a>Příkazy Declare v modulu nejde deklarovat jako \<specifier> .
`Declare`Deklarace má specifikátor, který není platný v rámci `Module` deklarace. Pro moduly nelze nikdy vytvořit instanci, nepodporují dědičnost a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30786  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
