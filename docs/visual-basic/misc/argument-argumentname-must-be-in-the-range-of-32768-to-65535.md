# <a name="argument-argumentname-must-be-in-the-range-of--32768-to-65535"></a>Argument \<argumentname> musí být v rozsahu-32768 až 65535.
Hodnota argumentu spadá mimo povolený rozsah-32 768 až 65 535.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že hodnota argumentu spadá do platného rozsahu.  
  
2. Pokud je argument dodaný výpočtem, podívejte se na hodnoty, které výpočet zpracovává.
