# <a name="option-option-cannot-be-followed-by--or---ignored"></a>za možností \<option> nemůže následovat znak + nebo-. ignoruje se.
Možnost kompilátoru byla určena hodnotou `+` nebo `-` ; tyto hodnoty nejsou podporovány.  
  
 Další informace o možnostech kompilátoru a hodnotách, které podporují, najdete v tématu [Visual Basic možností kompilátoru, které jsou uvedeny abecedně](../reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **ID chyby:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Opravte tuto chybu a použijte možnost  
  
- Zadejte možnost kompilátoru bez hodnoty.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Opravte tuto chybu a nepoužívejte možnost.  
  
- Odeberte možnost kompilátoru.
