# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a>Inicializátor pole nejde zadat pro nekonstantní dimenzi; použít prázdný inicializátor {}
Pole inicializuje dimenzi, která není známá v době kompilace.  
  
 Následující kód vygeneruje tuto chybu.  
  
```vb  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 Následující kód vylučuje chybu.  
  
```vb  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 **ID chyby:** BC30949  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pokud je to možné, zadejte v deklaraci pole konstantní dimenzi.  
  
- Pokud nemůžete zadat konstantní dimenzi, je nutné inicializovat pole pomocí smyčky, když se nekonstantní dimenze označuje jako známé.
