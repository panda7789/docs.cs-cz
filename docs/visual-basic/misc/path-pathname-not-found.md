# <a name="path-pathname-not-found"></a>Cesta ' \<pathname> ' nebyla nalezena.
Operační systém nemůže najít zadanou cestu během operace přístupu k souboru nebo přístupu k disku. Cesta k souboru musí obsahovat specifikaci jednotky a adresáře a podadresáře, které musí být provázány, aby bylo možné soubor najít. Cesta může být relativní nebo absolutní.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že cesta obsahuje požadované prvky.
