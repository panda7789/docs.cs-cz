# <a name="set-statement-must-end-with-a-matching-end-set"></a>Příkaz set musí být ukončen párovým příkazem End set.
`Set`procedury vlastnosti musí končit `End Set` příkazem.  
  
 **ID chyby:** BC30633  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že `Set` procedura vlastnosti končí `End Set` konstrukcí.
