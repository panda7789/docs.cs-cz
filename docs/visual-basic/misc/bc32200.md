# <a name="membername-cannot-be-declared-shadows-outside-of-a-class-structure-or-interface"></a>\<membername>nejde deklarovat klíčové slovo Shadows vně třídy, struktury nebo rozhraní.
`Shadows`Klíčové slovo se zobrazí v deklaraci na úrovni oboru názvů, modulu nebo souboru. Stíning je smysluplný pouze v rámci programovacího prvku, který může dědit ze základního prvku.  
  
 **ID chyby:** BC32200  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Shadows` klíčové slovo nebo přesuňte deklaraci uvnitř třídy, struktury nebo rozhraní.
