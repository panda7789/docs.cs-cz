# <a name="module-statement-must-end-with-a-matching-end-module"></a>Příkaz Module musí být ukončen párovým příkazem End Module.
Bloky modulu musí končit `End Module` příkazem.  
  
 **ID chyby:** BC30625  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `End Module` příkaz na konec `Module` bloku.
