# <a name="-expected"></a>očekával se znak).
Seznam, jako je například seznam argumentů v definici procedury nebo volání procedury, není ukončen znakem pravé závorky ( `)` ).  
  
 **ID chyby:** BC30198  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte znak pravé závorky na konec seznamu.
