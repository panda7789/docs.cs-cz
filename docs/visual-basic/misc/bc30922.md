# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="a09f3-102">'\<derivedtypename >' nemůže dědit od třídy \<typu >'\<constructedbasetypename >' protože rozšiřuje přístup typu '\<internaltypename >' mimo sestavení</span><span class="sxs-lookup"><span data-stu-id="a09f3-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>
<span data-ttu-id="a09f3-103">Odvozená třída nebo rozhraní se pokusí rozšířit úroveň přístupu omezeného typu tím, že ho použije jako argument typu pro základní třídu nebo rozhraní.</span><span class="sxs-lookup"><span data-stu-id="a09f3-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="a09f3-104">Následující kód může vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="a09f3-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="a09f3-105">Kód mimo sestavení nemá povolený přístup k `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="a09f3-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="a09f3-106">@No__t-0 však lze použít z libovolného kódu, který se na něj může odkazovat.</span><span class="sxs-lookup"><span data-stu-id="a09f3-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="a09f3-107">Proto `derivedClass` nemůže dědit `baseClass`, pokud používá `restrictedStructure` jako argument typu, protože to by mohlo vystavení `restrictedStructure` pro kód v jakémkoli sestavení.</span><span class="sxs-lookup"><span data-stu-id="a09f3-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="a09f3-108">**ID chyby:** BC30922</span><span class="sxs-lookup"><span data-stu-id="a09f3-108">**Error ID:** BC30922</span></span>
