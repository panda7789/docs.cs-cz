# <a name="membername-has-the-same-name-as-a-type-parameter"></a>' \<membername> má stejný název jako parametr typu
Člen obecné třídy nebo struktury má stejný název jako parametr typu této třídy nebo struktury.  
  
 Každá proměnná, konstanta, výčet, procedura, vlastnost, událost a delegát musí mít název odlišný od každého parametru typu.  
  
 **ID chyby:** BC32054  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte název člena třídy nebo struktury tak, aby se lišil od každého názvu parametru typu.
