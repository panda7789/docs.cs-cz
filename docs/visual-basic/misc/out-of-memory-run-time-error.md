# <a name="out-of-memory-visual-basic-run-time-error"></a>Nedostatek paměti (Visual Basic Běhová chyba)
Byla požadována více paměti, než je k dispozici.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zavřete nepotřebné aplikace, dokumenty nebo zdrojové soubory, které jsou otevřeny.  
  
2. Přerušit velké moduly nebo procedury do menších.  
  
3. Eliminujte rezidentní programy a ukončete činnost.  
  
4. Eliminujte zbytečné ovladače zařízení.  
  
5. Snižte počet veřejných proměnných.
