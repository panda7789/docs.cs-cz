# <a name="field-or-property-membername-is-not-found"></a>Pole nebo vlastnost \<membername> se nenašly.
Odkaz na nedefinované pole nebo vlastnost je odkazován na atribut.  
  
 **ID chyby:** BC30661  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte pravopis pole nebo vlastnosti.  
  
2. Ověřte, že váš projekt má přístup ke správné třídě atributů a v případě potřeby jej přidejte do odkazů projektu.
