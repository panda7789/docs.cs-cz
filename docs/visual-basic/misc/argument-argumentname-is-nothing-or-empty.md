# <a name="argument-argumentname-is-nothing-or-empty"></a>Argument \<argumentname> ' ' má hodnotu Nothing nebo je prázdný.
Název argumentu není platný, protože je inicializovaný `Nothing` nebo je prázdný. Argument musí mít konkrétní datový typ a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte platný název argumentu.
