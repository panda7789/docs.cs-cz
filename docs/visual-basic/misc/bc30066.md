# <a name="exit-property-is-not-valid-in-a-function-or-sub"></a>Klíčové slovo ' Exit Property ' není platné ve funkci ani v podmnožině.
`Exit Property`zobrazí se v `Function` proceduře nebo `Sub` proceduře. `Exit`Příkaz musí odpovídat bloku, ve kterém se vyskytuje.  
  
 **ID chyby:** BC30066  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte `Exit Property` `Exit Function` `Exit Sub` příkaz nebo podle potřeby.
