# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>Příkazy SELECT nejsou platné v příkazovém podokně.
`Select`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30718  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Select` příkazy z ladění kódu.
