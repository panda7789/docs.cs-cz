# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>Metoda nemůže obsahovat příkaz try a příkaz On Error nebo Resume.
Spojili jste `Try` příkaz s `On Error` nebo `Resume` .  
  
 **ID chyby:** BC30544  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte `On Error` nebo `Resume` .
