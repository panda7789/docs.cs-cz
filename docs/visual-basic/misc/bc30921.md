# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="94d1d-102">' \<derivedtypename> ' nemůže dědit z \<type> ' \<constructedbasetypename> ', protože rozšiřuje přístup typu \<internaltypename> na ' \<region> \<regionname> '.</span><span class="sxs-lookup"><span data-stu-id="94d1d-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="94d1d-103">Odvozená třída nebo rozhraní se pokusí rozšířit úroveň přístupu interního typu pomocí objektu jako argumentu typu základní třídy nebo rozhraní.</span><span class="sxs-lookup"><span data-stu-id="94d1d-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="94d1d-104">Následující kód může vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="94d1d-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="94d1d-105">Kód mimo nemá `containingClass` povolený přístup `internalStructure` .</span><span class="sxs-lookup"><span data-stu-id="94d1d-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="94d1d-106">Lze však `derivedClass` použít z libovolného kódu ve stejném sestavení.</span><span class="sxs-lookup"><span data-stu-id="94d1d-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="94d1d-107">Proto `derivedClass` nemůže dědit, `baseClass` Pokud používá `internalStructure` jako argument typu, protože by mohl vystavit `internalStructure` celou oblast definující kód.</span><span class="sxs-lookup"><span data-stu-id="94d1d-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="94d1d-108">**ID chyby:** BC30921</span><span class="sxs-lookup"><span data-stu-id="94d1d-108">**Error ID:** BC30921</span></span>
