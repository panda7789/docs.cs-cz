# <a name="instance-of-restricted-type-typename-cannot-be-used-in-a-query-expression"></a>\<typename>Ve výrazu dotazu nejde použít instanci omezeného typu.
Instance omezených typů, například <xref:System.ArgIterator> , <xref:System.RuntimeArgumentHandle> , a <xref:System.TypedReference> , nelze použít ve výrazu lambda nebo ve výrazu dotazu.  
  
 **ID chyby:** BC36598
