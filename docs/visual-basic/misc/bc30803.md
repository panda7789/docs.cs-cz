# <a name="currency-is-no-longer-a-supported-type-use-the-decimal-type-instead"></a>' Currency ' již není podporovaným typem; místo toho použijte typ Decimal.
Datový typ byl `Currency` nahrazen `Decimal` datovým typem.  
  
 **ID chyby:** BC30803  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Místo použijte `Decimal` datový typ `Currency` .
