# <a name="type-expected"></a>Očekáván typ
K znakům zdrojového kódu, které nelze rozpoznat jako název typu. Kontext volá buď základní datový typ, nebo deklarovaný název elementu představující typ.  
  
 **ID chyby:** BC30182  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že byl definován typ.  
  
- Ověřte, zda je název typu zadán správně.
