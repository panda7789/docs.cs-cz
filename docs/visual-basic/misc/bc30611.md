# <a name="array-dimensions-cannot-have-a-negative-size"></a>Rozměry pole nemohou mít zápornou velikost.
Jedna nebo více zadaných hranic pole je záporné číslo. Negativní dolní index lze zadat pouze v případě, že použijete horní mez-1 k deklaraci prázdného pole. Takové pole má nula prvků, ale není `Nothing` .  
  
 **ID chyby:** BC30611  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte záporné pole svázané s kladným číslem.
