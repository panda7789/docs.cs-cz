# <a name="user-interrupt-occurred"></a>Došlo k přerušení uživatele.
Uživatel stiskl kombinaci kláves CTRL + BREAK nebo jiný klíč přerušení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkuste operaci zopakovat.
