# <a name="if-operator-requires-either-two-or-three-operands"></a>Operátor if vyžaduje dva nebo tři operandy.
`If`Operátor přijímá pouze dva nebo tři argumenty.  
  
 **ID chyby:** BC33104  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Prohlédněte si `If` výraz, abyste viděli, zda lze počet operátorů snížit nebo rozšířit buď na dva, nebo tři argumenty.  
  
- Zvažte použití jiné podmíněné konstrukce, jako je například `If...Then...Else` příkaz nebo vnořený `If...Then...Else` příkaz pro uspořádání argumentů.
