# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>Sekvence znaků--není v komentáři XML povolena.
Literál komentáře XML obsahuje posloupnost znaků "--", což není podporováno.  
  
 **ID chyby:** BC31171  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebere sekvenci znaků--v literálu komentáře XML.
