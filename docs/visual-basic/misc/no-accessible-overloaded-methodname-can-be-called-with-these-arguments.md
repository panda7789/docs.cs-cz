# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>Bez \<methodname> zužujícího převodu se s těmito argumenty nedají volat žádné dostupné přetížené.
Byla volána přetížená metoda, ale nebyla porovnána žádná metoda se seznamem poskytnutých argumentů bez zužujícího převodu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte `Option Strict Off`.  
  
2. Změňte argumenty tak, aby odpovídaly jedné z signatur přetížené metody.
