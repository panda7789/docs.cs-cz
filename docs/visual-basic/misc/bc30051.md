# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>Parametrem ParamArray musí být jednorozměrné pole.
`ParamArray`Argument musí být deklarován jako jednorozměrné pole typu. Deklarace určuje více než jednu dimenzi pro `ParamArray` argument.  
  
 **ID chyby:** BC30051  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- V závorkách za názvem argumentu odeberte všechny čárky `ParamArray` .
