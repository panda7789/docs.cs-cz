# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Rozlišení přetěžování se nezdařilo, protože žádnou dostupnou metodu nelze \<method> volat s těmito argumenty:.\<error>
Provedli jste volání přetížené metody, ale kompilátor nemůže najít metodu se shodným seznamem argumentů.  
  
 **ID chyby:** BC30518  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Podívejte se na seznam argumentů, které jste zadali.
