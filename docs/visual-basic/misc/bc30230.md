# <a name="inherits-not-valid-in-modules"></a>Klíčové slovo Inherits není platné v modulech.
`Inherits`V modulu dojde k příkazu. Moduly nemůžou dědit z tříd.  
  
 **ID chyby:** BC30230  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Inherits` příkaz nebo znovu zadejte modul jako třídu.
