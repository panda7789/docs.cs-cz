# <a name="expected-one-of-dim-const-public-private-protected-friend-shadows-readonly-or-shared"></a>Očekávalo se jedno z "Dim", "const", "public", "Private", "protected", "Friend", "Shadows", "ReadOnly" nebo "Shared"
V příkazu deklarace chybí klíčové slovo deklarace. Jednou z možných příčin je, že deklarace atributu volá metodu.  
  
 **ID chyby:** BC30195  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, zda je metoda deklarována uvnitř deklarace atributu.  
  
2. Spusťte příkaz s příslušným klíčovým slovem deklarace.
