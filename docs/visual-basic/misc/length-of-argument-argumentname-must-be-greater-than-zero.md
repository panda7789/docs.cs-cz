# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>Délka argumentu \<argumentname> musí být větší než nula.
Argument má hodnotu, která je menší nebo rovna nule.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument, který je větší než nula.
