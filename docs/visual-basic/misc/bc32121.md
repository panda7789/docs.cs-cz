# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-for-some-type-arguments"></a><span data-ttu-id="55c64-102">Nejde zdědit rozhraní \<interfacename1> , protože rozhraní \<interfacename2> , ze kterého se dědí, by mohlo být \<interfacename3> pro některé argumenty typu identické s rozhraním.</span><span class="sxs-lookup"><span data-stu-id="55c64-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' for some type arguments</span></span>
<span data-ttu-id="55c64-103">Obecné rozhraní dědí ze dvou nebo více obecných rozhraní a dvě dědičnost by mohly kolidovat pro určité hodnoty argumentů typu.</span><span class="sxs-lookup"><span data-stu-id="55c64-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="55c64-104">Následující příkazy mohou vygenerovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="55c64-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="55c64-105">Pokud `derivedInterface` je vytvořena nebo implementováno poskytnutí stejného typu pro obojí `t1` a `t2` , musí dědit dvě verze `interfaceX` s identickými argumenty typu.</span><span class="sxs-lookup"><span data-stu-id="55c64-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="55c64-106">Tím by vznikla nejednoznačnost, o které verzi přistupuje.</span><span class="sxs-lookup"><span data-stu-id="55c64-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="55c64-107">**ID chyby:** BC32121</span><span class="sxs-lookup"><span data-stu-id="55c64-107">**Error ID:** BC32121</span></span>
