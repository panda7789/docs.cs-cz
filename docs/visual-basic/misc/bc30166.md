# <a name="cannot-create-an-instance-of-module-modulename"></a>Nejde vytvořit instanci \<modulename> modulu.
Modul existuje pouze jako jedna sdílená instance a další instance nelze vytvořit.  
  
 **ID chyby:** BC30166  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte modul na třídu nebo ji nahraďte v `New` klauzuli názvem třídy.
