# <a name="expected-closing--for-xml-cdata-section"></a>Bylo očekáváno ukončení oddílu CDATA kódu XML znakem ']] > '
V literálu CDATA XML chybí požadované znaky uzavíracího znaku ']] > '.  
  
 **ID chyby:** BC31162  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte požadované uzavírací znaky ']] > ' na konec literálu CDATA XML.
