# <a name="if-block-must-end-with-a-matching-end-if"></a>Blok ' #If ' musí končit párovým klíčovým #End If
`#If`je podmíněná direktiva kompilace. `#If`Blok není zakončen `#End If` direktivou.  
  
 **ID chyby:** BC30012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `#End If` direktivu na konec podmíněné kompilačního bloku.
