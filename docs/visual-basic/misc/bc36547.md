# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="3ea62-102">Člen anonymního typu nebo vlastnost ' \<propertyname> ' jsou již deklarovány.</span><span class="sxs-lookup"><span data-stu-id="3ea62-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="3ea62-103">Název vlastnosti lze v deklaraci anonymního typu vytvořit pouze jednou.</span><span class="sxs-lookup"><span data-stu-id="3ea62-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="3ea62-104">Například následující deklarace nejsou platné:</span><span class="sxs-lookup"><span data-stu-id="3ea62-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="3ea62-105">**ID chyby:** BC36547</span><span class="sxs-lookup"><span data-stu-id="3ea62-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3ea62-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="3ea62-106">To correct this error</span></span>  
  
- <span data-ttu-id="3ea62-107">Pro jednu z vlastností vyberte jiný název.</span><span class="sxs-lookup"><span data-stu-id="3ea62-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="3ea62-108">Zadejte nové názvy proměnných nebo názvů vlastností, ze kterých jsou odvozeny názvy a hodnoty.</span><span class="sxs-lookup"><span data-stu-id="3ea62-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
