# <a name="element-is-missing-an-end-tag"></a>V elementu chybí koncová značka.
V literálu elementu XML chybí koncová značka XML nebo prázdný identifikátor značky/(například `<tagName />` ).  
  
 **ID chyby:** BC31151  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte koncovou značku pro značku XML nebo přidejte prázdný identifikátor značky '/' do značky XML.
