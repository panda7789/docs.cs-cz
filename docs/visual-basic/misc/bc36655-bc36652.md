<span data-ttu-id="30c23-103">Z těchto argumentů nelze odvodit datové typy parametrů typu v metodě rozšíření \<methodname> definované v, \<typename> protože je přípustný více než jeden typ.</span><span class="sxs-lookup"><span data-stu-id="30c23-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="30c23-104">Je možné, že se chybu podaří odstranit explicitním zadáním datových typů.</span><span class="sxs-lookup"><span data-stu-id="30c23-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="30c23-105">Byl proveden pokus o použití odvození typu k určení typu (nebo typů) parametru typu (nebo parametrů) ve volání obecné metody rozšíření.</span><span class="sxs-lookup"><span data-stu-id="30c23-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="30c23-106">Kompilátor najde více než jeden možný datový typ pro jeden nebo více parametrů typu a oznámí tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="30c23-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="30c23-107">Pokud se zadáním argumentů nejedná o možnost (například pro operátory dotazu ve výrazech dotazu), zobrazí se chybová zpráva bez druhé věty.</span><span class="sxs-lookup"><span data-stu-id="30c23-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="30c23-108">Následující kód demonstruje chybu.</span><span class="sxs-lookup"><span data-stu-id="30c23-108">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Imports System.Runtime.CompilerServices
Module Module1
    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    <Extension()> _
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```
