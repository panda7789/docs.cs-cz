# <a name="module-modulename-cannot-be-used-as-a-type"></a>Modul se \<modulename> nedá použít jako typ.
Pokusili jste se použít modul jako typ. Modul nelze použít s výjimkou `Get` typu.  
  
 **ID chyby:** BC30371  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte modul něčím, který se dá použít jako platný typ.
