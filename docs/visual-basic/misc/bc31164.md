# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Očekávalo se párové koncové uvozovky hodnoty atributu XML.
Hodnota atributu, která je oddělena dvojitými uvozovkami, postrádá znak pravé uvozovky.  
  
 **ID chyby:** BC31164  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte uzavírací znak dvojité uvozovky k vymezení hodnoty atributu.
