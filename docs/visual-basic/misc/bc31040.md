# <a name="membername-exists-in-multiple-base-interfaces"></a>\<membername>existuje v několika základních rozhraních.
\<membername>existuje v několika základních rozhraních. Místo názvu odvozeného rozhraní použijte název rozhraní, které deklaruje \<membername> v klauzuli Implements.  
  
 Toto rozhraní dědí členy se stejným názvem z více rozhraní a vytváří nejednoznačnost.  
  
 **ID chyby:** BC31040  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `Implements`Místo názvu odvozeného rozhraní použijte definici názvu rozhraní v klauzulích.
