# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>Hodnotu ' \<valuename1> ' nelze převést na ' \<valuename2> '
Došlo k pokusu o převod typu mezi dvěma nekompatibilními typy.  
  
 **ID chyby:** BC30742  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte typy, které se mají přetypovat.
