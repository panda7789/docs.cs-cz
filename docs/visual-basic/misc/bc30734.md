# <a name="name-is-already-declared-as-a-parameter-of-this-method"></a>\<name>už je deklarovaný jako parametr této metody.
Zadaný název parametru již byl pro tuto metodu deklarován.  
  
 **ID chyby:** BC30734  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přejmenujte parametr tak, aby byl jedinečný.
