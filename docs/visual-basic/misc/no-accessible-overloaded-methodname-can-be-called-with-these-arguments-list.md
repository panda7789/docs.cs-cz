# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-list"></a>Pomocí těchto argumentů nelze volat žádné dostupné přetížené metody \<methodname> :\<list>
Je volána přetížená metoda, ale žádná metoda nemůže být shodná se seznamem poskytnutých argumentů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Porovná seznam argumentů s přetížením, které jste chtěli volat.  
  
- Upravte seznam argumentů tak, aby signatura odpovídala přetížení.
