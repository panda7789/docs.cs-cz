# <a name="argument-argumentname-cannot-be-converted-to-type-date"></a>Argument se \<argumentname> nedá převést na typ Date.
Převod se pokusil změnit proměnnou na typ `Date` , ale hodnota není rozpoznána jako platné datum.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Najděte smysluplný převod.
