# <a name="end-namespace-must-be-preceded-by-a-matching-namespace"></a>Příkazu End Namespace musí předcházet párové klíčové slovo Namespace.
Bloky oboru názvů začínají `Namespace` příkazem a končí `End Namespace` konstrukcí.  
  
 **ID chyby:** BC30623  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že blok oboru názvů začíná `Namespace` příkazem.
