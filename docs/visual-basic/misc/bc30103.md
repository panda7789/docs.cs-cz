# <a name="-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-type"></a>operátor! vyžaduje, aby levý operand byl parametr typu, třídy nebo typu rozhraní, ale tento operand má \<type> typ.
Přístup ke členu slovníku, který používá vykřičník ( `!` ) namísto tečky ( `.` ), je platný pouze pro třídu nebo rozhraní.  
  
 **ID chyby:** BC30103  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte výraz nalevo od vykřičníku s jedním, který se vyhodnotí na definovanou třídu nebo typ rozhraní.
