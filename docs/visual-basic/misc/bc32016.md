# <a name="procedurename-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>' \<procedurename> ' nemá žádné parametry a jeho návratový typ nelze indexovat.
Volání `Function` `Sub` metody nebo poskytuje jeden nebo více argumentů, ale procedura nepřijímá žádné argumenty a její návratový typ (Pokud se jedná o `Function` ) není typ pole.  
  
 **ID chyby:** BC32016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte argument nebo argumenty z volání procedury.
