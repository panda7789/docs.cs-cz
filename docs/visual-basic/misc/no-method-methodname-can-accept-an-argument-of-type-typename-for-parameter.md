# <a name="no-method-methodname-can-accept-an-argument-of-type-typename-for-parameter-parametername"></a>Žádná metoda ' \<methodname> ' nemůže přijmout argument typu ' ' \<typename> pro parametr ' \<parametername> '.
Volání metody obsahuje argument typu, který není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte typy argumentů.  
  
2. Porovná typy argumentů s typy parametrů metody, abyste se ujistili, že jsou ve smlouvě. Podle potřeby upravte typy.
