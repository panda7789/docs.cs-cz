# <a name="the-connectiontimeout-must-be-greater-than-0"></a>Hodnota ConnectionTimeout musí být větší než 0.
Když odesíláte a stahujete soubory pomocí [My. Computer. Network](xref:Microsoft.VisualBasic.Devices.Network), musíte zadat `connectionTimeout` větší než `0` .  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte `connectionTimeout` , který je větší než `0` .
