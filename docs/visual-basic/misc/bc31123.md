# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>Příkazu End Event musí předcházet párové klíčové slovo Custom Event.
`End Event`K příkazu dojde bez odpovídajícího `Custom Event` příkazu. `End Event`musí předcházet odpovídající `Custom Event` příkaz.  
  
 **ID chyby:** BC31123  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že předchozí `Custom Event` příkaz je platný a správně napsaný.
