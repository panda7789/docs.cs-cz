# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>Příkazy Declare ve struktuře nelze deklarovat \<specifier> jako.
`Declare`Deklarace má specifikátor, který není platný v rámci `Structure` deklarace.  
  
 **ID chyby:** BC30791  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
