# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Parametr komentáře XML \<parameter> neodpovídá parametru u odpovídajícího \<keyword> příkazu.
Parametr komentáře XML \<parameter> neodpovídá parametru u odpovídajícího \<keyword> příkazu. Komentář XML bude ignorován.  
  
 Značka komentáře XML byla zadána pro některé parametry v metodě, ale ne všechny.  
  
 **ID chyby:** BC42307  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte značku komentáře XML pro všechny parametry v metodě.
