# <a name="overload-resolution-failed-because-no-method-is-accessible"></a>Rozlišení přetěžování se nezdařilo, protože \<method> není přístupná žádná hodnota.
Provedli jste volání přetížené metody, ale kompilátor nemůže najít přístupnou metodu se seznamem parametrů, který odpovídá podmínkám počtu parametrů.  
  
 **ID chyby:** BC30517  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Podívejte se na seznam argumentů, které jste zadali.  
  
2. Zkontrolujte metodu, kterou se pokoušíte volat, abyste zjistili, jestli je přístupná.
