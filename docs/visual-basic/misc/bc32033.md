# <a name="end-get-must-be-the-first-statement-on-a-line"></a>Příkaz End Get musí být prvním příkazem na řádku.
`End Get`Příkaz sleduje dvojtečku (:) oddělovač příkazů `End Get`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC32033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
