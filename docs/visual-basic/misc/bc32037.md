# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a><span data-ttu-id="51081-102">Příkaz ' Next ' pojmenovává více proměnných, než jsou k dispozici párové příkazy ' for '</span><span class="sxs-lookup"><span data-stu-id="51081-102">'Next' statement names more variables than there are matching 'For' statements</span></span>
<span data-ttu-id="51081-103">Vnořené smyčky jsou ukončeny jediným `Next` příkazem, který určuje více proměnných smyčky, než jsou smyčky v vnořování, jako v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="51081-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```vb

   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
