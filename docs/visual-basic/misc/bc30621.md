# <a name="end-structure-must-be-preceded-by-a-matching-structure"></a>Příkazu End Structure musí předcházet párové klíčové slovo Structure.
`End Structure`Příkaz lze použít pouze na konci bloku struktury, který začíná `Structure` příkazem.  
  
 **ID chyby:** BC30621  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že aktuální blok struktury začíná `Structure` příkazem.
