# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>Návěstí \<labelname> je už definované v aktuální metodě nebo víceřádkovém výrazu lambda.
Název popisku je duplikován v rámci procedury. Všechny názvy v rámci procedury musí být jedinečné.  
  
 **ID chyby:** BC30094  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přejmenujte duplicitní popisek.
