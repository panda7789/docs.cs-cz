# <a name="argument-not-specified-for-parameter-parametername-of-extension-method-methodname-defined-in-typename"></a>Není zadán argument pro parametr \<parametername> metody rozšíření \<methodname> definované v ' \<typename> '.
Nebyl zadán argument pro požadovaný parametr v metodě rozšíření.  
  
 **ID chyby:** BC36586  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument pro zadaný parametr.
