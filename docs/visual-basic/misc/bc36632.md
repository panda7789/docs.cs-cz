# <a name="nullable-parameters-must-specify-a-type"></a>Parametry s možnou hodnotou null musí určovat typ.
Parametr obsahuje modifikátor typu s možnou hodnotou null (?), ale neurčuje typ.  
  
 **ID chyby:** BC36632  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte typ při deklaraci parametru typu hodnoty s možnou hodnotou null.
