# <a name="nullable-parameters-must-specify-a-type"></a>Parametry s možnou hodnotou Null musí určit typ.
Parametr obsahuje modifikátor typu null (?), ale neurčuje typ.  
  
 **ID chyby:** BC36632  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Při deklarování parametru typu hodnoty s možnou hodnotou zadejte typ.
