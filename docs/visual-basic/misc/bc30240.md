# <a name="exit-must-be-followed-by-sub-function-property-do-for-while-select-or-try"></a>Za klíčovým slovem Exit musí následovat klíčové slovo Sub, Function, Property, do, for, while, SELECT nebo try.
`Exit`Příkaz obsahuje neplatné klíčové slovo. `Exit`musíte určit blok, ze kterého se má ovládací prvek přenést do příkazu za blokem; například `End While` .  
  
 **ID chyby:** BC30240  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Níže přidejte platné klíčové slovo `Exit` nebo odeberte `Exit` příkaz.
