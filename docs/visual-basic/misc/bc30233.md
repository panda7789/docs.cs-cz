# <a name="specifier-is-not-valid-on-a-constant-declaration"></a>' \<specifier> ' není platný v deklaraci konstanty
`Const`Příkaz obsahuje neplatné klíčové slovo. `Const`Příkaz může obsahovat pouze `Friend` `Private` `Protected` `Public` klíčová slova,,, nebo `Shadows` .  
  
 **ID chyby:** BC30233  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že klíčové slovo je napsané správně.  
  
2. Odeberte z příkazu neplatné klíčové slovo `Const` .
