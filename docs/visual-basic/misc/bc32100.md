# <a name="as-comma-or--expected"></a>Byl očekáván operátor as, čárka nebo znak).
Zadaný parametr typu neobsahuje `As` čárku ( `,` ) nebo pravou závorku ( `)` ).  
  
 **ID chyby:** BC32100  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte chybějící `As` , čárku nebo pravou závorku.
