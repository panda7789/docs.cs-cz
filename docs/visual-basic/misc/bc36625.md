# <a name="lambda-expression-cannot-be-converted-to-typename-because-typename-is-not-a-delegate-type"></a>Výraz lambda nejde převést na, \<typename> protože není \<typename> typu delegáta.
Výrazy lambda lze použít tam, kde jsou delegáti platný. Lze je převést na kompatibilní typy delegátů, ale ne na jiný typ. Můžete například definovat typ delegáta a přiřadit k němu výraz lambda nebo odeslat výraz lambda jako argument <xref:System.Func%601> parametru. Tyto příklady jsou uvedeny v následujícím kódu.  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```  
  
 **ID chyby:** BC36625
