# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument \<argumentname> musí být větší nebo roven nule.
Argument je neplatný, protože je menší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte pravopis argumentů. Nesprávně napsaný název proměnné může implicitně vytvořit číselnou proměnnou, která je inicializovaná na nulu.  
  
2. Zkontroluje předchozí operace s proměnnými ve výrazu, zejména ty, které byly předány do procedury jako argumenty z jiných postupů.
