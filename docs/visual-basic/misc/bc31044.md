# <a name="events-declared-with-an-as-clause-must-have-a-delegate-type"></a>Události deklarované s klauzulí as musí mít typ delegáta.
Byl zadán typ, který není delegátem.  
  
 **ID chyby:** BC31044  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte `As` klauzuli.  
  
2. Zadejte delegáta s `As` klauzulí.
