# <a name="set-is-already-declared"></a>Klíčové slovo set je už deklarované.
`Set`Příkaz se v bloku kódu vyskytuje více než jednou. `Set`Příkaz deklaruje `Set` proceduru vlastnosti použitou k přiřazení hodnoty k vlastnosti.  
  
 **ID chyby:** BC30444  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte redundantní `Set` příkaz.
