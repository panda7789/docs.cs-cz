# <a name="exponent-is-not-valid"></a>Exponent není platný.
Pokusili jste se použít exponent, který není platný literál s plovoucí desetinnou čárkou. Tato chyba obvykle nastává, když je typ literálu s plovoucí desetinnou čárkou v lichá s velikostí požadovanou pro uchování literálu s plovoucí desetinnou čárkou.  
  
 **ID chyby:** BC30495  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte typ literálu s plovoucí desetinnou čárkou a změňte jej na platný typ.
