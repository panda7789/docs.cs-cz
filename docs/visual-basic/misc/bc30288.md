# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>Lokální proměnná \<variablename> už je v aktuálním bloku deklarovaná.
Dvě proměnné se stejným názvem jsou deklarovány ve stejném bloku (což může být procedura nebo řídicí blok, například `If...Then` ).  
  
 **ID chyby:** BC30288  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte jednu z deklarací na jiný blok nebo změňte název jedné z proměnných.
