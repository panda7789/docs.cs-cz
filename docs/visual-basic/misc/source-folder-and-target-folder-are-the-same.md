# <a name="source-folder-and-target-folder-are-the-same"></a>Zdrojová složka a cílová složka jsou stejné.
Byl proveden pokus o zkopírování nebo přesunutí souboru, ale zadaný zdrojový adresář a cílový adresář jsou stejné. Může to být způsobeno nesprávnou analýzou cesty k souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že jste zadali správný zdrojový a cílový adresář.
