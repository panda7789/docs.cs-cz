# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="93738-102">Pole se nedají deklarovat pomocí New.</span><span class="sxs-lookup"><span data-stu-id="93738-102">Arrays cannot be declared with 'New'</span></span>

<span data-ttu-id="93738-103">`New`Klíčové slovo se může zobrazit pouze v části inicializace deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="93738-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="93738-104">To znamená, že `New` se musí nacházet na pravé straně znaménka rovná se ( `=` ), takže může vytvořit nový typ pole k přiřazení proměnné pole.</span><span class="sxs-lookup"><span data-stu-id="93738-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>

<span data-ttu-id="93738-105">Zástupce pro inicializaci třídy není pro pole k dispozici.</span><span class="sxs-lookup"><span data-stu-id="93738-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="93738-106">Následující dva řádky kódu jsou platné a jsou ekvivalentní, protože inicializují objekt z třídy.</span><span class="sxs-lookup"><span data-stu-id="93738-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>

```vb
Dim formA as Form = New Form
Dim formA as New Form
```

<span data-ttu-id="93738-107">Inicializace pole ale nemůže používat stejný zástupce jako inicializace třídy.</span><span class="sxs-lookup"><span data-stu-id="93738-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>

<span data-ttu-id="93738-108">Všimněte si, že `New` klauzule pro pole musí obsahovat závorky, `()` i složené závorky, `{}` .</span><span class="sxs-lookup"><span data-stu-id="93738-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="93738-109">Závorky určují, že nový typ je pole a složené závorky poskytují inicializační hodnoty.</span><span class="sxs-lookup"><span data-stu-id="93738-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="93738-110">Kompilátor vyžaduje složené závorky i v případě, že jsou prázdné, tedy i v případě, že neinicializujete žádnou z hodnot pole.</span><span class="sxs-lookup"><span data-stu-id="93738-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>

<span data-ttu-id="93738-111">**ID chyby:** BC30053</span><span class="sxs-lookup"><span data-stu-id="93738-111">**Error ID:** BC30053</span></span>
