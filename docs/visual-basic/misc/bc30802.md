# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>Příkazy Type již nejsou podporovány. místo toho použijte příkazy Structure.
`Type`Klíčové slovo již nelze použít k definování uživatelsky definovaných typů.  
  
 **ID chyby:** BC30802  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pomocí `Structure` příkazu definujte nový typ.
