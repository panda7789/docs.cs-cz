# <a name="structure-structurename-cannot-be-indexed-because-it-has-no-default-property"></a>Strukturu se \<structurename> nedá indexovat, protože nemá žádnou výchozí vlastnost.
Byl proveden pokus o použití výchozí syntaxe vlastnosti se strukturou, která nemá žádnou výchozí vlastnost.  
  
 **ID chyby:** BC30690  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pro přístup k vlastnostem struktury použijte syntaxi standardní vlastnosti.
