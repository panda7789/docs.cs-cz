# <a name="typename-is-a-namespace-and-cannot-be-used-as-an-expression"></a>\<typename>je obor názvů a nedá se použít jako výraz.
K názvu typu dojde v případě, že je vyžadován výraz. Výraz se musí skládat z proměnných, konstant, literálů, vlastností a `Function` volání procedur.  
  
 **ID chyby:** BC30112  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte název typu a vytvořte výraz pomocí platných prvků.
