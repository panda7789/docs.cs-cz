# <a name="interface-must-end-with-a-matching-end-interface"></a>Klíčové slovo Interface musí být ukončené párovým klíčovým znakem End Interface.
`Interface`K příkazu dojde bez odpovídajícího `End Interface` příkazu. `End Interface`K ukončení definice rozhraní musíte použít příkaz.  
  
 **ID chyby:** BC30253  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že `Interface` jsou členové správně naformátované.  
  
2. Přidejte `End Interface` příkaz na konec definice rozhraní.
