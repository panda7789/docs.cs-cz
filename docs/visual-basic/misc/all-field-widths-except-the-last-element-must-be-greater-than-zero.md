# <a name="all-field-widths-except-the-last-element-must-be-greater-than-zero"></a>Šířka všech polí, s výjimkou posledního prvku, musí být větší než nula.
Šířka všech polí, s výjimkou posledního prvku, musí být větší než nula. Šířka pole menší nebo rovna nule v posledním prvku označuje, že poslední pole má proměnlivou délku.  
  
 Operace se nezdařila, protože šířka pole jiná než poslední prvek je nastavena na hodnotu nula nebo menší. Šířka pole menší nebo rovna nule se dá použít jako poslední prvek k označení toho, že poslední pole má proměnlivou délku, ale nedá se použít jako žádný jiný element.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nastavte šířku pole na správnou délku.
