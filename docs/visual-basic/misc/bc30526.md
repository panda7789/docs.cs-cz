# <a name="property-propertyname-is-readonly"></a>Vlastnost \<propertyname> je jen pro čtení.
Pokusili jste se přiřadit hodnotu k vlastnosti, která je deklarována `ReadOnly` .  
  
 **ID chyby:** BC30526  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `ReadOnly` specifikátor z deklarace vlastnosti.
