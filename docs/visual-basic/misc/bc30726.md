# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>Příkazy a kontexty with nejsou platné v ladicích oknech.
`With`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30726  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `With` příkazy z ladění kódu.
