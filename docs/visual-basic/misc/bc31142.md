# <a name="systemobsoleteattribute-cannot-be-applied-to-the-addhandler-removehandler-or-raiseevent-definitions"></a>Atribut System. ObsoleteAttribute nejde použít pro definice AddHandler, RemoveHandler nebo RaiseEvent.
Atribut System. ObsoleteAttribute nelze použít pro definice AddHandler, RemoveHandler a RaiseEvent. V případě potřeby použijte atribut přímo pro událost.  
  
 Vlastní událost platí <xref:System.ObsoleteAttribute> pro svůj `AddHandler` `RemoveHandler` postup, nebo `RaiseEvent` .  
  
 <xref:System.ObsoleteAttribute>Označuje programovací prvek jako již nepoužívá a informuje uživatele o tom, že je nutné odebrat element v budoucích verzích produktu.  
  
 Nedoporučuje se, aby některé části vlastní události byly stále používány, zatímco jiné se již nepoužívají.  
  
 **ID chyby:** BC31142  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte <xref:System.ObsoleteAttribute> z deklarace jednotlivé procedury a použijte ji pro celkovou deklaraci události.
