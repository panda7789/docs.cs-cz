# <a name="arguments-are-not-valid"></a>Argumenty nejsou platné.
Argumenty, které nejsou platné, jsou předávány funkci nebo podrutině.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte argumenty a určete, proč jsou neplatné.  
  
2. Pokud jsou argumenty dodány pomocí výpočtů, podívejte se na hodnoty, které výpočty zpracovávají.
