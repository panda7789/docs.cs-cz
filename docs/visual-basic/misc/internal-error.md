# <a name="internal-error-visual-basic"></a>Vnitřní chyba (Visual Basic)
V Visual Basic došlo k vnitřnímu selhání.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že tato chyba nebyla vygenerována `Error` příkazem nebo `Raise` metodou. Pokud to nebylo, obraťte se na oddělení Microsoft Product Support Services, které oznámí podmínky, za kterých se zobrazila zpráva.
