# <a name="next-expected"></a>Byl očekáván příkaz Next.
`On Error Resume`Konstrukce probíhá bez uzavírání `Next` .  
  
 **ID chyby:** BC30003  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Next` klíčové slovo na konec `On Error` příkazu.
