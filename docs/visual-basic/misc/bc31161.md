# <a name="expected-closing----for-xml-comment"></a>Očekávalo se ukončení komentáře XML-->.
V literálu komentáře XML chybí požadované uzavírací znaky-->.  
  
 **ID chyby:** BC31161  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte požadované uzavírací znaky '--> ' na konec literálu komentáře XML.
