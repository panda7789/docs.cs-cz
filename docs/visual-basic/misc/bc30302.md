# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Znak typu se \<character> nedá použít v deklaraci s explicitním typem.
Použili jste znak typu, například `$` , při explicitním určení typu.  
  
 **ID chyby:** BC30302  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte znak typu nebo odeberte `As <Type>` klauzuli z deklarace.
