# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>Příkaz end AddHandler musí být prvním příkazem na řádku.
`End AddHandler`Příkaz sleduje dvojtečku (:) oddělovač příkazů `End AddHandler`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC31118  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
