# <a name="argument-argumentname-cannot-be-converted-to-type-typename"></a>Argument ' \<argumentname> ' nelze převést na typ ' \<typename> '.
Převod se pokusí změnit argument na neplatnou hodnotu typu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte typ na platný.
