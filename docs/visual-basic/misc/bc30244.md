# <a name="modifier-is-not-valid-on-a-declare"></a>' \<modifier> ' není platný v deklaraci
`Declare`Příkaz obsahuje neplatné klíčové slovo, například `ReadOnly` .  
  
 **ID chyby:** BC30244  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte z příkazu neplatné klíčové slovo `Declare` .
