# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metoda nemůže mít současně parametr ParamArray i Optional.
`ParamArray`Argument je automaticky volitelný a musí být jediným volitelným argumentem v deklaraci procedury. Musí být vyžadovány všechny předchozí argumenty.  
  
 **ID chyby:** BC30046  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte volitelné argumenty v deklaraci.
