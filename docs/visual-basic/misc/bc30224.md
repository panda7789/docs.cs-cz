# <a name="is-expected"></a>Očekává se klíčové slovo is.
`TypeOf`Operátor se vyskytuje bez `Is` klauzule.  
  
 **ID chyby:** BC30224  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Is` klauzuli za `TypeOf` operátorem; například `TypeOf MyControl Is ComboBox` .
