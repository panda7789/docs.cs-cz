# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>Značka komentáře XML není \<tag> povolená u \<element> elementu jazyka.
Značka komentáře XML \<tag> není povolená u \<element> elementu. Komentář XML bude ignorován.  
  
 Značka komentáře XML není podporována prvkem jazyka, na který je použita.  
  
 **ID chyby:** BC42306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nepodporovanou značku komentáře XML.
