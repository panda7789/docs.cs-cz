# <a name="namespace-statement-must-end-with-a-matching-end-namespace"></a>Příkaz Namespace musí být ukončený párovým příkazem End Namespace.
`Namespace`bloky musí končit `End Namespace` příkazem.  
  
 **ID chyby:** BC30626  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `End Namespace` příkaz na konec `Namespace` bloku.
