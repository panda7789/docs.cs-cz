# <a name="class-not-registered-on-local-machine"></a>Třída není zaregistrovaná na místním počítači.
K této chybě obvykle dochází v důsledku poškozeného souboru. Možná budete muset sadu Visual Studio přeinstalovat.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přeinstalujte Visual Studio.
