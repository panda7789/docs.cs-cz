# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-object-class"></a>Název proměnné rozsahu se nesmí shodovat s názvem členu třídy Object.
Proměnná rozsahu v dotazu LINQ odpovídá názvu člena <xref:System.Object> třídy. Tím dojde ke konfliktu s objekty vytvořenými Visual Basic pro dotaz LINQ.  
  
 **ID chyby:** BC36606  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Vyberte nový název proměnné rozsahu, která neodpovídá názvu žádného člena <xref:System.Object> třídy.
