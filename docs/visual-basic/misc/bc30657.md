# <a name="procedurename-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>\<procedurename>má návratový typ, který není podporovaný, nebo typy parametrů, které nejsou podporované.
Procedura je deklarována s parametry nebo návratovým typem, který nemůže být reprezentován sadou Visual Studio.  
  
 **ID chyby:** BC30657  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Definujte návratové typy nebo parametry jako podporovaný typ.  
  
- Vytvořte nový typ, který popisuje data používaná tímto postupem.
