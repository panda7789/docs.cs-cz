# <a name="a-delimiter-cannot-be-nothing-or-an-empty-string"></a>Oddělovač nemůže být Nothing ani prázdný řetězec.
`TextFieldParser`Nelze číst ze souboru, protože `Delimiters` vlastnost je nastavena na `Nothing` nebo je prázdná `String` ("").  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu pro `Delimiters` .
