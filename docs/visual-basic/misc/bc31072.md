# <a name="warning-treated-as-error-error"></a>Upozornění je považováno za chybu:\<error>
Při spouštění sestavení zkompilovaného pomocí možnosti **považovat upozornění jako chyby** byla zjištěna varovná zpráva.  
  
 **ID chyby:** BC31072  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Opravte upozornění.  
  
2. Zkompilujte sestavení pomocí možnosti **považovat upozornění jako chyby** je vypnuto.
