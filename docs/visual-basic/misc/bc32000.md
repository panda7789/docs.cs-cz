# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>Na místní proměnnou \<variablename> nelze odkazovat dříve, než je deklarována.
Proměnná uvnitř procedury se používá před svou deklarací.  
  
 **ID chyby:** BC32000  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte deklaraci na předcházející první odkaz na proměnnou.
