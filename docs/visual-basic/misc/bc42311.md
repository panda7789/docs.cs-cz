# <a name="unable-to-create-xml-documentation-file-name-message"></a>Nelze vytvořit soubor dokumentace XML ' \<name> ':\<message>
Soubor dokumentace XML nelze vytvořit z důvodu chyby.  
  
 **ID chyby:** BC42311  
  
 S touto chybou se mohou zobrazit následující zprávy:  
  
- [Blok komentáře XML nelze přidružit k žádnému prvku jazyka, který podporuje použití dokumentačních komentářů XML](bc42312.md).  
  
- [Značka komentáře XML Returns není u vlastnosti WriteOnly povolena](bc42313.md).  
  
- [Komentář XML nelze použít více než jednou pro částečné \<type> ](bc42314.md).  
  
- [Značka komentáře XML Returns není povolená u elementu jazyka Declare Sub](bc42315.md).  
  
- [Chyba analýzy dokumentace XML: počáteční značka \<tag> nemá zarovnávací koncovou značku](bc42316.md).  
  
- [Parametr typu komentáře XML \<parameter> neodpovídá parametru typu u odpovídajícího \<keyword> příkazu](bc42317.md).  
  
- [Parametr typu komentáře XML musí mít atribut Name](bc42318.md).  
  
- [Výjimka komentáře XML musí mít atribut cref](../language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).
