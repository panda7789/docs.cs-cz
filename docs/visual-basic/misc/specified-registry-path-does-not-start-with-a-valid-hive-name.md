# <a name="specified-registry-path-does-not-start-with-a-valid-hive-name"></a>Zadaná cesta registru nezačíná platným názvem podregistru.
Zadaná cesta registru nezačíná platným názvem podregistru. Platné názvy úlů jsou:  
  
- `HKEY_CLASSES_ROOT`  
  
- `HKEY_CURRENT_CONFIG`  
  
- `HKEY_CURRENT_USER`  
  
- `HKEY_DYNDATA`  
  
- `HKEY_LOCAL_MACHINE`  
  
- `HKEY_PERFORMANCE_DATA`  
  
- `HKEY_USERS`  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda zadávajíte platný název podregistru.
