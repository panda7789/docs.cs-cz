# <a name="labels-are-not-valid-in-the-immediate-window"></a>Popisky nejsou platné v příkazovém podokně.
Popisky řádků umožňují větvení a nejsou v kontextu ladění povoleny.  
  
 **ID chyby:** BC30134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nezadávejte popisek čáry v **příkazovém** okně.
