# <a name="enum-must-end-with-a-matching-end-enum"></a>Klíčové slovo enum musí být ukončené párovým klíčovým znakem end Enum.
`Enum`K příkazu dojde bez odpovídajícího `End Enum` příkazu. `End Enum`K ukončení výčtu je nutné použít příkaz.  
  
 **ID chyby:** BC30185  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že `Enum` jsou členové správně naformátované.  
  
2. Přidejte `End Enum` příkaz na konec výčtu.
