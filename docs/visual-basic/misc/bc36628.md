# <a name="a-nullable-type-cannot-be-inferred-for-variable-variablename"></a>Typ s možnou hodnotou null\<nelze odvodit pro proměnnou "variablename>"
Typ s možnou hodnotou null nelze odvodit z typu odkazu, jako je například pole, třída nebo `String`. Hodnota, ze které je odvozen datový typ, musí být typ hodnoty. Následující kód ilustruje tuto chybu.  
  
```vb  
'' Not valid.
'Dim arrList? = New ArrayList  
'Dim except? = New Exception  
'Dim obj? = New Object  
'Dim stringVar? = "Open the application."  
  
' Valid.  
Dim intVar? = 10  
```  
  
 **ID chyby:** BC36628  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte označení s možnou hodnotou null.
