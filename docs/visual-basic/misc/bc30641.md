# <a name="byval-and-byref-cannot-be-combined"></a>Klíčová slova ByVal a ByRef nejde kombinovat.
`ByVal` `ByRef` Klíčová slova a se vzájemně vylučují.  
  
 **ID chyby:** BC30641  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `ByVal` nebo `ByRef` , ale ne obojí.
