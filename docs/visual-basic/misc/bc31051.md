# <a name="namespace-or-type-name-has-already-been-imported"></a>Obor názvů nebo typ \<name> už je naimportovaný.
Byl proveden pokus o import oboru názvů více než jednou.  
  
 **ID chyby:** BC31051  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nadbytečné `Imports` příkazy.
