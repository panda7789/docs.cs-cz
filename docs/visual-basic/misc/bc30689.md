# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>Příkaz se nemůže vyskytovat vně těla metody nebo víceřádkového výrazu lambda.
V rámci procedur se musí použít spustitelné příkazy.  
  
 **ID chyby:** BC30689  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte příkaz do procedury nebo subrutiny.
