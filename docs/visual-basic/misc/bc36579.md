# <a name="too-many-type-arguments"></a>Příliš mnoho argumentů typu
Byla vyvolána obecná metoda s více argumenty typu, než jsou parametry typu.  
  
 Při vyvolání obecné metody je nutné zadat jeden argument typu pro každý parametr typu definovaný touto metodou.  
  
 **ID chyby:** BC36579  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte argumenty typu ze seznamu argumentů typu tak, aby existoval jeden pro každý parametr typu Obecné metody, kterou vyvoláte.
