# <a name="me-cannot-be-the-target-of-an-assignment"></a>Hodnota ' já ' nemůže být cílem přiřazení.
`Me`Klíčové slovo vždy odkazuje na konkrétní instanci třídy, kde je kód právě spuštěn. Její hodnotu nelze změnit.  
  
 **ID chyby:** BC30062  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkaz, kterému přiřadíte hodnotu `Me` .
