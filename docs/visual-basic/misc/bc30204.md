# <a name="integer-constant-expected"></a>Byla očekávána celočíselná konstanta.
K `#ExternalSource` direktivě se dojde v případě, že druhý argument není celočíselný literál. V tomto kontextu je platný jenom celočíselný literál. Pojmenovaný konstanta nebo člen výčtu není platný.  
  
 **ID chyby:** BC30204  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Místo literálu použijte pojmenovaný konstantu nebo člena výčtu.  
  
2. Zadejte celočíselný literál jako druhý argument `#ExternalSource` direktivy.
