# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="52a84-102">Operandy operátoru if nemohou být pojmenované argumenty.</span><span class="sxs-lookup"><span data-stu-id="52a84-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="52a84-103">Použití pojmenovaných argumentů v operandech operátoru `If` je neplatné.</span><span class="sxs-lookup"><span data-stu-id="52a84-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="52a84-104">Následující příklad způsobí tuto chybu:</span><span class="sxs-lookup"><span data-stu-id="52a84-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="52a84-105">To se liší od funkce `IIf`, která umožňuje pojmenovaným argumentům, jak je znázorněno v následujícím kódu:</span><span class="sxs-lookup"><span data-stu-id="52a84-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="52a84-106">**ID chyby:** BC33105</span><span class="sxs-lookup"><span data-stu-id="52a84-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="52a84-107">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="52a84-107">To correct this error</span></span>

    ```vb  
    result = If(i > 0, "positive", "not positive")  
    ```
