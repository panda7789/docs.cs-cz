# <a name="redim-preserve-operand-cannot-be-nothing"></a>Operand příkazu ReDim nemůže být Nothing.
`ReDim`Příkaz se pokusí použít `Preserve` klíčové slovo pro změnu rozměru pole, které není poslední dimenzí, ale neposkytuje platnou hodnotu pro svůj operand.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte `Preserve` operand na platnou hodnotu.
