# <a name="white-space-may-not-appear-here"></a>Tady se nesmí vyskytovat prázdné znaky.
Název literálu elementu XML obsahuje mezery nebo mezery.  
  
 **ID chyby:** BC31177  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte mezeru nebo mezery z názvu literálu elementu XML.
