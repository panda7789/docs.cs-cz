# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Při vyhodnocování tohoto výrazu došlo k dělení nulou.
Při vyhodnocování tohoto výrazu jste se pokusili dělit číslo nulou.  
  
 **ID chyby:** BC30542  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte hodnoty konstanty a proměnné a určete, jak se v jmenovateli vašeho výrazu vyskytuje nula.
