# <a name="end-operator-must-be-preceded-by-a-matching-operator"></a>Příkazu End operator musí předcházet párové klíčové slovo operator.
`End Operator`Před výrazem se zobrazí příkaz bez odpovídajícího `Operator` prohlášení.  
  
 **ID chyby:** BC33007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `End Operator` příkaz, pokud je redundantní.  
  
- Pokud chybí některá z těchto chybějících `Operator` procedur, zadejte ji.  
  
- Přesuňte `End Operator` příkaz na příslušné místo v kódu.
