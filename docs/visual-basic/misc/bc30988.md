<span data-ttu-id="53a64-103">Při deklaraci instance třídy `New` musí být klíčové slovo následováno názvem typu nebo `With` .</span><span class="sxs-lookup"><span data-stu-id="53a64-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="53a64-104">Například následující příkazy jsou deklarovány jako `client` instance `Customer` třídy.</span><span class="sxs-lookup"><span data-stu-id="53a64-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="53a64-105">Následuje název typu `Customer` `New` .</span><span class="sxs-lookup"><span data-stu-id="53a64-105">The type name `Customer` follows `New`.</span></span>  
  
```vb

' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="53a64-106">Počínaje Visual Basic 2008 můžete deklarovat objekt, který bude instancí anonymního typu, v takovém případě neurčíte datový typ.</span><span class="sxs-lookup"><span data-stu-id="53a64-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="53a64-107">V deklaracích anonymního typu následuje klíčové slovo `With` `New` .</span><span class="sxs-lookup"><span data-stu-id="53a64-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>
