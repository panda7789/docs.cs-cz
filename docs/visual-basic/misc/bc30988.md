# <a name="type-or-with-expected"></a>Očekával se typ nebo with.
Při deklaraci instance třídy musí být klíčové slovo `New` následováno názvem typu nebo `With`. Například následující příkazy jsou deklarovány `client` jako instance třídy `Customer`. Název typu `Customer` následuje `New`.  
  
```vb  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 Počínaje Visual Basic 2008 můžete deklarovat objekt, který bude instancí anonymního typu, v takovém případě neurčíte datový typ. V deklaracích anonymního typu klíčové slovo `With` následuje `New`.  
  
```vb  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```  
  
 **ID chyby:** BC30988  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte deklaraci tak, že `With` nebo název typu se řídí `New`.
