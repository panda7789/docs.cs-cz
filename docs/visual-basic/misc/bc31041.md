# <a name="statement-is-not-valid-in-an-interface"></a>Příkaz není platný v rozhraní.
V rozhraní byl zjištěn neplatný příkaz.  
  
 **ID chyby:** BC31041  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že příkazy v rozhraní definují prvky, ale neposkytují implementaci.
