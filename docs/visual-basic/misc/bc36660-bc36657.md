<span data-ttu-id="b81c9-105">Došlo k pokusu o použití odvození typu k určení datového typu nebo typů parametru nebo parametrů typu při vyhodnocování volání generické procedury.</span><span class="sxs-lookup"><span data-stu-id="b81c9-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="b81c9-106">Kompilátor nemohl najít datový typ, který splňuje omezení všech argumentů.</span><span class="sxs-lookup"><span data-stu-id="b81c9-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="b81c9-107">Proto ohlásil tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="b81c9-107">Therefore, it reported this error.</span></span>

> <span data-ttu-id="b81c9-108">Pokud se zadáním argumentů nejedná o možnost (například pro operátory dotazu ve výrazech dotazu), zobrazí se chybová zpráva bez druhé věty.</span><span class="sxs-lookup"><span data-stu-id="b81c9-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

        '' Not valid. Integer and Date do not convert to the same type.
