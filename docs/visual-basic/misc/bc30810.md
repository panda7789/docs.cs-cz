# <a name="methods-cannot-be-declared-static"></a>Pro metody nejde provést deklarace static.
`Static`Modifikátor nelze použít pro úplné procedury.  
  
 **ID chyby:** BC30810  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte `Static` Modifikátor k deklaraci jednotlivých položek v proceduře jako static.
