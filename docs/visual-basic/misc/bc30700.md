# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Vedlejší účinky nejsou během vyhodnocení výrazu v tomto kontextu platné.
Příkazy, které mění stav objektu, nejsou v tomto kontextu povoleny.  
  
 **ID chyby:** BC30700  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte kód tak, aby se příkazy vyhodnotily beze změny stavu dalších položek.
