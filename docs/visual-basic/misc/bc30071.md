# <a name="case-else-can-only-appear-inside-a-select-case-statement"></a>Klíčové slovo ' Case Else ' může být použit pouze uvnitř příkazu ' Select Case '
`Case Else`Příkaz se vyskytuje mimo `Select` blok. `Case Else`Příkaz lze použít pouze mezi `Select` `Select Case` příkazem nebo a jeho odpovídajícím `End Select` příkazem.  
  
 **ID chyby:** BC30071  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Case Else` příkaz nebo ho přesuňte v rámci `Select` bloku.
