# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>Znak typu se nedá použít v deklaraci parametru typu.
Deklarace parametru typu obsahuje alespoň jeden znak typu identifikátoru.  
  
 Parametr typu obecného typu musí být platný název Visual Basic. Povolené znaky neobsahují žádné znaky typu identifikátoru ( `%` , `&` , `@` ,, a `!` `#` `$` ). Viz [deklarované názvy elementů](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC32041  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte znak nebo znaky identifikátoru typu z deklarace parametru typu.
