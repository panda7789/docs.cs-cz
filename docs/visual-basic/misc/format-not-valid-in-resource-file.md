# <a name="format-not-valid-in-resource-file"></a>Neplatný formát v souboru prostředků
Soubor prostředků v projektu nedodržuje standardní formát pro soubor prostředků systému Windows nebo se neshoduje mezi souborem prostředků a Visual Basic projektu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Znovu vytvořte zdroj a soubory prostředků prostředku a dodržujte syntaxi dokumentované v sadě Windows Software Development Kit.  
  
2. Odeberte z projektu neplatný soubor prostředků.
