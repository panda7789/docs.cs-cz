# <a name="expected-closing--for-xml-entity"></a>Očekávalo se ukončení entity XML s uzavíracím znakem.
V odkazu na entitu XML (například `>` ) chybí uzavírací středník (;).  
  
 **ID chyby:** BC31178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zahrňte středník na konci odkazu na entitu XML.
