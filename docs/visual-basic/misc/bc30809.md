# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Příkazy Wend již nejsou podporovány; použijte místo toho příkazy End While.
`End While`Konstrukce nahrazuje `Wend` klíčové slovo.  
  
 **ID chyby:** BC30809  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Slouží `End While` k ukončení `While` bloku.
