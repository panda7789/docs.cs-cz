# <a name="property-propertyname-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="3b2d4-102">Vlastnost '\<propertyname >' nelze inicializovat ve výrazu inicializátoru objektu, protože všechna dostupná přetížení vyžadují argumenty</span><span class="sxs-lookup"><span data-stu-id="3b2d4-102">Property '\<propertyname>' cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>
<span data-ttu-id="3b2d4-103">Členy inicializovány v seznamu inicializátorů objektu musí být pole nebo vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="3b2d4-104">Kromě toho vlastnosti v seznamu inicializátorů nemohou mít parametry.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="3b2d4-105">Vlastnost příčinou této chyby je přetížena a každý z jeho verze vyžaduje argumenty.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="3b2d4-106">Proto vlastnost nelze inicializovat v seznamu inicializátorů objektů.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="3b2d4-107">**ID chyby:** BC30993</span><span class="sxs-lookup"><span data-stu-id="3b2d4-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3b2d4-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="3b2d4-108">To correct this error</span></span>  
  
- <span data-ttu-id="3b2d4-109">Odeberte vlastnost, která vyžaduje argumenty v seznamu inicializátorů.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3b2d4-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="3b2d4-110">Example</span></span>  
 <span data-ttu-id="3b2d4-111">Následující třída obsahuje tři definice vlastností: jeden pro `TotalItems` a dvě pro `Item`, která je přetížena.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="3b2d4-112">`TotalItems` Vlastnost nevyžaduje argumenty a mohou být inicializovány v seznamu inicializace objektu, jak je znázorněno v následující deklaraci.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="3b2d4-113">`Item` Vlastnost je přetížena a jednotlivá přetížení vyžaduje argument.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="3b2d4-114">Proto `Item` se nemůže vyskytovat v seznamu inicializátorů objektů.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="3b2d4-115">K této chybě předejít, inicializovat `Item` vlastnost mimo inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="3b2d4-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
