<span data-ttu-id="5d34b-103">Členové inicializované v seznamu inicializátorů objektů musí být buď pole, nebo vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="5d34b-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="5d34b-104">Kromě toho vlastnosti v seznamu inicializátorů nemohou mít parametry.</span><span class="sxs-lookup"><span data-stu-id="5d34b-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="5d34b-105">Vlastnost, která způsobila tuto chybu, je přetížena a každá z jejích verzí vyžaduje argumenty.</span><span class="sxs-lookup"><span data-stu-id="5d34b-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="5d34b-106">Proto nelze vlastnost inicializovat v seznamu inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="5d34b-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>

## <a name="example"></a><span data-ttu-id="5d34b-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="5d34b-110">Example</span></span>  
 <span data-ttu-id="5d34b-111">Následující třída obsahuje tři definice vlastností: jeden pro `TotalItems` a dva pro `Item` , který je přetížený.</span><span class="sxs-lookup"><span data-stu-id="5d34b-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="5d34b-112">`TotalItems`Vlastnost nevyžaduje žádné argumenty a může být inicializována v seznamu inicializace objektu, jak je znázorněno v následující deklaraci.</span><span class="sxs-lookup"><span data-stu-id="5d34b-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="5d34b-113">`Item`Vlastnost je přetížena a každé přetížení vyžaduje argument.</span><span class="sxs-lookup"><span data-stu-id="5d34b-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="5d34b-114">Proto se `Item` nemůže objevit v seznamu inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="5d34b-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="5d34b-115">Chcete-li se této chybě vyhnout, inicializujte `Item` vlastnost mimo inicializátor objektu.</span><span class="sxs-lookup"><span data-stu-id="5d34b-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
