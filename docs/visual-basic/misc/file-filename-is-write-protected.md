# <a name="file-filename-is-write-protected"></a>Soubor \<filename> je chráněný proti zápisu.
Soubor, do kterého se pokoušíte zapisovat, je chráněn proti zápisu, a proto jej nelze změnit.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte oprávnění k souboru pro čtení/zápis nebo zápis do jiného souboru.
