# <a name="index-indexnumber-for-dimension-dimensionnumber-is-out-of-range"></a>Index \<indexnumber> pro dimenzi \<dimensionnumber> je mimo rozsah.
Jeden z indexů překračuje definovaný rozsah.  
  
 **ID chyby:** BC30702  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte logiku procedury tak, aby hranice pole nebyly překročeny.  
  
- Předefinujte pole tak, aby umožňovalo širší rozsah hodnot.
