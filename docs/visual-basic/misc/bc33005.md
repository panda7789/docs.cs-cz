# <a name="end-operator-expected"></a>Očekával se příkaz end operator.
`End`Příkaz musí odpovídat bloku, který končí. `Operator`Definice procedury nebyla správně ukončena.  
  
 **ID chyby:** BC33005  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte `End Operator` příkaz na konec procedury.
