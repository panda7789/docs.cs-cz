# <a name="functionname-is-not-declared-visual-basic-error"></a>' \<functionname> ' není deklarováno (chyba Visual Basic)
není \<functionname> deklarováno. Funkce pro vstupně-výstupní operace se soubory jsou k dispozici v oboru názvů Microsoft. VisualBasic.  
  
 **ID chyby:** BC30815  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte vstupně-výstupní funkce souborů definované v `My.Computer.FileSystem` , `System.IO` obor názvů, Visual Basic běhové funkce nebo `Microsoft.VisualBasic.FileSystem` objekt.
