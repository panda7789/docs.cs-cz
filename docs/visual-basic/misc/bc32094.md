# <a name="comma-or--expected-type-argument-list"></a>Očekává se čárka nebo znak) (typ seznamu argumentů).
Seznam argumentů typu obsahuje neočekávaný element. Použijte čárky ( `,` ) k oddělení argumentů typu v takovém seznamu a použijte pravou závorku ( `)` ) pro ukončení seznamu.  
  
 **ID chyby:** BC32094  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte čárku a pokračujte v seznamu argumentů typu nebo přidejte uzavírací závorku pro ukončení seznamu.
