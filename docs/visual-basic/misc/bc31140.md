# <a name="custom-modifier-can-only-be-used-immediately-before-an-event-declaration"></a>Modifikátor Custom lze použít pouze bezprostředně před deklarací Event.
`Custom`Klíčové slovo musí předcházet pouze `Event` klíčové slovo ve vlastní deklaraci události.  
  
 **ID chyby:** BC31140  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte `Custom` klíčové slovo před `Event` klíčové slovo.  
  
     —nebo—  
  
- Odeberte `Custom` klíčové slovo.
