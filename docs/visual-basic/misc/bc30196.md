# <a name="comma-expected"></a>Očekávána čárka
V seznamu argumentů chybí čárka (,) mezi dvěma argumenty.  
  
 **ID chyby:** BC30196  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zajistěte, aby všechny argumenty byly oddělené čárkami.
