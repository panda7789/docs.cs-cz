# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>Deklarace AddHandler musí být ukončena párovým klíčovým operátorem end AddHandler.
`AddHandler`Deklarace musí končit `End AddHandler` příkazem.  
  
 **ID chyby:** BC31115  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda `AddHandler` deklarace končí `End AddHandler` příkazem.
