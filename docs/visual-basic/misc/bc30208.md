# <a name="compare-expected"></a>Očekávalo se klíčové slovo Compare.
`Option`Příkaz určuje `Text` nebo `Binary` , ale ne `Compare` .  
  
 **ID chyby:** BC30208  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Compare` klíčové slovo do `Option` příkazu, například `Option Compare Binary` .
