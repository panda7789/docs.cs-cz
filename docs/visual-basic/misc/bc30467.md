# <a name="name1-for-the-imports-name2-does-not-refer-to-a-namespace-class-structure-enum-or-module"></a>\<name1>pro importy \<name2> neodkazuje na obor názvů, třídu, strukturu, výčet ani modul.
Pokusili jste se použít `Imports` příkaz na něco, co není,,, `Namespace` `Class` `Structure` `Enum` nebo `Module` . `Imports`Příkaz naimportuje názvy oborů názvů z odkazovaných projektů a sestavení nebo importuje názvy oborů názvů definované v rámci stejného projektu jako modul, ve kterém se zobrazí příkaz.  
  
 **ID chyby:** BC30467  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte entitu, kterou se pokoušíte importovat, a ujistěte se, že je platná pro použití s `Imports` příkazem.
