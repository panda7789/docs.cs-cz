# <a name="field-fieldname-is-of-an-unsupported-type"></a>Pole \<fieldname> ' ' je nepodporovaného typu.
Pole je deklarováno jako typ, který nemůže být reprezentován aplikací Visual Studio.  
  
 **ID chyby:** BC30656  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Definujte pole jako podporovaný typ.  
  
2. Vytvořte nový typ, který popisuje data v poli.
