# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>Příkazy Error nejsou platné v příkazovém podokně.
`Error`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30724  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Error` příkazy z ladění kódu.
