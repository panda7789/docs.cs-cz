<span data-ttu-id="5569a-103">Procedura nebo vlastnost se pokusí implementovat proceduru nebo vlastnost definovanou v implementovaném rozhraní, ale kompilátor najde více než jednu verzi procedury rozhraní nebo vlastnosti se stejným názvem a signaturou.</span><span class="sxs-lookup"><span data-stu-id="5569a-103">A procedure or property attempts to implement a procedure or property defined in an implemented interface, but the compiler finds more than one version of the interface procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="5569a-104">K této chybě může dojít v situaci s vytvořenými obecnými typy, jak ilustrují následující deklarace.</span><span class="sxs-lookup"><span data-stu-id="5569a-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>  
  
```vb  
Public Interface baseInterface(Of t)  
    Sub doSomething(ByVal inputValue As String)  
    Sub doSomething(ByVal inputValue As t)  
End Class  
Public Class implementingClass  
    Implements baseInterface(Of String)  
    Sub doSomething(ByVal inputValue As String) _  
        Implements baseInterface(Of String).doSomething  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="5569a-105">Vzhledem `implementingClass` k tomu, že implementace implementují `baseInterface` `String` na svůj parametr typu `t` , obě verze `doSomething` v v systému `baseInterface` převezmou stejné signatury, jak je to možné `implementingClass` .</span><span class="sxs-lookup"><span data-stu-id="5569a-105">Because `implementingClass` implements `baseInterface` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseInterface` take on identical signatures as far as `implementingClass` is concerned.</span></span> <span data-ttu-id="5569a-106">V důsledku toho kompilátor nemůže určit, která verze se má implementovat.</span><span class="sxs-lookup"><span data-stu-id="5569a-106">As a result, the compiler cannot determine which version to implement.</span></span>
