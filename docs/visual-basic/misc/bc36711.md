# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>Argument, který odpovídá parametru ' \<parametername> ', je zúžen na ' \<typename> '.
Bylo provedeno volání přetížené metody, ale kompilátor nemůže najít metodu, která může být volána bez zúžení převodu. *Zužující převod* mění hodnotu na datový typ, který nemusí být schopný uchovat některé z možných hodnot.  
  
 **ID chyby:** BC36711
