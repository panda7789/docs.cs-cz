# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Parametr \<parametername> v metodě rozšíření \<methodname> definované v \<typename> už má shodný argument.
Argument je předán pomocí názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, zda je v některých názvech argumentů uveden chybný název.  
  
- Odebere duplicitní argument ze seznamu argumentů.
