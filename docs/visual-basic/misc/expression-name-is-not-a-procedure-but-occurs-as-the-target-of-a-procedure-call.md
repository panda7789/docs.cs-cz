# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>Výraz není \<name> procedura, ale vyskytuje se jako cíl volání procedury.
Provedli jste volání procedury do výrazu, který není procedurou.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že jste nezadali název procedury.  
  
2. Zadejte proceduru, která <`name`> volání.
