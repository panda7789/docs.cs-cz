# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>Nelze zadat výchozí hodnoty pro parametry, které nejsou deklarovány jako volitelné.
Pokud je procedura volána a v deklaraci nevezmou výchozí hodnoty, musí být zadány povinné argumenty.  
  
 **ID chyby:** BC32024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklarujte argument, který má být `Optional` nebo odeberte výchozí hodnotu.
