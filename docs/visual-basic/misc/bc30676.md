# <a name="eventname-is-not-an-event-of-containername"></a>není \<eventname> událost pro \<containername> .
Zadaná událost není pro objekt deklarována.  
  
 **ID chyby:** BC30676  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte pravopis názvu události.  
  
2. Ověřte, zda přistupujete ke správnému objektu. Použijte plně kvalifikované odkazy na objekt nebo použijte `Imports` příkaz k importu příslušného oboru názvů v případě potřeby.
