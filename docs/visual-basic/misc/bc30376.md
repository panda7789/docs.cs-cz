# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>Klíčové slovo New nelze použít pro třídu \<classname> , protože obsahuje člen MustOverride, který nebyl přepsán.
Pokusili jste se použít `New` třídu, která obsahuje `MustOverride` člena, který nebyl přepsán.  
  
 **ID chyby:** BC30376  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `New` příkaz.
