# <a name="statement-cannot-appear-within-a-method-body"></a>Příkaz nemůže být použit v těle metody.
Procedura obsahuje neplatný příkaz, jako je například jiná deklarace procedury. Deklarace procedury nelze vnořovat.  
  
 **ID chyby:** BC30289  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte deklaraci procedury nebo ji přesuňte mimo uzavírací proceduru.
