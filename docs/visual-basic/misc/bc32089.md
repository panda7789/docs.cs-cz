# <a name="parametername-is-already-declared-as-a-type-parameter-of-this-method"></a>' \<parametername> ' je již deklarován jako parametr typu této metody.
Obecný postup definuje normální parametr nebo místní proměnnou se stejným názvem jako parametr typu.  
  
 Každý parametr procedury, včetně každého parametru typu Obecné procedury, musí mít název odlišný od všech ostatních parametrů. Vzhledem k tomu, že parametry procedury jsou používány jako lokální proměnné, musí mít všechny místní proměnné deklarované v rámci procedury také název odlišný od všech parametrů a parametrů typu.  
  
 **ID chyby:** BC32089  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte název normálního parametru nebo místní proměnné.
