# <a name="one-or-more-folders-in-the-target-path-do-not-exist"></a>Jedna nebo více složek v cílové cestě neexistují.
Minimálně jeden adresář zadaný v cílové cestě neexistuje. Může to být výsledek nesprávně analyzované cesty, zejména pokud je potřeba dodat relativní cestu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, zda jste zadali správnou cílovou cestu.
