# <a name="file-format-not-valid"></a>Formát souboru není platný.
Buď jste se pokusili načíst soubor do součásti, ale formát dat v souboru byl nekompatibilní s komponentou nebo jste se pokusili uložit data součásti do souboru, ale formát dat byl nekompatibilní s formátem souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pro určení správného formátu dat souborů na disku si projděte dokumentaci k této součásti.  
  
2. Pokud chcete zjistit, jestli podporuje převod z jednoho formátu na jiný, Projděte si dokumentaci ke komponentě.
