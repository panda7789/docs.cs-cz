# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>Pro vlastnost \<propertyname> musí být použita deklarace Default, protože přepisuje výchozí vlastnost v základní třídě \<name> .
`Property`Příkaz v kódu Přepisuje `Default` vlastnost základní třídy, struktury nebo rozhraní, a proto musí být deklarována `Default` .  
  
 **ID chyby:** BC30360  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklarujte vlastnost `Default` .
