# <a name="the-file-filename-is-not-a-text-file"></a>Soubor ' \<filename> ' není textový soubor.
Zadaný soubor nelze zapsat do souboru, protože se jedná o binární soubor.  
  
 **ID chyby:** BC2015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že zadáváte správný název a cestu k souboru.
