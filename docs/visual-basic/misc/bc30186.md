# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>Příkazy using nejsou platné v příkazovém podokně.
`Using`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30186  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Using` příkaz z příkazového **podokna** .
