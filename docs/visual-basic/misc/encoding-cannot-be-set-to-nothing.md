# <a name="encoding-cannot-be-set-to-nothing"></a>Kódování nelze nastavit na hodnotu Nothing.
Pokus o čtení nebo zápis do souboru se nezdařil, protože parametr byl `encoding` nastaven na, `Nothing` ale vyžaduje platnou hodnotu.  
  
 <xref:System.Text.Encoding>slouží k určení, jaké kódování se má použít při zápisu do souboru. Výchozí kódování je UTF-8.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu `encoding` parametru.
