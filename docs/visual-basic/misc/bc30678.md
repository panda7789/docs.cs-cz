# <a name="end-statement-not-valid"></a>Příkaz end není platný.
`End`Klíčové slovo se používá nesprávně nebo je v aktuálním kontextu nesprávné.  
  
 **ID chyby:** BC30678  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. `End`Samotným způsobem, který způsobí, že dojde ke ztrátě rozsahu.  
  
2. Použijte `End` s jinými klíčovými slovy k ukončení bloku kódu, jako je například struktura nebo vlastnost.
