<span data-ttu-id="5cf5d-103">Byl zadán argument typu v volání metody rozšíření, která buď nemá žádné obecné parametry, nebo nemá žádné obecné parametry, pro které již není zadán typ.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="5cf5d-104">Například následující kód způsobuje tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="5cf5d-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
