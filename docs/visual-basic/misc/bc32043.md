# <a name="too-many-type-arguments-to-genericmethodname"></a>Příliš mnoho argumentů typu pro \<genericMethodName>
Byla vyvolána obecná metoda s více argumenty typu, než jsou parametry typu.  
  
 Při vyvolání obecné metody je nutné zadat jeden argument typu pro každý parametr typu definovaný touto metodou.  
  
 **ID chyby:** BC32043  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte argumenty typu ze seznamu argumentů typu tak, aby existovala jedna pro každý parametr typu na obecné metodě, kterou vyvoláte.
