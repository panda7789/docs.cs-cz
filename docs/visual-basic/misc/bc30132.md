# <a name="label-labelname-is-not-defined"></a>Návěstí není \<labelname> definované.
Na řádek nebo číslo řádku se odkazuje, ale není definovaný v rámci oboru odkazu. Popisek musí být v rámci postupu, který obsahuje odkaz.  
  
 **ID chyby:** BC30132  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Restrukturalizujte kód tak, aby byl popisek řádku a odkaz shodný s tímto postupem.
