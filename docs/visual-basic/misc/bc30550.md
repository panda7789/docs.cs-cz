# <a name="constructors-cannot-implement-interface-methods"></a>Konstruktory nemůžou implementovat metody rozhraní.
Došlo k pokusu o použití `New` klíčového slova k implementaci metody rozhraní, která není platná.  
  
 **ID chyby:** BC30550  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Implementujte rozhraní s `Implements` klíčovým slovem.
