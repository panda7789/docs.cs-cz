# <a name="anonymous-type-property-propertyname-cannot-be-used-in-the-definition-of-a-lambda-expression-within-the-same-initialization-list"></a><span data-ttu-id="549b4-102">Vlastnost anonymního typu \<propertyname> nejde použít v definici výrazu lambda v rámci stejného inicializačního seznamu.</span><span class="sxs-lookup"><span data-stu-id="549b4-102">Anonymous type property '\<propertyname>' cannot be used in the definition of a lambda expression within the same initialization list</span></span>

<span data-ttu-id="549b4-103">Vlastnosti definované v inicializačním seznamu anonymního typu nemůžou být součástí definice výrazu lambda ve stejném seznamu.</span><span class="sxs-lookup"><span data-stu-id="549b4-103">Properties defined in the initialization list of an anonymous type cannot be part of a lambda expression definition in the same list.</span></span> <span data-ttu-id="549b4-104">Například v následujícím kódu `Num` nelze vlastnost zahrnout do definice `LambdaFun` .</span><span class="sxs-lookup"><span data-stu-id="549b4-104">For example, in the following code, property `Num` cannot be included in the definition of `LambdaFun`.</span></span>

```vb
' Not valid.
'Dim anon = New With {.Num = 4, .LambdaFun = Function() .Num > 0}
```

<span data-ttu-id="549b4-105">**ID chyby:** BC36549</span><span class="sxs-lookup"><span data-stu-id="549b4-105">**Error ID:** BC36549</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="549b4-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="549b4-106">To correct this error</span></span>

1. <span data-ttu-id="549b4-107">Zvažte rozdělení anonymního typu na dvě části:</span><span class="sxs-lookup"><span data-stu-id="549b4-107">Consider splitting the anonymous type into two parts:</span></span>

    ```vb
    Dim anon1 = New With {.Num = 4}
    Dim anon2 = New With {.LambdaFun = Function() anon1.Num > 0}
    ' - or -
    Dim anon3 = New With {.lambdaFun = Function(n As Integer) n > 0}
    Console.WriteLine((anon2.LambdaFun)())
    Console.WriteLine(anon3.lambdaFun(anon1.Num))
    anon1.Num = -5
    Console.WriteLine((anon2.LambdaFun)())
    Console.WriteLine(anon3.lambdaFun(anon1.Num))
    ```

    <span data-ttu-id="549b4-108">Všimněte si, že pokud deklarujete `anon1.Num` jako `Key` vlastnost, její hodnotu nelze změnit.</span><span class="sxs-lookup"><span data-stu-id="549b4-108">Note that if you declare `anon1.Num` as a `Key` property, its value cannot be changed.</span></span>

2. <span data-ttu-id="549b4-109">Alternativou je použití regulárního příkazu funkce pro přístup k vlastnosti anonymního typu:</span><span class="sxs-lookup"><span data-stu-id="549b4-109">An alternative is to use a regular function statement to access the anonymous type property:</span></span>

    ```vb
    Function testNum(ByVal n As Integer) As Boolean
        Return n > 0
    End Function
    Console.WriteLine(testNum(anon1.Num))
    ```

3. <span data-ttu-id="549b4-110">Podobně lze použít funkci lambda, která je definována mimo anonymní typ:</span><span class="sxs-lookup"><span data-stu-id="549b4-110">Similarly, you can use a lambda function that is defined outside the anonymous type:</span></span>

    ```vb
    Dim lambdaFun1 = Function() anon1.Num > 0
    Dim lambdaFun2 = Function(n As Integer) n > 0
    ```
