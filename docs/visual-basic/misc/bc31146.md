# <a name="xml-name-expected"></a>Byl očekáván název XML.
Literál XML nebo vlastnost osy XML byly zadány bez názvu.  
  
 **ID chyby:** BC31146  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte název pro literál XML nebo vlastnost Axis XML.
