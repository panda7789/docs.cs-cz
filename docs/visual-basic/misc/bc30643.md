# <a name="property-propertyname-is-of-an-unsupported-type"></a>Vlastnost ' \<propertyname> ' je nepodporovaného typu.
Vlastnost v odkazovaném sestavení obsahuje typ, který nelze reprezentovat.  
  
 **ID chyby:** BC30643  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte typ, který lze použít společně s vaší aplikací.
