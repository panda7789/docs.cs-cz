# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>V deklaracích proměnných rozsahu nejde používat znaky typu.
Znak typu byl zahrnut na konci názvu deklarace proměnné rozsahu v dotazu LINQ.  
  
 **ID chyby:** BC36601  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte znak typu z deklarace proměnné rozsahu.
