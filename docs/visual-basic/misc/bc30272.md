# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>není \<argumentname> parametrem \<methoddeclaration> .
Volání procedury dodá další argument, který není definován procedurou.  
  
 **ID chyby:** BC30272  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebere nadbytečný argument ze seznamu argumentů.
