# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>Cílová verze prostředí .NET Compact Framework nepodporuje příkaz end.
Verze prostředí .NET Compact Framework, se kterou pracujete, nepodporuje `End` příkaz.  
  
 **ID chyby:** BC30769  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte příkaz.
