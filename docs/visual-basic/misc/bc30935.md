<span data-ttu-id="2833d-103">Procedura nebo vlastnost se pokusí přepsat zděděnou proceduru nebo vlastnost, ale kompilátor najde více než jednu verzi základní procedury nebo vlastnosti se stejným názvem a signaturou.</span><span class="sxs-lookup"><span data-stu-id="2833d-103">A procedure or property attempts to override an inherited procedure or property, but the compiler finds more than one version of the base procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="2833d-104">K této chybě může dojít v situaci s vytvořenými obecnými typy, jak ilustrují následující deklarace.</span><span class="sxs-lookup"><span data-stu-id="2833d-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
    Public Overridable Sub doSomething(ByVal inputValue As String)  
    End Sub  
    Public Overridable Sub doSomething(ByVal inputValue As t)  
    End Sub  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of String)  
    Overrides Sub doSomething(ByVal inputValue As String)  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="2833d-105">Vzhledem `derivedClass` k tomu, že dědí `baseClass` `String` `t` , že převezmou na svůj parametr typu, obě verze `doSomething` v v `baseClass` procházejí identickými podpisy, které jsou v souladu s nimi `derivedClass` .</span><span class="sxs-lookup"><span data-stu-id="2833d-105">Because `derivedClass` inherits `baseClass` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseClass` take on identical signatures as far as `derivedClass` is concerned.</span></span> <span data-ttu-id="2833d-106">V důsledku toho kompilátor nemůže určit, která verze se má přepsat.</span><span class="sxs-lookup"><span data-stu-id="2833d-106">As a result, the compiler cannot determine which version to override.</span></span>
