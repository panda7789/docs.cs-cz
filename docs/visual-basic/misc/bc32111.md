<span data-ttu-id="94d67-103">Obecný typ je deklarovaný s konfliktními omezeními kvůli kombinaci přímých a nepřímých omezení.</span><span class="sxs-lookup"><span data-stu-id="94d67-103">A generic type is declared with conflicting constraints due to a combination of direct and indirect constraints.</span></span>

 `Public Class testClass(Of t1 As {t2, Class}, t2 As Structure)`  
  
 <span data-ttu-id="94d67-105">Nepřímým omezením `Structure` a přímým omezením `Class` způsobuje konflikt pro parametr typu `t1` , protože `Structure` omezení vyžaduje, aby odpovídající argument typu hodnotový typ byl, ale `Class` vyžaduje, aby se jedná o typ odkazu.</span><span class="sxs-lookup"><span data-stu-id="94d67-105">The indirect constraint `Structure` and the direct constraint `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
