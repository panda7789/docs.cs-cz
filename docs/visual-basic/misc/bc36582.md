# <a name="too-many-arguments-to-extension-method-methodname-defined-in-typename"></a>Příliš mnoho argumentů metody rozšíření \<methodname> definované v ' \<typename> '
Počet argumentů dodaných v volání metody rozšíření překračuje počet argumentů definovaných pro metodu.  
  
 **ID chyby:** BC36582  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Prohlédněte si definici metody rozšíření a odeberte nadbytečné argumenty volání.
