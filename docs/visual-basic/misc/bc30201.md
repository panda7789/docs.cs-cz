# <a name="expression-expected"></a>Očekáván výraz
Operátor, jako je například `+` nebo `=` vyžaduje výraz pro svůj pravý operand, ale žádný rozpoznatelný výraz se nezobrazí.  
  
 **ID chyby:** BC30201  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platný výraz pro pravý operand operátoru.
