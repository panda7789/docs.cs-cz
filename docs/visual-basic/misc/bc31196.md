# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>V této relaci ladění nejsou k dispozici literály XML a vlastnosti OS XML, protože se v tomto sestavení nepoužívají.
V okně **Watch** nebo **Immediate** se v průběhu relace ladění odkazuje na vlastnost literálu XML nebo osy XML, ve které nejsou k dispozici XML ve Visual Basic funkce. Toto je případ pro sestavení, které nepoužívá XML v Visual Basic funkce nebo je sestavení pro vydání.  
  
 **ID chyby:** BC31196  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Spusťte novou relaci ladění pomocí ladicího sestavení sestavení.
