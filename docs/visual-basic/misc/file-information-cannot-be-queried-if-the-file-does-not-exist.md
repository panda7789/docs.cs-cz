# <a name="file-information-cannot-be-queried-if-the-file-does-not-exist"></a>Nelze se dotazovat na informace o souboru, pokud soubor neexistuje.
Byl proveden pokus o dotazování objektu na <xref:System.IO.FileInfo> informace o souboru, který neexistuje. Obvykle je to způsobeno omylem při zadávání názvu souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda jste zadali správný název souboru.
