# <a name="cannot-delete-a-registry-hive"></a>Podregistr registru nelze odstranit.
Byl proveden pokus o odstranění podregistru. Podregistry jsou klíče registru nejvyšší úrovně, například `HKEY_CURRENT_USER` a a `HKEY_LOCAL_MACHINE` nelze je odstranit.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte klíč registru, který se pokoušíte odstranit, a ujistěte se, že zadáváte správnou cestu.
