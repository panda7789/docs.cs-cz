# <a name="raiseevent-is-already-declared"></a>RaiseEvent už je deklarovaný.
`RaiseEvent`V deklaraci vlastní události se objevilo víc než jedna deklarace. `RaiseEvent`Deklarace deklaruje proceduru používanou k vyvolání události.  
  
 **ID chyby:** BC31129  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte redundantní `RaiseEvent` příkaz.
