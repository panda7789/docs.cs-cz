# <a name="parameter-specifier-is-duplicated"></a>Specifikátor parametru je duplicitní.
Specifikátor parametru je pro parametr použit více než jednou.  
  
 **ID chyby:** BC30785  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte duplicitní specifikátory.
