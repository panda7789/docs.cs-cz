# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>Atribut XML attribute1 se musí vyskytovat před atributem XML attribute2.
Atributy v literálu dokumentu XML jsou uvedeny v neplatném pořadí. Platná objednávka atributu pro literál dokumentu XML je založena na specifikaci XML 1,0. Například `encoding` atribut literálu dokumentu XML musí hned následovat po `version` atributu.  
  
 **ID chyby:** BC31157  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Aktualizujte pořadí atributů pro literál dokumentu XML pomocí pokynů uvedených ve specifikaci XML 1,0.
