# <a name="field-fieldname-of-type-typename-is-readonly"></a>Pole \<fieldname> typu \<typename> je jen pro čtení.
Pokusili jste se zapsat do pole, které je deklarováno `ReadOnly` .  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte pole tak, aby již nebylo deklarováno `ReadOnly` .  
  
- —nebo—  
  
- Nepokoušejte se zapisovat do tohoto pole.
