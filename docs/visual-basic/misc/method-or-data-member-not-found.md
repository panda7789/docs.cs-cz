# <a name="method-or-data-member-not-found"></a>Metoda nebo datový člen nebyly nalezeny.
Metoda nebo datový člen, ke kterému jste se pokusili získat přístup, nebyl nalezen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že jste nechybně naplnili název metody nebo datového členu.
