# <a name="typename-is-an-enum-type-and-cannot-be-used-as-an-expression"></a>\<typename>je typ výčtu a nedá se použít jako výraz.
K názvu typu dojde v případě, že je vyžadován výraz. Výraz se musí skládat z kombinace proměnných, konstant, literálů, vlastností a `Function` volání procedur.  
  
 **ID chyby:** BC30107  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte název typu a vytvořte výraz pomocí platných prvků.
