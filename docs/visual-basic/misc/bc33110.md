# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="b41d6-102">Nejde odvodit společný typ pro první a druhý operand v binárním operátoru if.</span><span class="sxs-lookup"><span data-stu-id="b41d6-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="b41d6-103">Nejde odvodit společný typ pro první a druhý operand binárního operátoru if.</span><span class="sxs-lookup"><span data-stu-id="b41d6-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="b41d6-104">Jeden musí mít rozšiřující převod na druhý.</span><span class="sxs-lookup"><span data-stu-id="b41d6-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="b41d6-105">Binární `If` operátor vyžaduje, aby existoval rozšiřující převod mezi jedním z argumentů a druhým argumentem.</span><span class="sxs-lookup"><span data-stu-id="b41d6-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="b41d6-106">Například vzhledem k tomu, že není rozšiřující převod v obou směrech mezi `Integer` a `String` , následující kód způsobuje tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="b41d6-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="b41d6-107">**ID chyby:** BC33110</span><span class="sxs-lookup"><span data-stu-id="b41d6-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="b41d6-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="b41d6-108">To correct this error</span></span>

- <span data-ttu-id="b41d6-109">Poskytněte explicitní převod pro jeden z operandů, pokud je to možné v kódu:</span><span class="sxs-lookup"><span data-stu-id="b41d6-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="b41d6-110">Přepište kód pomocí jiné podmíněné konstrukce.</span><span class="sxs-lookup"><span data-stu-id="b41d6-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
