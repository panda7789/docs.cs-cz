# <a name="attribute-conditional-is-only-valid-on-sub-declarations"></a>Atribut Conditional je platný jenom v deklaracích sub.
<xref:System.Diagnostics.ConditionalAttribute>Atribut byl použit pro deklaraci metody jinou než `Sub` procedura.  
  
 **ID chyby:** BC41007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte <xref:System.Diagnostics.ConditionalAttribute> z neplatné deklarace.
