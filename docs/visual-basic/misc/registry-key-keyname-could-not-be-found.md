# <a name="registry-key-keyname-could-not-be-found"></a>Klíč registru ' \<keyname> ' se nepodařilo najít.
Klíč registru nebyl nalezen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte pravopis názvu klíče.  
  
2. Ujistěte se, že máte dostatečná oprávnění zabezpečení pro přístup ke klíči.
