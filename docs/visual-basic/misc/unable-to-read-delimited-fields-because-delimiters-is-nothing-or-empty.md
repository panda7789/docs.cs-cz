# <a name="unable-to-read-delimited-fields-because-delimiters-is-nothing-or-empty"></a>Pole s oddělovači nelze číst, protože oddělovače nejsou žádné nebo jsou prázdné.
`TextFieldParser`Nelze číst ze souboru, protože `Delimiters` vlastnost je nastavena na `Nothing` nebo je prázdná.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu pro `Delimiters` .
