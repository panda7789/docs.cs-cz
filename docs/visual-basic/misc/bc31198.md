<span data-ttu-id="ac160-103">Deklarace literálů XML je použít ve výrazu do umístění, které je nejednoznačné pro kompilátor jazyka Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="ac160-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="ac160-104">To znamená, kompilátor jazyka Visual Basic nelze určit, zda je menší – než znak (<) má sloužit jako srovnávací operátor nebo spuštění literál XML.</span><span class="sxs-lookup"><span data-stu-id="ac160-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="ac160-105">Následující kód představuje příklad:</span><span class="sxs-lookup"><span data-stu-id="ac160-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="ac160-107">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="ac160-107">To correct this error</span></span>
