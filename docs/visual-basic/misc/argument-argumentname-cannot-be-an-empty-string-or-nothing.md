# <a name="argument-argumentname-cannot-be-an-empty-string-or-nothing"></a>Argument \<argumentname> nemůže být prázdný řetězec nebo Nothing.
Argument vyžaduje jinou hodnotu než prázdný řetězec ("") nebo `Nothing` . Může to být výsledek nesprávně počítané hodnoty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu argumentu.
