# <a name="you-must-specify-a-path-that-is-under-the-current-folder-or-one-of-its-sub-folders"></a>Je nutné zadat cestu, která se nachází v rámci aktuální složky nebo jedné z jejích podsložek.
Zadaná cesta není v aktuálním adresáři. Může to být způsobeno chybnou analýzou cest k souborům při zadávání relativní cesty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že zadáváte správnou cestu.
