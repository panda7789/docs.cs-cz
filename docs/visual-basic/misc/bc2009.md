# <a name="option-optionname-can-be-followed-only-by--or--"></a>\<optionname>za možností může následovat jenom znak + nebo-.
Text, který následuje za možností, nepředstavuje znaménko plus (+) nebo znaménko mínus (-).  
  
 **ID chyby:** BC2009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte text, který následuje za možností znaménkem plus (+), aby se tato možnost zapnula.  
  
     ani  
  
- Nahraďte text, který následuje za možností znaménkem mínus (-), aby se možnost vypnula.
