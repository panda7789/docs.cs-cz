# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>Klíčovému příkazu End Interface musí předcházet párové klíčové slovo Interface.
`End Interface`K příkazu dojde bez odpovídajícího `Interface` příkazu. `End Interface`musí předcházet odpovídající `Interface` příkaz.  
  
 **ID chyby:** BC30252  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, zda `Interface` je předchozí příkaz špatně napsaný nebo jinak neplatný.  
  
2. Ujistěte se, že `Interface` Členové jsou správně naformátované.
