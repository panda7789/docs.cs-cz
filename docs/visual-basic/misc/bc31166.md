# <a name="expected-closing--for-xml-tag"></a>Očekávalo se ukončení značky XML u >.
Ve značce XML pro literál XML chybí požadovaný uzavírací znak >.  
  
 **ID chyby:** BC31166  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte uzavírací znak ">" na konec značky XML pro literál XML.
