# <a name="default-cannot-be-combined-with-specifier"></a>Klíčové slovo Default nelze kombinovat s \<specifier> hodnotou.
Došlo k pokusu o kombinování `Default` klíčového slova se specifikátorem, `Shared` jako `Private` je například nebo, který není v tomto kontextu platný.  
  
 **ID chyby:** BC30490  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebrat `Default` .  
  
     -nebo-  
  
- Odeberte konfliktní specifikátor.
