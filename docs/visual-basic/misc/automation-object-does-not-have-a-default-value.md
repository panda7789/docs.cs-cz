# <a name="automation-object-does-not-have-a-default-value"></a>Automatizační objekt nemá výchozí hodnotu.
Visual Basic nemůže určit výchozího člena pro zadaný objekt.  
  
### <a name="to-correct-the-error"></a>Oprava chyby  
  
- Podívejte se do dokumentace k objektu a poskytněte explicitní specifikaci vlastnosti nebo metody.
