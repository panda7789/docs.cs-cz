# <a name="could-not-complete-operation-since-target-directory-is-under-source-directory"></a>Operaci se nepovedlo dokončit, protože cílový adresář se nachází ve zdrojovém adresáři.
Cyklická operace se nezdařila. Cyklická cyklus operací, a proto nemůže být dokončena. Například objekt A se může pokusit o dědění z objektu B, který zase dědí z Object A.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Při dědění se ujistěte, že neexistují žádné cyklické odkazy.
