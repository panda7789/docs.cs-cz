# <a name="only-one-of-public-private-protected-friend-or-protected-friend-can-be-specified"></a>Je možné zadat jenom jedno z veřejných, Private, Protected, Friend nebo Protected Friend.
V deklaraci elementu došlo ke konfliktu přístupnosti. Je možné zadat pouze jednu přístupnost.  
  
 **ID chyby:** BC30176  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vyberte jednu přístupnost a odeberte další klíčová slova z deklarace.
