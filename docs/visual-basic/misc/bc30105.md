# <a name="number-of-indices-is-less-than-the-number-of-dimensions-of-the-indexed-array"></a>Počet indexů je menší než počet rozměrů indexovaného pole.
Počet indexů použitých pro přístup k prvku pole musí být přesně stejný jako rozměr pole, tj. počet dimenzí, které jsou pro něj deklarovány.  
  
 **ID chyby:** BC30105  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte do odkazu na pole dolní indexy, dokud celkový počet dolních indexů nerovná rozměru pole.
