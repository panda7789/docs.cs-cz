# <a name="addhandler-is-already-declared"></a>Metoda AddHandler již byla deklarována.
`AddHandler`V deklaraci vlastní události se zobrazí více než jedna deklarace. `AddHandler`Deklarace deklaruje proceduru použitou k přidání obslužné rutiny události.  
  
 **ID chyby:** BC31127  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte redundantní `AddHandler` příkaz.
