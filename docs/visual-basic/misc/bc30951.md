# <a name="wrong-number-of-type-arguments-passed-to-method-genericprocedurename"></a>Metodě se předal nesprávný počet argumentů typu \<genericprocedurename> .
Obecný postup se volá s počtem argumentů typu, které se neshodují s počtem parametrů typu, se kterými je definovaný.  
  
 **ID chyby:** BC30951  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument typu pro každý parametr typu definovaný pro obecný postup.  
  
     -nebo-  
  
- Zavolejte obecný postup bez argumentů typu vůbec a nechejte kompilátor pokoušet odvodit argumenty typu.
