# <a name="static-local-variable-variablename-is-already-declared"></a>Statická lokální proměnná \<variablename> už je deklarovaná.
Již byla deklarována jiná statická místní proměnná se stejným názvem.  
  
 **ID chyby:** BC31401  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte redundantní statické místní deklarace.  
  
2. Každé statické lokální proměnné zadejte jedinečný název.
