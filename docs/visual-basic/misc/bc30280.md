# <a name="enum-enumname-must-contain-at-least-one-member"></a>Výčtový typ \<enumname> musí obsahovat alespoň jeden člen.
Dojde k prázdnému výčtu. V rámci příkazu a příkazu musí být deklarovaný aspoň jeden člen `Enum` `End Enum` .  
  
 **ID chyby:** BC30280  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte do výčtu deklaraci člena.
