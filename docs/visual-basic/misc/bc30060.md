# <a name="conversion-from-type1-to-type2-cannot-occur-in-a-constant-expression"></a>\<type1> \<type2> V konstantním výrazu nemůže být převod z na.
Inicializační výraz v `Const` příkazu je vyhodnocen jako datový typ, který nelze převést na deklarovaný typ konstanty.  
  
 **ID chyby:** BC30060  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Inicializujte konstantu pomocí výrazu datového typu, který lze převést na typ deklarovaný pro konstantu.
