# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Hodnotu typu \<type1> nejde převést na, \<type2> protože typy polí mají různý počet rozměrů.
Pokusili jste se převést jedno pole na jiný s jiným počtem rozměrů.  
  
 **ID chyby:** BC30414  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte jedno nebo obě pole tak, aby se počet odpovídajících dimenzí shodoval.
