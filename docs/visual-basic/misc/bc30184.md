# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>Příkazu End enum musí předcházet párové klíčové slovo enum.
`End Enum`K příkazu dojde bez odpovídajícího `Enum` příkazu. `End Enum`musí předcházet odpovídající `Enum` příkaz.  
  
 **ID chyby:** BC30184  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zajistěte, aby předchozí `Enum` příkaz nebyl špatně napsaný nebo jinak neplatný.  
  
2. Ujistěte se, že `Enum` Členové jsou správně naformátované.
