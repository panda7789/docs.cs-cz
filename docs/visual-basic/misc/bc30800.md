# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Argumenty metody musí být uzavřeny v závorkách.

Pravidla řídící volání procedur jsou v novějších verzích Visual Basic jednodušší. Všechny argumenty musí být uzavřeny v závorkách.

**ID chyby:** BC30800

## <a name="to-correct-this-error"></a>Oprava této chyby

- Seznam argumentů uveďte v závorkách. například:

  ```vb
  MySub(ArrayIndex, NewValue)
  ```
