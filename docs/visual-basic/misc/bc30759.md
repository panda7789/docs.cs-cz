# <a name="referenced-membername-has-a-value-of-nothing"></a>Odkazovaný objekt \<membername> má hodnotu Nothing.
K položce, na kterou se odkazuje, se buď nepřiřadila hodnota, nebo jí byla přiřazena hodnota `Nothing` .  
  
 **ID chyby:** BC30759  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, že je název položky napsaný a správně nastavený.
