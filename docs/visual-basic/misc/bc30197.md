# <a name="as-expected"></a>Bylo očekáváno klíčové slovo As.
Dojde k deklaraci, která neposkytuje `As` klauzuli pro jeden nebo více deklarovaných prvků.  
  
 **ID chyby:** BC30197  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `As`Podle potřeby klauzuli Doveďte nebo zapněte `Option Strict Off` .
