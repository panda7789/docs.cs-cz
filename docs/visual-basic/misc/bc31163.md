# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Očekávalo se, že pro hodnotu atributu XML odpovídá uzavírací jednoduchá uvozovka.
Hodnota atributu, která je oddělená jednoduchými uvozovkami, postrádá znak pravé uvozovky.  
  
 **ID chyby:** BC31163  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte uzavírací znak jednoduché uvozovky (') pro vymezení hodnoty atributu.
