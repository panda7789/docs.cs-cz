# <a name="the-file-is-currently-closed"></a>Soubor je aktuálně uzavřený.
Došlo k pokusu o provedení operace, která může být provedena pouze pro otevřený soubor, ale soubor je aktuálně uzavřen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Otevřete soubor a zkuste operaci zopakovat.
