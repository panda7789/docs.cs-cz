# <a name="modules-cannot-be-declared-specifier"></a>Pro moduly nelze provést deklaraci \<specifier> .
Modul byl deklarován s neplatným modifikátorem přístupu.  
  
 **ID chyby:** BC31052  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte modifikátor přístupu platným modifikátorem přístupu, například `Public` nebo `Friend` .
