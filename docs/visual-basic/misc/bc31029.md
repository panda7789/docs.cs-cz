# <a name="method-methodname-cannot-handle-event-eventname-because-they-do-not-have-the-same-signature"></a>Metoda \<methodname> nemůže zpracovat událost \<eventname> , protože nemá stejnou signaturu.
Parametry nebo typy parametrů pro určenou obslužnou rutinu události se neshodují s hodnotami definovanými pro zadanou událost.  
  
 **ID chyby:** BC31029  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Předefinování události nebo obslužné rutiny události tak, aby seznam parametrů odpovídal.
