# <a name="too-few-type-arguments-to-extension-method-methodname-defined-in-typename"></a>Nedostatečný počet argumentů metody rozšíření \<methodname> definované v ' \<typename> '
Obecnou metodu rozšíření vyvolala méně argumentů typu, než je v definici metody typ parametru.  
  
 Při vyvolání obecné metody je nutné zadat jeden argument typu pro každý parametr typu definovaný touto metodou.  
  
 **ID chyby:** BC36590  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte argumenty typu do seznamu argumentů typu tak, aby existoval jeden pro každý parametr typu metody rozšíření, kterou vyvoláte.
