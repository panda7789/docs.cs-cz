# <a name="alias-namespacename-is-already-declared"></a>Alias \<namespacename> už je deklarovaný.
Pokusili jste se použít `Imports` pro import oboru názvů, který je již deklarován.  
  
 **ID chyby:** BC30572  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte obor názvů, který se pokoušíte importovat.
