# <a name="statementname-statement-requires-an-array"></a>\<statementname>příkaz vyžaduje pole.
`Erase`Příkazy a `ReDim` vyžadují datové typy pole. Zadaný typ není pole.  
  
 **ID chyby:** BC30049  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte datový typ na typ pole.
