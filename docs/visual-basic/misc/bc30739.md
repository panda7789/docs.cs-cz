# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Příkazy RaiseEvent nejsou platné v příkazovém podokně.
`RaiseEvent`příkazy nelze použít v **příkazovém** okně.  
  
 **ID chyby:** BC30739  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `RaiseEvent`Příkazy testu z kódu v **editoru kódu**.
