# <a name="name-is-ambiguous-in-the-application-objects-list"></a>' \<name> ' je v aplikačních objektech dvojznačný. \<list> ' '
Zadali jste název, který je nejednoznačný, a proto je v konfliktu s jiným názvem. Kompilátor Visual Basic nemá žádná pravidla řešení konfliktů; názvy je nutné jednoznačně odlišit. Seznam uvedený v chybové zprávě podrobně popisuje objekty aplikace, kde je název dvojznačný.  
  
 **ID chyby:** BC30563  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Plně kvalifikované jméno.
