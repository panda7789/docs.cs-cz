# <a name="properties-in-a-module-cannot-be-declared-specifier"></a>Pro vlastnosti v modulu nejde provést \<specifier> deklarace.
Pokusili jste se použít specifikátor, jako `MustOverride` je, `NotOverridable` nebo `Overridable` , na vlastnost v rámci modulu.  
  
 **ID chyby:** BC30503  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
