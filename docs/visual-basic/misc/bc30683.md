# <a name="inherits-statement-must-precede-all-declarations-in-a-class"></a>Příkaz Inherits musí předcházet všem deklaracím ve třídě.
`Inherits`Příkazy musí být na začátku definice třídy.  
  
 **ID chyby:** BC30683  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte `Inherits` příkaz na začátek vaší třídy.
