# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>' \<specifier> ' není platný v deklaraci struktury.
Použili jste specifikátor, jako `Dim` je například nebo `Override` , který není platný v `Structure` deklaraci.  
  
 **ID chyby:** BC30395  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
