# <a name="end-set-must-be-the-first-statement-on-a-line"></a>Příkaz end set musí být prvním příkazem na řádku.
`End Set`Příkaz sleduje dvojtečku (:) oddělovač příkazů `End Set`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC32034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
