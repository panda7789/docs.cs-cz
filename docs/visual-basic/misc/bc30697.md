# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>' \<membername1> ' nemůže přepsat ' \<membername2> ', protože se liší typy nepovinných parametrů.
Přetížené členy musí mít jiné datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30697  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Definujte aspoň jeden povinný parametr jako jiný datový typ.
