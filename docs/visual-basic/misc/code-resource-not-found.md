# <a name="code-resource-not-found"></a>Prostředek kódu nebyl nalezen.
Bylo provedeno volání procedury v prostředku kódu, ale nepovedlo se najít prostředek kódu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda je prostředek k dispozici a zda je na něj správně odkazováno.
