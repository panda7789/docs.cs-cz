# <a name="unable-to-open-resource-file-filename-error"></a>Nepovedlo se otevřít soubor prostředků \<filename> :\<error>
Při otevírání zadaného souboru prostředků došlo k chybě.  
  
 **ID chyby:** BC31509  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, zda je zadaný název a umístění správné.  
  
2. Ověřte, že soubor není poškozený nebo poškozený.
