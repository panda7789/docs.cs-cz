# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>Příkaz end operator musí být prvním příkazem na řádku.
`End Operator`Příkaz následuje `:` oddělovačem příkazů dvojtečka (). `End Operator`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC33006  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
