# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>Pro metody v modulu nelze provést deklaraci \<specifier> .
Použili jste specifikátor, který není platný pro metodu v rámci `Module` příkazu. Pro moduly nelze nikdy vytvořit instanci, nepodporují dědičnost a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30433  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
