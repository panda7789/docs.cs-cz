# <a name="type-characters-are-not-allowed-on-imports-aliases"></a>Pro aliasy import nejsou povoleny znaky typu.
Alias importu v `Imports` příkazu obsahuje alespoň jeden znak typu identifikátoru.  
  
 Alias importu musí být platný název Visual Basic. Povolené znaky neobsahují žádné znaky typu identifikátoru ( `%` , `&` , `@` ,, a `!` `#` `$` ). Viz [deklarované názvy elementů](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC31398  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte znak nebo znaky identifikátoru typu z aliasu importu.
