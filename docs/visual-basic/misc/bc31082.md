# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>není \<variablename> lokální proměnná ani parametr, a proto se nedá použít jako proměnná catch.
Proměnné v `Try...Catch...Finally` příkazech musí být deklarovány místně, nebo musí být parametry pro aktuální proceduru.  
  
 **ID chyby:** BC31082  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarujte místní proměnné nebo parametry pro `Try...Catch...Finally` příkazy.
