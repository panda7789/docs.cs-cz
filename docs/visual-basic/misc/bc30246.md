# <a name="modifier-is-not-valid-on-a-local-constant-declaration"></a>' \<modifier> ' není platný v deklaraci lokální konstanty.
`Const`Příkaz v proceduře obsahuje neplatné klíčové slovo, například `Overloads` .  
  
 **ID chyby:** BC30246  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte z příkazu neplatné klíčové slovo `Const` .
