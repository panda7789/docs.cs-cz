# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>Vlastnost \<propertyname> nemůže být nastavena na prázdný řetězec nebo Nothing.
Vlastnost vyžaduje jinou hodnotu než prázdný řetězec ("") nebo `Nothing` . Může to být výsledek nesprávně počítané hodnoty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu.
