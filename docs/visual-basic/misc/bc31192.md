# <a name="error-embedding-win32-manifest-manifest-file-file-cannot-be-found"></a>Chyba při vkládání manifestu Win32: soubor manifestu \<file> nejde najít.
Soubor zadaný pro `/win32manifest` možnost kompilátoru nejde najít.  
  
 **ID chyby:** BC31192  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zadejte platný název souboru a cestu pro `/win32manifest` možnost kompilátoru.
