# <a name="named-argument-argumentname-specified-multiple-times"></a>Pojmenovaný argument se \<argumentname> zadal víckrát.
Pojmenovaný argument jste zadali více než jednou.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte všechny kromě jedné ze specifikací.
