# <a name="disk-not-ready"></a>Disk není připravený
Pro zadanou jednotku buď není k dispozici žádný disk, nebo dvířka jednotky nejsou otevřená.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte zadanou jednotku a vložte disk, pokud je to potřeba.  
  
2. Zavřete dvířka jednotky.
