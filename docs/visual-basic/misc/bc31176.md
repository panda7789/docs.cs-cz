# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>Znak ' znak ' (0x \<hexadecimal> ) není v literálu XML povolen.
V literálu XML byl použit nepodporovaný znak.  
  
 **ID chyby:** BC31176  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nepodporovaný znak.
