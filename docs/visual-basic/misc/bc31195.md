# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>Operátor ' operator ' není definován pro typy ' typ1 ' a ' typ2 '
Operátor ' operator ' není definován pro typy ' typ1 ' a ' typ2 '. Vlastnost Value lze použít k získání řetězcové hodnoty prvního prvku \<parentElement> .  
  
 V binárním souboru nebo porovnání byl použit literál XML `Boolean` .  
  
 **ID chyby:** BC31195  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `Value`Pro binární nebo porovnání použijte vlastnost literálu XML `Boolean` .
