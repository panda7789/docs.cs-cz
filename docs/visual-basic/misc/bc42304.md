# <a name="xml-documentation-parse-error-error"></a>Chyba analýzy dokumentace XML:\<error>
S touto chybou se mohou zobrazit následující zprávy. Další informace najdete v jednotlivých tématech:  
  
- [Značka komentáře XML \<tag> s identickými atributy se vyskytuje ve stejném bloku komentáře XML víc než jednou.](bc42305.md)  
  
- [Značka komentáře XML není \<tag> povolená u \<element> elementu jazyka.](bc42306.md)  
  
- [Parametr komentáře XML \<parameter> neodpovídá parametru u odpovídajícího \<keyword> příkazu.](bc42307.md)  
  
- [Parametr komentáře XML musí mít atribut Name.](bc42308.md)  
  
- [Komentář XML má značku s atributem cref \<attribute> , který nebylo možné přeložit.](bc42309.md)  
  
- [Značka include komentáře XML musí mít \<attribute> atribut.](bc42310.md)  
  
 **ID chyby:** BC42304
