# <a name="on-goto-and-on-gosub-statements-are-no-longer-supported"></a>Příkazy On GoTo a On GoSub už nejsou podporované.
Již není platný pro použití hodnoty proměnné nebo výrazu s `GoTo` `GoSub` příkazy a pro řízení toku programu.  
  
 **ID chyby:** BC30817  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Restrukturování aplikace pro použití `If...Then...Else` `Case` příkazů nebo.
