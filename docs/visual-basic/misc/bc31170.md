# <a name="character-character-0xhexadecimal-is-not-allowed-in-an-xml-name"></a>Znak "znak" (0x \<hexadecimal> ) není v názvu XML povolen
Název literálu XML nebo vlastnost osy XML obsahuje nepodporovaný znak.  
  
 **ID chyby:** BC31170  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nepodporovaný znak z názvu literálu XML nebo vlastnosti osy XML.
