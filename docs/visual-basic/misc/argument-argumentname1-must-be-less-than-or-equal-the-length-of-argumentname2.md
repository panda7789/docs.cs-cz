# <a name="argument-argumentname1-must-be-less-than-or-equal-the-length-of-argument-argumentname2"></a>Argument \<argumentname1> musí být menší nebo roven délce argumentu \<argumentname2> .
Délka zadaného argumentu překračuje délku argumentu, který nahrazuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nastavte délku prvního argumentu, který je menší nebo roven délce druhého argumentu.
