# <a name="statement-cannot-appear-within-an-interface-body-visual-basic-error"></a>Příkaz nemůže být použit v těle rozhraní (chyba Visual Basic).
Byla zjištěna neočekávaná konstrukce jazyka. Předpokládá se, že `End Interface` konstrukce chybí a že jakýkoliv zdrojový kód po tomto bodu je mimo rozhraní.  
  
 **ID chyby:** BC30604  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte syntaxi kódu používaného uvnitř definice rozhraní.  
  
2. Ujistěte se, že definice rozhraní končí `End Interface` konstrukcí.
