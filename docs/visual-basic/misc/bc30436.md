# <a name="members-in-a-module-cannot-be-declared-specifier"></a>Pro členy v modulu nejde \<specifier> deklarovat.
Použili jste specifikátor, který není platný pro člena v rámci `Module` příkazu. Pro moduly nelze nikdy vytvořit instanci, nepodporují dědičnost a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30436  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
