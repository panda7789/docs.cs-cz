# <a name="registry-key-keyname-could-not-be-created"></a>Klíč registru ' \<keyname> ' se nepodařilo vytvořit.
Zadaný klíč registru nelze vytvořit, pravděpodobně z důvodu nedostatečných oprávnění zabezpečení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že máte dostatečná oprávnění zabezpečení pro přístup ke klíči.
