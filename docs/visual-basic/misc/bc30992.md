## <a name="example"></a><span data-ttu-id="3f40a-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="3f40a-109">Example</span></span>  
 <span data-ttu-id="3f40a-110">Následující třída obsahuje výchozí vlastnost `defaultProp`, která vyžaduje celočíselný argument.</span><span class="sxs-lookup"><span data-stu-id="3f40a-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="3f40a-111">Vzhledem k tomu, že výchozí vlastnost vyžaduje argument, vyvolá následující deklarace chybu.</span><span class="sxs-lookup"><span data-stu-id="3f40a-111">Because the default property requires an argument, the following declaration causes an error.</span></span>  
  
```vb

```
