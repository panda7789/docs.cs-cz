# <a name="address-is-not-a-valid-remote-file-address"></a>\<address>není platná adresa vzdáleného souboru.
\<address>není platná adresa vzdáleného souboru. Platná adresa by měla obsahovat protokol, cestu a název souboru.  
  
 Pro operaci odeslání nebo stažení byla zadána neplatná adresa. Adresy musí zahrnovat protokol, jako je například `ftp` nebo `http` , platnou cestu a název souboru. Tato chyba může být způsobena nesprávnou analýzou cest souborů.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte zadanou adresu a určete, kde se tato adresa nezdařila, a zadejte potřebné informace.
