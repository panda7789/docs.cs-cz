# <a name="files-open-mode-wasnt-set-to-a-valid-value"></a>Režim otevření souboru se nestavil na platnou hodnotu.
Hodnota zadaná pro režim otevření souboru není platná. V následující tabulce jsou uvedeny platné hodnoty pro <xref:Microsoft.VisualBasic.OpenMode> výčet.  
  
|Hodnota|Mode|  
|-----------|----------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda je hodnota zadaná pro režim otevření souboru.
