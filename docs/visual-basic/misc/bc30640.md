# <a name="option-explicit-can-be-followed-only-by-on-or-off"></a>Za operátorem Option Explicit může následovat jenom klíčové slovo on nebo off.
`Option Explicit`příkazy musí určovat buď `On` nebo `Off` . Všechny ostatní hodnoty jsou neplatné.  
  
 **ID chyby:** BC30640  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `Option Explicit On` nebo `Option Explicit Off` .
