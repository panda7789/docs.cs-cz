# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>Literály XML a vlastnosti OS XML nejsou k dispozici.
Literály XML a vlastnosti OS XML nejsou k dispozici. Přidejte odkazy na System. XML, System. XML. Linq a System. Core.  
  
 Kompilovaný kód zahrnuje literály XML nebo vlastnosti osy XML, ale neodkazuje na sestavení, která jsou potřebná pro kompilování literálů XML nebo vlastností osy XML.  
  
 **ID chyby:** BC31190  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte odkazy na sestavení System. XML. dll, System. XML. Linq. dll a System. Core. dll.
