# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>Literály XML a vlastnosti OS XML nejsou k dispozici
Literály XML a vlastnosti OS XML nejsou k dispozici. Přidejte odkazy do System.Xml, System.Xml.Linq a System.Core.  
  
 Kompilovaný kód zahrnuje literály XML a vlastnosti OS XML, ale neobsahuje odkaz na sestavení, které jsou potřebné ke kompilaci literály XML a vlastnosti OS XML.  
  
 **ID chyby:** BC31190  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte odkazy na sestavení System.Xml.dll, knihovně System.Xml.Linq.dll a System.Core.dll.
