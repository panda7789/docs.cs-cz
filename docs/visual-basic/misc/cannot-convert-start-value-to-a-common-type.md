# <a name="cannot-convert-start-value-of-type-type1-limit-value-of-type2-and-step-value-of-type3-to-a-common-type"></a>Počáteční hodnotu typu \<type1> , hodnotu omezení \<type2> a hodnotu kroku nejde převést \<type3> na společný typ.
Zadané `For...Next` hodnoty ovládacího prvku mají jiné typy, a proto nejsou platné, protože nemohou být převedeny na společný typ.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte typ jedné nebo více hodnot tak, aby všechny mohly být převedeny na společný typ.
