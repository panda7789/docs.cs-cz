# <a name="imports-alias-name-conflicts-with-name2-declared-in-the-root-namespace"></a>Alias importu je \<name> v konfliktu s \<name2> deklarací deklarovaným v kořenovém oboru názvů.
Název aliasu použitý v `Imports` příkazu koliduje s jiným názvem deklarovaným v kořenovém oboru názvů.  
  
 **ID chyby:** BC31403  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vyberte jedinečný název aliasu používaného `Imports` příkazem.
