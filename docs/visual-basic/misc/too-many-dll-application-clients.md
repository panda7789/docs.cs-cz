# <a name="too-many-dll-application-clients"></a>Příliš mnoho aplikačních klientů knihovny DLL
Dynamická knihovna (DLL) pro Visual Basic může využívat pouze omezený počet hostitelských aplikací. Vaše aplikace a další aplikace, které jsou Visual Basic hostitele (některé z nich mohou být přístupné vaší aplikací), se snaží získat přístup k Visual Basic knihovně DLL ve stejnou dobu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Snižte počet otevřených aplikací, které přistupují k Visual Basic.
