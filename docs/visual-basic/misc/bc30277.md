# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>Znak typu neodpovídá \<charactername> deklarovanému datovému typu \<type> .
Proměnná je deklarována s jedním datovým typem, ale odkazuje se znakem typu, který představuje nekompatibilní datový typ, například `K$ = 10` při `K` deklaraci `Integer` .  
  
 **ID chyby:** BC30277  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte deklarovaný datový typ proměnné nebo změňte nebo odeberte znak typu v odkazu.
