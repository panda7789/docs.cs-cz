# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>' \<qualifiedmethod> ' není v tomto kontextu k dispozici, protože je ' \<modifier> '
Došlo k pokusu o přístup k metodě, která není v tomto kontextu k dispozici, protože byl použit modifikátor, který se na něj použil.  
  
 **ID chyby:** BC30390  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte modifikátor.
