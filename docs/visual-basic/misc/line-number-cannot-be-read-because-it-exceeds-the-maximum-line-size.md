# <a name="line-number-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>Řádek \<number> nelze číst, protože přesahuje maximální velikost řádku.
Řádek nelze číst, protože přesahuje maximální velikost řádku. Obvykle to znamená, že se v souboru nachází levá uvozovka, ale ne uzavírací uvozovky.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vložte do souboru uzavírací uvozovky.
