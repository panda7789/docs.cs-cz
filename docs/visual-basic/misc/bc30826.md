# <a name="endif-statements-are-no-longer-supported-use-end-if-instead"></a>Příkazy EndIf již nejsou podporovány; místo toho použijte příkaz End If.
`EndIf`Příkaz již není podporován.  
  
 **ID chyby:** BC30826  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte `End If` k ukončení `If` příkazů.
