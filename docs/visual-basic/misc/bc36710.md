# <a name="extension-method-methodname-defined-in-modulename-does-not-have-a-signature-compatible-with-delegate-delegatename"></a>Metoda rozšíření \<methodname> definovaná v ' ' nemá \<modulename> signaturu kompatibilní s delegátem ' \<delegatename> '.
Signatura metody rozšíření a delegáta, který se pokoušíte použít, se neshodují. `Delegate`Příkaz definuje typy parametrů a návratové typy třídy delegáta. K vytvoření instance tohoto typu delegáta lze použít jakýkoli postup s kompatibilními parametry, typy parametrů a návratový typ.  
  
 **ID chyby:** BC36710
