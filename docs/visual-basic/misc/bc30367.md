# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>Třídu se \<classname> nedá indexovat, protože nemá žádnou výchozí vlastnost.
Nepovedlo se dodat výchozí vlastnost pro třídu.  
  
 **ID chyby:** BC30367  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte vlastnost jako výchozí vlastnost spuštěním deklarace s `Default` klíčovým slovem.
