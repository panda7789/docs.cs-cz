# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>' \<typename> ' je typ v ' \<containername> ' a nelze jej použít jako výraz.
Jeden z používaných názvů je již deklarován jako název typu a nelze jej použít ve výrazu.  
  
 **ID chyby:** BC30691  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Jedinečně pojmenujte člena.  
  
- Použijte hranaté závorky kolem názvu člena.
