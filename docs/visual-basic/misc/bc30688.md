# <a name="events-in-interfaces-cannot-be-declared-implements"></a>Pro události v rozhraní nelze provést deklaraci \<implements> .
Události deklarované v rozhraních nemohou implementovat události jiných rozhraní.  
  
 **ID chyby:** BC30688  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte `Implements` příkaz.  
  
2. Implementujte událost v rámci třídy nebo struktury.
