# <a name="public-member-membername-on-type-typename-not-found"></a>Veřejný člen \<membername> typu \<typename> nebyl nalezen.
Člen, ke kterému se pokoušíte získat přístup, nebyl nalezen. K této chybě může dojít při přístupu k neveřejným členům objektů s pozdní vazbou.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Pokud člen existuje, ujistěte se, že je deklarován jako `Public` .
