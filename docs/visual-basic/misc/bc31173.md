# <a name="missing-required-white-space"></a>Chybí požadované prázdné znaky.
Byl zadán literál XML bez požadovaného oddělovacího prostoru. Příkladem požadovaného prostoru oddělovače je mezera mezi následujícími atributy v literálu elementu XML.  
  
 **ID chyby:** BC31173  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte oddělovač požadované místo.
