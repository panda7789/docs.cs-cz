# <a name="typename1-cannot-be-converted-to-typename2"></a>' \<typename1> ' nelze převést na ' \<typename2> '
Došlo k pokusu o převod typu mezi dvěma nekompatibilními typy.  
  
 **ID chyby:** BC30741  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte typy, které se mají přetypovat.
