# <a name="name1-conflicts-with-a-type-by-the-same-name-defined-in-name2"></a>' \<name1> ' je v konfliktu s a se \<type> stejným názvem definovaným v ' \<name2> '
Byl zjištěn název, který je v konfliktu s typem se stejným názvem.  
  
 **ID chyby:** BC31073  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarujte jedinečný název.  
  
2. Použijte hranaté závorky kolem konfliktního názvu člena.
