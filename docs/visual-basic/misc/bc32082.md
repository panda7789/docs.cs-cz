<span data-ttu-id="f3484-103">Obecný typ je vyvolán s `MustInherit` třídou jako argument typu, zatímco odpovídající parametr typu je deklarován s `New` omezením.</span><span class="sxs-lookup"><span data-stu-id="f3484-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="f3484-104">`New`Omezení vyžaduje, aby typ předaný v odpovídajícím argumentu typu měl podporovat vytváření objektů.</span><span class="sxs-lookup"><span data-stu-id="f3484-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="f3484-105">*Abstraktní* třída, která je deklarována jako `MustInherit` , ale nezveřejňuje žádné konstruktory, protože z ní nelze vytvořit žádné objekty.</span><span class="sxs-lookup"><span data-stu-id="f3484-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>
