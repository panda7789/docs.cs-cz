# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>Řádek \<number> nejde analyzovat pomocí současných oddělovačů.
Zadaný řádek nelze analyzovat, protože používá oddělovače jiné než zadané.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Upravit, `Delimiters` aby řádek bylo možné správně analyzovat, nebo vložit kód pro zpracování výjimek, aby se řádek zpracoval.
