# <a name="try-must-have-at-least-one-catch-or-a-finally"></a>Příkaz try musí mít aspoň jedno klíčové slovo Catch nebo finally.
`Try`Blok musí zahrnovat buď blok, `Finally` nebo alespoň jeden `Catch` blok před `End Try` příkazem.  
  
 **ID chyby:** BC30030  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Catch` blok nebo `Finally` v rámci `Try` bloku.
