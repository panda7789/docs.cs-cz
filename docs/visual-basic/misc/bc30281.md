# <a name="structure-structurename-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Struktura \<structurename> musí obsahovat aspoň jednu členskou proměnnou instance nebo deklaraci události.
Dojde k funkčně prázdné struktuře. Nejméně jeden člen proměnné musí být deklarovaný mezi `Structure` příkazem a `End Structure` příkazem.  
  
 **ID chyby:** BC30281  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Dim` do struktury příkaz nebo `Event` příkaz.
