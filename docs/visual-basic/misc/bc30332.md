# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-derived-from-type4"></a>Hodnotu typu \<type1> nejde převést na, \<type2> protože není \<type3> odvozená od \<type4> :.
Příkaz se pokusí převést typ pole na jiný typ pole v situaci, kde datové typy prvků polí nejsou oba odkazové typy, nebo kde konverze, buď rozšiřující nebo zúžení, není možné mezi typy prvků dvou polí.  
  
 **ID chyby:** BC30332  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte datové typy prvků obou polí a určete zdroj chyby.
