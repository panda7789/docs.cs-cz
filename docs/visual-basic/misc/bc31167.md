# <a name="expected--for-xml-end-tag"></a>Pro koncovou značku XML se očekával znak/.
Koncová značka XML pro literál XML neobsahuje požadovaný znak/.  
  
 **ID chyby:** BC31167  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte znak '/' na začátek značky end XML pro literál XML.
