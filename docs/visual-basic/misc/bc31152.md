# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Předpona oboru názvů XML prefix je vyhrazena pro použití v kódu XML a identifikátor URI oboru názvů nelze změnit.
V literálu XML byla použita vyhrazená předpona oboru názvů XML, například `xmlns` .  
  
 **ID chyby:** BC31152  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte předponu oboru názvů XML na hodnotu, která není vyhrazena.
