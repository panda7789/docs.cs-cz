# <a name="type-parameter-with-a-structure-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="51826-102">Parametr typu s omezením Structure nejde použít jako omezení.</span><span class="sxs-lookup"><span data-stu-id="51826-102">Type parameter with a 'Structure' constraint cannot be used as a constraint</span></span>
<span data-ttu-id="51826-103">Parametr typu s `Structure` omezením se používá jako omezení pro jiný parametr typu.</span><span class="sxs-lookup"><span data-stu-id="51826-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="51826-104">`Structure`Omezení vyžaduje, aby argument typu předaný parametru typu byl typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="51826-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="51826-105">Typ hodnoty však nelze implementovat nebo zdědit, proto není smysluplný ho použít jako omezení, což by vyžadovalo, aby jej implementoval jiný typ parametru.</span><span class="sxs-lookup"><span data-stu-id="51826-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="51826-106">Jediným smysluplným výkladem této situace je, že oba argumenty typu musí být přesně stejného typu.</span><span class="sxs-lookup"><span data-stu-id="51826-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="51826-107">Pokud je to tento případ, váš obecný typ vyžaduje pouze jeden parametr typu.</span><span class="sxs-lookup"><span data-stu-id="51826-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="51826-108">Následující příkaz může tuto chybu vygenerovat.</span><span class="sxs-lookup"><span data-stu-id="51826-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="51826-109">Typ předaný do `t2` nemůže implementovat nebo zdědit typ předaný do `t1` , protože předaný typ `t1` musí být typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="51826-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="51826-110">**ID chyby:** BC32114</span><span class="sxs-lookup"><span data-stu-id="51826-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="51826-111">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="51826-111">To correct this error</span></span>  
  
- <span data-ttu-id="51826-112">Odeberte parametr typu omezený na `Structure` ze seznamu omezení na jiném parametru typu.</span><span class="sxs-lookup"><span data-stu-id="51826-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
- <span data-ttu-id="51826-113">Pokud oba parametry typu vyžadují stejný typ hodnoty, definujte obecný typ pouze s jedním parametrem typu.</span><span class="sxs-lookup"><span data-stu-id="51826-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
