# <a name="nothing-cannot-be-evaluated"></a>Hodnotu Nothing nejde vyhodnotit.
`Nothing`Klíčové slovo bylo použito v okně ladění.  
  
 **ID chyby:** BC30701  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nepoužívejte `Nothing` při ladění.
