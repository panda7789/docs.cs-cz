# <a name="xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property"></a>Značka komentáře XML Returns není u vlastnosti WriteOnly povolená.
Značka komentáře XML Returns není u vlastnosti WriteOnly povolena. Komentář XML bude ignorován.  
  
 Komentář XML pro deklaraci vlastnosti jen pro zápis nemůže obsahovat \<returns> značku.  
  
 **ID chyby:** BC42313  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nepodporovanou \<returns> značku.
