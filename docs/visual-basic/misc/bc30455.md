# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>Není zadán argument pro parametr ' \<parametername> ' pro ' \<methodname> '.
Nezadali jste argument pro požadovaný parametr.  
  
 **ID chyby:** BC30455  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte argument pro zadaný parametr.
