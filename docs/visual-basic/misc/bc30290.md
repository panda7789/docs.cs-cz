# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Lokální proměnná nemůže mít stejný název jako funkce, která ji obsahuje.
`Dim`Příkaz určuje proměnnou se stejným názvem, jako má procedura, `Function` která ji obsahuje.  
  
 **ID chyby:** BC30290  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte deklaraci proměnné nebo změňte název proměnné.
