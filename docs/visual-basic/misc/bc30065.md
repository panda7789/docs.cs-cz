# <a name="exit-sub-is-not-valid-in-a-function-or-property"></a>Klíčové slovo Exit Sub není ve funkci nebo vlastnosti platné.
`Exit Sub`zobrazí se v `Function` proceduře nebo `Property` proceduře. `Exit`Příkaz musí odpovídat bloku, ve kterém se vyskytuje.  
  
 **ID chyby:** BC30065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte `Exit Sub` `Exit Function` `Exit Property` příkaz nebo podle potřeby.
