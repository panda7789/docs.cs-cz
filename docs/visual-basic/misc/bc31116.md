# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>Deklarace RemoveHandler musí končit párovým klíčovým znakem end RemoveHandler.
`RemoveHandler`Deklarace musí končit `End RemoveHandler` příkazem.  
  
 **ID chyby:** BC31116  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda `RemoveHandler` deklarace končí `End RemoveHandler` příkazem.
