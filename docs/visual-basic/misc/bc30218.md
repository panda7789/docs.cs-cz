# <a name="lib-expected"></a>Bylo očekáváno klíčové slovo lib.
`Declare`Příkaz určuje název knihovny bez použití `Lib` klíčového slova.  
  
 **ID chyby:** BC30218  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Lib` klíčové slovo před název knihovny.
