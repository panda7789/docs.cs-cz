# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-a-reference-type"></a>Hodnotu typu \<type1> nejde převést na, \<type2> protože \<type3> není odkazový typ.
Příkaz se pokusí převést typ pole na jiný typ pole, kde datový typ prvků cílového pole není odkazový typ.  
  
 **ID chyby:** BC30333  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte datové typy prvků obou polí a určete zdroj chyby.
