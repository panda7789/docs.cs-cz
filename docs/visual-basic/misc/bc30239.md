# <a name="relational-operator-expected"></a>Byl očekáván relační operátor
`Case`Příkaz obsahuje klauzuli, `Is` ale žádný relační operátor, například `=` nebo `>` . Pokud `Case` příkaz neobsahuje operátor, `=` je předpokládána a není `Is` použit.  
  
 **ID chyby:** BC30239  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Is` klíčové slovo nebo ho Sledujte pomocí operátoru porovnání.
