# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Řídicí proměnná smyčky nemůže být vlastnost ani indexované pole s pozdní vazbou.
Proměnná použitá pro iteraci `For` smyčkou musí být číselného datového typu.  
  
 **ID chyby:** BC30039  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte deklaraci řídicí proměnné smyčky tak, aby určovala `Integer` , `Short` ,, `Long` `Byte` , `Single` , `Double` nebo `Decimal` .
