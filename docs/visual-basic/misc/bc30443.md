# <a name="get-is-already-declared"></a>Příkaz Get už je deklarovaný.
`Get`Příkaz se v bloku kódu vyskytuje více než jednou. `Get`Příkaz deklaruje `Get` proceduru vlastnosti použitou k načtení hodnoty vlastnosti.  
  
 **ID chyby:** BC30443  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte redundantní `Get` příkaz.
