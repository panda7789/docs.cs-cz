# <a name="optional-parameters-cannot-have-structure-types"></a>Volitelné parametry nemůžou mít typy struktury.
Volitelné parametry nemůžou být typy struktury.  
  
 **ID chyby:** BC31405  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklaruje parametry typu struktury podle potřeby.  
  
2. Deklarujete statického člena třídy, aby bylo možné převzít místo struktury.
