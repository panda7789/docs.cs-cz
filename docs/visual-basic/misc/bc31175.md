# <a name="xml-dtds-are-not-supported"></a>Soubory DTD XML nejsou podporované.
Jako literál XML byla zadána definice typu dokumentu (DTD).  
  
 **ID chyby:** BC31175  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte DTD. Použijte definici schématu XML (XSD) k určení schématu pro dokument XML.
