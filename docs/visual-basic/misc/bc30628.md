# <a name="structures-cannot-have-inherits-statements"></a>Příkazy Inherits nemůžou obsahovat struktury.
Na rozdíl od tříd nepodporují struktury dědičnost.  
  
 **ID chyby:** BC30628  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Inherits` příkaz ze struktury.  
  
- Místo toho změňte návrh aplikace tak, aby používala třídy.
