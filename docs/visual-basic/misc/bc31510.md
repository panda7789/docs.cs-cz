# <a name="attribute-constant-constantname-cannot-be-the-target-of-an-assignment"></a>Konstanta atributu \<constantname> nemůže být cílem přiřazení.
Byl proveden pokus o přiřazení hodnoty konstantě deklarované v atributu.  
  
 **ID chyby:** BC31510  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte chybné přiřazení.  
  
- Pokud používáte vlastní atributy, které jste vyvinuli, předefinujte člena v atributu jako pole.
