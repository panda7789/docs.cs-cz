# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Názvy jiných než vnitřních typů nejsou ve výrazech podmíněné kompilace povoleny.
Výraz podmíněné kompilace odkazuje na nevnitřní typ, ale jsou povoleny pouze vnitřní typy.  
  
 Další informace o vnitřním typu v Visual Basic najdete v tématu [Přehled datových typů](../language-reference/keywords/data-types-summary.md).  
  
 **ID chyby:** BC31426  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte nevnitřní typ vnitřním typem.
