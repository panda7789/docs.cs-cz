# <a name="file-io-of-a-structure-with-field-fieldname-of-type-typename-is-not-valid"></a>I/O soubor struktury s polem \<fieldname> typu není \<typename> platný.
Struktura, kterou se pokoušíte zapsat do souboru, obsahuje nepodporovaný typ.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte strukturu a určete, kde dojde k nepodporovanému typu, a pak ho změňte na platný typ.
