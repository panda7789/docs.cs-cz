# <a name="argument-period-must-be-less-than-or-equal-to-argument-life"></a>Argument period musí být menší nebo roven argumentu Life.
Hodnota `Period` argumentu, která určuje období, pro které se počítá odpisy assetu, je větší než hodnota `Life` argumentu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zajistěte, aby `Life` `Period` argumenty a byly vyjádřeny ve stejných jednotkách. Například pokud `Life` se měří v měsících, `Period` měla by být také.
