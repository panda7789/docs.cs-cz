# <a name="new-cannot-be-used-on-a-class-that-is-declared-mustinherit"></a>Klíčové slovo New nejde použít pro třídu s deklarací MustInherit.
U `MustInherit` třídy nelze vytvořit instanci přímo, a proto `New` operátor nelze použít pro `MustInherit` třídu. Přestože je možné mít proměnné a hodnoty, jejichž typy času kompilace jsou `MustInherit` , takové proměnné a hodnoty budou nutně buď hodnotou null, nebo obsahovat odkazy na instance regulárních tříd odvozených z těchto `MustInherit` typů.  
  
 **ID chyby:** BC30569  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `New` operátor.
