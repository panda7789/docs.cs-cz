# <a name="option-infer-can-be-followed-only-by-on-or-off"></a>U možnosti "odvození" můžou následovat jenom klíčové slovo on nebo off.
`Option Infer`příkazy musí určovat buď `On` nebo `Off` . Jiné hodnoty nejsou platné.  
  
 **ID chyby:** BC30979  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `Option Infer On` nebo `Option Infer Off` .
