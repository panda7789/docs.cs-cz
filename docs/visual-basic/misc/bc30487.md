# <a name="operator-operatorname-is-not-defined-for-type-typename"></a>Operátor ' \<operatorname> ' není definován pro typ ' \<typename> '
Pokusili jste se použít unární operátor ( `+` , `-` nebo `Not` ) pro typ, například `String` , pro který není definován.  
  
 **ID chyby:** BC30487  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte unární operátor odpovídajícím operátorem pro výraz.
