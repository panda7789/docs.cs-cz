# <a name="the-path-has-not-been-set"></a>Cesta nebyla nastavena.
Došlo k pokusu o operaci se souborem, jako je například přesunutí nebo zkopírování souboru, bez poskytnutí potřebné cesty. To může vést k nesprávné analýze cesty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda byla zadána cesta k souboru a zda je správná.
