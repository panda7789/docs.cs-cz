# <a name="structure-statement-must-end-with-a-matching-end-structure"></a>Příkaz Structure musí být ukončený párovým příkazem End Structure.
Struktury začínají `Structure` příkazem a končí `End Structure` konstrukcí.  
  
 **ID chyby:** BC30624  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že blok struktury končí `End Structure` příkazem.
