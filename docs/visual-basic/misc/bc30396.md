# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>' \<specifier> ' není platný v deklaraci výčtu.
Použili jste specifikátor, který není platný v `Enum` deklaraci.  
  
 **ID chyby:** BC30396  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte specifikátor.
