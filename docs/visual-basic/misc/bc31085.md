# <a name="date-constant-is-not-valid"></a>Konstanta data není platná.
Konstanta data je deklarována s neplatným datem.  
  
 **ID chyby:** BC31085  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Opravte datum použité v konstantě.
