# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>Cílová verze prostředí .NET Compact Framework nepodporuje řešení přetížení lateBound.
Cílová verze prostředí .NET Compact Framework nepodporuje řešení přetížení s pozdní vazbou.  
  
 **ID chyby:** BC30764  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přetypování parametrů typu Object na příslušný typ.
