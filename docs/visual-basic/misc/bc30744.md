# <a name="referenced-array-element-has-a-value-of-nothing"></a>Odkazovaný element Array má hodnotu Nothing.
Odkazovaný element pole nebyl nastaven nebo byl dříve nastaven na hodnotu `Nothing` .  
  
 **ID chyby:** BC30744  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda jsou hodnoty v poli nastaveny správně.
