# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Typ argumentu ' \<argumentname> ' je ' \<typename> ', který není numerický.
K nenumerickému argumentu dochází v situaci, kdy je očekáván číselný argument.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zajistěte, aby byl argument numerický.
