# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Řídicí proměnná smyčky for se \<variablename> už používá v nadřazeném cyklu for.
`For`Všechny vnořené smyčky musí používat jedinečné proměnné ovládacího prvku.  
  
 **ID chyby:** BC30069  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte jeden z názvů proměnných ovládacího prvku.
