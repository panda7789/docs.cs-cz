# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>Znaky typů nejsou povolené v identifikátorech popisků.
Popisek příkazu obsahuje alespoň jeden znak typu identifikátoru.  
  
 Popisek příkazu musí být platný název Visual Basic. Povolené znaky neobsahují žádné znaky typu identifikátoru ( `%` , `&` , `@` ,, a `!` `#` `$` ). Viz [deklarované názvy elementů](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC31395  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte znak nebo znaky identifikátoru typu z popisku příkazu.
