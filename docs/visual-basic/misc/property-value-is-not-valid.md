# <a name="property-value-is-not-valid"></a>Hodnota vlastnosti není platná.
Jedna z vlastností objektu nebo ovládacího prvku byla nastavena na hodnotu mimo rozsah.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte hodnotu vlastnosti na platné nastavení.
