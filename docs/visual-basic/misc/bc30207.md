# <a name="option-compare-must-be-followed-by-text-or-binary"></a>Za přepínačem Option Compare musí následovat klíčové slovo text nebo Binary.
`Option Compare`Příkaz obsahuje nesprávné nastavení nebo žádné nastavení. Jediná nastavení povolená v nástroji `Option Compare` jsou `Text` a `Binary` .  
  
 **ID chyby:** BC30207  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte, zda je u specifikátoru nastavení nesprávně napsaný.  
  
2. Přidejte buď `Text` nebo `Binary` do `Option Compare` příkazu, například `Option Compare Text` .
