# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="666c8-102">Proměnná typu ReadOnly nemůže být cílem přiřazení ve výrazu lambda uvnitř konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="666c8-102">'ReadOnly' variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>
<span data-ttu-id="666c8-103">`ReadOnly`V rámci výrazu lambda se odkazovalo na proměnnou, která není povolená.</span><span class="sxs-lookup"><span data-stu-id="666c8-103">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="666c8-104">Následující kód způsobuje tuto chybu odesláním `ReadOnly` proměnné `m` jako argumentu `ByRef` parametru.</span><span class="sxs-lookup"><span data-stu-id="666c8-104">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="666c8-105">**ID chyby:** BC36602</span><span class="sxs-lookup"><span data-stu-id="666c8-105">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="666c8-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="666c8-106">To correct this error</span></span>  
  
- <span data-ttu-id="666c8-107">Pokud je to možné, změňte parametr `n` ve funkci `Test` na `ByVal` parametr, jak je znázorněno v následujícím kódu.</span><span class="sxs-lookup"><span data-stu-id="666c8-107">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="666c8-108">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="666c8-108">To correct this error</span></span>  
  
- <span data-ttu-id="666c8-109">Přiřaďte `ReadOnly` proměnnou `m` místní proměnné v konstruktoru a místo toho použijte místní proměnnou `m` , jak je znázorněno v následujícím kódu.</span><span class="sxs-lookup"><span data-stu-id="666c8-109">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
