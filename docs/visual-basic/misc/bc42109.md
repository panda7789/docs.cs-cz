# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Proměnná \<variablename> se používá dřív, než se jí přiřadí hodnota (Visual Basic chyba).
Proměnná se používá dřív, než se jí přiřadí hodnota, která má za následek <xref:System.NullReferenceException> .  
  
 **ID chyby:** BC42109  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že k proměnné je přiřazena hodnota.
