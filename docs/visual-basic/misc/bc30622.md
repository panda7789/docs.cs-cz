# <a name="end-module-must-be-preceded-by-a-matching-module"></a>Příkazu End Module musí předcházet párové klíčové slovo Module.
Bloky modulu začínají `Module` příkazem a končí `End Module` konstrukcí.  
  
 **ID chyby:** BC30622  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že blok modulu začíná `Module` příkazem.
