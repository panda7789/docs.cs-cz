# <a name="sub-or-function-expected-after-delegate"></a>Po operátoru Delegate bylo očekáváno klíčové slovo Sub nebo Function.
`Delegate`Příkaz neurčuje `Sub` `Function` proceduru or. `Sub` `Function` Klíčové slovo or musí následovat hned za `Delegate` klíčovým slovem.  
  
 **ID chyby:** BC30278  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Přidejte `Sub` `Function` klíčové slovo or hned po `Delegate` .  
  
2. Zadejte název procedury, seznam argumentů a podle potřeby návratový typ.
