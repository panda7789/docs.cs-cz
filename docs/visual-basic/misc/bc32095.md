# <a name="3939-unexpected"></a>&#39;(&#39; neočekávané
'(' Unexpected. Pole obecných typů bez instancí nejsou povolena.  
  
 Visual Basic nejde zkompilovat pole, pokud je konkrétní datového typu. Datový typ parametru obecného typu nelze použít jako datový typ pole.  
  
 **ID chyby:** BC32095  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je potřeba použít pole, je třeba deklarovat bude konkrétní datového typu. Datový typ parametru nelze použít.  
  
-   Pokud budete potřebovat skupinu prvky datového typu, který je pro datový typ parametru, musíte použít kolekci místo pole.
