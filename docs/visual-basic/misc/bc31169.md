# <a name="character-character-0xhexadecimal-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Znak "znak" (0x \<hexadecimal> ) není povolen na začátku názvu XML.
Na začátku názvu literálu XML nebo vlastnosti osy XML je zahrnut nepodporovaný znak.  
  
 **ID chyby:** BC31169  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebere nepodporovaný znak z začátku názvu vlastnosti osy XML nebo osy XML.
