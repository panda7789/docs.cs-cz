# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>' \<implementsclause> ' nemůže implementovat ' \<typename> ', protože ' \<typename> ' je vyhrazený název.
`Implements`Příkaz zadal vyhrazený název.  
  
 **ID chyby:** BC31415  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přejmenujte člena rozhraní, které nebylo možné implementovat.
