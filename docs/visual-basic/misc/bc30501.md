# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>Klíčové slovo Shared nejde kombinovat s \<specifier> deklarací v deklaraci metody.
Došlo k pokusu o kombinování `Shared` modifikátoru se specifikátorem, jako například `Overridable` , `NotOverridable` nebo `MustOverride` , který není platný v takové kombinaci v deklaraci metody.  
  
 **ID chyby:** BC30501  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
