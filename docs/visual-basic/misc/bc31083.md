# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Metody v modulu nemohou implementovat členy rozhraní.
Členy rozhraní musí být implementovány třídami nebo strukturami.  
  
 **ID chyby:** BC31083  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Implementujte členy rozhraní z tříd nebo struktur.
