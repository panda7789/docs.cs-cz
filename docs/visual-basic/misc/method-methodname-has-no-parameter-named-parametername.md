# <a name="method-methodname-has-no-parameter-named-parametername"></a>Metoda \<methodname> nemá parametr s názvem \<parametername> .
Zadaný pojmenovaný argument neodpovídá žádnému z argumentů cílové metody.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že pojmenovaný argument byl zadán správně.
