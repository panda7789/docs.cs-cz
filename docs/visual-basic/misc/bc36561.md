# <a name="extension-method-methodname-has-type-constraints-that-can-never-be-satisfied"></a><span data-ttu-id="98142-102">Metoda rozšíření \<methodname> má omezení typu, která nelze nikdy splnit.</span><span class="sxs-lookup"><span data-stu-id="98142-102">Extension method '\<methodname>' has type constraints that can never be satisfied</span></span>
<span data-ttu-id="98142-103">Parametry typu této metody pracují způsobem, který jim brání v jejich nedodržení.</span><span class="sxs-lookup"><span data-stu-id="98142-103">The type parameters of this method interact in a way that prevents them from ever being satisfied.</span></span> <span data-ttu-id="98142-104">Následující rozšiřující metoda je příklad.</span><span class="sxs-lookup"><span data-stu-id="98142-104">The following extension method is an example.</span></span>  
  
```vb  
'' Not valid.  
'<Extension()> _  
'Sub extensionExample(Of T As U, U)(ByVal para1 As T, ByVal para2 As U)  
'End Sub  
```  
  
 <span data-ttu-id="98142-105">Vzhledem k tomu, že metoda je metoda rozšíření, musí být kompilátor schopný určit datový typ nebo typy, které metoda rozšiřuje pouze na první parametr v deklaraci metody, `para1` a argumentem, který byl pro tento parametr odeslán.</span><span class="sxs-lookup"><span data-stu-id="98142-105">Because the method is an extension method, the compiler must be able to determine the data type or types that the method extends based only on the first parameter in the method declaration, `para1`, and the argument sent in for that parameter.</span></span> <span data-ttu-id="98142-106">Když první parametr odkazuje na parametry obecného typu, `para1 as T` omezení na obecných parametrech omezují sadu typů, na které se metoda vztahuje.</span><span class="sxs-lookup"><span data-stu-id="98142-106">When the first parameter refers to generic type parameters, `para1 as T`, the constraints on the generic parameters restrict the set of types to which the method applies.</span></span>  
  
 <span data-ttu-id="98142-107">Použitelnost metody rozšíření je určena od argumentu zadaného pro první parametr, který je `arg1` v následujícím kódu.</span><span class="sxs-lookup"><span data-stu-id="98142-107">Applicability of an extension method is determined from the argument provided for the first parameter, which is `arg1` in the following code.</span></span>  
  
 `'' Not valid.`  
  
 `'arg1.extensionExample(arg2)`  
  
 <span data-ttu-id="98142-108">Je nutné ověřit omezení pro všechny parametry obecného typu odkazované prvním parametrem, `para1` a to tak, že prohlížíte pouze první argument, `arg1` .</span><span class="sxs-lookup"><span data-stu-id="98142-108">It must be possible to verify the constraints on all generic type parameters referred to by the first parameter, `para1`, by looking at only the first argument, `arg1`.</span></span> <span data-ttu-id="98142-109">V `extensionExample` sadě nelze sadu typů, které lze rozšířit, určit pouze z prvního parametru.</span><span class="sxs-lookup"><span data-stu-id="98142-109">In `extensionExample`, the set of types that is being extended cannot be determined from the first parameter alone.</span></span> <span data-ttu-id="98142-110">Parametr typu `T` je omezený parametrem typu `U` , na který se neodkazuje `para1` a nedá se odvodit `arg1` .</span><span class="sxs-lookup"><span data-stu-id="98142-110">Type parameter `T` is constrained by type parameter `U`, which is not referenced by `para1` and cannot be inferred from `arg1`.</span></span> <span data-ttu-id="98142-111">Proto nelze ověřit použitelnost metody pro libovolný možný typ a metodu nelze nikdy volat.</span><span class="sxs-lookup"><span data-stu-id="98142-111">Therefore, the applicability of the method to any possible type cannot be verified, and the method can never be called.</span></span>
