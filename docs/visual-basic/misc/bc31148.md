# <a name="xml-namespace-prefix-prefix-is-not-defined"></a>Předpona oboru názvů XML prefix není definovaná.
Pro literál XML nebo vlastnost Axis XML byla zadána nedefinovaná předpona oboru názvů XML.  
  
 **ID chyby:** BC31148  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pomocí příkazu definujte předponu oboru názvů XML `Imports` .  
  
 -nebo-  
  
- Změňte předponu oboru názvů XML pro literál XML nebo vlastnost Axis XML na dříve definovanou předponu oboru názvů XML.
