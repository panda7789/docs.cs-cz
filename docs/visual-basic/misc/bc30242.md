# <a name="modifier-is-not-valid-on-a-method-declaration"></a>' \<modifier> ' není platný v deklaraci metody.
`Function`Příkaz or `Sub` obsahuje neplatné klíčové slovo.  
  
 **ID chyby:** BC30242  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte klíčové slovo neplatné z `Function` příkazu OR `Sub` .
