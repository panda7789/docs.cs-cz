# <a name="no-default-member-found-for-type-typename"></a>Pro typ nebyl nalezen žádný výchozí člen \<typename> .
Došlo k pokusu o vytvoření indexu s pozdní vazbou `Get` nebo `Set` volání.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že objekt má výchozí vlastnost s `Get` nebo `Set` , v závislosti na tom, který se pokoušíte vytvořit.
