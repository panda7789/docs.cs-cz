# <a name="unable-to-include-xml-fragment-elementpath-of-file-commentfile"></a>Nejde zahrnout fragment XML souboru. \<elementpath> \<commentfile>
`<include>`Značka v komentáři XML obsahuje výraz XPath pro element, který neexistuje, nebo soubor obsahuje kód XML s chybou ve správném formátu.  
  
 **ID chyby:** BC42320  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, zda je soubor XML v souboru platný.  
  
2. Ověřte, zda v souboru existuje fragment XML.
