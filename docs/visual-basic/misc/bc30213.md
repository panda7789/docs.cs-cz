# <a name="comma-or--expected-procedure-parameter-list"></a>Očekává se čárka nebo znak) (seznam parametrů procedury).
Seznam parametrů obsahuje neočekávaný element. Použijte čárky ( `,` ) k oddělení parametrů v takovém seznamu a použijte pravou závorku ( `)` ) pro ukončení seznamu.  
  
 **ID chyby:** BC30213  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte čárku a pokračujte v seznamu parametrů typu nebo přidejte uzavírací závorku pro ukončení seznamu.
