# <a name="comma--or-a-valid-expression-continuation-expected"></a>Očekává se čárka, znak) nebo platné pokračování výrazu.
Seznam, jako je například seznam argumentů v definici procedury nebo volání procedury, nesprávně odděluje nebo neukončuje členy. Jednou z možných příčin je, že dva členové seznamu jsou odděleni pouze mezerami nebo mezerami.  
  
 **ID chyby:** BC32017  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- K oddělení členů seznamu a uzavíracích závorek () použijte čárku (,) a seznam ukončete.
