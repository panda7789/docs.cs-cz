# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modifikátory pole nejde zadat u proměnné i jejího typu.
Modifikátory pole jsou k dispozici v proměnné i v jejím typu, což značí pole polí.  
  
 **ID chyby:** BC31087  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte modifikátor pole ze specifikátoru typu.
