# <a name="typename-is-an-unsupported-type"></a>\<typename>je nepodporovaný typ.
Zadaný typ nemůže být reprezentován aplikací Visual Studio.  
  
 **ID chyby:** BC30649  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte podporovaný typ.  
  
- Vývoj nového typu.
