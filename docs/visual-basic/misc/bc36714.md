# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="75bf3-102">Rozšířené vlastnosti nelze inicializovat.</span><span class="sxs-lookup"><span data-stu-id="75bf3-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="75bf3-103">Automaticky implementovaná vlastnost může být inicializována jako součást její deklarace, ale rozšířená vlastnost nemůže být.</span><span class="sxs-lookup"><span data-stu-id="75bf3-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="75bf3-104">**ID chyby:** BC36714</span><span class="sxs-lookup"><span data-stu-id="75bf3-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="75bf3-105">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="75bf3-105">To correct this error</span></span>  
  
- <span data-ttu-id="75bf3-106">Buď použijte automaticky implementovanou vlastnost, nebo odeberte inicializaci z deklarace vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="75bf3-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="75bf3-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="75bf3-107">Example</span></span>  
 <span data-ttu-id="75bf3-108">Následující příklady znázorňují automaticky implementované a rozšířené vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="75bf3-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="75bf3-109">Automaticky implementované vlastnosti lze inicializovat pomocí přiřazení nebo `New` klauzule, ale rozšířené vlastnosti nemohou být.</span><span class="sxs-lookup"><span data-stu-id="75bf3-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
