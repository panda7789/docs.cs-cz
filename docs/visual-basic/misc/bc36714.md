# <a name="expanded-properties-cannot-be-initialized"></a>Rozšířené vlastnosti nelze inicializovat.
Automaticky implementovaná vlastnost může být inicializována jako součást její deklarace, ale rozšířená vlastnost nemůže být.  
  
 **ID chyby:** BC36714  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Buď použijte automaticky implementovanou vlastnost, nebo odeberte inicializaci z deklarace vlastnosti.  
  
## <a name="example"></a>Příklad  
 Následující příklady znázorňují automaticky implementované a rozšířené vlastnosti. Automaticky implementované vlastnosti lze inicializovat pomocí přiřazení nebo `New` klauzule, ale rozšířené vlastnosti nemohou být.  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
