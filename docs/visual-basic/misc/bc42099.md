# <a name="unused-local-constant-constant"></a>Nepoužitá lokální konstanta: ' \<constant> '
Byla deklarována lokální konstanta, která se nepoužívá kdekoli v kódu.  
  
 **ID chyby:** BC42099  
  
### <a name="to-remove-this-warning"></a>Odebrání tohoto upozornění  
  
1. Tato chyba je upozornění a lze ji ignorovat. Chcete-li odstranit upozornění, odeberte nepoužitou konstantní deklaraci z kódu.
