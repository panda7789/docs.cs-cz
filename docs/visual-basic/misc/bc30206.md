# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>Za možností ' Option ' musí následovat klíčové slovo ' Compare ', ' Explicit ', ' odvodí ' nebo ' Strict '.
Jediné možnosti, které lze nastavit pomocí `Option` příkazu `Compare` , jsou, `Explicit` , `Infer` a `Strict` .  
  
 **ID chyby:** BC30206  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte, zda není napsané klíčové slovo Option ( `Compare` , `Explicit` , `Infer` nebo `Strict` ).  
  
- Zkontrolujte, zda `Option` je příkaz neúplný.
