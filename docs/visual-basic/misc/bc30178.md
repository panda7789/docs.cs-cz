# <a name="specifier-is-duplicated"></a>Specifikátor je duplicitní.
Klíčové slovo se ve stejné deklaraci vyskytuje více než jednou.  
  
 **ID chyby:** BC30178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odstraňte duplicitní vzhledy klíčového slova z deklarace.
