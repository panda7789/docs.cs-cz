# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>Příkaz SyncLock musí být ukončen párovým příkazem End SyncLock.
`SyncLock`bloky začínají `SyncLock` klíčovým slovem a končí `End SyncLock` konstrukcí.  
  
 **ID chyby:** BC30675  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že `SyncLock` blok končí `End SyncLock` konstrukcí.
