<span data-ttu-id="94568-103">[Příkaz Get](../language-reference/statements/get-statement.md) nebo [set](../language-reference/statements/set-statement.md) ve `NotOverridable` vlastnosti zahrnuje `Private` klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="94568-103">A [Get Statement](../language-reference/statements/get-statement.md) or [Set Statement](../language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="94568-104">Následující řádek vysvětlení vysvětluje `NotOverridable` , proč a `Private` nelze kombinovat v [příkazu vlastnosti](../language-reference/statements/property-statement.md):</span><span class="sxs-lookup"><span data-stu-id="94568-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../language-reference/statements/property-statement.md):</span></span>  
  
1. <span data-ttu-id="94568-105">Vlastnost nebo procedura, která nepřepisuje vlastnost nebo proceduru základní třídy, má výchozí nastavení [NotOverridable](../language-reference/modifiers/notoverridable.md).</span><span class="sxs-lookup"><span data-stu-id="94568-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../language-reference/modifiers/notoverridable.md).</span></span>  
  
2. <span data-ttu-id="94568-106">Nicméně vlastnost nebo procedura v odvozené třídě, která přepisuje vlastnost nebo proceduru základní třídy, má výchozí nastavení [přepsatelné](../language-reference/modifiers/overridable.md).</span><span class="sxs-lookup"><span data-stu-id="94568-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="94568-107">Chcete-li ukončit hierarchii přepsání, můžete ji deklarovat `NotOverridable` .</span><span class="sxs-lookup"><span data-stu-id="94568-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="94568-108">Toto je jediný kontext, ve kterém můžete použít `NotOverridable` .</span><span class="sxs-lookup"><span data-stu-id="94568-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="94568-109">To znamená, že můžete použít `NotOverridable` pouze v kombinaci s [přepsáními](../language-reference/modifiers/overrides.md).</span><span class="sxs-lookup"><span data-stu-id="94568-109">That is, you can use `NotOverridable` only in combination with [Overrides](../language-reference/modifiers/overrides.md).</span></span>  
  
3. <span data-ttu-id="94568-110">Pokud je vlastnost nebo procedura základní třídy deklarována jako [soukromá](../language-reference/modifiers/private.md), nemůže odvozená třída tuto vlastnost nebo proceduru přepsat, protože k ní nelze přistupovat.</span><span class="sxs-lookup"><span data-stu-id="94568-110">If a base class property or procedure is declared [Private](../language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="94568-111">Z tohoto důvodu nemůžete použít `Private` v kombinaci s `Overridable` .</span><span class="sxs-lookup"><span data-stu-id="94568-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4. <span data-ttu-id="94568-112">Chcete-li přepsat vlastnost nebo proceduru, překrytá vlastnost nebo procedura musí mít nejen stejný podpis, ale také stejnou úroveň přístupu.</span><span class="sxs-lookup"><span data-stu-id="94568-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="94568-113">To znamená, že překrytá vlastnost nebo procedura nemůže být zadána `Private` , protože nelze určit přepisující vlastnost nebo proceduru `Private` .</span><span class="sxs-lookup"><span data-stu-id="94568-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5. <span data-ttu-id="94568-114">Vzhledem k tomu, že lze zadat `NotOverridable` pouze pro přepisující vlastnost nebo proceduru, nelze ji kombinovat s `Private` .</span><span class="sxs-lookup"><span data-stu-id="94568-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="94568-115">Stejným důvodem je, že jednotlivé procedury vlastností ( `Get` a `Set` ) překryté vlastnosti nemůžou být `Private` .</span><span class="sxs-lookup"><span data-stu-id="94568-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
