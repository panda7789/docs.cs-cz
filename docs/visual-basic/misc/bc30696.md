# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>\<membername1>a se \<membername2> nemůžou vzájemně přetěžovat, protože se liší jenom typy nepovinných parametrů.
Přetížené členy musí mít jiné datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30696  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Definujte aspoň jeden povinný parametr jako jiný datový typ.
