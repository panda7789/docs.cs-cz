# <a name="too-few-type-arguments-to-genericmethodname"></a>Příliš málo argumentů typu pro \<genericMethodName>
Obecná metoda byla vyvolána s menším počtem argumentů než parametry typu.  
  
 Při vyvolání obecné metody je nutné zadat jeden argument typu pro každý parametr typu definovaný touto metodou.  
  
 **ID chyby:** BC32042  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte argumenty typu do seznamu argumentů typu tak, aby existoval jeden pro každý parametr typu na obecné metodě, kterou vyvoláte.
