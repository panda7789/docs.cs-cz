# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Připojení k knihovně typů nebo knihovně objektů pro vzdálený proces bylo ztraceno.
Ztratili jste připojení k knihovně objektů nebo knihovně typů vzdáleného procesu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Restartujte `Application` objekt.  
  
2. V dialogovém okně chyby, ve kterém jste zadali toto téma nápovědy, kliknutím na tlačítko **OK** zobrazte dialogové okno **odkazy** . Ztracený odkaz se zobrazí se slovem chybějící na levé straně.  
  
3. Odstraňte ztracený odkaz.  
  
4. V dialogovém okně **odkazy** zaškrtněte políčko pro daný `Application` objekt.
