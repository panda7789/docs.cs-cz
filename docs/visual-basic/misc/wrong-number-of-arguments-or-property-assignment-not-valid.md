# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>Nesprávný počet argumentů nebo přiřazení vlastnosti je neplatné.
Došlo k pokusu o přiřazení, které není platné.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, že počet argumentů, které jste zadali, odpovídá číslu požadovanému cílem.  
  
2. Ověřte přiřazení vlastnosti.
