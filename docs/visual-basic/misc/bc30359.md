# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>Klíčové slovo Default se dá použít jenom pro jeden název vlastnosti v.\<type>
Pokusili jste se použít `Default` na více než jeden název vlastnosti v rámci třídy, rozhraní nebo struktury.  
  
 **ID chyby:** BC30359  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte všechny kromě jedné z `Default` deklarací.
