# <a name="optional-and-paramarray-cannot-be-combined"></a>Možnost Optional a ParamArray nejde kombinovat.
`Optional` `ParamArray` Modifikátory a se vzájemně vylučují.  
  
 **ID chyby:** BC30642  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `ParamArray` nebo `Optional` , ale ne obojí.
