# <a name="-expected"></a>Očekával se znak >.
Syntaxe zadaného kódu je nesprávná. Tato chyba často znamená, že chybí uzavírací lomená závorka pro atribut nebo literál elementu XML.  
  
 **ID chyby:** BC30636  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte syntaxi používaného kódu.
