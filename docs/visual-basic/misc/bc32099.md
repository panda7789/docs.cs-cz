# <a name="comma-or--expected-type-parameter-list"></a>Očekává se čárka nebo znak) (seznam parametrů typu).
Seznam parametrů typu obsahuje neočekávaný element. Použijte čárky ( `,` ) k oddělení parametrů typu v takovém seznamu a použijte pravou závorku ( `)` ) pro ukončení seznamu.  
  
 **ID chyby:** BC32099  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte čárku a pokračujte v seznamu parametrů typu nebo přidejte uzavírací závorku pro ukončení seznamu.
