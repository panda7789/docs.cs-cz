# <a name="methodname-cannot-override-a-method-that-has-been-shadowed"></a>' \<methodname> ' nemůže přepsat metodu, která byla Stínová
Metoda se pokusila přepsat metodu, která vystínuje jiného člena.  
  
 **ID chyby:** BC31406  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte `Shadows` modifikátor pro přepsání stínovaných členů.
