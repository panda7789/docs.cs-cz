# <a name="end-function-must-be-the-first-statement-on-a-line"></a>Příkaz End Function musí být prvním příkazem na řádku.
`End Function`Příkaz sleduje dvojtečku (:) oddělovač příkazů `End Function`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC32032  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
