# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modifikátor Nullable nejde zadat u proměnné i jejího typu.
Modifikátor typu s možnou hodnotou null (?) byl zahrnut pro deklarovanou proměnnou a její deklarovaný typ.  
  
 **ID chyby:** BC33100  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte modifikátor typu s možnou hodnotou null (?) buď z proměnné, nebo z jeho deklarovaného typu.
