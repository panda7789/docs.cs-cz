# <a name="a-log-has-already-been-created-with-this-name-on-this-machine"></a>V tomto počítači už byl vytvořen protokol s tímto názvem.
Byl proveden pokus o vytvoření protokolu s názvem, který se již používá.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte konfliktní protokol.  
  
- Pro protokol použijte jiný název.
