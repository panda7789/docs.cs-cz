# <a name="sub-or-function-expected"></a>Očekávalo se klíčové slovo Sub nebo Function.
`Declare`Příkaz neurčuje buď `Sub` nebo `Function` . `Declare`identifikuje externí proceduru a musí určit, který typ procedury.  
  
 **ID chyby:** BC30215  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- `Sub`Do příkazu přidejte buď `Function` klíčové slovo or `Declare` .
