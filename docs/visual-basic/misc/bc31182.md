# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Hodnota atributu není platná. očekává se hodnota1 nebo hodnota2.
Byla zadána neplatná hodnota pro atribut v literálu dokumentu XML. Platné hodnoty atributu pro literál dokumentu XML jsou založeny na specifikaci XML 1,0.  
  
 **ID chyby:** BC31182  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte hodnotu atributu na hodnotu, která je platná podle specifikace XML 1,0.
