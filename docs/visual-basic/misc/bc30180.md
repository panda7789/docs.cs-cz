# <a name="keyword-does-not-name-a-type"></a>Klíčové slovo nejmenuje typ.
Klíčové slovo se zobrazí tam, kde je požadován název typu. Kontext volá buď základní datový typ, nebo deklarovaný název elementu představující typ.  
  
 **ID chyby:** BC30180  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Nahraďte klíčové slovo platným názvem typu.
