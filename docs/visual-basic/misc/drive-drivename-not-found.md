# <a name="drive-drivename-not-found"></a>Jednotka ' \<drivename> ' nebyla nalezena.
Jednotku nejde najít. Buď došlo ke ztrátě síťového připojení, nebo jednotka, ke které se pokoušíte získat přístup, je offline nebo neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte napájení jednotky.  
  
2. Zajistěte, aby všechny kabely připojující počítač k síti byly správně připojeny.  
  
3. Pokud chcete získat přístup k jednotce přes síť, ujistěte se, že mezi počítačem a jednotkou existuje logické připojení.  
  
4. Znovu se připojte k síti a zkuste to znovu.
