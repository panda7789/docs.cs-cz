# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>Značka include komentáře XML musí mít \<attribute> atribut.
Značka include komentáře XML musí mít \<attribute> atribut. Komentář XML bude ignorován.  
  
 `file`Atribut nebo `path` předaný do `<` `>` tagu include nebyl nalezen nebo je neúplný.  
  
 **ID chyby:** BC42310  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Do značky přidejte nebo dokončete požadovaný atribut \<include> .
