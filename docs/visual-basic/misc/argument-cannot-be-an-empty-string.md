# <a name="argument-cannot-be-an-empty-string"></a>Argument nemůže být prázdný řetězec.
Argument byl definován jako prázdný řetězec ("").  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte platnou hodnotu argumentu.
