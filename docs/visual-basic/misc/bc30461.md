# <a name="classes-cannot-be-declared-specifier"></a>Pro třídy nejde provést deklarace \<specifier> .
Došlo k pokusu o deklaraci `Class` nebo `Module` s neplatným specifikátorem.  
  
 **ID chyby:** BC30461  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
