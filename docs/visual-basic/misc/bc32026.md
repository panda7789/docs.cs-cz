# <a name="specifiers-and-attributes-are-not-valid-on-namespace-statements"></a>Specifikátory a atributy nejsou platné pro příkazy Namespace.
Obory názvů se používají pro hierarchickou organizaci bloků programu a programovacích prvků. Nevyužívají žádné modifikátory ani atributy.  
  
 **ID chyby:** BC32026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odebere všechna klíčová slova a bloky atributů z `Namespace` příkazu.
