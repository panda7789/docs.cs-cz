# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Příkazy Get již nejsou podporovány (Visual Basic)
`Get`příkazy již nejsou podporovány. Funkce vstupně-výstupní operace se soubory jsou v `Microsoft.VisualBasic` oboru názvů běžně dostupné, ale cílová verze prostředí .NET Compact Framework ji nepodporuje.  
  
 **ID chyby:** BC30767  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Provádět operace se soubory pomocí funkcí definovaných v <xref:System.IO> oboru názvů.
