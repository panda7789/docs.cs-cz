# <a name="get-statements-are-no-longer-supported-visual-basic"></a>Příkazy Get již nejsou podporovány (Visual Basic)
příkazy `Get` již nejsou podporovány. Funkce pro vstupně-výstupní operace se soubory jsou obvykle k dispozici v oboru názvů `Microsoft.VisualBasic`, ale cílová verze prostředí .NET Compact Framework ji nepodporuje.  
  
 **ID chyby:** BC30767  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Provádět operace se soubory pomocí funkcí definovaných v oboru názvů <xref:System.IO>.
