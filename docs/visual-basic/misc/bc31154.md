# <a name="xml-declaration-does-not-allow-attribute-attributename"></a>Deklarace XML nepovoluje atribut ' attributeName '.
Byl zadán neplatný atribut pro literál dokumentu XML. Platné hodnoty atributu pro literál dokumentu XML jsou založeny na specifikaci XML 1,0.  
  
 **ID chyby:** BC31154  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte neplatný atribut z literálu dokumentu XML.
