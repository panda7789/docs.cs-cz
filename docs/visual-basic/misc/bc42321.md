# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>Chybně vytvořený kód XML. Soubor ' \<commentfile> ' nelze zahrnout.
`<include>`Značka v jednom z komentářů XML odkazuje na soubor, který neexistuje, nebo na soubor, který obsahuje chybně vytvořený kód XML.  
  
 **ID chyby:** BC42321  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, zda soubor existuje.  
  
2. Ověřte, zda je soubor XML v souboru platný.
