<span data-ttu-id="911ee-103">Z těchto argumentů nelze odvodit datové typy parametrů typu v metodě rozšíření \<methodname> definované v typu typeName, protože nejsou převoditelné na stejný typ.</span><span class="sxs-lookup"><span data-stu-id="911ee-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="911ee-104">Je možné, že se chybu podaří odstranit explicitním zadáním datových typů.</span><span class="sxs-lookup"><span data-stu-id="911ee-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="911ee-105">Byl proveden pokus o použití odvození typu k určení datového typu (nebo typů) parametru typu (nebo parametrů) při vyhodnocování volání obecné metody rozšíření.</span><span class="sxs-lookup"><span data-stu-id="911ee-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="911ee-106">Kompilátor nemohl najít datový typ, který splňuje omezení všech argumentů.</span><span class="sxs-lookup"><span data-stu-id="911ee-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="911ee-107">Proto ohlásil tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="911ee-107">Therefore, it reported this error.</span></span>
