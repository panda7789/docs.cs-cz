# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Pro metodu \<methodname> nejde deklarovat klíčové slovo Partial, protože pro ni \<methodname> se dá označit jenom jedna metoda.
Nemůžete mít dvě deklarace částečné metody, které mají stejný název a signaturu.  
  
 **ID chyby:** BC31433  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Změňte název nebo podpis jedné z deklarací částečné metody.
