# <a name="argument-argumentname-must-be-greater-than-0-or-equal-to--1"></a>Argument \<argumentname> musí být větší než 0 nebo roven-1.
Hodnota argumentu je menší než 0, ale není rovna-1.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že hodnota argumentu spadá do platného rozsahu.  
  
2. Pokud je argument dodaný výpočtem, podívejte se na hodnoty, které výpočet zpracovává.
