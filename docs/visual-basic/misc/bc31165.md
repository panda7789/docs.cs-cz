# <a name="expected-beginning--for-an-xml-tag"></a>Byl očekáván začátek \< pro značku XML.
Značka XML pro literál XML neobsahuje požadovaný počáteční znak <.  
  
 **ID chyby:** BC31165  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte znak ' < ' na začátek značky XML pro literál XML.
