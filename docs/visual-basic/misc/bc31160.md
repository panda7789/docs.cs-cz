# <a name="expected-closing--for-xml-processor-instruction"></a>Bylo očekáváno ukončení instrukce procesoru XML znaky? >.
V literálu instrukcí pro zpracování XML chybí požadované uzavírací `?>` znaky.  
  
 **ID chyby:** BC31160  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte požadované uzavírací `?>` znaky na konec řetězcového literálu instrukcí pro zpracování XML.
