# <a name="get-statement-must-end-with-a-matching-end-get"></a>Příkaz Get musí být ukončen párovým příkazem End Get.
`Get`procedury vlastnosti musí končit `End Get` příkazem.  
  
 **ID chyby:** BC30631  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že `Get` procedura vlastnosti končí `End Get` konstrukcí.
