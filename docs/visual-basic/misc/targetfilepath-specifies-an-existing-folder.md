# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath určuje existující složku.
Parametr `TargetFilePath` Určuje existující adresář, ale musí určovat název nového adresáře.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte `TargetFilePath` zadáním nového adresáře místo existujícího.
