# <a name="maximum-number-of-errors-has-been-exceeded"></a>Byl překročen maximální počet chyb.
Maximální počet chyb kompilace je 100. Kompilátor ukládá toto omezení na omezení velikosti zpráv o chybách.  
  
 **ID chyby:** BC30041  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zkontrolujte ostatní chyby generované v této kompilaci a co nejvíce vyřešte. Pokud zdrojový kód obsahuje více než 100 chyb, mohou se v další kompilaci zobrazit další chyby.
