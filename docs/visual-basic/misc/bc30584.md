# <a name="type-cannot-be-inherited-more-than-once"></a>' \<type> ' nelze dědit více než jednou.
Příkaz byl duplikován `Inherits` .  
  
 **ID chyby:** BC30584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte redundantní `Inherits` příkaz.
