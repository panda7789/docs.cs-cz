# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>Částečná metoda \<methodname> nemůže používat klíčové slovo Implements.
Částečná definice metody nemůže implementovat rozhraní nebo člena rozhraní.  
  
 **ID chyby:** BC31439  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte `Implements` příkaz z definice částečné metody.
