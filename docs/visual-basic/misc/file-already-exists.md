# <a name="file-already-exists"></a>Soubor už existuje.
K této chybě dochází v důsledku pojmenování nového souboru s názvem existujícího souboru nebo pomocí příkazu **Uložit jako** k uložení aktuálně načteného projektu s názvem, který již existuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte název a buď ho změňte na takový, který nekoliduje, nebo odstraňte již existující soubor nebo projekt, pokud již není potřeba.
