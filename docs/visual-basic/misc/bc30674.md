# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>Příkazu End SyncLock musí předcházet párové klíčové slovo SyncLock.
`SyncLock`bloky začínají `SyncLock` klíčovým slovem a končí `End SyncLock` konstrukcí.  
  
 **ID chyby:** BC30674  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že `SyncLock` blok začíná `SyncLock` příkazem.
