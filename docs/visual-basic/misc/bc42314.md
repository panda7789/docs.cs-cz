# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-lttypegt"></a>Komentář XML nelze použít více než jednou pro částečné &lt;typu&gt;
Komentář XML nelze použít více než jednou pro částečné \<typ >. Komentáře XML pro tuto \<typ > se bude ignorovat.  
  
 Blok komentáře XML můžete zobrazovat nad pouze jedné části částečného typu.  
  
 Pokud blok komentáře XML se zobrazí nad více než jedné části částečného typu, toto upozornění se vytvoří pro každý blok komentáře a nejvyšší úrovně Komentář XML bude ignorován.  
  
 **ID chyby:** BC42314  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné komentář bloky.
