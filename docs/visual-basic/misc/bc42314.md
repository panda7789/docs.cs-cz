# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-type"></a>Komentář XML nelze použít více než jednou pro částečný\<type>
Komentář XML nelze použít více než jednou pro částečné \<type> . Komentáře XML pro tuto volbu \<type> budou ignorovány.  
  
 Blok komentáře XML se může vyskytovat nad rámec pouze jedné části částečného typu.  
  
 Pokud se blok komentáře XML zobrazí nad více než jednou částí částečného typu, toto upozornění je vytvořeno pro každý blok komentáře a komentář XML nejvyšší úrovně bude ignorován.  
  
 **ID chyby:** BC42314  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte nadbytečné bloky komentářů.
