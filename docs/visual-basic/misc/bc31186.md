# <a name="-is-not-allowed"></a>znak: není povolený.
znak ': ' není povolen. V tomto kontextu nelze použít kvalifikované názvy XML.  
  
 Literál XML byl zadán s kvalifikovaným názvem v neplatném umístění, například cíl v literálu instrukcí pro zpracování.  
  
 **ID chyby:** BC31186  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte literál XML pomocí kvalifikovaného názvu nebo ho přesuňte do platného umístění.  
  
     Informace o tom, které znaky jsou platné v názvech XML, naleznete v tématu [jazyk XML (Extensible Markup Language) (XML) 1,0](https://www.w3.org/TR/xml).
