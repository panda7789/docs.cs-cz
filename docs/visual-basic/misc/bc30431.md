# <a name="end-property-must-be-preceded-by-a-matching-property"></a>Příkazu End Property musí předcházet párové klíčové slovo Property.
`End Property`Příkaz se zobrazí ve vašem kódu bez odpovídajícího prohlášení, které `Property` před ním.  
  
 **ID chyby:** BC30431  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `End Property` příkaz, pokud je redundantní.  
  
- Pokud chybí některá z těchto chybějících `Property` procedur, zadejte ji.  
  
- Přesuňte na `End Property` příslušné místo v kódu.
