# <a name="overloading-methods-declared-in-multiple-base-interfaces-is-not-valid"></a>Přetěžování metod deklarovaných ve více základních rozhraních není platné.
Více zděděných rozhraní implicitně přetížit stejnou metodu.  
  
 **ID chyby:** BC31410  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte `Shadows` Modifikátor namísto `Overloads` modifikátoru.
