# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Parametry metody AddHandler a RemoveHandler nemůžou být deklarované jako ByRef.
Parametry `AddHandler` `RemoveHandler` metod a nelze deklarovat s `ByRef` modifikátorem.  
  
 **ID chyby:** BC31134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `ByRef` klíčové slovo z parametru.
