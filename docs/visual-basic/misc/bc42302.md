# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>Komentář XML musí být prvním příkazem na řádku.
Komentář XML musí být prvním příkazem na řádku. Komentář XML bude ignorován.  
  
 Komentář XML byl umístěn za jiným prvkem jazyka, což není povoleno.  
  
 **ID chyby:** BC42302  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesune komentář XML na nový řádek.
