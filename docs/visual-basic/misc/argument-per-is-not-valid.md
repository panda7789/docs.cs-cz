# <a name="argument-per-is-not-valid"></a>Argument per není platný.
Argument zadaný pro `Per` , který musí být `Double` určující platební období v rozsahu 1 až `NPer` , není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že argument spadá do platného rozsahu.
