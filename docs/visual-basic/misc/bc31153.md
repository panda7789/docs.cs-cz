# <a name="required-attribute-version-missing-from-xml-declaration"></a>V deklaraci XML chybí vyžadovaný atribut Version.
`version`Chybí atribut pro literál dokumentu XML.  
  
 **ID chyby:** BC31153  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `version` atribut do deklarace dokumentu XML, jak je uvedeno v následujícím příkladu:`<?xml version="1.0"?>`
