# <a name="member-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>' \<member> ' je v konfliktu s vyhrazeným členem podle tohoto názvu, který je implicitně deklarován ve všech výčtech
Název člena typu je v konfliktu s názvem členu implicitně deklarovaným ve všech výčtech.  
  
 Kompilátor Visual Basic vytvoří implicitní členy odpovídající určitým programovacím prvkům, které deklarujete. Výčty implicitně deklaruje člen `value__member` .  
  
 **ID chyby:** BC31420  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Upravte název člena.
