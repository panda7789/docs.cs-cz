# <a name="withevents-variables-cannot-be-typed-as-arrays"></a>Proměnné WithEvents nejde zadat jako pole.
Pokusili jste se deklarovat pole pomocí `WithEvents` . Můžete deklarovat libovolný počet individuálních proměnných, které chcete použít `WithEvents` , ale nelze deklarovat pole, která jsou tímto způsobem.  
  
 **ID chyby:** BC30476  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Deklarujte proměnné jednotlivě.
