# <a name="character-constant-must-contain-exactly-one-character"></a>Znaková konstanta musí obsahovat právě jeden znak.
Znaková konstanta není platná, protože buď obsahuje více než jeden znak, nebo je znak null (všechny bity jsou nula).  
  
 **ID chyby:** BC30004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte znakovou konstantu na jeden znak nebo změňte znak null na platný znak.
