# <a name="classes-can-inherit-only-from-other-classes"></a>Třídy mohou dědit pouze od jiných tříd.
`Inherits`Příkaz určuje jiný typ než třídu jako svůj základní typ.  
  
 **ID chyby:** BC30258  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte základní typ na třídu, která je již definována.
