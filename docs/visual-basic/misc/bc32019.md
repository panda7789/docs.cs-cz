# <a name="resume-or-goto-expected"></a>Očekávalo se klíčové slovo Resume nebo GoTo.
`On Error`Příkaz neobsahuje `Resume` klauzuli ani `GoTo` klauzuli. Musí používat jednu klauzuli nebo druhou.  
  
 **ID chyby:** BC32019  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `Resume` `GoTo` do příkazu klauzuli nebo klauzuli `On Error` .
