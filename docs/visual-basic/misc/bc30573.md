# <a name="xml-namespace-prefix-prefix-is-already-declared"></a>Předpona oboru názvů XML prefix je už deklarovaná.
Předpona oboru názvů XML definovaná `Imports` příkazem se shoduje s dříve definovanou předponou oboru názvů XML.  
  
 **ID chyby:** BC30573  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Pro předponu oboru názvů XML použijte jedinečný název.
