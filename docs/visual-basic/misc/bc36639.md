# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a>\<parametername>Ve výrazu lambda nejde použít parametr ByRef.
Výraz lambda deklarovaný v rámci `Sub` funkce or nemůže používat žádné `ByRef` parametry `Sub` funkce or. Například následující kód způsobí tuto chybu, protože `ByRef` parametr `n` je použit ve výrazu lambda.  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 **ID chyby:** BC36639  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přiřaďte `ByRef` parametr místní proměnné a použijte místní proměnnou ve výrazu lambda, jak je znázorněno v následujícím kódu:  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
