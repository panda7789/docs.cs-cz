<span data-ttu-id="bbc28-103">Výraz lambda deklarovaný v rámci `Sub` funkce nebo nemůže použít žádné `ByRef` parametry této `Sub` funkce nebo funkce.</span><span class="sxs-lookup"><span data-stu-id="bbc28-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="bbc28-104">Například následující kód způsobí tuto `ByRef` chybu, `n` protože parametr se používá ve výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="bbc28-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>
