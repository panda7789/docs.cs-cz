# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>Převody z ' \<typename1> ' na ' \<typename2> ' musí být explicitní
Tento převod nelze provést implicitně.  
  
 **ID chyby:** BC30664  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte `CType` nebo jednu z konkrétních funkcí pro převod, například `CInt` nebo `CDec` .
