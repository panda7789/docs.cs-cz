# <a name="cast-from-type-typename1-to-type-typename2-is-not-valid"></a>Přetypování z typu \<typename1> na typ \<typename2> není platné.
Pokus o převod proměnné z jednoho typu na jiný není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte typy zahrnuté v převodu a ujistěte se, že se snažíte převést na smysluplný typ.
