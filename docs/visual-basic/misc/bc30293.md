# <a name="error-structurename1-contains-structurename2"></a>\<error>: \<structurename1> obsahuje \<structurename2>
Cyklická definice struktury nastane, pokud struktura označí sebe jako jeden z vlastních členů.  
  
 **ID chyby:** BC30293  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte název členu struktury.
