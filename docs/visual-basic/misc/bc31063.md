# <a name="set-method-cannot-have-more-than-one-parameter"></a>Metoda Set nemůže mít víc než jeden parametr.
Pro procedury vlastnosti je definováno více než jeden parametr `Set` .  
  
 **ID chyby:** BC31063  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte všechny další parametry z `Set` procedur vlastností.
