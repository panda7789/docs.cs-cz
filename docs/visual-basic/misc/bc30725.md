# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>Příkazy throw nejsou platné v příkazovém podokně.
`Throw`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30725  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Throw` příkazy z ladění kódu.
