# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>Atribut MyBase nejde použít s metodou \<methodname> , protože je deklarovaný jako MustOverride.
Pokusili jste se použít `MyBase` s metodou, která byla deklarována `MustOverride` .  
  
 **ID chyby:** BC30399  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `MustOverride` deklaraci.
