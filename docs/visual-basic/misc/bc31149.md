# <a name="duplicate-xml-attribute-attributename"></a>Duplicitní atribut XML ' attributeName '
Atribut zadaný pro literál elementu XML odpovídá atributu, který byl dříve definován pro literál elementu XML. Názvy atributů v literálech elementu XML musí být jedinečné.  
  
 **ID chyby:** BC31149  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přejmenujte atribut na název, který je jedinečný pro literál elementu XML.
