# <a name="add-failed-duplicate-key-value-supplied"></a>Přidání selhalo. Byla zadána duplicitní hodnota klíče.
`Add`Operace se nezdařila, protože hodnota klíče je shodná s jinou hodnotou klíče. Hodnoty klíče musí být jedinečné.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ujistěte se, že hodnota klíče je jedinečná.
