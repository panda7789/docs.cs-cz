# <a name="modifier-is-not-valid-on-a-delegate-declaration"></a>atribut \<modifier> není platný v deklaraci delegáta.
Pokusili jste se použít modifikátor, například `Shared` , v deklaraci delegáta. Delegáti jsou objekty používané pro volání metod jiných objektů definováním konstruktoru, který je předán specifikaci metody objektu. Zadání modifikátoru v deklaraci delegáta není platné.  
  
 **ID chyby:** BC30385  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Odeberte modifikátor.
