# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>Vnořování závorek v seznamu parametrů výrazu lambda překračuje maximální povolenou hloubku.
Vnořené kulaté závorky v seznamu parametrů výrazu lambda nemohou překročit hloubku 128.  
  
 **ID chyby:** BC36664
