# <a name="implements-not-valid-in-modules"></a>Klíčové slovo Implements není platné v modulech.
`Implements`V modulu se vyskytuje příkaz nebo klauzule. Moduly nemůžou implementovat rozhraní.  
  
 **ID chyby:** BC30231  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `Implements` příkaz nebo klauzuli nebo znovu zadejte modul jako třídu.
