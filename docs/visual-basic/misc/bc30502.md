# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>Klíčové slovo Shared nejde kombinovat s atributem \<specifier> v deklaraci vlastnosti.
Došlo k pokusu o kombinování `Shared` modifikátoru se specifikátorem, který není platný v takové kombinaci v deklaraci vlastnosti.  
  
 **ID chyby:** BC30502  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte specifikátor.
