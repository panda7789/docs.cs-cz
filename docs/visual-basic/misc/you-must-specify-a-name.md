# <a name="you-must-specify-a-name"></a>Je nutné zadat název.
Při provádění operací se soubory, jako je například přejmenování souboru, je nutné zadat název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte název.
