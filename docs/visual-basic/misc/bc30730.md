# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>Pro metody s deklarací Overrides nejde provést deklaraci Overridable, protože jsou implicitně přepsatelné.
Na rozdíl od většiny metod jsou metody označené `Overrides` modifikátorem ve výchozím nastavení přepsatelné.  
  
 **ID chyby:** BC30730  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vynechejte `Overridable` Modifikátor z metod označených `Overrides` modifikátorem.
