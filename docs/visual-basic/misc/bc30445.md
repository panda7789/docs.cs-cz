# <a name="const-declaration-cannot-have-an-array-initializer"></a>Deklarace const nemůže mít inicializátor pole.
Pokusili jste se použít inicializátor pole v `Const` deklaraci.  
  
 **ID chyby:** BC30445  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Změňte inicializátor na platný výraz odebráním okolních složených závorek.
