# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>Deklaraci end RemoveHandler musí předcházet párové klíčové slovo RemoveHandler.
`End RemoveHandler`Došlo k příkazu bez odpovídajícího `RemoveHandler` příkazu. `End RemoveHandler`musí předcházet odpovídající `RemoveHandler` příkaz.  
  
 **ID chyby:** BC31125  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Ověřte, zda `RemoveHandler` je předchozí příkaz platný a správně napsaný.
