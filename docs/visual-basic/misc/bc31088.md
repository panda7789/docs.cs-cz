# <a name="notoverridable-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>Klíčové slovo NotOverridable nelze zadat pro metody, které nepřepisují jinou metodu.
Vlastnosti a metody jsou `NotOverridable` ve výchozím nastavení. `NotOverridable`Modifikátor lze použít pouze pro metody, které přepíší jinou vlastnost nebo metodu.  
  
 **ID chyby:** BC31088  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte `NotOverridable` Modifikátor z vlastností a metod, které nepřepisují jiného člena.
