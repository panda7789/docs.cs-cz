# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Blok komentáře XML nelze přidružit k žádnému prvku jazyka, který podporuje použití dokumentačních komentářů XML.
Blok komentáře XML nelze přidružit k žádnému platnému prvku jazyka. Komentář XML bude ignorován. K této chybě dochází také v případě, že značka XML v bloku komentáře nemá žádný shodný prvek jazyka.  
  
 **ID chyby:** BC42312  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vyhledejte a odstraňte cizí značku XML nebo zadejte prvek jazyka, který popisuje.
