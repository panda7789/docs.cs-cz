# <a name="39join39-expected"></a>& č. 39; Připojení k & č. 39; očekávání
A `Group Join` byla zadána klauzule bez `Join` – klíčové slovo.  
  
 **ID chyby:** BC36631  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `Join` – klíčové slovo chcete `Group Join` klauzule, jak je znázorněno v následujícím příkladu:  
  
```vb  
Dim query = From var1 In collection1 _  
            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
