# <a name="end-function-expected"></a>Očekává se příkaz End Function.
`End`Příkaz musí odpovídat bloku, který končí. `Function`Definice procedury nebyla správně ukončena.  
  
 **ID chyby:** BC30027  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `End Function` příkaz na konec procedury.
