# <a name="property-missing-end-property"></a>Ve vlastnosti chybí příkaz end Property.
`End`Příkaz musí odpovídat bloku, který končí. `Property`Definice nebyla správně ukončena.  
  
 **ID chyby:** BC30025  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `End Property` příkaz na konec definice vlastnosti.
