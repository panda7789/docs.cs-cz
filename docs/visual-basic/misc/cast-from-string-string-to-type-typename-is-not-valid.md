# <a name="cast-from-string-quotstringquot-to-type-typename-is-not-valid"></a>Přetypování z řetězce &quot; \<string> &quot; na typ \<typename> není platné.
Pokus o převod řetězcové proměnné na jiný typ není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Zkontrolujte typy zahrnuté v převodu a ujistěte se, že se snažíte převést na smysluplný typ.
