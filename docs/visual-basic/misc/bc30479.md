# <a name="shared-sub-new-cannot-have-any-parameters"></a>Sdílená procedura Sub New nemůže mít žádné parametry.
Do sdíleného příkazu jste zahrnuli parametr nebo parametry `Sub New` , které nejsou platné.  
  
 **ID chyby:** BC30479  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte parametr nebo parametry.
