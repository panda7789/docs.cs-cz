# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argument \<argumentname> musí být větší nebo roven nule.
Hodnota argumentu je menší než nula. Hodnoty argumentu musí být větší nebo rovny nule.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že hodnota argumentu je větší nebo rovna nule.  
  
2. Pokud je argument dodaný výpočtem, ověřte hodnoty zpracovávané výpočtem.
