# <a name="typename-membername-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>\<typename>\<membername>pro musí být použita deklarace Shadows, protože jiný člen s tímto názvem je deklarovaný jako Shadows.
Tato vlastnost nebo procedura má stejný název jako jiná vlastnost nebo procedura označená `Shadows` modifikátorem.  
  
 **ID chyby:** BC30695  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Použijte `Shadows` modifikátor.
