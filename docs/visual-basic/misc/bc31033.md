# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>Rozhraní se \<interfacename> dá pomocí tohoto typu implementovat jenom jednou.
Zadané rozhraní je již tímto typem implementováno a nelze ho implementovat znovu.  
  
 **ID chyby:** BC31033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Implementovat toto rozhraní pouze jednou pomocí tohoto typu.
