# <a name="argument-argumentname-must-be-greater-than-or-equal-to-1"></a>Argument \<argumentname> musí být větší nebo roven 1.
Hodnota argumentu je menší než 1.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ujistěte se, že hodnota argumentu spadá do platného rozsahu.  
  
2. Pokud je argument dodaný výpočtem, podívejte se na hodnoty, které výpočet zpracovává.
