# <a name="xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement"></a>Parametr typu komentáře XML \<parameter> neodpovídá parametru typu u odpovídajícího \<keyword> příkazu.
Parametr komentáře XML \<parameter> neodpovídá parametru typu u odpovídajícího \<keyword> příkazu. Komentář XML bude ignorován.  
  
 Značka komentáře XML byla zadána pro některé, ale ne všechny, parametry typu v metodě.  
  
 **ID chyby:** BC42317  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Poskytněte značky komentáře XML pro všechny parametry typu v metodě.
