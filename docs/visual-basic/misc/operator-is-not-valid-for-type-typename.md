# <a name="operator-is-not-valid-for-type-typename"></a>Operátor není platný pro \<typename> typ.
Operátor není vhodný pro typy.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zajistěte, aby byl operátor vhodný pro typy, na kterých je prováděno.
