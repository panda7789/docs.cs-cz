# <a name="end-tag-tagname-expected"></a>Očekává se koncová značka. \</tagName>
Koncová značka v literálu elementu XML se neshoduje s odpovídající počáteční značkou.  
  
 **ID chyby:** BC31150  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Upravte značku end XML tak, aby odpovídala počáteční značce XML pro literál elementu XML.
