# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>Příkazy přiřazení let a set už nejsou podporované.
`Let`a `Set` nelze je použít v přiřazeních.  
  
 **ID chyby:** BC30807  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vynechejte `Let` a `Set` z příkazů přiřazení, například:  
  
     `X = 2`
