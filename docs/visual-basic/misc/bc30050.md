# <a name="paramarray-parameter-must-be-an-array"></a>Parametrem ParamArray musí být pole.
`ParamArray`Argument musí být deklarován jako jednorozměrné pole typu. Deklarace nezahrnuje závorky za `ParamArray` názvem argumentu.  
  
 **ID chyby:** BC30050  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Za název argumentu přidejte dvojici závorek `ParamArray` .
