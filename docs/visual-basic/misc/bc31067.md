# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Pro metodu ve struktuře nejde provést deklarace typu Protected nebo Protected Friend.
Metoda ve struktuře byla deklarována jako `Protected` nebo `Protected Friend` .  
  
 **ID chyby:** BC31067  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Deklarujte metody uvnitř struktur jako `Public` nebo `Private` .
