# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>Příkaz end RaiseEvent musí být prvním příkazem na řádku.
`End RaiseEvent`Příkaz sleduje dvojtečku (:) oddělovač příkazů `End RaiseEvent`musí se jednat o jediný příkaz na jeho zdrojovém řádku.  
  
 **ID chyby:** BC31120  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Vícenásobné příkazy oddělte na jiné řádky.
