# <a name="too-few-type-arguments"></a>Příliš málo argumentů typu
Obecná metoda byla vyvolána s menším počtem argumentů než parametry typu.  
  
 Při vyvolání obecné metody je nutné zadat jeden argument typu pro každý parametr typu definovaný touto metodou.  
  
 **ID chyby:** BC36578  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte argumenty typu do seznamu argumentů typu tak, aby existoval jeden pro každý parametr typu Obecné metody, kterou vyvoláte.
