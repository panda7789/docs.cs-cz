# <a name="readonly-and-writeonly-cannot-be-combined"></a>Typ ReadOnly a WriteOnly nejde kombinovat.
`ReadOnly` `WriteOnly` Modifikátory a se vzájemně vylučují.  
  
 **ID chyby:** BC30635  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Zadejte buď `ReadOnly` nebo `WriteOnly` , ale ne obojí.
