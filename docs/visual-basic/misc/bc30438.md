# <a name="constants-must-have-a-value"></a>Konstanty musí mít hodnotu.
Deklarovali jste konstantu bez přiřazení hodnoty.  
  
 **ID chyby:** BC30438  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přiřaďte hodnotu konstantě.
