# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>' \<modifier> ' není platný v deklaraci lokální proměnné.
`Dim`Příkaz v proceduře obsahuje neplatné klíčové slovo, například `Overloads` .  
  
 **ID chyby:** BC30247  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Odeberte z příkazu neplatné klíčové slovo `Dim` .
