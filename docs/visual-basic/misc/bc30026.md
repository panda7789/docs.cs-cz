# <a name="end-sub-expected"></a>Očekával se příkaz End Sub.
`End`Příkaz musí odpovídat bloku, který končí. `Sub`Definice procedury nebyla správně ukončena.  
  
 **ID chyby:** BC30026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte `End Sub` příkaz na konec procedury.
