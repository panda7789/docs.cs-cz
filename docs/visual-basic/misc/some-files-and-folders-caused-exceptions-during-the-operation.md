# <a name="some-files-and-folders-caused-exceptions-during-the-operation"></a>Některé soubory a složky způsobily během operace výjimky.
Adresáře a soubory mohou způsobit výjimky při pokusu o operaci, ale uživatel nemá potřebná oprávnění.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přidejte oprávnění potřebná pro přístup ke všem souborům a složkám.
