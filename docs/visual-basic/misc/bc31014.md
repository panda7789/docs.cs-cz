# <a name="unable-to-find-entry-point-name-in-dll-filename-error"></a>Nelze najít vstupní bod \<name> v knihovně DLL ' \<filename> ':\<error>
Název procedury nebyl nalezen v zadané knihovně DLL.  
  
 **ID chyby:** BC31014  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Ověřte, zda je správně zadán název procedury a zda velikost písmen odpovídá funkci v knihovně DLL.
