# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-delegatename"></a><span data-ttu-id="2895b-102">Možnost Strict On nepovoluje zúžení v implicitních převodech typů mezi výrazem lambda a delegátem \<delegatename> .</span><span class="sxs-lookup"><span data-stu-id="2895b-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate '\<delegatename>'</span></span>
<span data-ttu-id="2895b-103">V `Option Strict` systému na portálu nemůžete mít zužující převod mezi datovým typem parametru v delegátu a odpovídajícím parametrem výrazu lambda přiřazeného proměnné daného typu delegáta.</span><span class="sxs-lookup"><span data-stu-id="2895b-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="2895b-104">Například v následujícím kódu `Del` má delegát jeden parametr typu `Integer` .</span><span class="sxs-lookup"><span data-stu-id="2895b-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="2895b-105">Proto odpovídající parametr jakéhokoli výrazu lambda přiřazeného proměnné typu `Del` může být `Integer` nebo libovolný datový typ, pro který existuje rozšiřující převod z `Integer` .</span><span class="sxs-lookup"><span data-stu-id="2895b-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="2895b-106">**ID chyby:** BC36662</span><span class="sxs-lookup"><span data-stu-id="2895b-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2895b-107">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="2895b-107">To correct this error</span></span>  
  
- <span data-ttu-id="2895b-108">Změňte datový typ parametru v delegátu nebo výrazu lambda tak, aby existovala požadovaná rozšiřující relace.</span><span class="sxs-lookup"><span data-stu-id="2895b-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
- <span data-ttu-id="2895b-109">Nezadávejte datové typy parametrů ve výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="2895b-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="2895b-110">Typy budou odvozeny z odpovídajících parametrů v delegátovi.</span><span class="sxs-lookup"><span data-stu-id="2895b-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"  
```
