# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>Příkazy Inherits musí předcházet před všemi deklaracemi v rozhraní.
`Inherits`Příkaz musí být prvním neprázdným řádkem bez komentářů v definici třídy.  
  
 **ID chyby:** BC30357  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přesuňte `Inherits` příkaz tak, aby předcházel všechny deklarace.
