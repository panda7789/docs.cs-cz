# <a name="xml-attribute-version-must-be-the-first-attribute-in-xml-declaration"></a>Atribut XML Version musí být prvním atributem v deklaraci XML.
`version`Atribut literálu dokumentu XML není prvním zadaným atributem. `version`Atribut musí být prvním atributem zadaným pro literál dokumentu XML podle specifikace xml 1,0.  
  
 **ID chyby:** BC31156  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
- Přemístěte `version` atribut literálu dokumentu XML tak, aby byl prvním atributem.
