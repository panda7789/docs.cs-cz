<span data-ttu-id="6573b-159">Jak bylo uvedeno dříve, nastavení <xref:System.Windows.Style.TargetType%2A> vlastnosti na hodnotu `TextBlock` bez přiřazení stylu `x:Key` způsobí, že styl bude použit pro všechny <xref:System.Windows.Controls.TextBlock> elementy.</span><span class="sxs-lookup"><span data-stu-id="6573b-159">As previously shown, setting the <xref:System.Windows.Style.TargetType%2A> property to `TextBlock` without assigning the style an `x:Key` causes the style to be applied to all <xref:System.Windows.Controls.TextBlock> elements.</span></span> <span data-ttu-id="6573b-160">V tomto případě `x:Key` je implicitně nastaven na `{x:Type TextBlock}` .</span><span class="sxs-lookup"><span data-stu-id="6573b-160">In this case, the `x:Key` is implicitly set to `{x:Type TextBlock}`.</span></span> <span data-ttu-id="6573b-161">To znamená, že pokud explicitně nastavíte `x:Key` hodnotu na jinou než `{x:Type TextBlock}` , <xref:System.Windows.Style> není použita na všechny `TextBlock` prvky automaticky.</span><span class="sxs-lookup"><span data-stu-id="6573b-161">This means that if you explicitly set the `x:Key` value to anything other than `{x:Type TextBlock}`, the <xref:System.Windows.Style> isn't applied to all `TextBlock` elements automatically.</span></span> <span data-ttu-id="6573b-162">Místo toho je nutné použít styl (pomocí `x:Key` hodnoty) na `TextBlock` elementy explicitně.</span><span class="sxs-lookup"><span data-stu-id="6573b-162">Instead, you must apply the style (by using the `x:Key` value) to the `TextBlock` elements explicitly.</span></span> <span data-ttu-id="6573b-163">Pokud je váš styl v části Resources (prostředky) a nenastavíte `TargetType` vlastnost na styl, je nutné nastavit `x:Key` atribut.</span><span class="sxs-lookup"><span data-stu-id="6573b-163">If your style is in the resources section and you don't set the `TargetType` property on your style, then you must set the `x:Key` attribute.</span></span>

<span data-ttu-id="6573b-164">Kromě poskytování výchozí hodnoty pro `x:Key` , `TargetType` vlastnost určuje typ, pro který se aplikuje vlastnosti setter.</span><span class="sxs-lookup"><span data-stu-id="6573b-164">In addition to providing a default value for the `x:Key`, the `TargetType` property specifies the type to which setter properties apply.</span></span> <span data-ttu-id="6573b-165">Pokud nezadáte, je `TargetType` nutné kvalifikovat vlastnosti <xref:System.Windows.Setter> objektů pomocí názvu třídy pomocí syntaxe `Property="ClassName.Property"` .</span><span class="sxs-lookup"><span data-stu-id="6573b-165">If you don't specify a `TargetType`, you must qualify the properties in your <xref:System.Windows.Setter> objects with a class name by using the syntax `Property="ClassName.Property"`.</span></span> <span data-ttu-id="6573b-166">Například místo nastavení `Property="FontSize"` musíte nastavit <xref:System.Windows.Setter.Property%2A> na `"TextBlock.FontSize"` nebo `"Control.FontSize"` .</span><span class="sxs-lookup"><span data-stu-id="6573b-166">For example, instead of setting `Property="FontSize"`, you must set <xref:System.Windows.Setter.Property%2A> to `"TextBlock.FontSize"` or `"Control.FontSize"`.</span></span>
