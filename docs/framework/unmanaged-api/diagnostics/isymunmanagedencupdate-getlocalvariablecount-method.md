# <a name="isymunmanagedencupdategetlocalvariablecount-method"></a>ISymUnmanagedENCUpdate::GetLocalVariableCount – metoda
Získá počet místních proměnných.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetLocalVariableCount(  
    [in]  mdMethodDef  mdMethodToken,  
    [out] ULONG        *pcLocals);  
```  
  
## <a name="parameters"></a>Parametry  
 `mdMethodToken`  
 pro Token metadat metod.  
  
 `pcLocals`  
 mimo Ukazatel na `ULONG32` , který obdrží velikost vyrovnávací paměti, která je nutná k omezení počtu místních proměnných.  
  
## <a name="return-value"></a>Návratová hodnota  
 S_OK, pokud je metoda úspěšná; v opačném případě E_FAIL nebo nějaký jiný kód chyby.
