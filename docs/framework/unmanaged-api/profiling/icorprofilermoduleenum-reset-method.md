# <a name="icorprofilermoduleenumreset-method"></a>ICorProfilerModuleEnum::Reset – metoda
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Reset();  
```
