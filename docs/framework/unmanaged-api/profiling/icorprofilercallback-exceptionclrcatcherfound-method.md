# <a name="icorprofilercallbackexceptionclrcatcherfound-method"></a>ICorProfilerCallback::ExceptionCLRCatcherFound – metoda
Volá se, když `catch` se v samotném modulu CLR (Common Language Runtime) najde blok pro výjimku. Tato metoda je zastaralá ve verzi .NET Framework 2,0.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT ExceptionCLRCatcherFound();  
```
