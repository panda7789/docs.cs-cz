# <a name="icorprofilerinfo2getthreadappdomain-method"></a>ICorProfilerInfo2::GetThreadAppDomain – metoda
Získá ID domény aplikace, ve které zadané vlákno aktuálně spouští kód.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetThreadAppDomain(  
    [in]  ThreadID threadId,  
    [out] AppDomainID *pAppDomainId);  
```  
  
## <a name="parameters"></a>Parametry  
 `threadId`  
 pro ID určující vlákno.  
  
 `pAppDomainId`  
 mimo Ukazatel na ID domény aplikace.
