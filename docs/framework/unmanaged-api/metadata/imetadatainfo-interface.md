# <a name="imetadatainfo-interface"></a>IMetaDataInfo – rozhraní
Poskytuje metodu, která získává informace o mapování metadat ze souboru na disku do paměti.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[GetFileMapping – metoda](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|Získá oblast paměti mapovaného souboru a druh mapování.|
