# <a name="imetadataconvertergettypelibfrommetadata-method"></a>IMetaDataConverter::GetTypeLibFromMetaData – metoda
Získá ukazatel na `ITypeLib` instanci, která představuje knihovnu typů, která má zadané názvy knihovny a modulu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetTypeLibFromMetaData (  
    [in]  BSTR     strModule,
    [in]  BSTR     strTlbName,
    [out] ITypeLib **ppITL  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `strModule`  
 [v] Název modulu knihovny typů.  
  
 `strTlbName`  
 [v] Název knihovny typů.  
  
 `ppITL`  
 [out] Ukazatel na umístění, které přijímá adresu `ITypeLib` instance, která představuje knihovnu typů.
