# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString – metoda
Získá index dalšího řetězce v aktuálním sloupci tabulky.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetNextString (
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ixString`  
 [v] Hodnota indexu ze sloupce tabulky řetězců.  
  
 `pNext`  
 [out] Ukazatel na index dalšího řetězce ve sloupci.
