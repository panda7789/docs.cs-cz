# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString – metoda
Získá token metadat pro zadaný řetězec literálu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>Parametry  
 `szString`  
 [v] Uživatelský řetězec k uložení.  
  
 `cchString`  
 [v] Počet širokých znaků `szString`v .  
  
 `pstk`  
 [out] Přiřazený token řetězce.
