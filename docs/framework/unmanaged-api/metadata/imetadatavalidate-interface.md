# <a name="imetadatavalidate-interface"></a>IMetaDataValidate – rozhraní
Poskytuje metody pro ověření signatur metadat.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Description|  
|------------|-----------------|  
|[ValidateMetaData – metoda](imetadatavalidate-validatemetadata-method.md)|Ověří podpisy metadat objektů v aktuálním oboru metadat.|  
|[ValidatorInit – metoda](imetadatavalidate-validatorinit-method.md)|Nastaví příznak, který určuje typ modulu v aktuálním oboru metadat a registruje zadanou metodu zpětného volání pro chyby ověření.|
