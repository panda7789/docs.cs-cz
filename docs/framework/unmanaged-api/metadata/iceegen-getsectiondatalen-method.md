# <a name="iceegengetsectiondatalen-method"></a>ICeeGen::GetSectionDataLen – metoda
Získá délku zadaného oddílu.  
  
 Tato metoda je zastaralá a neměla by se používat.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetSectionDataLen (  
    [in]  HCEESECTION  section,  
    [out] ULONG        *dataLen  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `section`  
 pro Datová část, jejíž délka bude načtena.  
  
 `dataLen`  
 mimo Vrácená délka zadaného oddílu.  
  
## <a name="remarks"></a>Poznámky  
 Volejte `GetSectionDataLen` pouze v případě, že máte zvláštní požadavky na oddíly, které nejsou zpracovány jinými metodami.
