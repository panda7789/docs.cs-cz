# <a name="imetadataimportcountenum-method"></a>IMetaDataImport::CountEnum – metoda
Získá počet prvků ve výčtu, který byl načten zadaným čítač výčtu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT CountEnum (  
   [in]  HCORENUM    hEnum,
   [out] ULONG       *pulCount  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `hEnum`  
 [v] Popisovač pro čítač výčtu.  
  
 `pulCount`  
 [out] Počet prvků ve výčtu.  
  
## <a name="remarks"></a>Poznámky  
 Popisovač zadaný `hEnum` podle je `Enum`získán z *předchozího* name volání (například [IMetaDataImport::EnumTypeDefs](../../../../docs/framework/unmanaged-api/metadata/imetadataimport-enumtypedefs-method.md)).
