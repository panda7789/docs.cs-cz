# <a name="iceegengetimaptokeniface-method"></a>ICeeGen::GetIMapTokenIface – metoda
Získá rozhraní, na které odkazuje zadaný token.  
  
 Tato metoda je zastaralá a neměla by se používat.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetIMapTokenIface (  
    [in, out] IUnknown   **pIMapToken  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pIMapToken`  
 [in, out] Token metadat pro rozhraní, které má být vráceno.
