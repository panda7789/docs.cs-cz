# <a name="imetadataimportcloseenum-method"></a>IMetaDataImport::CloseEnum – metoda
Ukončí enumerátor, který je identifikován specifikovaným popisovačem.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
void CloseEnum (  
   [in] HCORENUM hEnum  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `hEnum`  
 pro Popisovač pro zavření čítače.  
  
## <a name="remarks"></a>Poznámky  
 Popisovač určený pomocí `hEnum` je získán z předchozího `Enum` *názvu* volání (například [IMetaDataImport:: enumtypedefs –](imetadataimport-enumtypedefs-method.md)).
