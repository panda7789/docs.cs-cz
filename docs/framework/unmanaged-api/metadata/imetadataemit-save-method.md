# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save – metoda
Uloží všechna metadata v aktuálním oboru do souboru na zadanou adresu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `wzFile`  
 [v] Název souboru, do který chcete uložit. Pokud je tato hodnota null, bude kopie v paměti uložena do posledního umístění, které bylo použito.  
  
 `dwSaveFlags`  
 [v] Vyhrazena. Musí být nula.
