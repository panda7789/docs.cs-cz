# <a name="icordebugassemblyenum-interface"></a>ICorDebugAssemblyEnum – rozhraní

Implementuje metody ICorDebugEnum a vytváří výčet polí ICorDebugAssembly.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[Next – metoda](icordebugassemblyenum-next-method.md)|Získá zadaný počet `ICorDebugAssembly` instancí ve výčtu, počínaje od aktuální pozice.|  
  
## <a name="remarks"></a>Poznámky  
  
> [!NOTE]
> Toto rozhraní nepodporuje vzdálené volání, a to buď mezi počítačem, nebo mezi procesy.
