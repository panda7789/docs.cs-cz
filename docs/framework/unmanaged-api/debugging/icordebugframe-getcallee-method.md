# <a name="icordebugframegetcallee-method"></a>ICorDebugFrame::GetCallee – metoda
Získá ukazatel na objekt ICorDebugFrame v aktuálním řetězu, který tento rámec volal.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetCallee (  
    [out] ICorDebugFrame     **ppFrame  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppFrame`  
 mimo Ukazatel na adresu `ICorDebugFrame` objektu, který představuje pojmenovaný rámec. Tato hodnota je null, pokud je volajícím snímkem vnitřní rámec aktuálního řetězce.
