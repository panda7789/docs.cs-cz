# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose – metoda
Uvolňuje popisovač, na který odkazuje tento objekt ICorDebugHandleValue, bez explicitního uvolnění ukazatele rozhraní.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Dispose ();  
```
