## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>Parametry

 mimo Ukazatel na počet rozměrů.
