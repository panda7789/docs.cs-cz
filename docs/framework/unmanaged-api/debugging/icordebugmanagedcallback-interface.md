## <a name="remarks"></a><span data-ttu-id="fe012-160">Poznámky</span><span class="sxs-lookup"><span data-stu-id="fe012-160">Remarks</span></span>  
 <span data-ttu-id="fe012-161">Všechna zpětná volání jsou serializována, volána ve stejném vlákně a volána s procesem v synchronizovaném stavu.</span><span class="sxs-lookup"><span data-stu-id="fe012-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>  
  
 <span data-ttu-id="fe012-162">Každé implementaci zpětného volání musí volat metodu [ICorDebugController:: Continue](icordebugcontroller-continue-method.md) , aby bylo možné pokračovat v provádění.</span><span class="sxs-lookup"><span data-stu-id="fe012-162">Each callback implementation must call [ICorDebugController::Continue](icordebugcontroller-continue-method.md) to resume execution.</span></span> <span data-ttu-id="fe012-163">Pokud `ICorDebugController::Continue` není volána před vrácením zpětného volání, proces zůstane zastaven a žádné další zpětné volání události nebude provedeno, dokud `ICorDebugController::Continue` není volána.</span><span class="sxs-lookup"><span data-stu-id="fe012-163">If `ICorDebugController::Continue` is not called before the callback returns, the process will remain stopped and no more event callbacks will occur until `ICorDebugController::Continue` is called.</span></span>  
  
 <span data-ttu-id="fe012-164">Ladicí program musí implementovat [ICorDebugManagedCallback2](icordebugmanagedcallback2-interface.md) , pokud ladí aplikace .NET Framework aplikací verze 2,0.</span><span class="sxs-lookup"><span data-stu-id="fe012-164">A debugger must implement [ICorDebugManagedCallback2](icordebugmanagedcallback2-interface.md) if it is debugging .NET Framework version 2.0 applications.</span></span> <span data-ttu-id="fe012-165">Instance `ICorDebugManagedCallback` nebo `ICorDebugManagedCallback2` je předána jako objekt zpětného volání do [ICorDebug:: SetManagedHandler –](icordebug-setmanagedhandler-method.md).</span><span class="sxs-lookup"><span data-stu-id="fe012-165">An instance of `ICorDebugManagedCallback` or `ICorDebugManagedCallback2` is passed as the callback object to [ICorDebug::SetManagedHandler](icordebug-setmanagedhandler-method.md).</span></span>
