# <a name="icordebugclassgetmodule-method"></a>ICorDebugClass::GetModule – metoda
Získá modul, který definuje tuto třídu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetModule (  
    [out] ICorDebugModule    **pModule  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pModule`  
 mimo Ukazatel na adresu objektu ICorDebugModule, který představuje modul, ve kterém je tato třída definována.
