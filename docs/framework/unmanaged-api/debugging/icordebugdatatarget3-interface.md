# <a name="icordebugdatatarget3-interface"></a>Rozhraní ICorDebugDataTarget3
Logicky rozšiřuje rozhraní [ICorDebugDataTarget](icordebugdatatarget-interface.md) , aby poskytovala informace o načtených modulech.  
  
## <a name="method"></a>Metoda  
  
|Metoda|Popis|  
|------------|-----------------|  
|[GetLoadedModules – metoda](icordebugdatatarget3-getloadedmodules-method.md)|Načte seznam modulů, které byly doposud načteny.|  
  
## <a name="remarks"></a>Poznámky  
  
> [!NOTE]
> Toto rozhraní je dostupné jenom pro .NET Native. Pokud implementujete Toto rozhraní pro ICorDebug scénáře mimo .NET Native, modul CLR (Common Language Runtime) bude toto rozhraní ignorovat.
