# <a name="cordebuggenerationtypes-enumeration"></a>CorDebugGenerationTypes – výčet
Určuje generaci oblasti paměti na spravované haldě.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
typedef enum CorDebugGenerationTypes {  
    CorDebug_Gen0 = 0,  
    CorDebug_Gen1 = 1,  
    CorDebug_Gen2 = 2,  
    CorDebug_LOH  = 3,  
} CorDebugRegionTypes;  
```  
  
## <a name="members"></a>Členové  
  
|Název členu|Popis|  
|-----------------|-----------------|  
|`CorDebug_Gen0`|Generace 0.|  
|`CorDebug_Gen1`|Generace 1.|  
|`CorDebug_Gen2`|Generace 2.|  
|`CorDebug_LOH`|Halda velkých objektů.|
