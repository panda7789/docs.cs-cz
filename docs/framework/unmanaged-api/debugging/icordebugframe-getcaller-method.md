# <a name="icordebugframegetcaller-method"></a>ICorDebugFrame::GetCaller – metoda
Získá ukazatel na objekt ICorDebugFrame v aktuálním řetězci, který se nazývá tento rámec.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetCaller (  
    [out] ICorDebugFrame     **ppFrame  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppFrame`  
 mimo Ukazatel na adresu `ICorDebugFrame` objektu, který představuje volající rámec. Tato hodnota je null, pokud pojmenovaný rámec je nejvzdálenějším snímkem v aktuálním řetězu.
