# <a name="icordebugmergedassemblyrecordgetversion-method"></a>ICorDebugMergedAssemblyRecord::Metoda GetVersion
Získá informace o verzi sestavení.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetVersion(  
   [out] USHORT *pMajor,
   [out] USHORT *pMinor,
   [out] USHORT *pBuild,
   [out] USHORT *pRevision  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pMajor`  
 [out] Ukazatel na číslo hlavní verze.  
  
 `pMinor`  
 [out] Ukazatel na číslo dílčí verze.  
  
 `pBuild`  
 [out] Ukazatel na číslo sestavení.  
  
 `pRevision`  
 [out] Ukazatel na číslo revize.  
  
## <a name="remarks"></a>Poznámky  
 Informace o číslech verzí <xref:System.Version> sestavení naleznete v tématu třídy.  
  
> [!NOTE]
> Tato metoda je k dispozici pouze s nativní .NET.
