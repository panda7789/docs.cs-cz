# <a name="icordebuggetprocess-method"></a>ICorDebug::GetProcess – metoda
Získá ukazatel na instanci "ICorDebugProcess" zadaného procesu.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetProcess (  
    [in] DWORD               dwProcessId,  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `dwProcessId`  
 pro ID procesu  
  
 `ppProcess`  
 mimo Ukazatel na adresu instance `ICorDebugProcess` pro zadaný proces.
