# <a name="debugging-unmanaged-api-reference"></a>Ladění (referenční dokumentace nespravovaného rozhraní API)
Ladění rozhraní API umožňuje ladicí program ladit kód, který běží v prostředí CLR (COMMON Language Runtime). Kód, který má být laděn může být libovolný typ kódu, který podporuje CLR.
