|Metoda|Popis|  
|------------|-----------------|  
|[GetID – metoda](icorpublishappdomain-getid-method.md)|Získá jedinečný identifikátor pro tento `ICorPublishAppDomain`.|  
|[GetName – metoda](icorpublishappdomain-getname-method.md)|Získá název domény aplikace, na kterou odkazuje tento `ICorPublishAppDomain`.|
