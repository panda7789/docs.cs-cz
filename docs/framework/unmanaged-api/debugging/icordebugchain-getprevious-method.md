# <a name="icordebugchaingetprevious-method"></a>ICorDebugChain::GetPrevious – metoda
Načte předchozí řetězec snímků pro vlákno.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT GetPrevious (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `ppChain`  
 mimo Ukazatel na adresu objektu ICorDebugChain, který představuje předchozí řetězec snímků pro toto vlákno. Pokud je tento řetěz prvním řetězcem, `ppChain` má hodnotu null.
