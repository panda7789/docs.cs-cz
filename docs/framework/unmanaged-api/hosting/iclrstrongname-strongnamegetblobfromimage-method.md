# <a name="iclrstrongnamestrongnamegetblobfromimage-method"></a>ICLRStrongName::StrongNameGetBlobFromImage – metoda
Načte binární reprezentaci image sestavení v zadané adrese paměti.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT StrongNameGetBlobFromImage (  
    [in]  BYTE        *pbBase,  
    [in]  DWORD       dwLength,  
    [in]  BYTE        *pbBlob,  
    [in, out] DWORD   *pcbBlob  
);  
```  
  
## <a name="parameters"></a>Parametry  
 `pbBase`  
 pro Adresa paměti namapovaného manifestu sestavení.  
  
 `dwLength`  
 pro Velikost obrázku v bajtech `pbBase` .  
  
 `pbBlob`  
 pro Vyrovnávací paměť obsahující binární reprezentaci obrázku.  
  
 `pcbBlob`  
 [in, out] Požadovaná maximální velikost (v bajtech) `pbBlob` . Po vrácení se skutečnou velikostí v bajtech `pbBlob` .  
  
## <a name="return-value"></a>Návratová hodnota  
 `S_OK`Pokud byla metoda úspěšně dokončena; v opačném případě hodnota HRESULT, která označuje selhání (viz [společné hodnoty HRESULT](/windows/win32/seccrypto/common-hresult-values) pro seznam).
