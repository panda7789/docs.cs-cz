# <a name="clrruntimehost-coclass"></a>CLRRuntimeHost – třída typu coclass
Poskytuje rozhraní pro správu provádění kódu modulem runtime.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
coclass CLRRuntimeHost {  
    [default] interface  ICLRRuntimeHost;  
    interface            ICLRValidator;  
};  
```  
  
## <a name="interfaces"></a>Rozhraní  
  
|Rozhraní|Popis|  
|---------------|-----------------|  
|[ICLRRuntimeHost – rozhraní](iclrruntimehost-interface.md)|Poskytuje metody pro řízení provádění aplikací modulem runtime.|  
|[ICLRValidator – rozhraní](iclrvalidator-interface.md)|Poskytuje metody pro ověření přenosných spustitelných imagí a podrobné hlášení chyb ověřování.|
