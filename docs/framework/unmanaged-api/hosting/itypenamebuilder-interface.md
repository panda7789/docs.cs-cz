# <a name="itypenamebuilder-interface"></a>ITypeNameBuilder – rozhraní
Poskytuje metody pro sestavení názvu typu.  
  
 Toto rozhraní podporuje infrastrukturu .NET Framework a není určeno pro použití přímo v kódu.
