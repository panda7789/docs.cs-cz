# <a name="deprecated-clr-hosting-interfaces-and-coclasses"></a>Zastaralá rozhraní a třídy typu Coclass rozhraní hostování CLR
Tato část popisuje rozhraní, která nespravované hostitele můžou použít k integraci modulu CLR (Common Language Runtime) v .NET Framework verzích 1,0 a 1,1 do svých aplikací. Tato rozhraní poskytují metody pro hostitele ke konfiguraci a načtení modulu runtime do procesu.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 IAppDomainSetup –  
 Poskytuje metody pro hostitele ke konfiguraci <xref:System.AppDomain> .  
  
 [ICeeFileGen – třída](iceefilegen-class.md)  
 Zastaralé Poskytuje funkce pro vytvoření nativního přenositelného spustitelného souboru (PE).  
  
 [ICorRuntimeHost – rozhraní](icorruntimehost-interface.md)  
 Poskytuje metody pro hostitele ke konfiguraci nastavení CLR.
