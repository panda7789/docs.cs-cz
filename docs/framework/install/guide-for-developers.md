# <a name="install-the-net-framework-for-developers"></a>Instalace rozhraní .NET Framework pro vývojáře

.NET je nedílnou součástí mnoha aplikací spuštěných v systému Windows a poskytuje běžné funkce pro tyto aplikace ke spuštění. Pro vývojáře rozhraní .NET Framework poskytuje komplexní a konzistentní programovací model pro vytváření aplikací, které mají vizuálně ohromující uživatelské prostředí a bezproblémovou a zabezpečenou komunikaci.

> [!NOTE]
> Toto téma je určeno **pro vývojáře,** kteří buď chtějí nainstalovat rozhraní .NET Framework do vlastního systému, nebo ho chtějí nainstalovat se svými aplikacemi. Informace **o uživatelích, kteří** mají zájem o instalaci rozhraní .NET Framework, naleznete v jednotlivých tématech, která popisují instalaci rozhraní .NET Framework v konkrétních operačních systémech, jako je [například Instalace rozhraní .NET Framework v systému Windows 10 a Windows Server 2016](on-windows-10.md).

Tento článek obsahuje odkazy pro instalaci všech verzí rozhraní .NET Framework z rozhraní .NET Framework 4.5 do rozhraní .NET Framework 4.8 v počítači. Pokud jste vývojář, můžete také použít tyto odkazy ke stažení a redistribuci rozhraní .NET Framework s vašimi aplikacemi. Informace o nasazení verze rozhraní .NET Framework s vaší aplikací naleznete v [příručce pro vývojáře pro nasazení rozhraní .NET Framework](../deployment/deployment-guide-for-developers.md).

## <a name="installation-choices"></a>Volby instalace

Nainstalujte balíček cílení pro vývojáře, který se bude vyvíjet podle nejnovější verze rozhraní .NET Framework v sadě Visual Studio nebo jiného vývojového prostředí, nebo si stáhněte rozhraní .NET Framework redistributable pro distribuci s vaší aplikací nebo ovládacím prvkem.

### <a name="to-install-the-net-framework-developer-pack-or-targeting-pack"></a>Instalace sady .NET Framework Developer Pack nebo Balíčku cílení

*Balíček cílení* umožňuje vaší aplikaci cílit na konkrétní verzi rozhraní .NET Framework při vývoji v sadě Visual Studio a v některých dalších vývojových prostředích. *Vývojářská sada* obsahuje konkrétní verzi rozhraní .NET Framework a její doprovodnou sadu SDK spolu s odpovídající sadou cílení.

Vývojářská sada pro rozhraní .NET Framework 4.5.1 nebo 4.5.2, balíček cílení pro rozhraní .NET Framework 4.6 a vývojářský balíček pro rozhraní .NET Framework 4.6.1, 4.6.2, 4.7, 4.7.1, 4.7.2 nebo 4.8 poskytuje konkrétní verzi referenčních sestavení, jazyk rozhraní .NET Framework. a soubory IntelliSense pro použití v integrovaném vývojovém prostředí, jako je Visual Studio.  Pokud používáte Visual Studio, vývojářská sada nebo balíček cílení také přidá nainstalovanou verzi rozhraní .NET Framework do cílových voleb při vytváření nového projektu.  Vyberte jednu z následujících možností:

- [Rozhraní .NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)
- [Rozhraní .NET 4.7.2](https://dotnet.microsoft.com/download/dotnet-framework/net472)
- [Rozhraní .NET 4.7.1](https://dotnet.microsoft.com/download/dotnet-framework/net471)
- [Rozhraní .NET Framework 4.7](https://dotnet.microsoft.com/download/dotnet-framework/net47)
- [.NET Framework 4.6.2](https://dotnet.microsoft.com/download/dotnet-framework/net462)
- [.NET Framework 4.6.1](https://dotnet.microsoft.com/download/dotnet-framework/net461)
- [.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46)
- [.NET Framework 4.5.2](https://dotnet.microsoft.com/download/dotnet-framework/net452) pro instalaci verze 4.5.2 ve Windows 8.1 nebo starší, Visual Studio 2013, Visual Studio 2012 nebo jiné INEŽ.
- [Rozhraní .NET Framework 4.5.1](https://dotnet.microsoft.com/download/dotnet-framework/net451) pro instalaci verze 4.5.1 v sadě Visual Studio 2012 nebo jiných ines.

Na stránce pro stažení sady developer pack zvolte **Stáhnout**. Dále zvolte **Spustit** nebo **Uložit**a postupujte podle pokynů po zobrazení výzvy. Sadu Developer Pack nebo balíček cílení můžete nainstalovat také pro konkrétní verzi rozhraní .NET Framework tak, že jej vyberete z volitelných součástí ve vývojovém zatížení **.NET** pro stolní počítače v Instalační službě sady Visual Studio, jak ukazuje následující obrázek.

   ![Instalační program sady Visual Studio s úlohou vývoje plochy rozhraní .NET](./media/visual-studio-installer.jpg)

Když cílíte na určitou verzi rozhraní .NET Framework, vaše aplikace je vytvořena pomocí referenčních sestavení, která jsou součástí sady developer pack této verze. Za běhu jsou sestavení vyřešena z globální mezipaměti sestavení a referenční sestavení se nepoužívají.

Při vytváření aplikace z visual studio nebo pomocí MSBuild z příkazového řádku MSBuild může zobrazit chybu MSB3644, "referenční sestavení pro rozhraní framework *-version*" nebyly nalezeny. Chcete-li chybu vyřešit, stáhněte si sadu developer pack nebo balíček cílení pro tuto verzi rozhraní .NET Framework.

### <a name="to-install-or-download-the-net-framework-redistributable"></a>Instalace nebo stažení rozhraní .NET Framework redistributable

Instalační služby stáhnou součásti rozhraní .NET Framework pro aplikaci nebo ovládací prvek, který cílí na tyto verze rozhraní .NET Framework. Tyto součásti musí být nainstalovány v každém počítači, kde je aplikace nebo ovládací prvek spuštěn. Tyto instalační programy jsou redistribuovatelné, takže je můžete zahrnout do instalačního programu pro vaši aplikaci.

Stránka ke stažení je k dispozici v několika jazycích, ale většina souborů ke stažení je poskytována pouze v angličtině. Pro další jazykovou podporu je nutné nainstalovat jazykovou sadu.

K dispozici jsou dva typy redistribuovatelných instalačních techniků:

- **Webový instalační program** (webový zaváděcí nástroj) stáhne požadované součásti a jazykovou sadu, která odpovídá operačnímu systému instalačního počítače z webu. Tento balíček je mnohem menší než offline instalační program, ale vyžaduje konzistentní připojení k Internetu. Můžete si stáhnout [samostatné jazykové sady](#to-install-language-packs) a nainstalovat další jazykovou podporu.

- **Offline instalační program** (samostatný redistribuovatelný) obsahuje všechny požadované součásti pro instalaci rozhraní .NET Framework, ale neobsahuje jazykové sady. Tento soubor ke stažení je větší než webový instalační program. Offline instalační program nevyžaduje připojení k Internetu. Po spuštění offline instalačního programu si můžete stáhnout [samostatné jazykové sady](#to-install-language-packs) a nainstalovat jazykovou podporu. Pokud se nemůžete spolehnout na konzistentní připojení k Internetu, použijte offline instalační program.

Webové i offline instalační programy jsou určeny pro počítače s procesory x86 a x64 (viz [systémové požadavky),](../get-started/system-requirements.md)ale nepodporují počítače s procesory Itanium.

1. Otevřete stránku pro stažení verze rozhraní .NET Framework, kterou chcete nainstalovat:

   - [Rozhraní .NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)
   - [Rozhraní .NET 4.7.2](https://dotnet.microsoft.com/download/dotnet-framework/net472)
   - [Rozhraní .NET 4.7.1](https://dotnet.microsoft.com/download/dotnet-framework/net471)
   - [Rozhraní .NET Framework 4.7](https://dotnet.microsoft.com/download/dotnet-framework/net47)
   - [.NET Framework 4.6.2](https://dotnet.microsoft.com/download/dotnet-framework/net462)
   - [.NET Framework 4.6.1](https://dotnet.microsoft.com/download/dotnet-framework/net461)
   - [.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46)
   - [.NET Framework 4.5.2](https://dotnet.microsoft.com/download/dotnet-framework/net452)
   - [.NET Framework 4.5.1](https://dotnet.microsoft.com/download/dotnet-framework/net451)
   - [.NET Framework 4.5](https://dotnet.microsoft.com/download/dotnet-framework/net45)

1. Vyberte jazyk pro stránku pro stažení. Tato možnost nestáhne lokalizované prostředky rozhraní .NET Framework. ovlivňuje pouze text zobrazený na stránce pro stahování.

1. Zvolte **Stáhnout**.

1. Pokud se zobrazí výzva, vyberte stahování, které odpovídá architektuře systému, a pak zvolte **Další**.

1. Po zobrazení výzvy ke stažení proveďte **jednu** z následujících akcí:

   - Pokud chcete do počítače nainstalovat rozhraní .NET Framework, zvolte **Spustit**a postupujte podle pokynů na obrazovce.

   - Pokud chcete stáhnout rozhraní .NET Framework pro další distribuci, zvolte **Uložit**a postupujte podle pokynů na obrazovce.

1. Chcete-li stáhnout zdroje pro další jazyky, nainstalujte jednu nebo více jazykových sad podle pokynů v další části.

> Pokud během instalace narazíte na nějaké problémy, [přečtěte si článek Poradce při potížích](troubleshoot-blocked-installations-and-uninstallations.md).

**Poznámky k instalaci:**

- Rozhraní .NET Framework 4.5 a novější verze nahrazují rozhraní .NET Framework 4.0. Při instalaci těchto verzí do systému, který má nainstalovanou rozhraní .NET Framework 4, jsou sestavení nahrazena.

- Odinstalace rozhraní .NET Framework 4.5 nebo novějších verzí také odebere již existující soubory rozhraní .NET Framework 4. Pokud se chcete vrátit do rozhraní .NET Framework 4, je nutné ji přeinstalovat a všechny její aktualizace. Viz [Instalace rozhraní .NET Framework 4](https://go.microsoft.com/fwlink/p/?LinkId=230665).

- K instalaci rozhraní .NET Framework 4.5 nebo novějších verzí musíte mít pověření pro správu.

- 9. října 2012 byla aktualizována redistribuovatelná architektura .NET Framework 4.5, aby opravila problém související s nesprávným časovým razítkem na digitálním certifikátu, který způsobil předčasné vypršení platnosti digitálního podpisu na souborech vytvořených a podepsaných společností Microsoft. Pokud jste dříve nainstalovali balíček .NET Framework 4.5 redistributable ze srpna 16, 2012, doporučujeme aktualizovat kopii s nejnovější redistribuovatelné ze [stránky pro stažení rozhraní .NET Framework](https://dotnet.microsoft.com/download/dotnet-framework/net45). Další informace o tomto problému naleznete [v informačním zpravodaji zabezpečení společnosti Microsoft 2749655](https://docs.microsoft.com/security-updates/SecurityAdvisories/2012/2749655) a [v článku 2770445 znalostní báze Knowledge Base](https://support.microsoft.com/kb/2770445).

## <a name="to-install-language-packs"></a>Instalace jazykových sad

Jazykové sady jsou spustitelné soubory, které obsahují lokalizované prostředky (například přeložené chybové zprávy a text uj. ) pro podporované jazyky. Pokud nenainstalujete jazykovou sadu, zobrazí se chybové zprávy rozhraní .NET Framework a další text v angličtině.  Všimněte si, že webový instalační program automaticky nainstaluje jazykovou sadu, která odpovídá vašemu operačnímu systému, ale do počítače si můžete stáhnout další jazykové sady. Instalátory offline neobsahují žádné jazykové sady.

> [!IMPORTANT]
> Jazykové sady neobsahují součásti rozhraní .NET Framework, které jsou nutné ke spuštění aplikace, takže je nutné před instalací jazykové sady spustit webovou nebo offline instalační službu. Pokud jste již nainstalovali jazykovou sadu, odinstalujte ji, nainstalujte rozhraní .NET Framework a znovu nainstalujte jazykovou sadu.

1. Otevřete stránku pro stažení jazykové sady pro nainstalovanou verzi rozhraní .NET Framework:

   - [Rozhraní .NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)
