# <a name="install-the-net-framework-on-windows-xp-and-windows-server-2003"></a>Instalace rozhraní .NET Framework v systému Windows XP a Windows Server 2003

> [!NOTE]
> Systém Windows XP již není podporován společností Microsoft. Doporučujeme upgradovat na Windows 10, který je podporovaný a obsahuje nejnovější verzi rozhraní .NET Framework. Tento dokument je k dispozici pouze jako užitečný průvodce odstraňováním potíží.

Rozhraní .NET Framework je nutné spustit mnoho aplikací v systému Windows. K instalaci můžete použít následující pokyny. Pravděpodobně jste se dostali na tuto stránku po pokusu o spuštění aplikace a zobrazení následujícího dialogového okna v počítači.

![Tuto aplikaci nelze spustit.](./media/this-application-could-not-be-started.png)

Tyto pokyny vám pomohou nainstalovat potřebné verze rozhraní .NET Framework. [Rozhraní .NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) je nejnovější verze. Není podporována v systémech Windows XP a Windows Server 2003. Je součástí [aktualizace Windows 10 Květen 2019](https://support.microsoft.com/help/4028685/windows-10-get-the-update).

## <a name="net-framework-403"></a>Rozhraní .NET Framework 4.0.3

Rozhraní [.NET Framework 4.0.3](https://www.microsoft.com/download/details.aspx?id=29053) je nejnovější podporovanou verzí rozhraní .NET Framework v systémech Windows XP a Windows Server 2003. Rozhraní .NET Framework 4.0.3 vyžaduje, aby byla nejprve nainstalována [rozhraní .NET Framework 4.](https://dotnet.microsoft.com/download/dotnet-framework/net40) Obě tyto verze rozhraní .NET Framework již nejsou společností Microsoft podporovány.

## <a name="net-framework-4"></a>.NET Framework 4

[Rozhraní .NET Framework 4](https://dotnet.microsoft.com/download/dotnet-framework/net40) můžete nainstalovat do systému Windows XP. Společnost Microsoft ji již nepodporuje.

## <a name="net-framework-35"></a>.NET Framework 3.5

[Rozhraní .NET Framework 3.5](https://dotnet.microsoft.com/download/dotnet-framework/net35-sp1) můžete nainstalovat do systému Windows XP.

Rozhraní .NET Framework 3.5 lze použít ke spuštění aplikací vytvořených pro rozhraní .NET Framework 1.0 až 3.5.
