# <a name="install-the-net-framework-on-windows-7-sp1-and-windows-server-2008-r2"></a>Instalace rozhraní .NET Framework v systému Windows 7 SP1 a Windows Server 2008 R2

Rozhraní .NET Framework je nutné spustit mnoho aplikací v systému Windows. K instalaci můžete použít následující pokyny. Pravděpodobně jste se dostali na tuto stránku po pokusu o spuštění aplikace a zobrazení následujícího dialogového okna v počítači.

![Tuto aplikaci nelze spustit.](./media/this-application-could-not-be-started.png)

Tyto pokyny vám pomohou nainstalovat potřebné verze rozhraní .NET Framework. [Rozhraní .NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) je nejnovější verze. Je podporován v systémech Windows 7 SP1 a Windows Server 2008 R2 a je součástí [aktualizace Windows 10 May 2019 Update](https://support.microsoft.com/help/4028685/windows-10-get-the-update).

## <a name="net-framework-48"></a> .NET Framework 4.8

> [!div class="button"]
> [Stáhnout .NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)

Rozhraní [.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) lze použít ke spuštění aplikací vytvořených pro rozhraní .NET Framework 4.0 nebo novější.

## <a name="net-framework-35"></a>.NET Framework 3.5

Rozhraní [.NET Framework 3.5](https://dotnet.microsoft.com/download/dotnet-framework/net35-sp1) je součástí systému Windows 7.

Rozhraní .NET Framework 3.5 podporuje aplikace vytvořené pro rozhraní .NET Framework 1.0 až 3.5.

## <a name="help"></a>Nápověda

Pokud nelze nainstalovat správnou verzi rozhraní .NET Framework, můžete [se obrátit na společnost Microsoft.](mailto:dotnet-install-help@service.microsoft.com?subject=Install-Help)
