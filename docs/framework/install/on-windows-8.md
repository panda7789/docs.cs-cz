# <a name="install-the-net-framework-on-windows-8-and-windows-server-2012"></a>Instalace rozhraní .NET Framework v systému Windows 8 a Windows Server 2012

Rozhraní .NET Framework je nutné spustit mnoho aplikací v systému Windows. K instalaci můžete použít následující pokyny. Pravděpodobně jste se dostali na tuto stránku po pokusu o spuštění aplikace a zobrazení následujícího dialogového okna v počítači.

![Tuto aplikaci nelze spustit.](./media/this-application-could-not-be-started.png)

Tyto pokyny vám pomohou nainstalovat potřebné verze rozhraní .NET Framework. [Rozhraní .NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) je nejnovější verze. Je podporován v systému Windows Server 2012, ale není podporován v systému Windows 8. Je součástí [aktualizace Windows 10 Květen 2019](https://support.microsoft.com/help/4028685/windows-10-get-the-update)).

## <a name="net-framework-48"></a> .NET Framework 4.8

> [!div class="button"]
> [Stáhnout .NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)

Rozhraní .NET Framework 4.8 je v systému Windows Server 2012 podporováno. Není podporován v systému Windows 8.

Rozhraní [.NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) lze použít ke spuštění aplikací vytvořených pro rozhraní .NET Framework 4.0 nebo novější.

## <a name="net-framework-46"></a>.NET Framework 4.6

Rozhraní [.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46) je nejnovější podporovaná verze rozhraní .NET Framework v systému Windows 8.

Rozhraní .NET Framework 4.6 podporuje aplikace vytvořené pro rozhraní .NET Framework 4.0 až 4.6.

## <a name="net-framework-35"></a>.NET Framework 3.5

Podle pokynů nainstalujte [rozhraní .NET Framework 3.5 do windows 8 a 10](dotnet-35-windows-10.md).

Rozhraní .NET Framework 3.5 podporuje aplikace vytvořené pro rozhraní .NET Framework 1.0 až 3.5.

## <a name="help"></a>Nápověda

Pokud nelze nainstalovat správnou verzi rozhraní .NET Framework, můžete [se obrátit na společnost Microsoft.](mailto:dotnet-install-help@service.microsoft.com?subject=Install-Help)
