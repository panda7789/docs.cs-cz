# <a name="install-the-net-framework-on-windows-vista-and-windows-server-2008"></a>Instalace rozhraní .NET Framework v systému Windows Vista a Windows Server 2008

Rozhraní .NET Framework je nutné spustit mnoho aplikací v systému Windows. K instalaci můžete použít následující pokyny. Pravděpodobně jste se dostali na tuto stránku po pokusu o spuštění aplikace a zobrazení následujícího dialogového okna v počítači.

![Tuto aplikaci nelze spustit.](./media/this-application-could-not-be-started.png)

Tyto pokyny vám pomohou nainstalovat potřebné verze rozhraní .NET Framework. [Rozhraní .NET Framework 4.8](https://github.com/Microsoft/dotnet/tree/master/releases/net48) je nejnovější verze. Není podporována v systémech Windows Vista a Windows Server 2008. Je součástí [aktualizace Windows 10 Květen 2019](https://support.microsoft.com/help/4028685/windows-10-get-the-update).

## <a name="net-framework-46"></a>.NET Framework 4.6

Rozhraní [.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46) je nejnovější podporovaná verze rozhraní .NET Framework v systémech Windows Vista a Windows Server 2008.

Rozhraní [.NET Framework 4.6](https://dotnet.microsoft.com/download/dotnet-framework/net46) podporuje aplikace vytvořené pro rozhraní .NET Framework 4.0 až 4.6.

## <a name="net-framework-35"></a>.NET Framework 3.5

[Rozhraní .NET Framework 3.5](https://dotnet.microsoft.com/download/dotnet-framework/net35-sp1) můžete nainstalovat do systému Windows Vista.

Rozhraní .NET Framework 3.5 podporuje aplikace vytvořené pro rozhraní .NET Framework 1.0 až 3.5.
