- <span data-ttu-id="a3532-108">Zadejte cíl pro trasování výstupu.</span><span class="sxs-lookup"><span data-stu-id="a3532-108">Specify a destination for tracing output.</span></span>

 <span data-ttu-id="a3532-111">Nejběžnějšími cíli trasování, označovaných také jako naslouchací procesy trasování, jsou výchozí naslouchací proces a soubor protokolu.</span><span class="sxs-lookup"><span data-stu-id="a3532-111">The most common trace destinations, also referred to as trace listeners, are the default listener and the log file.</span></span>  
  
 <span data-ttu-id="a3532-112">Trasování používá výchozí naslouchací proces, pokud nezadáte naslouchací proces trasování.</span><span class="sxs-lookup"><span data-stu-id="a3532-112">Tracing uses the default listener if you do not specify a trace listener.</span></span> <span data-ttu-id="a3532-113">Zprávy odeslané do výchozího naslouchacího procesu můžete zobrazit spuštěním kódu v ladicím programu s povoleným kódem, jako je například ladicí program CLR dodávaný s .NET Framework SDK nebo DBwin32. exe dodaný s Windows SDK.</span><span class="sxs-lookup"><span data-stu-id="a3532-113">You can view messages sent to the default listener by executing your code in a managed code-enabled debugger such as the CLR Debugger shipped with the .NET Framework SDK, or DBwin32.exe shipped with the Windows SDK.</span></span> <span data-ttu-id="a3532-114">Pomocí ladicího programu CLR se zprávy trasování zobrazí v okně **výstup** .</span><span class="sxs-lookup"><span data-stu-id="a3532-114">Using the CLR Debugger, the trace messages appear in the **Output** window.</span></span>  
  
 <span data-ttu-id="a3532-115">Pokud dáváte přednost použití souboru pro příjem trasování, můžete určit soubor protokolu pomocí nastavení konfigurace, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="a3532-115">If you prefer to use a file to receive traces, you can specify a log file by using configuration settings, as shown in the following example.</span></span> <span data-ttu-id="a3532-116">(Obecné informace o konfiguračních souborech najdete v tématu [konfigurační soubory](../configure-apps/index.md).)</span><span class="sxs-lookup"><span data-stu-id="a3532-116">(For a general discussion of configuration files, see [Configuration Files](../configure-apps/index.md).)</span></span>  
  
 <span data-ttu-id="a3532-117">Chcete-li odeslat trasování do souboru protokolu, přidejte následující uzel do `<system.diagnostics>` uzlu příslušného konfiguračního souboru (aplikace nebo počítač).</span><span class="sxs-lookup"><span data-stu-id="a3532-117">To send traces to a log file, add the following node to the `<system.diagnostics>` node of the appropriate configuration file (application or machine).</span></span> <span data-ttu-id="a3532-118">Název souboru (Trace. log) můžete změnit tak, aby vyhovoval vašim potřebám.</span><span class="sxs-lookup"><span data-stu-id="a3532-118">You can change the name of the file (trace.log) to suit your needs.</span></span>  
  
```xml

  <trace autoflush="true" indentsize="4">  
    <listeners>  
      <add name="file" type="System.Diagnostics.TextWriterTraceListener" initializeData="trace.log"/>  
    </listeners>
  </trace>  
</system.diagnostics>  
```
