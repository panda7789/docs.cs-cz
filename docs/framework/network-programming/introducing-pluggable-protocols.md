## <a name="identifying-resources"></a><span data-ttu-id="323d2-112">Identifikace zdrojů</span><span class="sxs-lookup"><span data-stu-id="323d2-112">Identifying Resources</span></span>

 <span data-ttu-id="323d2-116">Poté, co server obdržel požadavek a zpracoval odpověď, vrátí odpověď klientské aplikaci.</span><span class="sxs-lookup"><span data-stu-id="323d2-116">After the server has received the request and processed the response, it returns the response to the client application.</span></span> <span data-ttu-id="323d2-117">Odpověď obsahuje doplňující informace, například typ obsahu (například nezpracovaný text nebo data XML).</span><span class="sxs-lookup"><span data-stu-id="323d2-117">The response includes supplemental information, such as the type of the content (raw text or XML data, for example).</span></span>
