## <a name="pnrp-ids"></a><span data-ttu-id="72250-118">ID PNRP</span><span class="sxs-lookup"><span data-stu-id="72250-118">PNRP IDs</span></span>  
 <span data-ttu-id="72250-119">![PNRP ID](./media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span><span class="sxs-lookup"><span data-stu-id="72250-119">![PNRP ID](./media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span></span>  
  
 <span data-ttu-id="72250-120">ID PNRP se skládají z následujících:</span><span class="sxs-lookup"><span data-stu-id="72250-120">PNRP IDs are composed of the following:</span></span>  
  
- <span data-ttu-id="72250-121">High-order 128 bitů, označované jako ID peer-to-peer (P2P), jsou hash názvu partnera přiřazeného ke koncovému bodu.</span><span class="sxs-lookup"><span data-stu-id="72250-121">The high-order 128 bits, known as the peer-to-peer (P2P) ID, are a hash of a peer name assigned to the endpoint.</span></span> <span data-ttu-id="72250-122">Název partnera má následující formát: *Authority.Classifier*.</span><span class="sxs-lookup"><span data-stu-id="72250-122">The peer name has the following format: *Authority.Classifier*.</span></span> <span data-ttu-id="72250-123">Pro zabezpečené názvy *authority* je algoritmus secure hash 1 (SHA1) hash veřejného klíče názvu partnera v šestnáctkové znaky.</span><span class="sxs-lookup"><span data-stu-id="72250-123">For secured names, *Authority* is the Secure Hash Algorithm 1 (SHA1) hash of the public key of the peer name in hexadecimal characters.</span></span> <span data-ttu-id="72250-124">Pro nezabezpečené názvy je *úřad* jediným znakem "0".</span><span class="sxs-lookup"><span data-stu-id="72250-124">For unsecured names, the *Authority* is the single character "0".</span></span> <span data-ttu-id="72250-125">*Třídění* je řetězec, který identifikuje aplikaci.</span><span class="sxs-lookup"><span data-stu-id="72250-125">*Classifier* is a string that identifies the application.</span></span> <span data-ttu-id="72250-126">Žádný klasifikátor názvu partnera může být větší `null` než 149 znaků, včetně zakončení.</span><span class="sxs-lookup"><span data-stu-id="72250-126">No peer name classifier can be greater than 149 characters long, including the `null` terminator.</span></span>  
  
- <span data-ttu-id="72250-127">Nízké pořadí 128 bitů se používají pro umístění služby, což je generované číslo, které identifikuje různé instance stejného ID P2P ve stejném cloudu.</span><span class="sxs-lookup"><span data-stu-id="72250-127">The low-order 128 bits are used for the Service Location, which is a generated number that identifies different instances of the same P2P ID in the same cloud.</span></span>  
  
 <span data-ttu-id="72250-128">Tato kombinace P2P ID a umístění služby umožňuje více PNRP ID, které mají být registrovány z jednoho počítače.</span><span class="sxs-lookup"><span data-stu-id="72250-128">This combination of P2P ID and Service Location allows multiple PNRP IDs to be registered from a single computer.</span></span>
