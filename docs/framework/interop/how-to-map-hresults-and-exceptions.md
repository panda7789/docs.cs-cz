1.  <span data-ttu-id="c988c-115">Pomocí následujícího kódu vytvořte novou třídu výjimek s názvem `NoAccessException` a jejich mapování na HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="c988c-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>

    ```cpp  
        NoAccessException () {

    throw new NoAccessException();  
    }  
    ```
