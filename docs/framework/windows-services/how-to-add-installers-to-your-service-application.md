 <span data-ttu-id="c4635-111">V rámci instalačních programů není nutné provádět žádné speciální kódování, aby se vaše služby nainstalovaly správně.</span><span class="sxs-lookup"><span data-stu-id="c4635-111">You do not need to do any special coding within the installers to make your services install correctly.</span></span> <span data-ttu-id="c4635-112">Někdy však může být nutné změnit obsah instalačních programů, pokud potřebujete do procesu instalace přidat speciální funkce.</span><span class="sxs-lookup"><span data-stu-id="c4635-112">However, you may occasionally need to modify the contents of the installers if you need to add special functionality to the installation process.</span></span>

3. <span data-ttu-id="c4635-119">V návrháři se zaostřením klikněte pravým tlačítkem myši a pak klikněte na **Přidat instalační program**.</span><span class="sxs-lookup"><span data-stu-id="c4635-119">With the designer in focus, right-click, and then click **Add Installer**.</span></span>

4. <span data-ttu-id="c4635-121">Klikněte na <xref:System.ServiceProcess.ServiceInstaller> součást a ověřte, zda je hodnota <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> vlastnosti nastavena na stejnou hodnotu jako <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> vlastnost v samotné službě.</span><span class="sxs-lookup"><span data-stu-id="c4635-121">Click the <xref:System.ServiceProcess.ServiceInstaller> component and verify that the value of the <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> property is set to the same value as the <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> property on the service itself.</span></span>

6. <span data-ttu-id="c4635-129">Chcete-li určit kontext zabezpečení, ve kterém bude služba spuštěna, klikněte <xref:System.ServiceProcess.ServiceProcessInstaller> na součást a nastavte příslušné hodnoty vlastností.</span><span class="sxs-lookup"><span data-stu-id="c4635-129">To determine the security context in which your service will run, click the <xref:System.ServiceProcess.ServiceProcessInstaller> component and set the appropriate property values.</span></span> <span data-ttu-id="c4635-130">Další informace naleznete v tématu [How to: Určete kontext zabezpečení pro služby](how-to-specify-the-security-context-for-services.md).</span><span class="sxs-lookup"><span data-stu-id="c4635-130">For more information, see [How to: Specify the Security Context for Services](how-to-specify-the-security-context-for-services.md).</span></span>
