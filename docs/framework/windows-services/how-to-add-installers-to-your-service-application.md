 <span data-ttu-id="9764b-110">Přidáte-li do projektu instalační program, je v projektu vytvořena nová třída (ve výchozím nastavení je pojmenována `ProjectInstaller` ) a v rámci ní jsou vytvořeny instance příslušných instalačních komponent.</span><span class="sxs-lookup"><span data-stu-id="9764b-110">When you add an installer to your project, a new class (which, by default, is named `ProjectInstaller`) is created in the project, and instances of the appropriate installation components are created within it.</span></span> <span data-ttu-id="9764b-111">Tato třída slouží jako centrální bod pro všechny součásti instalace, které váš projekt potřebuje.</span><span class="sxs-lookup"><span data-stu-id="9764b-111">This class acts as a central point for all of the installation components your project needs.</span></span> <span data-ttu-id="9764b-112">Pokud například přidáte do aplikace druhou službu a kliknete na odkaz Přidat instalační program, není vytvořena druhá třída instalačního programu. místo toho je do existující třídy přidána nutná dodatečná instalační součást pro druhou službu.</span><span class="sxs-lookup"><span data-stu-id="9764b-112">For example, if you add a second service to your application and click the Add Installer link, a second installer class is not created; instead, the necessary additional installation component for the second service is added to the existing class.</span></span>  
  
 <span data-ttu-id="9764b-113">V rámci instalačních programů není nutné provádět žádné speciální kódování, aby se vaše služby nainstalovaly správně.</span><span class="sxs-lookup"><span data-stu-id="9764b-113">You do not need to do any special coding within the installers to make your services install correctly.</span></span> <span data-ttu-id="9764b-114">Někdy však může být nutné změnit obsah instalačních programů, pokud potřebujete do procesu instalace přidat speciální funkce.</span><span class="sxs-lookup"><span data-stu-id="9764b-114">However, you may occasionally need to modify the contents of the installers if you need to add special functionality to the installation process.</span></span>

1. <span data-ttu-id="9764b-119">V **Průzkumník řešení**zobrazení **návrhu** přístupu ke službě, pro kterou chcete přidat instalační komponentu.</span><span class="sxs-lookup"><span data-stu-id="9764b-119">In **Solution Explorer**, access **Design** view for the service for which you want to add an installation component.</span></span>

3. <span data-ttu-id="9764b-121">V návrháři se zaostřením klikněte pravým tlačítkem myši a pak klikněte na **Přidat instalační program**.</span><span class="sxs-lookup"><span data-stu-id="9764b-121">With the designer in focus, right-click, and then click **Add Installer**.</span></span>  
  
     <span data-ttu-id="9764b-122">Nové třídy, a `ProjectInstaller` dvě instalační komponenty <xref:System.ServiceProcess.ServiceProcessInstaller> a <xref:System.ServiceProcess.ServiceInstaller> , jsou přidány do projektu a hodnoty vlastností služby jsou zkopírovány do komponent.</span><span class="sxs-lookup"><span data-stu-id="9764b-122">A new class, `ProjectInstaller`, and two installation components, <xref:System.ServiceProcess.ServiceProcessInstaller> and <xref:System.ServiceProcess.ServiceInstaller>, are added to your project, and property values for the service are copied to the components.</span></span>  
  
4. <span data-ttu-id="9764b-123">Klikněte na <xref:System.ServiceProcess.ServiceInstaller> součást a ověřte, zda <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> je hodnota vlastnosti nastavena na stejnou hodnotu jako <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> vlastnost v samotné službě.</span><span class="sxs-lookup"><span data-stu-id="9764b-123">Click the <xref:System.ServiceProcess.ServiceInstaller> component and verify that the value of the <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> property is set to the same value as the <xref:System.ServiceProcess.ServiceInstaller.ServiceName%2A> property on the service itself.</span></span>

6. <span data-ttu-id="9764b-131">Chcete-li určit kontext zabezpečení, ve kterém bude služba spuštěna, klikněte na <xref:System.ServiceProcess.ServiceProcessInstaller> součást a nastavte příslušné hodnoty vlastností.</span><span class="sxs-lookup"><span data-stu-id="9764b-131">To determine the security context in which your service will run, click the <xref:System.ServiceProcess.ServiceProcessInstaller> component and set the appropriate property values.</span></span> <span data-ttu-id="9764b-132">Další informace naleznete v tématu [How to: Určete kontext zabezpečení pro služby](how-to-specify-the-security-context-for-services.md).</span><span class="sxs-lookup"><span data-stu-id="9764b-132">For more information, see [How to: Specify the Security Context for Services](how-to-specify-the-security-context-for-services.md).</span></span>
