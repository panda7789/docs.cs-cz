## <a name="lifetime-of-collectible-assemblies"></a><span data-ttu-id="5b425-110">Doba života kolekční sestavení</span><span class="sxs-lookup"><span data-stu-id="5b425-110">Lifetime of collectible assemblies</span></span>

- <span data-ttu-id="5b425-121">Statický odkaz na `T` jiným dynamicky definovaným typem `T1`, který je stále dosažitelný spuštěním kódu.</span><span class="sxs-lookup"><span data-stu-id="5b425-121">A static reference to `T` by another dynamically defined type `T1` that is still reachable by executing code.</span></span> <span data-ttu-id="5b425-122">Například `T1` může odvozovat z `T`nebo `T` může být typ parametru v metodě `T1`.</span><span class="sxs-lookup"><span data-stu-id="5b425-122">For example, `T1` might derive from `T`, or `T` might be the type of a parameter in a method of `T1`.</span></span>
