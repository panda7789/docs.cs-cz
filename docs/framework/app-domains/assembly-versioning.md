## <a name="version-information"></a><span data-ttu-id="65b63-117">Informace o verzi</span><span class="sxs-lookup"><span data-stu-id="65b63-117">Version Information</span></span>

### <a name="assembly-version-number"></a><span data-ttu-id="65b63-122">Číslo verze sestavení</span><span class="sxs-lookup"><span data-stu-id="65b63-122">Assembly Version Number</span></span>

 <span data-ttu-id="65b63-126">\<*hlavní verze*>.\< *podverze*>.\< *číslo sestavení*>.\< *revize*></span><span class="sxs-lookup"><span data-stu-id="65b63-126">\<*major version*>.\<*minor version*>.\<*build number*>.\<*revision*></span></span>  
  
 <span data-ttu-id="65b63-127">Verze 1.5.1254.0 například označuje jako hlavní verze 5 jako vedlejší verze 1, 1254 jako číslo sestavení a 0 jako číslo revize.</span><span class="sxs-lookup"><span data-stu-id="65b63-127">For example, version 1.5.1254.0 indicates 1 as the major version, 5 as the minor version, 1254 as the build number, and 0 as the revision number.</span></span>  
  
 <span data-ttu-id="65b63-128">Číslo verze je uloženo v manifestu sestavení spolu s dalšími informacemi o identitě, včetně názvu sestavení a veřejný klíč, stejně jako informace o relacích a identit jiných sestavení spojených s aplikací.</span><span class="sxs-lookup"><span data-stu-id="65b63-128">The version number is stored in the assembly manifest along with other identity information, including the assembly name and public key, as well as information on relationships and identities of other assemblies connected with the application.</span></span>  
  
 <span data-ttu-id="65b63-129">Pokud sestavení vytvořené, vývojové nástroje záznamy informace o závislostech pro každé sestavení, která je popsána v manifestu sestavení.</span><span class="sxs-lookup"><span data-stu-id="65b63-129">When an assembly is built, the development tool records dependency information for each assembly that is referenced in the assembly manifest.</span></span> <span data-ttu-id="65b63-130">Modul runtime používá tato čísla verze ve spojení s informace o konfiguraci nastavení správce nebo aplikace, vydavatel, načtení správná verze odkazovaného sestavení.</span><span class="sxs-lookup"><span data-stu-id="65b63-130">The runtime uses these version numbers, in conjunction with configuration information set by an administrator, an application, or a publisher, to load the proper version of a referenced assembly.</span></span>
