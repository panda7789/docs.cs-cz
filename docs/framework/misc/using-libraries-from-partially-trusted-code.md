- <span data-ttu-id="ff075-115">Chcete-li zakázat automatické **LinkDemand** a zabránit vyvolání výjimky, můžete umístit **AllowPartiallyTrustedCallersAttribute** atribut na rozsah sestavení sdílené knihovny.</span><span class="sxs-lookup"><span data-stu-id="ff075-115">In order to disable the automatic **LinkDemand** and prevent the exception from being thrown, you can place the **AllowPartiallyTrustedCallersAttribute** attribute on the assembly scope of a shared library.</span></span> <span data-ttu-id="ff075-116">Tento atribut umožňuje volání knihoven z částečně důvěryhodného spravovaného kódu.</span><span class="sxs-lookup"><span data-stu-id="ff075-116">This attribute allows your libraries to be called from partially trusted managed code.</span></span>  
  
- <span data-ttu-id="ff075-117">Částečně důvěryhodný kód, kterému je udělen přístup ke knihovně s <xref:System.AppDomain>tímto atributem, stále podléhá dalším omezením definovaným programem .</span><span class="sxs-lookup"><span data-stu-id="ff075-117">Partially trusted code that is granted access to a library with this attribute is still subject to further restrictions defined by the <xref:System.AppDomain>.</span></span>

 <span data-ttu-id="ff075-119">Knihovny, které jsou soukromé pro konkrétní aplikaci, nevyžadují silný název nebo atribut **AllowPartiallyTrustedCallersAttribute** a nelze na ně odkazovat potenciálně škodlivým kódem mimo aplikaci.</span><span class="sxs-lookup"><span data-stu-id="ff075-119">Libraries that are private to a specific application do not require a strong name or the **AllowPartiallyTrustedCallersAttribute** attribute and cannot be referenced by potentially malicious code outside the application.</span></span> <span data-ttu-id="ff075-120">Takový kód je chráněn proti úmyslnému nebo neúmyslnému zneužití částečně důvěryhodným mobilním kódem, aniž by vývojář musel dělat něco navíc.</span><span class="sxs-lookup"><span data-stu-id="ff075-120">Such code is protected against intentional or unintentional misuse by partially trusted mobile code without the developer having to do anything extra.</span></span>

- <span data-ttu-id="ff075-123">Knihovny kódu se silným názvem, které jsou speciálně napsány pro částečně důvěryhodné scénáře.</span><span class="sxs-lookup"><span data-stu-id="ff075-123">Strong-named code libraries that are specifically written for partially trusted scenarios.</span></span>
