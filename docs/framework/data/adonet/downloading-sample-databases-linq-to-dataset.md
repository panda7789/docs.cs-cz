# <a name="download-sample-databases-linq-to-dataset"></a>Stažení ukázkových databází (LINQ to DataSet)

Ukázky a návody v dokumentaci LINQ to DataSet používají ukázkovou databázi AdventureWorks. Tuto ukázkovou databázi si můžete zdarma stáhnout. Přejděte do [ukázkové databáze AdventureWorks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks) a stáhněte si databázi. Potom postupujte podle pokynů na této stránce pro připojení nebo obnovení databáze.
  
## <a name="sql-server-express"></a>SQL Server Express

Ukázky a návody v části LINQ to DataSet používají sql server 2005 jako úložiště dat, ale můžete je upravit tak, aby místo nich [používaly SQL Server Express.](https://go.microsoft.com/fwlink/?linkid=866658) SQL Server Express je zdarma a můžete jej dále distribuovat s aplikacemi.
