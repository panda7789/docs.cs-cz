-   <span data-ttu-id="59b59-113">Ověřování je proces přihlášení k systému SQL Server, pomocí kterého požaduje objekt zabezpečení přístup odesláním přihlašovací údaje, které vyhodnotí server.</span><span class="sxs-lookup"><span data-stu-id="59b59-113">Authentication is the process of logging on to SQL Server by which a principal requests access by submitting credentials that the server evaluates.</span></span> <span data-ttu-id="59b59-114">Ověření určí identitu uživatele nebo proces ověřuje.</span><span class="sxs-lookup"><span data-stu-id="59b59-114">Authentication establishes the identity of the user or process being authenticated.</span></span>
