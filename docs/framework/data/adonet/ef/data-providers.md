## <a name="in-this-section"></a>V tomto oddílu  
 [Zprostředkovatel EntityClient pro Entity Framework](entityclient-provider-for-the-entity-framework.md)  
 Popisuje poskytovatele dat EntityClient. Tento zprostředkovatel transformuje dotazy na datový model na kanonický strom příkazů. Strom příkazů lze následně spotřebovat .NET Framework poskytovatel dat pro Entity Framework.  
  
 [SqlClient pro Entity Framework](sqlclient-for-the-entity-framework.md)
