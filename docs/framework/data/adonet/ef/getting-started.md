# <a name="getting-started-entity-framework"></a>Začínáme (Entity Framework)
ADO.NET Entity Framework podporuje aplikace a služby zaměřené na data a poskytuje platformu pro programování na základě dat, která vyvolává úroveň abstrakce z logické relační úrovně až po koncepční úroveň. Díky tomu, že vývojáři umožňují pracovat s daty na větší úrovni abstrakce, Entity Framework podporuje kód, který je nezávislý na konkrétním modulu úložiště dat nebo relačním schématu. Další informace najdete v tématu [přehled Entity Framework](overview.md).  
  
 Pokud chcete rychle začít používat nejnovější verzi Entity Framework, přečtěte si téma Začínáme [s Entity Framework 6](/ef/ef6/get-started).
