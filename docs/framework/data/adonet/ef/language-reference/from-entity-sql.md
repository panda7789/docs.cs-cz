[!INCLUDE[esql](../../../../../../includes/esql-md.md)] <span data-ttu-id="292a7-176">zavádí alias pro každou jednoduchou `FROM` položku klauzule v oboru.</span><span class="sxs-lookup"><span data-stu-id="292a7-176">introduces an alias for each simple `FROM` clause item in scope.</span></span> <span data-ttu-id="292a7-177">Například v následujícím fragmentu klauzule FROM, názvy zavedené do rozsahu jsou c, d a e.</span><span class="sxs-lookup"><span data-stu-id="292a7-177">For example, in the following FROM clause snippet, The names introduced into scope are c, d, and e.</span></span>
