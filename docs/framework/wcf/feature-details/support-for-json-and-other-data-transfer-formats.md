# <a name="support-for-json-and-other-data-transfer-formats"></a>Podpora formátu JSON a dalších formátů přenosu dat
JSON (JavaScript Object Notation) je efektivní formát kódování dat, který umožňuje rychlé výměny malých objemů dat mezi klientskými prohlížeči a webovými službami AJAX (asynchronní pro JavaScript a XML).

 Popisuje samostatné serializace JSON.  
  
 [Postupy: Serializace a deserializace dat protokolu JSON](how-to-serialize-and-deserialize-json-data.md)  
 Ukazuje, jak serializovat objekty typu .NET do dat zakódovaných v kódování JSON a potom deserializovat data ve formátu JSON zpátky do instancí typů .NET pomocí <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer> .  
  
 [Mapování mezi JSON a XML](mapping-between-json-and-xml.md)  
 Popisuje korespondenci mezi kódováním JavaScript Object Notation (JSON) a dokumentovou informační sadu XML, která je interně používána Windows Communication Foundation (WCF) k reprezentaci dat a dokumentů kódovaných ve formátu JSON.
