# <a name="correlation"></a>Korelace
Když aplikace služby pracovních postupů komunikují s jinými službami, je důležité, aby byly zprávy mezi nimi odesílány do příslušné instance pracovního postupu. Korelace poskytuje mechanismus pro toto. Témata v této části poskytují přehled korelace a jejich použití v různých scénářích služby pracovního postupu.
