# <a name="message-security-with-issued-tokens"></a>Zabezpeční zpráv pomocí vydaných tokenů
Klient vydaných tokenů umožňuje scénář federovaného zabezpečení ve službě Windows Communication Foundation (WCF).  
  
 Federované zabezpečení umožňuje vytvoření vztahu důvěryhodnosti mezi nezávislými doménami. Každá doména procentralizovaná ověřování a autorizaci ve službě tokenů zabezpečení. Při navázání vztahu důvěryhodnosti mezi dvě takové služby můžou členové každé domény používat služby druhé.  
  
 Podrobné vysvětlení federovaného scénáře najdete v tématu [federace a vystavené tokeny](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md).
