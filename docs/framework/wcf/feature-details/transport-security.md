# <a name="transport-security"></a>Zabezpečení přenosu
Zabezpečení přenosu v Windows Communication Foundation (WCF) závisí na vybrané vazbě. Přenos, který implementuje vazba, určuje skutečný bezpečnostní mechanismus. Témata v této části popisují implementované mechanismy a jejich možnosti.

 Vysvětluje základy zabezpečení přenosu ve službě WCF.  
  
 [Zabezpečení přenosu HTTP](http-transport-security.md)  
 Vysvětluje, jak WCF implementuje SSL (Secure Sockets Layer) (SSL nebo HTTPS).  
  
 [Princip ověřování HTTP](understanding-http-authentication.md)  
 Popisuje schémata ověřování HTTP, například Basic, Digest, NT LAN Manager (NTLM) a další.  
  
 [Použití zosobnění se zabezpečením přenosu](using-impersonation-with-transport-security.md)  
 Vysvětluje pět úrovní zosobnění, které je možné v režimu zabezpečení přenosu.  
  
 [Postupy: Konfigurace portu s certifikátem SSL](how-to-configure-a-port-with-an-ssl-certificate.md)  
 Provede vás základy konfigurace portu na počítači s certifikátem X. 509 pro zabezpečení SSL (Transport).
