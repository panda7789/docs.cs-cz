# <a name="interoperability-with-web-services-enhancements-30"></a>Interoperabilita s Web Services Enhancements 3.0
Klienti a služby Windows Communication Foundation (WCF) jsou kompatibilní s rozšířeními webových služeb 3,0 pro klienty a služby Microsoft .NET (WSE), když jsou klienti a služby WCF nakonfigurovaní tak, aby používali specifikaci WS-Addressing ve verzi ze srpna 2004.
