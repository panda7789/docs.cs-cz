# <a name="integrating-with-com-applications"></a>Integrace s aplikacemi modelu COM+
Windows Communication Foundation (WCF) poskytuje bohatou prostředí pro vytváření distribuovaných aplikací. Pokud máte významnou investici do aplikační logiky založené na komponentách, která je hostována v modelu COM+, můžete pomocí WCF prodloužit stávající logiku a nemusíte je přepisovat. Témata v této části popisují, jak používat model COM+ s WCF.
