# <a name="partial-trust"></a>Částečná důvěryhodnost

Počínaje .NET Framework 3,5 mohou částečně důvěryhodní volající přistupovat k veřejným typům a metodám implementovaným v <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>a <xref:System.ServiceModel.Web>. Tato část popisuje podporované scénáře použití Windows Communication Foundation (WCF) v částečně důvěryhodné aplikaci a také omezenou podmnožinu funkcí WCF, které jsou k dispozici pro aplikace běžící s omezeným zabezpečením přístupu kódu (CAS). nastaven.
