# <a name="partial-trust"></a>Částečná důvěryhodnost

Počínaje .NET Framework 3,5 mohou částečně důvěryhodní volající přistupovat k veřejným typům a metodám implementovaným v <xref:System.ServiceModel> , <xref:System.Runtime.Serialization> a <xref:System.ServiceModel.Web> . Tato část popisuje podporované scénáře použití Windows Communication Foundation (WCF) v částečně důvěryhodné aplikaci a také omezenou podmnožinu funkcí WCF, která je dostupná pro aplikace spuštěné s omezenými oprávněními zabezpečení přístupu kódu (CAS).

 Popisuje funkce služby WCF, které nelze použít s částečnou důvěryhodností.
