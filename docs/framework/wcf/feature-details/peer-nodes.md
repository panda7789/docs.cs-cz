# <a name="peer-nodes"></a>Partnerské uzly
Uzel PeerNode je jedním koncovým bodem v síti s rovnocenným kanálem. Aplikace může mít několik partnerských uzlů, které se účastní různých sítí. Například aplikace chat peer-to-peer může být součástí dvou oddělených sítí: jedna síť pro chatování s přáteli s vysokými školami; a další pro chatování s pracovními kolegy.
