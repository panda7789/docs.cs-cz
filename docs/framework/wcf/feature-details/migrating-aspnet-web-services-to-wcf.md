<span data-ttu-id="914eb-103">ASP.NET poskytuje .NET Framework knihovny tříd a nástroje pro vytváření webových služeb a také zařízení pro hostování služeb v rámci služby Internetová informační služba (IIS).</span><span class="sxs-lookup"><span data-stu-id="914eb-103">ASP.NET provides .NET Framework class libraries and tools for building Web services, as well as facilities for hosting services within Internet Information Services (IIS).</span></span> <span data-ttu-id="914eb-104">Windows Communication Foundation (WCF) poskytuje .NET Framework knihoven tříd, nástrojů a hostitelských prostředí pro umožnění komunikace softwarových entit pomocí libovolných protokolů, včetně těch, které používá webová služba.</span><span class="sxs-lookup"><span data-stu-id="914eb-104">Windows Communication Foundation (WCF) provides .NET Framework class libraries, tools and hosting facilities for enabling software entities to communicate using any protocols, including those used by Web services.</span></span>  <span data-ttu-id="914eb-105">Migrace webových služeb ASP.NET do WCF umožňuje vašim aplikacím využívat nové funkce a vylepšení, které jsou jedinečné pro službu WCF.</span><span class="sxs-lookup"><span data-stu-id="914eb-105">Migrating ASP.NET Web Services to WCF allows your applications to take advantage of new features and improvements that are unique to WCF.</span></span>  
  
 <span data-ttu-id="914eb-106">Služba WCF má vzhledem k ASP.NET webovým službám několik důležitých výhod.</span><span class="sxs-lookup"><span data-stu-id="914eb-106">WCF has several important advantages relative to ASP.NET Web services.</span></span> <span data-ttu-id="914eb-107">I když jsou nástroje webové služby ASP.NET výhradně pro vytváření webových služeb, služba WCF poskytuje nástroje, které lze použít, pokud je třeba, aby mohly komunikovat s jinými softwarovými entitami.</span><span class="sxs-lookup"><span data-stu-id="914eb-107">While ASP.NET Web services tools are solely for building Web services, WCF provides tools that can be used when software entities must be made to communicate with one another.</span></span> <span data-ttu-id="914eb-108">Tím se sníží počet technologií, které vývojáři potřebuje k tomu, aby pokryly různé scénáře komunikace softwaru. tím se sníží náklady na prostředky vývoje softwaru a také čas potřebný k dokončení projektů vývoje softwaru.</span><span class="sxs-lookup"><span data-stu-id="914eb-108">This will reduce the number of technologies that developers are required to know in order to accommodate different software communication scenarios, which in turn will reduce the cost of software development resources, as well as the time to complete software development projects.</span></span>  
  
 <span data-ttu-id="914eb-109">I v případě projektů vývoje webové služby podporuje WCF více protokolů webové služby než podpora webových služeb ASP.NET.</span><span class="sxs-lookup"><span data-stu-id="914eb-109">Even for Web service development projects, WCF supports more Web service protocols than ASP.NET Web services support.</span></span> <span data-ttu-id="914eb-110">Tyto další protokoly poskytují složitější řešení zahrnující mimo jiné i spolehlivé relace a transakce.</span><span class="sxs-lookup"><span data-stu-id="914eb-110">These additional protocols provide for more sophisticated solutions involving, amongst other things, reliable sessions and transactions.</span></span>  
  
 <span data-ttu-id="914eb-111">Služba WCF podporuje další protokoly pro přenos zpráv, než ASP.NET webové služby.</span><span class="sxs-lookup"><span data-stu-id="914eb-111">WCF supports more protocols for transporting messages than ASP.NET Web services.</span></span> <span data-ttu-id="914eb-112">Webové služby ASP.NET podporují posílání zpráv jenom pomocí protokolu HTTP (Hypertext Transfer Protocol).</span><span class="sxs-lookup"><span data-stu-id="914eb-112">ASP.NET Web services only support sending messages by using the Hypertext Transfer Protocol (HTTP).</span></span> <span data-ttu-id="914eb-113">Služba WCF podporuje odesílání zpráv pomocí protokolu HTTP a také protokolu TCP (Transmission Control Protocol), pojmenovaných kanálů a služby Microsoft řízení front zpráv (MSMQ).</span><span class="sxs-lookup"><span data-stu-id="914eb-113">WCF supports sending messages by using HTTP, as well as the Transmission Control Protocol (TCP), named pipes, and Microsoft Message Queuing (MSMQ).</span></span> <span data-ttu-id="914eb-114">Důležitější je, že WCF se dá rozšířit na podporu dalších přenosových protokolů.</span><span class="sxs-lookup"><span data-stu-id="914eb-114">More important, WCF can be extended to support additional transport protocols.</span></span> <span data-ttu-id="914eb-115">Software vyvinutý pomocí WCF se proto dá přizpůsobit tak, aby spolupracoval s širší škálou jiného softwaru, což zvyšuje potenciální návratnost investic.</span><span class="sxs-lookup"><span data-stu-id="914eb-115">Therefore, software developed using WCF can be adapted to work together with a wider variety of other software, thereby increasing the potential return on the investment.</span></span>  
  
 <span data-ttu-id="914eb-116">Služba WCF nabízí mnohem rozsáhlejší možnosti pro nasazování a správu aplikací než ASP.NET webové služby.</span><span class="sxs-lookup"><span data-stu-id="914eb-116">WCF provides much richer facilities for deploying and managing applications than ASP.NET Web services provides.</span></span> <span data-ttu-id="914eb-117">Kromě konfiguračního systému, který ASP.NET má také, WCF nabízí Editor konfigurací, trasování aktivity od odesílatelů a zpětně prostřednictvím libovolného počtu zprostředkovatelů, prohlížeče trasování, protokolování zpráv, rozsáhlého počtu čítačů výkonu a podporu rozhraní WMI (Windows Management Instrumentation).</span><span class="sxs-lookup"><span data-stu-id="914eb-117">In addition to a configuration system, which ASP.NET also has, WCF offers a configuration editor, activity tracing from senders to receivers and back through any number of intermediaries, a trace viewer, message logging, a vast number of performance counters, and support for Windows Management Instrumentation.</span></span>  
  
 <span data-ttu-id="914eb-118">Vzhledem k tomu, že tyto potenciální výhody WCF souvisí s ASP.NET webovými službami, pokud používáte nebo zvažujete použití webových služeb ASP.NET, máte několik možností:</span><span class="sxs-lookup"><span data-stu-id="914eb-118">Given these potential benefits of WCF relative to ASP.NET Web services, if you are using, or are considering using ASP.NET Web services you have several options:</span></span>  
  
- <span data-ttu-id="914eb-119">Můžete dál používat webové služby ASP.NET a forego výhody proffered službou WCF.</span><span class="sxs-lookup"><span data-stu-id="914eb-119">Continue to use ASP.NET Web services, and forego the benefits proffered by WCF.</span></span>  
  
- <span data-ttu-id="914eb-120">Používejte webové služby ASP.NET s úmyslem přijmout WCF v určitou dobu v budoucnu.</span><span class="sxs-lookup"><span data-stu-id="914eb-120">Keep using ASP.NET Web services with the intention of adopting WCF at some time in the future.</span></span> <span data-ttu-id="914eb-121">Témata v této části vysvětlují, jak maximalizovat potenciální zákazníky, aby mohli používat nové aplikace webové služby ASP.NET spolu s budoucími aplikacemi WCF.</span><span class="sxs-lookup"><span data-stu-id="914eb-121">The topics in this section explain how to maximize the prospects for being able to use new ASP.NET Web service applications together with future WCF applications.</span></span> <span data-ttu-id="914eb-122">Témata v této části také vysvětlují, jak vytvořit nové webové služby ASP.NET, aby je bylo snazší migrovat do WCF.</span><span class="sxs-lookup"><span data-stu-id="914eb-122">The topics in this section also explain how to build new ASP.NET Web services so as to make it easier to migrate them to WCF.</span></span> <span data-ttu-id="914eb-123">Pokud je však důležité zabezpečení služeb, nebo jsou vyžadovány spolehlivost či transakce, nebo pokud bude nutné vytvořit vlastní zařízení správy, je lepší volbou WCF přijmout.</span><span class="sxs-lookup"><span data-stu-id="914eb-123">However, if securing the services is important, or reliability or transaction assurances are required, or if custom management facilities will have to be constructed, then it is a better option to adopt WCF.</span></span> <span data-ttu-id="914eb-124">WCF je navržené pro tyto scénáře přesně.</span><span class="sxs-lookup"><span data-stu-id="914eb-124">WCF is designed for precisely such scenarios.</span></span>

- <span data-ttu-id="914eb-130">Přijme WCF a migruje stávající aplikace webové služby ASP.NET do WCF.</span><span class="sxs-lookup"><span data-stu-id="914eb-130">Adopt WCF and migrate existing ASP.NET Web service applications to WCF.</span></span> <span data-ttu-id="914eb-131">Tuto možnost můžete zvolit, chcete-li vylepšit existující aplikace pomocí funkcí poskytovaných službou WCF nebo reprodukování funkcí stávajících webových služeb ASP.NET v rámci nových, výkonnějších aplikací WCF.</span><span class="sxs-lookup"><span data-stu-id="914eb-131">You may choose this option to enhance the existing applications with features provided by WCF, or to reproduce the functionality of existing ASP.NET Web services within new, more powerful WCF applications.</span></span>
