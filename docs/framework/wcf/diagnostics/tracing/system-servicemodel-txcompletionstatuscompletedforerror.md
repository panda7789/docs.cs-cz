# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
Zadaná transakce pro zadanou operaci byla dokončena z důvodu neošetřené výjimky spuštění.  
  
## <a name="description"></a>Popis  
 Trasovat se, když dojde k chybě během pokusu o dokončení aktuální transakce. K tomu dojde před odesláním odpovědi nebo chyby volajícímu.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Zkontrolujte sledovanou zprávu pro zprávu výjimky a všechny akce, které jsou k.
