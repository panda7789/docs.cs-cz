# <a name="microsofttransactionstransactionbridgeenlistmentidentitycheckfailed"></a>Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
Identita příchozího odesílatele zprávy nesouhlasí s identitou odvozenou z reference koncového bodu zápisu.  
  
## <a name="description"></a>Popis  
 Sledováno, pokud se nezdařilo ověření zabezpečení zprávy.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Zajistěte, aby byly správně nakonfigurovány zabezpečení transakcí pro všechny počítače účastnící se transakce.
