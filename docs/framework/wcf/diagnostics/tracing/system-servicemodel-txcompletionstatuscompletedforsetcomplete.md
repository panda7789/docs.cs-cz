# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Zadaná transakce pro zadanou operaci byla dokončena z důvodu volání SetTransactionComplete.  
  
## <a name="description"></a>Popis  
 Sledováno, pokud se uživatel explicitně rozhodl dokončit aktuální transakci prostřednictvím OperationContext.
