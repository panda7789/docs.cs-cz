# <a name="systemservicemodelactivationwebhostprotocolmisconfigured"></a>System.ServiceModel.Activation.WebHostProtocolMisconfigured
Protokol je nesprávně nakonfigurovaný v aktivační službě procesů systému Windows (WAS).  
  
## <a name="description"></a>Popis  
 K této chybě dochází, pokud je transportní protokol nesprávně nakonfigurován.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Ověřte a opravte parametry konfigurace přenosu v souboru Web. config.
