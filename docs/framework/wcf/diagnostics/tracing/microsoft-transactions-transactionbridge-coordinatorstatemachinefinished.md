# <a name="microsofttransactionstransactionbridgecoordinatorstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.CoordinatorStateMachineFinished
Stavový počítač pro zařazení koordinátora přešel do stavu dokončeno.  
  
## <a name="description"></a>Popis  
 Sledováno, když místní správce transakce považuje za dokončenou 2PC zpracování. Výsledek zařazení se dá potvrdit nebo zrušit nebo zapomenout. Je také sledováno, pokud má správce místní transakce hlasy ReadOnly během přípravy.
