# <a name="systemservicemodeltxcompletionstatusremainsattached"></a>System.ServiceModel.TxCompletionStatusRemainsAttached
Zadaná transakce pro zadanou operaci zůstane připojena ke službě InstanceContext.  
  
## <a name="description"></a>Popis  
 Sledováno po dokončení ServiceOperation, ale transakce není nastavena na dokončeno.
