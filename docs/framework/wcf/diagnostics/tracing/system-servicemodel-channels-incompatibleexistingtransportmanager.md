# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Pro zadaný identifikátor URI byl nalezen stávající nekompatibilní správce přenosu.  
  
## <a name="description"></a>Popis  
 Toto informační trasování znamená, že pro naslouchací proces, který je otevřen, byla nalezena odpovídající registrace, ale stávající správce přenosu je nekompatibilní. Naslouchací proces se pokusí použít jiného správce přenosu a vyvolat výjimku, pokud je neúspěšný.
