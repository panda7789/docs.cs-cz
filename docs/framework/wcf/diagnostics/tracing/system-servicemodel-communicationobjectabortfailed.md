# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Nepovedlo se přerušit objekt komunikace.  
  
## <a name="description"></a>Popis  
 Komunikace objektu se nezdařila, protože při provádění metody Abort došlo k výjimce. Další okolí trasování může obsahovat podrobnosti o výjimce.
