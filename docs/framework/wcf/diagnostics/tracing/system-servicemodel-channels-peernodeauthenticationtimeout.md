# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Bezpečnostní Metoda handshake neuspěla v zadaném časovém limitu.  
  
## <a name="description"></a>Popis  
 K tomuto trasování dojde při pokusu o navázání zabezpečeného sousedního připojení.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Ujistěte se, že všechny sousední sousedi mají příslušná zabezpečovací pověření a reagují na všechny zprávy za méně než 1 minutu.
