# <a name="microsofttransactionstransactionbridgerecoveredcoordinatorinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredCoordinatorInvalidMetadata
Reference koncového bodu nalezená v položce protokolu obnovení koordinátora obsahovala neplatná nebo nekompatibilní metadata.  
  
## <a name="description"></a>Popis  
 Systém nemůže vytvořit proxy server pro obnovený zařazení koordinátoru.

 Pokud potíže potrvají, obraťte se na společnost Microsoft.
