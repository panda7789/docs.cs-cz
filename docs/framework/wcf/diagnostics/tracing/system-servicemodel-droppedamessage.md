# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Popis  
 Byla přijata zpráva odpovědi, ale neexistují žádné nedokončené žádosti čekající na tuto zprávu. Zpráva je vyhozena modulem runtime služby Windows Communication Foundation (WCF).  
  
 Toto trasování lze použít pro ladění ztracených zpráv.
