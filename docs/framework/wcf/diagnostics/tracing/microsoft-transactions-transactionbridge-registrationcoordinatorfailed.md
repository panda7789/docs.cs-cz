# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfailed"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
Službě protokolu WS-AT se nepodařilo odeslat do svého koordinátora registrační zprávu.  
  
## <a name="description"></a>Popis  
 Sledováno, pokud místní funkce TransactionManager není schopen registrovat se svým nadřazeným funkce TransactionManager z důvodu neschopnosti odeslat zprávu.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Kontrola zprávy trasování pro výjimku způsobující selhání odeslání zprávy
