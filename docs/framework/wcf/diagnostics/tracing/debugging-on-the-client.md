# <a name="debugging-on-the-client"></a>Ladění na klientovi
Aby uživatelé mohli pro službu WCF psát klientské aplikace, můžete [\<serviceDebug>](../../../configure-apps/file-schema/wcf/servicedebug.md) do konfiguračního souboru služby Přidat chování služby. Toto chování lze použít k publikování stránek s informacemi o správě a vrácení informací o spravované výjimce v podrobnostech o chybách protokolu SOAP vrácených klientovi.
