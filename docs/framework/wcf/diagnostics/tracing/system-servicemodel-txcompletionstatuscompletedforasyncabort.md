# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Zadaná transakce pro zadanou operaci byla dokončena z důvodu asynchronního přerušení.  
  
## <a name="description"></a>Popis  
 Aktuální transakce byla přerušena z důvodu dalšího účastníka pro přerušení, vypršení časového limitu nebo jiné chyby uvnitř účastníka transakce.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Pokud je toto přerušení neočekávané, zkontrolujte všechny systémové protokoly a určete skutečný důvod přerušení.
