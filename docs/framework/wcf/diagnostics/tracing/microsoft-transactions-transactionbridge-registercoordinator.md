# <a name="microsofttransactionstransactionbridgeregistercoordinator"></a>Microsoft.Transactions.TransactionBridge.RegisterCoordinator
Služba protokolu WS-Atomic Transaction Protocol se úspěšně zaregistrovala se svým koordinátorem pro kontrolní protokol.  
  
## <a name="description"></a>Popis  
 Sledováno, když se místní funkce TransactionManager úspěšně zaregistroval s jeho nadřazenou TransactionManageou.
