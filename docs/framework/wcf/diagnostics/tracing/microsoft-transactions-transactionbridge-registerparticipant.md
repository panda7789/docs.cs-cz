# <a name="microsofttransactionstransactionbridgeregisterparticipant"></a>Microsoft.Transactions.TransactionBridge.RegisterParticipant
Služba WS-Atomic Transaction Protocol úspěšně zaregistrovala účastníka pro kontrolní protokol.  
  
## <a name="description"></a>Popis  
 Sledováno v případě, že místní správce transakcí úspěšně potvrdí požadavek na registraci od účastníka pro protokol 2PC.
