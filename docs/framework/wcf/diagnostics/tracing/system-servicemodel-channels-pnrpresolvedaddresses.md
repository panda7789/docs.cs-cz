# <a name="systemservicemodelchannelspnrpresolvedaddresses"></a>System.ServiceModel.Channels.PnrpResolvedAddresses
Překladač PNRP dokončil operaci řešení.  
  
## <a name="description"></a>Popis  
 Toto upozornění signalizuje, že překladač PNRP nemůže najít žádné adresy. K tomuto trasování dojde během operace řešení překladačem PNRP.
