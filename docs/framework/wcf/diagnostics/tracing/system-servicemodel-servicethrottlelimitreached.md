# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Popis  
 Bylo dosaženo jednoho z omezení MaxConcurrentCalls/instance/relace a zpráva se zařadí do fronty.  
  
 Toto trasování můžete použít k úpravě hodnot omezení na základě očekávaného a skutečného zatížení v produkčním prostředí.
