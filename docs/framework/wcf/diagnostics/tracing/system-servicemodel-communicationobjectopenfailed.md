# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Nepovedlo se otevřít objekt komunikace.  
  
## <a name="description"></a>Popis  
 Nepovedlo se otevřít objekt komunikace, protože při provádění jeho otevřené metody byla vyvolána výjimka. Další okolí trasování může obsahovat podrobnosti o výjimce.
