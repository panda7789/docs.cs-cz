# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Popis  
 Deserializátor nemá žádné znalosti typu, který se má použít během deserializace. Zvažte přidání typu zadaného jako typ objektu pro vytváření do souboru XML do seznamu známých typů – například pomocí atributu KnownTypeAttribute.
