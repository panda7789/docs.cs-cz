# <a name="systemservicemodeltxsourcetxscoperequiredistransactedtransport"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactedTransport
Zadaná transakce byla pro zadanou operaci přijata z transakčního přenosu, například MSMQ.  
  
## <a name="description"></a>Popis  
 Sledováno, pokud byla transakce zpřístupněna pro uživatelský kód vytvořená samotným přenosem.
