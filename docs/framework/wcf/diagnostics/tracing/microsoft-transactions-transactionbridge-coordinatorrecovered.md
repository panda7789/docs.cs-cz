# <a name="microsofttransactionstransactionbridgecoordinatorrecovered"></a>Microsoft.Transactions.TransactionBridge.CoordinatorRecovered
Zařazení koordinátora bylo obnoveno z protokolu správce transakcí.  
  
## <a name="description"></a>Popis  
 Sledováno, když místní správce transakce úspěšně provedl obnovení pro daný zápis nadřazeného koordinátora.
