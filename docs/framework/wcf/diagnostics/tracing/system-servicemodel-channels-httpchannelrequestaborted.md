# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Odeslání zprávy požadavku prostřednictvím protokolu HTTP se nezdařilo.  
  
## <a name="description"></a>Popis  
 Toto trasování úrovně upozornění indikuje, že požadavek kanálu HTTP byl přerušen. Další informace v tomto trasování budou odhalovat důvod nebo zprávu předanou metodě Abort.
