# <a name="systemservicemodeltxsourcetxscoperequiredisattachedtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
Zadaná transakce byla přijata pro zadanou operaci z transakce InstanceContext.  
  
## <a name="description"></a>Popis  
 Sledováno v případě, že transakce zpřístupněná pro uživatelský kód byla ponechána za metodou AutoComplete = false volanou před tímto voláním.
