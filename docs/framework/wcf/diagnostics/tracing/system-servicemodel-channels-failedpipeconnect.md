# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Pokus o připojení ke koncovému bodu pojmenovaného kanálu se nezdařil. V zadaném časovém limitu se provede další pokus.  
  
## <a name="description"></a>Popis  
 Toto informační trasování indikuje selhání připojení ke koncovému bodu pojmenovaného kanálu. K tomu může dojít, pokud koncový bod pojmenovaného kanálu nebyl nalezen nebo je zaneprázdněný. Dojde k dalším pokusům, které jsou odděleny krátkou dobu, dokud jeden neuspěje nebo OpenTimeout nevyprší.
