# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Pokus o otevření sousedního připojení se nezdařil.  
  
## <a name="description"></a>Popis  
 K tomuto trasování dochází po pokusu o navázání sousedního připojení. Možné důvody mohou zahrnovat nereagující překladač, nesprávné nastavení vazby nebo přechodné chyby sítě.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Ujistěte se, že oba uzly mají platné adresy a jsou správně nakonfigurovány.
