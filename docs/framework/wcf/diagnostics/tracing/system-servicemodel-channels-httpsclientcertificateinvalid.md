# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certifikát klienta je neplatný.  
  
## <a name="description"></a>Popis  
 Toto trasování indikuje, že certifikát, který poskytl klient, byl nalezen jako neplatný naslouchací proces HTTPS. Naslouchací proces HTTPS se pokusil ověřit pravost klienta pomocí tohoto certifikátu. Certifikát může být neplatný, pokud jeho certifikační autorita není rozpoznána serverem hostujícím službu.
