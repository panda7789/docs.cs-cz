# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Rovnocenný soused nebyl přijat.  
  
## <a name="description"></a>Popis  
 K tomuto trasování dochází při zpracování připojení k sousednímu uzlu. Mezi možné příčiny patří neshoda nastavení zabezpečení, překročení časového limitu, sousední směrovač, který nemůže přijmout připojení nebo přechodné problémy se sítí.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Ověřte neshodná nastavení zabezpečení, překročení časového limitu, neschopnost souseda přijmout připojení nebo přechodné problémy se sítí.
