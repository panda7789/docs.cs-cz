# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Certifikát klienta je povinný. V požadavku nebyl nalezen žádný certifikát.  
  
## <a name="description"></a>Popis  
 Toto trasování indikuje, že naslouchací proces HTTPS přijal požadavek HTTPS, který nebyl přidružen k klientskému certifikátu. Vzhledem k tomu, že naslouchací proces byl nakonfigurován tak, aby vyžadoval klientské certifikáty u všech požadavků HTTPS, naslouchací proces nemohl ověřit pravost klienta.
