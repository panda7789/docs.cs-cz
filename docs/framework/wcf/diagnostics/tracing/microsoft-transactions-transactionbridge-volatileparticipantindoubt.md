# <a name="microsofttransactionstransactionbridgevolatileparticipantindoubt"></a>Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
Služba protokolu WS-AT obdržela zprávu o přípravě nebo přehrání z nerozpoznaného účastníka. Účastník vrátil chybu, deklaruje, že výsledek transakce je nejistý.  
  
## <a name="description"></a>Popis  
 Sledováno, když místní správce transakce obdrží přípravnou nebo přepracovanou zprávu z nestálého zařazení, který již byl zapomenut.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Prozkoumejte možné příčiny zpožděných zpráv přicházejících od účastníka.
