# <a name="microsofttransactionstransactionbridgecreatetransactionfailure"></a>Microsoft.Transactions.TransactionBridge.CreateTransactionFailure
Transakci se nepovedlo vytvořit.  
  
## <a name="description"></a>Popis  
 Toto trasování se vygeneruje, když MSDTC nemůže vytvořit transakci. Příčinou může být nedostatek prostředků, nedostatečné místo v protokolu nebo jiné chyby.  
  
## <a name="troubleshooting"></a>Řešení potíží  
 Zkontrolujte řetězec stavu v rámci zprávy trasování a určete, zda existuje nějaká položka, která je k dispozici.
