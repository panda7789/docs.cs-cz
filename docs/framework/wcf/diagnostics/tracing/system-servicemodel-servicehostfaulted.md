System.ServiceModel.ServiceHostFaulted  
  
## <a name="description"></a>Popis  
 Došlo k chybě hostitele ServiceHost.  
  
 Toto trasování indikuje, že u jednoho z posluchačů došlo k chybě. Toto trasování je užitečné při ladění.
