# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
Konkrétní transakce byla dokončena při zavření relace.  
  
## <a name="description"></a>Popis  
 Sledováno v případě, že aktuální aktivní relace byla zavřena a `TransactionAutoCompleteOnSessionClose` je nastavena na `true` .
