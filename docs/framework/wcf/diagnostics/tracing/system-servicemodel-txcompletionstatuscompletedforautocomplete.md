# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
Zadaná transakce pro zadanou operaci byla dokončena, protože `OperationBehaviorAttribute.TransactionAutoComplete` člen je nastaven na hodnotu `true` .  
  
## <a name="description"></a>Popis  
 Sledováno, když infrastruktura automaticky dokončí transakci jménem uživatele.
