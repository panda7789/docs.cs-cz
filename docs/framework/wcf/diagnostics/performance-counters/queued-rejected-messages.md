# <a name="queued-rejected-messages"></a>Odmítnuté zprávy zařazené do fronty
Název čítače: Zprávy zařazené do fronty byly odmítnuty.  
  
## <a name="description"></a>Popis  
 Počet zpráv, které jsou odmítnuty přenosem ve frontě v této službě.  
  
 Další informace o tom, kdy jsou zprávy odmítnuty, naleznete v [tématu Zpracování zpráv poison](../../feature-details/poison-message-handling.md).
