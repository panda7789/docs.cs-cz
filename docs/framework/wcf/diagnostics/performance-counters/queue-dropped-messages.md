# <a name="queue-dropped-messages"></a>Vynechané zprávy fronty
Název čítače: Zprávy zařazené do fronty byly vynechány.  
  
## <a name="description"></a>Popis  
 Počet zpráv, které jsou vynechány ve frontě přenosu v této službě.  
  
 Další informace o tom, kdy jsou zprávy vynechány, naleznete v [tématu Zpracování poškozená zpráva](../../feature-details/poison-message-handling.md).
