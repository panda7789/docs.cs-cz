# <a name="queued-poison-messages"></a>Počet poškozených zpráv
Název čítače: Poškozená zpráva zařazená do fronty.  
  
## <a name="description"></a>Popis  
 Počet zpráv, které jsou označeny poisoned ve frontě přenosu v této službě.  
  
 Další informace o tom, kdy jsou zprávy označeny jako otrávené, naleznete v [tématu Zpracování zpráv poison](../../feature-details/poison-message-handling.md).
