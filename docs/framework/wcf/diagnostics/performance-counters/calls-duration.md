# <a name="calls-duration"></a>Doba volání
Název čítače: Doba trvání volání  
  
## <a name="description"></a>Popis  
 Průměrná doba trvání volání této operace. Průměrná doba trvání se vypočítá na základě této rovnice: (N1-N0)/(D1-D0).  
  
> [!WARNING]
> Při použití na asynchronní WCF služby volání duration čítač vždy vrátí -1.
