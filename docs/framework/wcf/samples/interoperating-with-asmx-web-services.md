 <span data-ttu-id="f46db-105">Tato ukázka se skládá z programu klientské konzole (.exe) a knihovny služeb (.dll) hostované Internetovou informační službou (IIS).</span><span class="sxs-lookup"><span data-stu-id="f46db-105">This sample consists of a client console program (.exe) and a service library (.dll) hosted by Internet Information Services (IIS).</span></span> <span data-ttu-id="f46db-106">Služba je webová služba ASMX, která implementuje smlouvu, která definuje vzor komunikace požadavek odpověď.</span><span class="sxs-lookup"><span data-stu-id="f46db-106">The service is an ASMX Web Service that implements a contract that defines a request-reply communication pattern.</span></span> <span data-ttu-id="f46db-107">Služba zpřístupňuje matematické`Add` `Subtract`operace `Multiply`( `Divide`, , , a ).</span><span class="sxs-lookup"><span data-stu-id="f46db-107">The service exposes math operations (`Add`, `Subtract`, `Multiply`, and `Divide`).</span></span> <span data-ttu-id="f46db-108">Klient provádí synchronní požadavky na matematické operace a služby odpovědi s výsledkem.</span><span class="sxs-lookup"><span data-stu-id="f46db-108">The client makes synchronous requests to a math operation and the service replies with the result.</span></span> <span data-ttu-id="f46db-109">Aktivita klienta je viditelná v okně konzoly.</span><span class="sxs-lookup"><span data-stu-id="f46db-109">Client activity is visible in the console window.</span></span>  
  
 <span data-ttu-id="f46db-110">Implementace webové služby ASMX zobrazená v následujícím ukázkovém kódu vypočítá a vrátí příslušný výsledek.</span><span class="sxs-lookup"><span data-stu-id="f46db-110">The ASMX Web service implementation shown in the following sample code calculates and returns the appropriate result.</span></span>  
  
```csharp  
[WebService(Namespace="http://Microsoft.ServiceModel.Samples")]  
public class CalculatorService : System.Web.Services.WebService  
    {  
        [WebMethod]  
        public double Add(double n1, double n2)  
        {  
            return n1 + n2;  
        }  
        [WebMethod]  
        public double Subtract(double n1, double n2)  
        {  
            return n1 - n2;  
        }  
        [WebMethod]  
        public double Multiply(double n1, double n2)  
        {  
            return n1 * n2;  
        }  
        [WebMethod]  
        public double Divide(double n1, double n2)  
        {  
            return n1 / n2;  
        }  
    }  
```  
  
 <span data-ttu-id="f46db-111">Jak je nakonfigurováno, služba `http://localhost/servicemodelsamples/service.asmx` je přístupná klientem ve stejném počítači.</span><span class="sxs-lookup"><span data-stu-id="f46db-111">As configured, the service can be accessed at `http://localhost/servicemodelsamples/service.asmx` by a client on the same machine.</span></span> <span data-ttu-id="f46db-112">Pro klienty na vzdálených počítačích pro přístup ke službě musí být zadán kvalifikovaný název domény namísto localhost.</span><span class="sxs-lookup"><span data-stu-id="f46db-112">For clients on remote machines to access the service, a qualified domain name must be specified instead of localhost.</span></span>  
  
 <span data-ttu-id="f46db-113">Komunikace se provádí prostřednictvím klienta generované [servicemodel metadata utility tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span><span class="sxs-lookup"><span data-stu-id="f46db-113">Communication is done through a client generated by the [ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).</span></span> <span data-ttu-id="f46db-114">Klient je obsažen v souboru generatedClient.cs.</span><span class="sxs-lookup"><span data-stu-id="f46db-114">The client is contained in the file generatedClient.cs.</span></span> <span data-ttu-id="f46db-115">Služba ASMX musí být k dispozici pro generování proxy kódu, protože se používá k načtení aktualizovaných metadat.</span><span class="sxs-lookup"><span data-stu-id="f46db-115">The ASMX service must be available to generate the proxy code, because it is used to retrieve the updated metadata.</span></span> <span data-ttu-id="f46db-116">Spusťte následující příkaz z příkazového řádku v adresáři klienta a vygenerujte zadaný proxy server.</span><span class="sxs-lookup"><span data-stu-id="f46db-116">Run the following command from a command prompt in the client directory to generate the typed proxy.</span></span>  
  
```console  
svcutil.exe /n:http://Microsoft.ServiceModel.Samples,Microsoft.ServiceModel.Samples http://localhost/servicemodelsamples/service.svc?wsdl /out:generatedClient.cs  
```  
  
 <span data-ttu-id="f46db-117">Pomocí generovaného klienta můžete získat přístup ke koncovému bodu služby konfigurací příslušné adresy a vazby.</span><span class="sxs-lookup"><span data-stu-id="f46db-117">By using the generated client, you can access a service endpoint by configuring the appropriate address and binding.</span></span> <span data-ttu-id="f46db-118">Podobně jako služba klient používá konfigurační soubor (App.config) k určení koncového bodu pro komunikaci.</span><span class="sxs-lookup"><span data-stu-id="f46db-118">Like the service, the client uses a configuration file (App.config) to specify the endpoint to communicate with.</span></span> <span data-ttu-id="f46db-119">Konfigurace koncového bodu klienta se skládá z absolutní adresy pro koncový bod služby, vazbu a smlouvu, jak je znázorněno v následující ukázce konfigurace.</span><span class="sxs-lookup"><span data-stu-id="f46db-119">The client endpoint configuration consists of an absolute address for the service endpoint, the binding, and the contract, as shown in the following sample configuration.</span></span>  
  
```xml  
<client>  
   <endpoint
      address="http://localhost/ServiceModelSamples/service.asmx"
      binding="basicHttpBinding"
      contract="Microsoft.ServiceModel.Samples.CalculatorServiceSoap" />  
</client>  
```  
  
 <span data-ttu-id="f46db-120">Implementace klienta vytvoří instanci generovaného klienta.</span><span class="sxs-lookup"><span data-stu-id="f46db-120">The client implementation constructs an instance of the generated client.</span></span> <span data-ttu-id="f46db-121">Vygenerovaný klient pak lze ke komunikaci se službou.</span><span class="sxs-lookup"><span data-stu-id="f46db-121">The generated client can then be used to communicate with the service.</span></span>  
  
```csharp  
// Create a client.  
CalculatorServiceSoapClient client = new CalculatorServiceSoapClient();  
  
// Call the Add service operation.  
double value1 = 100.00D;  
double value2 = 15.99D;  
double result = client.Add(value1, value2);  
Console.WriteLine("Add({0},{1}) = {2}", value1, value2, result);  
  
// Call the Subtract service operation.  
value1 = 145.00D;  
value2 = 76.54D;  
result = client.Subtract(value1, value2);  
Console.WriteLine("Subtract({0},{1}) = {2}", value1, value2, result);  
  
// Call the Multiply service operation.  
value1 = 9.00D;  
value2 = 81.25D;  
result = client.Multiply(value1, value2);  
Console.WriteLine("Multiply({0},{1}) = {2}", value1, value2, result);  
  
// Call the Divide service operation.  
value1 = 22.00D;  
value2 = 7.00D;  
result = client.Divide(value1, value2);  
Console.WriteLine("Divide({0},{1}) = {2}", value1, value2, result);  
  
//Closing the client gracefully closes the connection and cleans up resources.  
client.Close();  
  
Console.WriteLine();  
Console.WriteLine("Press <ENTER> to terminate client.");  
Console.ReadLine();  
```
