# <a name="uritemplate-sample"></a>Ukázka UriTemplate
Třída <xref:System.UriTemplate> poskytuje metody pro práci s sadami IDENTIFIKÁTORŮ, které sdílejí společnou strukturu. Tento vzorek ukazuje následující klíčové pojmy `UriTemplate`týkající se :  
  
- Syntaxe pro vytváření šablon.  
  
- Vytváření instanci identifikátorů `UriTemplate` <xref:System.UriTemplate.BindByName%2A> URI <xref:System.UriTemplate.BindByPosition%2A>z using a .  
  
- <xref:System.UriTemplateTable.Match%2A>, což je inverzní provoz `BindByName` a `BindByPosition`.
