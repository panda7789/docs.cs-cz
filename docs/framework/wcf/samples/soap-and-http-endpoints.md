 <span data-ttu-id="4117a-121">Klientský projekt ukazuje přístupu ke službě pomocí proxy serveru SOAP (vygenerované pomocí **přidat odkaz na službu**) a přístup pomocí služby <xref:System.Net.WebClient>.</span><span class="sxs-lookup"><span data-stu-id="4117a-121">The client project demonstrates accessing the service using a SOAP proxy (generated through **Add Service Reference**) and accessing the service using <xref:System.Net.WebClient>.</span></span>
