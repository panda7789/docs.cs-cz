# <a name="behavior-security"></a>Zabezpečení chování
V této části najdete ukázky, které ukazují konfiguraci zabezpečení pro chování služby.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Chování auditování služby](service-auditing-behavior.md)  
 Tato ukázka předvádí, jak použít <xref:System.ServiceModel.Description.ServiceSecurityAuditBehavior> k povolení auditování událostí zabezpečení během operací služby.  
  
 [Členství a poskytovatel rolí](membership-and-role-provider.md)  
 Tato ukázka předvádí, jak může služba používat členství v ASP.NET a poskytovatele rolí k ověřování a autorizaci klientů.  
  
 [Autorizace přístupu k operacím služby](authorizing-access-to-service-operations.md)  
 Tato ukázka předvádí, jak použít [\<serviceAuthorization>](../../configure-apps/file-schema/wcf/serviceauthorization-element.md) k povolení použití <xref:System.Security.Permissions.PrincipalPermissionAttribute> atributu k autorizaci přístupu k operacím služby.  
  
 [Zosobnění klienta](impersonating-the-client.md)  
 Tato ukázka předvádí, jak zosobnit volající aplikaci ve službě, aby služba mohla přistupovat k systémovým prostředkům jménem volajícího.
