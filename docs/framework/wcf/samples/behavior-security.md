# <a name="behavior-security"></a>Zabezpečení chování
Tato část obsahuje ukázky, které předvádějí konfiguraci zabezpečení pro chování služby.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Chování při auditování služby](../../../../docs/framework/wcf/samples/service-auditing-behavior.md)  
 Tato ukázka předvádí, jak používat <xref:System.ServiceModel.Description.ServiceSecurityAuditBehavior> povolení auditování událostí zabezpečení během operací služby.  
  
 [Členství a zprostředkovatel rolí](../../../../docs/framework/wcf/samples/membership-and-role-provider.md)  
 Tato ukázka předvádí, jak můžete použít službu [!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)] zprostředkovatele členství a rolí k ověřování a autorizaci klientů.  
  
 [Autorizace přístupu k operacím služby](../../../../docs/framework/wcf/samples/authorizing-access-to-service-operations.md)  
 Tato ukázka předvádí, jak používat [ \<serviceAuthorization >](../../../../docs/framework/configure-apps/file-schema/wcf/serviceauthorization-element.md) povolit používání <xref:System.Security.Permissions.PrincipalPermissionAttribute> atribut k autorizaci přístupu k operacím služby.  
  
 [Zosobnění klienta](../../../../docs/framework/wcf/samples/impersonating-the-client.md)  
 Tento příklad ukazuje, jak zosobnit volající aplikace na službu tak, aby službě můžete získat přístup k systémovým prostředkům jménem volající.
