# <a name="behaviors"></a>Chování
Tato část obsahuje ukázky, které ukazují chování služby Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Souběžnost](concurrency.md)  
 Ukazuje použití <xref:System.ServiceModel.ServiceBehaviorAttribute> s <xref:System.ServiceModel.ConcurrencyMode> výčtem, který určuje, zda instance služby zpracovává zprávy sekvenčně nebo souběžně.  
  
 [Výchozí chování služby](default-service-behavior.md)  
 Ukazuje, jak lze nakonfigurovat nastavení chování služby.  
  
 [Vytváření instancí](instancing.md)  
 Ukazuje nastavení chování vytváření instancí, které určuje, jak se vytvářejí instance třídy služby v reakci na požadavky klientů.  
  
 [Chování publikování metadat](metadata-publishing-behavior.md)  
 Ukazuje, jak řídit funkce publikování metadat služby.  
  
 [Chování transakce služby](service-transaction-behavior.md)  
 Ukazuje použití transakce s koordinovaným klientem a nastavení <xref:System.ServiceModel.ServiceBehaviorAttribute> a <xref:System.ServiceModel.OperationBehaviorAttribute> k řízení chování transakce služby.  
  
 [Chování ladění služby](service-debug-behavior.md)  
 Ukazuje, jak lze nakonfigurovat nastavení chování ladění služby.  
  
 [Omezování](throttling.md)  
 Ukazuje použití ovládacích prvků omezování.  
  
 [Zabezpečení chování](behavior-security.md)  
 Ukazuje konfiguraci zabezpečení chování služby.
