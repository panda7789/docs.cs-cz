 <span data-ttu-id="1a14d-108">Ve službě operace načte pole celá čísla předaná ve vstupní zprávě a pak vypočítá součet.</span><span class="sxs-lookup"><span data-stu-id="1a14d-108">On the service, the operation retrieves the array of integers passed in the input message and then computes the sum.</span></span> <span data-ttu-id="1a14d-109">Chcete-li odeslat zprávu odpovědi, ukázka vytvoří novou zprávu s příslušnou verzí zprávy a akce a přidá vypočítaný součet jako jeho tělo.</span><span class="sxs-lookup"><span data-stu-id="1a14d-109">To send a response message, the sample creates a new message with the appropriate message version and Action and adds the computed sum as its body.</span></span> <span data-ttu-id="1a14d-110">Následující ukázkový kód to ukazuje.</span><span class="sxs-lookup"><span data-stu-id="1a14d-110">The following sample code demonstrates this.</span></span>  
  
```csharp
public Message ComputeSum(Message request)  
{  
    //The body of the message contains a list of numbers which will be
    //read as a int[] using GetBody<T>  
    int result = 0;  
  
    int[] inputs = request.GetBody<int[]>();  
    foreach (int i in inputs)  
    {  
        result += i;  
    }  
  
    Message response = Message.CreateMessage(request.Version,
                                      ReplyAction, result);  
    return response;  
}  
```

verzi zprávy naplněný v jeho vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="1a14d-113">Thus, it creates a new <xref:System.ServiceModel.OperationContextScope> scoped to the proxy data-ttu-id="1a14d-114">Klient předá vstupní pole jako tělo zprávy požadavku a `ComputeSum` potom vyvolá na proxy serveru.</span><span class="sxs-lookup"><span data-stu-id="1a14d-114">The client passes an input array as the body to the request message and then invokes the `ComputeSum` on the proxy.</span></span> <span data-ttu-id="1a14d-115">Klient pak načte součet vstupů, které předal `GetBody<T>` přístupem k metodě na odpovědi.</span><span class="sxs-lookup"><span data-stu-id="1a14d-115">The client then retrieves the sum of the inputs it passed by accessing the `GetBody<T>` method demonstrates this.</span></span>  
  
```csharp
using (new OperationContextScope(client.InnerChannel))  
{  
    // Call the Sum service operation.  
    int[] values = { 1, 2, 3, 4, 5 };  
    Message request = Message.CreateMessage(  
        OperationContext.Current.OutgoingMessageHeaders.MessageVersion,
        RequestAction, values);  
    Message reply = client.ComputeSum(request);  
    int response = reply.GetBody<int>();  
  
    Console.WriteLine("Sum of numbers passed (1,2,3,4,5) = {0}",
                                                       response);  
}  
```
