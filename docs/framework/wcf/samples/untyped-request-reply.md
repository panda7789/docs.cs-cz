```  
Message ComputeSum(Message request);  
```  
  
 <span data-ttu-id="6a853-108">Ve službě tato operace načte pole celých čísel předaný vstupní zprávy a pak vypočítá součet.</span><span class="sxs-lookup"><span data-stu-id="6a853-108">On the service, the operation retrieves the array of integers passed in the input message and then computes the sum.</span></span> <span data-ttu-id="6a853-109">Odeslat zprávu odpovědi, ukázka vytvoří novou zprávu s verzí odpovídající zprávu a akce a přidá vypočítaný součet jako jeho obsah.</span><span class="sxs-lookup"><span data-stu-id="6a853-109">To send a response message, the sample creates a new message with the appropriate message version and Action and adds the computed sum as its body.</span></span> <span data-ttu-id="6a853-110">Následující ukázkový kód ukazuje to.</span><span class="sxs-lookup"><span data-stu-id="6a853-110">The following sample code demonstrates this.</span></span>  
  
```  
public Message ComputeSum(Message request)  
{  
    //The body of the message contains a list of numbers which will be   
    //read as a int[] using GetBody<T>  
    int result = 0;  
  
    int[] inputs = request.GetBody<int[]>();  
    foreach (int i in inputs)  
    {  
        result += i;  
    }  
  
    Message response = Message.CreateMessage(request.Version,   
                                      ReplyAction, result);  
    return response;  
}  
```  
  
 <span data-ttu-id="6a853-111">Klient používá kód, který je generován [ServiceModel Metadata Utility Tool data-stu-id="6a853-114">The client passes an input array as the body to the request message and then invokes the `ComputeSum` on the proxy.</span></span> <span data-ttu-id="6a853-115">Klient pak načte součet vstupy jí předán díky přístupu `GetBody<T>` metoda u zprávy s odpovědí.</span><span class="sxs-lookup"><span data-stu-id="6a853-115">The client then retrieves the sum of the inputs it passed by accessing the `GetBody<T>` method on the reply message.</span></span> <span data-ttu-id="6a853-116">Následující ukázkový kód ukazuje to.</span><span class="sxs-lookup"><span data-stu-id="6a853-116">The following sample code demonstrates this.</span></span>  
  
```  
using (new OperationContextScope(client.InnerChannel))  
{  
    // Call the Sum service operation.  
    int[] values = { 1, 2, 3, 4, 5 };  
    Message request = Message.CreateMessage(  
        OperationContext.Current.OutgoingMessageHeaders.MessageVersion,   
        RequestAction, values);  
    Message reply = client.ComputeSum(request);  
    int response = reply.GetBody<int>();  
  
    Console.WriteLine("Sum of numbers passed (1,2,3,4,5) = {0}",   
                                                       response);  
}  
```
