<span data-ttu-id="b292a-103">Tato ukázka předvádí, jak implementovat službu RPC založenou na protokolu HTTP, která je oblíbená jako "POX" (obyčejná stará XML) – pomocí programovacího modelu REST služby Windows Communication Foundation (WCF).</span><span class="sxs-lookup"><span data-stu-id="b292a-103">This sample demonstrates how to implement an HTTP-based, RPC-based service - popularly referred to as "POX" (Plain Old XML) service – using the Windows Communication Foundation (WCF) REST Programming model.</span></span> <span data-ttu-id="b292a-104">Tato ukázka se skládá ze dvou součástí: Samoobslužná služba WCF HTTP (Service.cs) a Konzolová aplikace (Program.cs), která vytvoří službu a provede na ni volání.</span><span class="sxs-lookup"><span data-stu-id="b292a-104">This sample consists of two components: a self-hosted WCF HTTP service (Service.cs) and a console application (Program.cs) that creates the service and makes calls to it.</span></span>

## <a name="sample-details"></a><span data-ttu-id="b292a-105">Podrobnosti ukázky</span><span class="sxs-lookup"><span data-stu-id="b292a-105">Sample Details</span></span>

<span data-ttu-id="b292a-106">Služba WCF zveřejňuje 2 operace `EchoWithGet` a `EchoWithPost`, která vrací řetězec, který byl předán jako vstup.</span><span class="sxs-lookup"><span data-stu-id="b292a-106">The WCF service exposes 2 operations, `EchoWithGet` and `EchoWithPost`, which returns the string that was passed as input.</span></span>

<span data-ttu-id="b292a-107">Operace je opatřena poznámkou <xref:System.ServiceModel.Web.WebGetAttribute>, která indikuje, že operace zpracovává `GET` požadavky HTTP. `EchoWithGet`</span><span class="sxs-lookup"><span data-stu-id="b292a-107">The `EchoWithGet` operation is annotated with <xref:System.ServiceModel.Web.WebGetAttribute>, which indicates that the operation processes HTTP `GET` requests.</span></span> <span data-ttu-id="b292a-108">Vzhledem k <xref:System.ServiceModel.Web.WebGetAttribute> tomu <xref:System.UriTemplate>, že explicitně nespecifikuje, operace očekává, že vstupní řetězec bude předán pomocí parametru řetězce dotazu s názvem `s`.</span><span class="sxs-lookup"><span data-stu-id="b292a-108">Because the <xref:System.ServiceModel.Web.WebGetAttribute> does not explicitly specify a <xref:System.UriTemplate>, the operation expects the input string to be passed in using a query string parameter with name `s`.</span></span> <span data-ttu-id="b292a-109">Všimněte si, že formát identifikátoru URI, který služba očekává, lze přizpůsobit pomocí <xref:System.ServiceModel.Web.WebGetAttribute.UriTemplate%2A> vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="b292a-109">Note that the format of the URI that the service expects can be customized using the <xref:System.ServiceModel.Web.WebGetAttribute.UriTemplate%2A> property.</span></span>

<span data-ttu-id="b292a-110">Tato `EchoWithPost` operace je opatřena poznámkami <xref:System.ServiceModel.Web.WebInvokeAttribute>, což znamená, že se `GET` nejedná o operaci (má vedlejší účinky).</span><span class="sxs-lookup"><span data-stu-id="b292a-110">The `EchoWithPost` operation is annotated with <xref:System.ServiceModel.Web.WebInvokeAttribute>, which indicates it is not a `GET` operation (it has side effects).</span></span> <span data-ttu-id="b292a-111">Vzhledem k <xref:System.ServiceModel.Web.WebInvokeAttribute> tomu `Method`, že explicitně nespecifikuje, operace zpracovává `POST` požadavky HTTP, které mají řetězec v textu požadavku (například ve formátu XML, například).</span><span class="sxs-lookup"><span data-stu-id="b292a-111">Because the <xref:System.ServiceModel.Web.WebInvokeAttribute> does not explicitly specify a `Method`, the operation processes HTTP `POST` requests that have the string in the request body (in the XML format, for instance).</span></span> <span data-ttu-id="b292a-112">Všimněte si, že metoda HTTP a formát identifikátoru URI pro požadavek lze přizpůsobit pomocí <xref:System.ServiceModel.Web.WebInvokeAttribute.Method%2A> vlastností a. <xref:System.ServiceModel.Web.WebInvokeAttribute.UriTemplate></span><span class="sxs-lookup"><span data-stu-id="b292a-112">Note that the HTTP method and the format of the URI for the request can be customized using the <xref:System.ServiceModel.Web.WebInvokeAttribute.Method%2A> and <xref:System.ServiceModel.Web.WebInvokeAttribute.UriTemplate> properties respectively.</span></span>

<span data-ttu-id="b292a-113">Soubor App. config nakonfiguruje službu WCF s výchozím <xref:System.ServiceModel.Description.WebHttpEndpoint> nastavením, které <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> má vlastnost nastavenou na `true`.</span><span class="sxs-lookup"><span data-stu-id="b292a-113">The App.config file configures the WCF service with a default <xref:System.ServiceModel.Description.WebHttpEndpoint> that has the <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> property set to `true`.</span></span> <span data-ttu-id="b292a-114">Výsledkem je, že infrastruktura WCF vytvoří na `http://localhost:8000/Customers/help` stránce pro automatickou nápovědu založenou na HTML, která poskytuje informace o tom, jak vytvořit požadavky HTTP ke službě a jak využívat odpověď HTTP služby.</span><span class="sxs-lookup"><span data-stu-id="b292a-114">As a result, the WCF infrastructure creates an automatic HTML based help page at `http://localhost:8000/Customers/help` that provides information about how to construct HTTP requests to the service and how to consume the service’s HTTP response.</span></span>

<span data-ttu-id="b292a-115">Program.cs ukazuje, jak lze pomocí objektu pro vytváření kanálů WCF volat služby a zpracovávat odpovědi.</span><span class="sxs-lookup"><span data-stu-id="b292a-115">Program.cs demonstrates how a WCF channel factory can be used to make calls to the service and process responses.</span></span> <span data-ttu-id="b292a-116">Všimněte si, že toto je pouze jeden způsob, jak získat přístup ke službě WCF.</span><span class="sxs-lookup"><span data-stu-id="b292a-116">Note that this is just one way to access a WCF service.</span></span> <span data-ttu-id="b292a-117">Ke službě je také možné přistupovat pomocí jiných tříd .NET Framework, jako <xref:System.Net.HttpWebRequest> jsou a. <xref:System.Net.WebClient></span><span class="sxs-lookup"><span data-stu-id="b292a-117">It is also possible to access the service using other .NET Framework classes like <xref:System.Net.HttpWebRequest> and <xref:System.Net.WebClient>.</span></span>

<span data-ttu-id="b292a-118">Ukázka se skládá ze samoobslužné služby a klienta spouštěného v konzolové aplikaci.</span><span class="sxs-lookup"><span data-stu-id="b292a-118">The sample consists a self-hosted service and a client that both run within a console application.</span></span> <span data-ttu-id="b292a-119">Při spuštění konzolové aplikace klient provede požadavky na službu a zapíše příslušné informace z odpovědí do okna konzoly.</span><span class="sxs-lookup"><span data-stu-id="b292a-119">As the console application runs, the client makes requests to the service and writes the pertinent information from the responses to the console window.</span></span>
