## <a name="remarks"></a><span data-ttu-id="e5d70-130">Poznámky</span><span class="sxs-lookup"><span data-stu-id="e5d70-130">Remarks</span></span>

<span data-ttu-id="e5d70-136">Namísto použití nastavení konfiguračního souboru můžete povolit starší 64 kompilátor JIT dvěma způsoby:</span><span class="sxs-lookup"><span data-stu-id="e5d70-136">Instead of using a configuration file setting, you can enable the legacy 64-bit JIT compiler in two other ways:</span></span>

  <span data-ttu-id="e5d70-148">Přidání hodnoty do klíče `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework` má vliv na všechny aplikace spuštěné v počítači.</span><span class="sxs-lookup"><span data-stu-id="e5d70-148">Adding the value to the `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework` key affects all apps running on the machine.</span></span> <span data-ttu-id="e5d70-149">Přidání hodnoty do klíče `HKEY_CURRENT_USER\SOFTWARE\Microsoft\.NETFramework` má vliv na všechny aplikace spuštěné aktuálním uživatelem.</span><span class="sxs-lookup"><span data-stu-id="e5d70-149">Adding the value to the `HKEY_CURRENT_USER\SOFTWARE\Microsoft\.NETFramework` key affects all apps run by the current user.</span></span> <span data-ttu-id="e5d70-150">Pokud je počítač nakonfigurovaný s několika uživatelskými účty, ovlivní se jenom aplikace spuštěné aktuálním uživatelem, pokud se hodnota nepřidá do klíčů registru pro ostatní uživatele.</span><span class="sxs-lookup"><span data-stu-id="e5d70-150">If a machine is configured with multiple user accounts, only apps run by the current user are affected, unless the value is added to the registry keys for other users as well.</span></span> <span data-ttu-id="e5d70-151">Přidání prvku `<useLegacyJit>` do konfiguračního souboru přepíše nastavení registru, pokud jsou k dispozici.</span><span class="sxs-lookup"><span data-stu-id="e5d70-151">Adding the `<useLegacyJit>` element to a configuration file overrides the registry settings, if they're present.</span></span>
