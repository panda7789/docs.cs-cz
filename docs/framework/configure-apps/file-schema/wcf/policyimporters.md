# \<policyImporters>
Tento prvek konfigurace určuje všechny nástroje pro import zásad, které řídí import vlastních výrazů zásad o vazbách. Každý podřízený prvek je <`policyImporter`>, který se používá k hledání vlastních kontrolních výrazů zásad o funkcích vazby a také k připojení vlastního elementu vazby, který implementuje funkce vyžadované kontrolním výrazem.
