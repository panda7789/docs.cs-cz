# <a name="transports"></a>Přenosy
Transportní vrstva je na nejnižší úrovni zásobníku kanálů. Hlavní přenos používaný v Windows Communication Foundation (WCF) jsou HTTP, HTTPS, TCP a pojmenované kanály. V této části jsou uvedené všechny konfigurační prvky, které umožňují nakonfigurovat různé typy přenosů.
