## <a name="remarks"></a><span data-ttu-id="a996c-126">Poznámky</span><span class="sxs-lookup"><span data-stu-id="a996c-126">Remarks</span></span>  
 <span data-ttu-id="a996c-127">Ve výchozím <xref:System.Uri?displayProperty=nameWithType> nastavení třída un-escapes procent procent kódované cesty oddělovače před spuštěním komprese cesty.</span><span class="sxs-lookup"><span data-stu-id="a996c-127">By default, the <xref:System.Uri?displayProperty=nameWithType> class un-escapes percent encoded path delimiters before executing path compression.</span></span> <span data-ttu-id="a996c-128">To to bylo implementováno jako mechanismus zabezpečení proti útokům, jako je následující:</span><span class="sxs-lookup"><span data-stu-id="a996c-128">This was implemented as a security mechanism against attacks like the following:</span></span>  
  
 `http://www.contoso.com/..%2F..%2F/Windows/System32/cmd.exe?/c+dir+c:\`  
  
 <span data-ttu-id="a996c-129">Pokud tento identifikátor URI bude předán modulům, které nezpracovávají méně kódované znaky, může dojít k provedení následujícího příkazu serverem:</span><span class="sxs-lookup"><span data-stu-id="a996c-129">If this URI gets passed down to modules not handling percent encoded characters correctly, it could result in the following command being executed by the server:</span></span>  
  
 `c:\Windows\System32\cmd.exe /c dir c:\`  
  
 <span data-ttu-id="a996c-130">Z tohoto <xref:System.Uri?displayProperty=nameWithType> důvodu třída první un-escapes oddělovače cesty a potom použije kompresi cesty.</span><span class="sxs-lookup"><span data-stu-id="a996c-130">For this reason, <xref:System.Uri?displayProperty=nameWithType> class first un-escapes path delimiters and then applies path compression.</span></span> <span data-ttu-id="a996c-131">Výsledkem předání výše uvedené <xref:System.Uri?displayProperty=nameWithType> škodlivé adresy URL konstruktoru třídy je výsledkem následující identifikátor URI:</span><span class="sxs-lookup"><span data-stu-id="a996c-131">The result of passing the malicious URL above to <xref:System.Uri?displayProperty=nameWithType> class constructor results in the following URI:</span></span>

 <span data-ttu-id="a996c-132">Toto výchozí chování lze upravit tak, aby neodcházaly procentově kódované oddělovače cest pomocí možnosti konfigurace schemeSettings pro určité schéma.</span><span class="sxs-lookup"><span data-stu-id="a996c-132">This default behavior can be modified to not un-escape percent encoded path delimiters using the schemeSettings configuration option for a specific scheme.</span></span>

## <a name="example"></a><span data-ttu-id="a996c-135">Příklad</span><span class="sxs-lookup"><span data-stu-id="a996c-135">Example</span></span>  
 <span data-ttu-id="a996c-136">Následující příklad ukazuje konfiguraci <xref:System.Uri> používanou třídou pro podporu neunikající oddělovače cest kódovaných procenty pro schéma http.</span><span class="sxs-lookup"><span data-stu-id="a996c-136">The following example shows a configuration used by the <xref:System.Uri> class to support not escaping percent-encoded path delimiters for the http scheme.</span></span>
