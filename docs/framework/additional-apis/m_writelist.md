# <a name="connectionm_writelist-field"></a>Pole Connection.m\_WriteList

`Connection.m_WriteList`je <xref:System.Collections.ArrayList> objekty, <xref:System.Net.HttpWebRequest> které jsou zařazeny do fronty k odeslání přes PROTOKOL HTTP.

## <a name="syntax"></a>Syntaxe
  
```csharp  
private ArrayList m_WriteList
```

> [!WARNING]
> Pole `Connection.m_WriteList` je soukromé a není určeno k použití přímo ve vašem kódu.
>
> Společnost Microsoft nepodporuje použití tohoto pole v produkční aplikaci za žádných okolností.
