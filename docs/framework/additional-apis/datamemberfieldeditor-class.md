# <a name="datamemberfieldeditor-class"></a>Třída DataMemberFieldEditor

Poskytuje uživatelské rozhraní pro úpravy vlastností objektů vázaných `DataSource` na data tím, že uvádí všechny vlastnosti objektu, ze kterého má být zdroj hodnoty vynalezen.  
  
## <a name="syntax"></a>Syntaxe
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> Třída `DataMemberFieldEditor` je interní a není určena pro použití přímo ve vašem kódu.
>
> Společnost Microsoft nepodporuje použití této třídy v produkční aplikaci za žádných okolností.
