# <a name="datamemberfieldeditor-class"></a>DataMemberFieldEditor – třída

Poskytuje uživatelské rozhraní pro úpravu vlastností objektů vázaných na data zobrazením všech vlastností objektu `DataSource`, ze kterého se má vybrat zdroj hodnoty.  
  
## <a name="syntax"></a>Syntaxe
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> Třída `DataMemberFieldEditor` je interní a není určena pro použití přímo v kódu.
> 
> Společnost Microsoft v žádné situaci nepodporuje použití této třídy v produkční aplikaci.
