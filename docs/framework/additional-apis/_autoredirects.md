# <a name="httpwebrequest_autoredirects-field"></a>HttpWebRequest. \_ Pole autoredirect

`HttpWebRequest._AutoRedirects`je <xref:System.Int32> , který odráží počet přesměrování, která jsou pro ně určena <xref:System.Net.HttpWebRequest> .

## <a name="syntax"></a>Syntax  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> `HttpWebRequest._AutoRedirects`Pole je interní a není určeno pro použití přímo v kódu.
>
> Společnost Microsoft v žádné situaci nepodporuje použití tohoto pole v produkční aplikaci.
