# <a name="httpwebrequest_autoredirects-field"></a>HttpWebRequest. \_Pole Automatické přesměrování

`HttpWebRequest._AutoRedirects`<xref:System.Int32> je, který odráží počet přesměrování provedených <xref:System.Net.HttpWebRequest>pro toto .

## <a name="syntax"></a>Syntaxe  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> Pole `HttpWebRequest._AutoRedirects` je interní a není určeno k použití přímo ve vašem kódu.
>
> Společnost Microsoft nepodporuje použití tohoto pole v produkční aplikaci za žádných okolností.
