# <a name="rtcstate-class"></a>Třída RtcState

Třída `RtcState` představuje data stavu pro požadavek komunikace v reálném čase (RTC).

## <a name="syntax"></a>Syntaxe
  
```csharp  
internal class RtcState
```

> [!WARNING]
> Třída `RtcState` je interní a není určena pro použití přímo ve vašem kódu.
>
> Společnost Microsoft nepodporuje použití této třídy v produkční aplikaci za žádných okolností.

## <a name="requirements"></a>Požadavky

**Obor názvů:**<xref:System.Net>

**Sestava:** Systém (v souboru System.dll)

**Verze rozhraní .NET Framework:** K dispozici od 4.5.
