# <a name="connectiongroupm_connectionlist-field"></a>Pole connectioncollection. m \_ ConnectionList

`ConnectionGroup.m_ConnectionList`je <xref:System.Collections.ArrayList> objekt připojení, který slouží stejný identifikátor URI a sdílí stejné hodnoty pro některé další vlastnosti, jako je například vypršení platnosti a ověřování.

## <a name="syntax"></a>Syntax
  
```csharp  
private ArrayList m_ConnectionList
```

> [!WARNING]
> `ConnectionGroup.m_ConnectionList`Pole je soukromé a není určeno pro použití přímo v kódu.
>
> Společnost Microsoft v žádné situaci nepodporuje použití tohoto pole v produkční aplikaci.
