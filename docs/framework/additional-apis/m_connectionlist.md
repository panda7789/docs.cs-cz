# <a name="connectiongroupm_connectionlist-field"></a>Pole ConnectionGroup.m\_ConnectionList

`ConnectionGroup.m_ConnectionList`<xref:System.Collections.ArrayList> je objekty připojení, které slouží stejný identifikátor URI a sdílet stejné hodnoty pro některé další vlastnosti, jako je vypršení platnosti a ověřování.

## <a name="syntax"></a>Syntaxe
  
```csharp  
private ArrayList m_ConnectionList
```

> [!WARNING]
> Pole `ConnectionGroup.m_ConnectionList` je soukromé a není určeno k použití přímo ve vašem kódu.
>
> Společnost Microsoft nepodporuje použití tohoto pole v produkční aplikaci za žádných okolností.
