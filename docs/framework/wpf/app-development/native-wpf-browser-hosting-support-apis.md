# <a name="native-wpf-browser-hosting-support-apis"></a>Nativní Prohlížeč WPF – rozhraní API pro podporu hostování
Hostování wpf aplikací ve webových prohlížečích je usnadněno serverem Active Document (označovaným také jako DocObject) registrovaným mimo wpf hostitele. Aplikace Internet Explorer může přímo aktivovat a integrovat s aktivním dokumentem. Pro hostování XBAPs a volné dokumenty XAML v prohlížečích Mozilla, WPF poskytuje NPAPI plugin, který poskytuje podobné hostitelské prostředí wpf Active Document server jako Internet Explorer dělá. Nejjednodušší praktický způsob, jak hostovat dokumenty XBAPs a XAML v jiných prohlížečích a samostatných aplikacích, je však prostřednictvím ovládacího prvku Webového prohlížeče Internet Explorer. Ovládací prvek webový prohlížeč poskytuje komplexní hostitelské prostředí serveru Active Document, ale umožňuje vlastnímu hostiteli přizpůsobit a rozšířit toto prostředí a komunikovat přímo s aktuálním objektem Active Document.  
  
 Server WPF Active Document implementuje několik běžných hostitelských rozhraní, včetně [IOleObject](/windows/win32/api/oleidl/nn-oleidl-ioleobject), [IOleDocument](/windows/win32/api/docobj/nn-docobj-ioledocument), [IOleInPlaceActiveObject](/windows/win32/api/oleidl/nn-oleidl-ioleinplaceactiveobject), [IPersistMoniker](https://docs.microsoft.com/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms775042(v=vs.85)), [IOleCommandTarget](/windows/win32/api/docobj/nn-docobj-iolecommandtarget). Při hostování v ovládacím prvku webový prohlížeč, tato rozhraní mohou být dotazy z objektu vrácené [IWebBrowser2::Document](https://docs.microsoft.com/previous-versions/aa752116(v=vs.85)) vlastnost.  
  
## <a name="iolecommandtarget"></a>IOleCommandTarget  
 Implementace [iOleCommandTarget](/windows/win32/api/docobj/nn-docobj-iolecommandtarget) serverem Active Document serveru WPF podporuje mnoho příkazů souvisejících s navigací a prohlížečem standardní skupiny příkazů OLE (s identifikátorem GUID skupiny příkazů null). Kromě toho rozpozná vlastní skupinu příkazů s názvem CGID_PresentationHost. V současné době je v této skupině definován pouze jeden příkaz.  
  
```cpp  
DEFINE_GUID(CGID_PresentationHost, 0xd0288c55, 0xd6, 0x4f5e, 0xa8, 0x51, 0x79, 0xde, 0xc5, 0x1b, 0x10, 0xec);  
enum PresentationHostCommands {
   PHCMDID_TABINTO = 1
};  
```  
  
 PHCMDID_TABINTO instruuje PresentationHost přepnout fokus na první nebo poslední zaostřitelný prvek v jeho obsahu, v závislosti na stavu klávesy Shift.
