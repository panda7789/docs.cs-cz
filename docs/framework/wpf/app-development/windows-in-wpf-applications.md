# <a name="windows-in-wpf-applications"></a>Systém Windows v aplikacích WPF
Uživatelé komunikují s aplikacemi prostřednictvím Windows. Základním účelem okna je hostování a zobrazení obsahu. Typ obsahu, který je hostitelem okna, závisí na typu dat, která aplikace pracuje, což může zahrnovat média, stránky jazyk Extensible Application Markup Language (XAML) (XAML), webové stránky, dokumenty, databázové tabulky a záznamy a systémové informace.
