# <a name="hosting-wpf-applications"></a>Hostování aplikací WPF
WPF – aplikace prohlížeče XAML (XBAP) jsou bohatě klientské aplikace, které je možné nasadit na webový server a spustit v prohlížeči. Hostitel WPF (PresentationHost. exe) je zaregistrován jako prostředí a obslužná rutina MIME pro soubory XBAP a XAML. Proto aplikace Internet Explorer ví, že při spuštění XBAP spustí hostitele WPF. Uživatelé Firefox můžou nainstalovat Doplňky Firefox, které umožní hostování Firefox i v prohlížeči Firefox. XBAP lze hostovat v jiných prohlížečích nebo v samostatných aplikacích pomocí rozhraní API hostování v nativním prohlížeči, které poskytuje WPF.
