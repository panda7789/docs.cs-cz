# <a name="dialog-boxes-in-windows-forms"></a>Dialogová okna ve Windows Forms
Dialogová okna slouží k interakci s uživatelem a načítání informací. V jednoduchých výrazech je dialogové okno formuláře s jeho <xref:System.Windows.Forms.FormBorderStyle> vlastností výčtu nastavenou na hodnotu `FixedDialog`. Vlastní dialogová okna můžete vytvořit pomocí Návrhář formulářů v aplikaci Visual Studio. Přidejte ovládací prvky, například `Label`, `Textbox`a `Button` pro přizpůsobení dialogových oken vašim konkrétním potřebám. .NET Framework také obsahuje předdefinovaná dialogová okna, jako je například **otevření souboru** a okna se zprávami, které můžete přizpůsobit pro vlastní aplikace. Další informace naleznete v tématu [ovládací prvky a součásti dialogového okna](./controls/dialog-box-controls-and-components-windows-forms.md).
