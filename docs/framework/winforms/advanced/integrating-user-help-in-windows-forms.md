# <a name="integrating-user-help-in-windows-forms"></a>Integrace uživatelské nápovědy do formulářů Windows
Zásadní, ale často přehlédnutíný aspekt sestavování aplikací založených na systému Windows je systém nápovědy, protože je v tomto případě uživatelům zapnuta pomoc v časech nejasností. Model Windows Forms podporují dva různé typy pomoci, které jsou poskytovány [komponentou HelpProvider –](../controls/helpprovider-component-windows-forms.md). První z nich spočívá v tom, že uživatel odkazuje na soubor Help HTML nebo HTML Help 1. formát *x* nebo vyšší. Druhý může zobrazit stručný popis "What to this" na jednotlivých ovládacích prvcích; To je užitečné hlavně pro dialogová okna. Oba typy nápovědě lze použít ve stejném formuláři.  
  
 Kromě toho lze [komponentu ToolTip](../controls/tooltip-component-windows-forms.md) použít k poskytnutí individuální nápovědy pro ovládací prvky na model Windows Forms.
