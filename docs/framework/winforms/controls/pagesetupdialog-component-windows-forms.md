# <a name="pagesetupdialog-component-windows-forms"></a>PageSetupDialog – komponenta (Windows Forms)
Komponenta <xref:System.Windows.Forms.PageSetupDialog> Windows Forms je předem nakonfigurované dialogové okno, které slouží k nastavení podrobností stránky pro tisk v aplikacích systému Windows. Použijte jej v aplikaci založené na systému Windows jako jednoduché řešení pro uživatele nastavit předvolby stránky namísto konfigurace vlastní dialogové okno. Uživatelům můžete povolit nastavení úprav ohraničení a okrajů, záhlaví a zápatí a orientaci na výšku vs. orientace na šířku. Spoléháním se na standardní dialogová okna systému Windows vytvoříte aplikace, jejichž základní funkce jsou uživatelům okamžitě známé.
