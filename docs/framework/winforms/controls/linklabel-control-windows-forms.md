# <a name="linklabel-control-windows-forms"></a>LinkLabel – ovládací prvek (Windows Forms)
Ovládací prvek `LinkLabel` Windows Forms umožňuje přidat odkazy ve stylu webu do aplikací systému Windows Forms. Ovládací `LinkLabel` prvek můžete použít pro vše, <xref:System.Windows.Forms.Label> co můžete použít ovládací prvek; Můžete také nastavit část textu jako odkaz na objekt nebo webovou stránku.
