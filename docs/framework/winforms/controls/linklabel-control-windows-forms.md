# <a name="linklabel-control-windows-forms"></a>LinkLabel – ovládací prvek (Windows Forms)
Ovládací prvek model Windows Forms `LinkLabel` umožňuje přidat odkazy na webový styl do aplikací model Windows Forms. Ovládací prvek `LinkLabel` lze použít pro vše, co lze použít pro možnost <xref:System.Windows.Forms.Label> ovládací prvek pro. Můžete také nastavit část textu jako odkaz na objekt nebo webovou stránku.
