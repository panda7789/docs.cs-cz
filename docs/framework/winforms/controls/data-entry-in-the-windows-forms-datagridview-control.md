# <a name="data-entry-in-the-windows-forms-datagridview-control"></a>Zadávání dat v ovládacím prvku Windows Forms DataGridView
Ovládací prvek `DataGridView` poskytuje několik funkcí, které umožňují změnit způsob, jakým uživatelé přidávají nebo upravují data v ovládacím prvku. Například můžete zvýšit efektivitu zadávání dat tím, že poskytnete výchozí hodnoty pro nové řádky a upozorníte uživatele, když dojde k chybám.
