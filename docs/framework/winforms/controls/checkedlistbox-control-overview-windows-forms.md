# <a name="checkedlistbox-control-overview-windows-forms"></a>CheckedListBox – přehled ovládacího prvku (Windows Forms)
Ovládací prvek model Windows Forms <xref:System.Windows.Forms.CheckedListBox> rozšiřuje ovládací prvek <xref:System.Windows.Forms.ListBox>. Má skoro všechno, co je pole se seznamem, a také může zobrazit zaškrtnutí vedle položek v seznamu. Mezi tyto dva ovládací prvky patří další rozdíly, které mají zaškrtnuté políčko pouze <xref:System.Windows.Forms.DrawMode.Normal?displayProperty=nameWithType>. a zaškrtnutá pole seznamu mohou mít pouze jednu položku nebo žádnou vybranou. Všimněte si, že vybraná položka se zvýrazní ve formuláři a není stejná jako zaškrtnutá položka.  
  
 Zaškrtnutá pole seznamu mohou mít položky přidané v době návrhu pomocí **editoru kolekce řetězců** nebo jejich položky lze dynamicky přidat z kolekce v době běhu pomocí vlastnosti <xref:System.Windows.Forms.CheckedListBox.Items%2A>. Další informace naleznete v tématu [How to: Add and Remove Items from model Windows Forms ComboBox, ListBox nebo CheckedListBox Control](add-and-remove-items-from-a-wf-combobox.md).
