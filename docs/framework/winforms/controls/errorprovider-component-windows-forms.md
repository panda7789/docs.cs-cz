# <a name="errorprovider-component-windows-forms"></a>ErrorProvider – komponenta (Windows Forms)
Komponenta model Windows Forms `ErrorProvider` se používá k zobrazení uživatele nerušivým způsobem, že je něco špatně. Obvykle se používá ve spojení s ověřováním vstupu uživatele na formuláři nebo při zobrazení chyb v rámci datové sady.
