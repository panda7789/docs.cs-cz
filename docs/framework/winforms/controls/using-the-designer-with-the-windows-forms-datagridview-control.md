# <a name="using-the-designer-with-the-windows-forms-datagridview-control"></a>Používání Návrháře s ovládacím prvkem Windows Forms DataGridView
Visual Studio poskytuje podporu návrháře pro ovládací prvek `DataGridView`, který umožňuje provádět mnoho úloh nastavení bez psaní kódu. Tyto úlohy zahrnují vazbu ovládacího prvku na zdroj dat, úpravu sloupců používaných k zobrazení dat a úpravu vzhledu a základního chování ovládacího prvku.
