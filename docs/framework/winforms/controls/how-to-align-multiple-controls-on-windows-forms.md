# <a name="how-to-align-multiple-controls-on-windows-forms"></a>Postupy: Zarovnání více ovládacích prvků ve Windows Forms

Pro standardizaci rozložení uživatelského rozhraní (UI) aplikace založené na systému Windows můžete umístit skupiny ovládacích prvků jediným příkazem.

## <a name="to-align-multiple-controls-on-a-form"></a>Zarovnání více ovládacích prvků ve formuláři

1. V aplikaci Visual Studio otevřete formulář obsahující ovládací prvky, které chcete umístit do **Návrhář formulářů**.

2. Vyberte ovládací prvky, které chcete zarovnat. Prvním vybraným ovládacím prvkem je primární ovládací prvek, na který se budou zarovnávat ostatní.

3. V nabídce **Formát** vyberte **Zarovnat**a pak vyberte jednu z sedmi dostupných možností.
