# <a name="splitter-control-windows-forms"></a>Rozdělovač – ovládací prvek (Windows Forms)
Ovládací prvky model Windows Forms `Splitter` slouží k změně velikosti ukotvených ovládacích prvků za běhu. `Splitter` ovládací prvek se často používá ve formulářích s ovládacími prvky, které mají různou délku dat, jako je Průzkumník Windows, jejichž podokna dat obsahují informace o různých šířkách v různých časech.  
  
> [!NOTE]
> I když <xref:System.Windows.Forms.SplitContainer> nahrazuje a přidává funkce pro `Splitter` kontrolu nad předchozími verzemi, `Splitter` se zachová pro zpětnou kompatibilitu i pro budoucí použití, pokud zvolíte.
