 <span data-ttu-id="5839f-108">Každý sloupec může mít jiný režim velikosti, ale všechny sloupce s režimem velikosti <xref:System.Windows.Forms.DataGridViewAutoSizeColumnMode.Fill> budou sdílet šířku oblasti zobrazení, která se nepoužívá v ostatních sloupcích.</span><span class="sxs-lookup"><span data-stu-id="5839f-108">Each column can have a different size mode, but any columns with a size mode of <xref:System.Windows.Forms.DataGridViewAutoSizeColumnMode.Fill> will share the display-area width that is not used by the other columns.</span></span> <span data-ttu-id="5839f-109">Tato šířka je rozdělena mezi sloupce v režimu výplně v poměrech k jejich <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> hodnotám vlastností.</span><span class="sxs-lookup"><span data-stu-id="5839f-109">This width is divided among the fill-mode columns in proportions relative to their <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> property values.</span></span> <span data-ttu-id="5839f-110">Pokud například dva sloupce mají <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> hodnoty 100 a 200, bude první sloupec poloviční stejně velký jako druhý sloupec.</span><span class="sxs-lookup"><span data-stu-id="5839f-110">For example, if two columns have <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values of 100 and 200, the first column will be half as wide as the second column.</span></span>

 <span data-ttu-id="5839f-117">Po změně velikosti sloupce v režimu výplně se <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> hodnoty pro všechny změněné sloupce upraví proporcionálně.</span><span class="sxs-lookup"><span data-stu-id="5839f-117">After resizing a fill-mode column, the <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values for all columns that changed are adjusted proportionally.</span></span> <span data-ttu-id="5839f-118">Pokud například čtyři sloupce v režimu výplně mají <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> hodnoty 100, změna velikosti druhého sloupce na polovinu jeho původní šířky bude mít za následek <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> hodnoty 100, 50, 125 a 125.</span><span class="sxs-lookup"><span data-stu-id="5839f-118">For example, if four fill-mode columns have <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values of 100, resizing the second column to half its original width will result in <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values of 100, 50, 125, and 125.</span></span> <span data-ttu-id="5839f-119">Změna velikosti sloupce, který není v režimu Fill, nezmění žádné <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> hodnoty, protože sloupce v režimu výplně budou jednoduše měnit velikost, aby bylo možné kompenzovat stejné poměry.</span><span class="sxs-lookup"><span data-stu-id="5839f-119">Resizing a column that is not in fill mode will not change any <xref:System.Windows.Forms.DataGridViewColumn.FillWeight%2A> values because the fill-mode columns will simply resize to compensate while retaining the same proportions.</span></span>
