# <a name="datetimepicker-control-windows-forms"></a>DateTimePicker – ovládací prvek (Windows Forms)
`DateTimePicker`Ovládací prvek model Windows Forms umožňuje uživateli vybrat jednu položku ze seznamu kalendářních dat nebo časů. Když se používá k reprezentaci data, zobrazí se ve dvou částech: rozevírací seznam s datem, který je reprezentován textem, a mřížku, která se zobrazí po kliknutí na šipku dolů vedle seznamu.
