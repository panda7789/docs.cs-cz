3. <span data-ttu-id="c105c-117">V editoru kolekce klikněte na tlačítko **Přidat** a vložte tak styl tabulky.</span><span class="sxs-lookup"><span data-stu-id="c105c-117">In the collection editor, click **Add** to insert a table style.</span></span>
