# <a name="fontdialog-component-windows-forms"></a>FontDialog – komponenta (Windows Forms)
Součást model Windows Forms <xref:System.Windows.Forms.FontDialog> je předem nakonfigurovaným dialogovým oknem. Jedná se o stejné dialogové okno **písma** , které je zveřejněné operačním systémem Windows. Komponenta dědí z třídy <xref:System.Windows.Forms.CommonDialog>.
