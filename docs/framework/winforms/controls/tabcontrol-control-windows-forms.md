# <a name="tabcontrol-control-windows-forms"></a>TabControl – ovládací prvek (Windows Forms)
`TabControl` model Windows Forms zobrazí více karet, jako jsou například oddělovače v poznámkovém bloku nebo štítky v sadě složek v kartotéce. Karty mohou obsahovat obrázky a další ovládací prvky. Stránky vlastností můžete vytvořit pomocí `TabControl`.
