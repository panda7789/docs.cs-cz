# <a name="combobox-control-windows-forms"></a>ComboBox – ovládací prvek (Windows Forms)
Ovládací prvek model Windows Forms `ComboBox` slouží k zobrazení dat v rozevíracím poli se seznamem. Ve výchozím nastavení se ovládací prvek `ComboBox` zobrazí ve dvou částech: horní část je textové pole, které uživateli umožňuje zadat položku seznamu. Druhá část je seznam, ve kterém se zobrazuje seznam položek, ze kterých si uživatel může vybrat jednu.
