## <a name="to-bind-the-control-to-a-data-source"></a><span data-ttu-id="6bffb-114">Svázání ovládacího prvku se zdrojem dat</span><span class="sxs-lookup"><span data-stu-id="6bffb-114">To bind the control to a data source</span></span>

1. <span data-ttu-id="6bffb-115">V pravém horním rohu ovládacího prvku <xref:System.Windows.Forms.DataGridView> klikněte na glyf akcí návrháře (![malou černou šipku](./media/designer-actions-glyph.gif)).</span><span class="sxs-lookup"><span data-stu-id="6bffb-115">Click the designer actions glyph (![Small black arrow](./media/designer-actions-glyph.gif)) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control.</span></span>

2. <span data-ttu-id="6bffb-116">Klikněte na šipku rozevíracího seznamu u možnosti **Zvolit zdroj dat** .</span><span class="sxs-lookup"><span data-stu-id="6bffb-116">Click the drop-down arrow for the **Choose Data Source** option.</span></span>

3. <span data-ttu-id="6bffb-117">Pokud projekt ještě nemá zdroj dat, klikněte na **Přidat zdroj dat projektu** a postupujte podle kroků uvedených v průvodci.</span><span class="sxs-lookup"><span data-stu-id="6bffb-117">If your project does not already have a data source, click **Add Project Data Source** and follow the steps indicated by the wizard.</span></span>

     <span data-ttu-id="6bffb-118">Další informace najdete v tématu [Průvodce konfigurací zdroje dat](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/w4dd7z6t(v=vs.120)).</span><span class="sxs-lookup"><span data-stu-id="6bffb-118">For more information, see [Data Source Configuration Wizard](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/w4dd7z6t(v=vs.120)).</span></span> <span data-ttu-id="6bffb-119">Nový zdroj dat se zobrazí v rozevíracím okně **Zvolit zdroj dat** .</span><span class="sxs-lookup"><span data-stu-id="6bffb-119">Your new data source will appear in the **Choose Data Source** drop-down window.</span></span> <span data-ttu-id="6bffb-120">Pokud nový zdroj dat obsahuje pouze jednoho člena, například jedinou databázovou tabulku, bude ovládací prvek automaticky svázán s tímto členem.</span><span class="sxs-lookup"><span data-stu-id="6bffb-120">If your new data source contains only one member, such as a single database table, the control will automatically bind to that member.</span></span> <span data-ttu-id="6bffb-121">V opačném případě pokračujte k dalšímu kroku.</span><span class="sxs-lookup"><span data-stu-id="6bffb-121">Otherwise, continue to the next step.</span></span>

4. <span data-ttu-id="6bffb-122">Rozbalte **Další uzly zdroje dat** a **zdroje dat projektu** , pokud ještě nejsou rozbalené, a pak vyberte zdroj dat, ke kterému chcete ovládací prvek navazovat.</span><span class="sxs-lookup"><span data-stu-id="6bffb-122">Expand the **Other Data Sources** and **Project Data Sources** nodes if they are not already expanded, and then select the data source to bind the control to.</span></span>

5. <span data-ttu-id="6bffb-123">Pokud zdroj dat obsahuje více než jednoho člena, například pokud jste vytvořili <xref:System.Data.DataSet?displayProperty=nameWithType>, který obsahuje více tabulek, rozbalte zdroj dat a pak vyberte konkrétního člena, ke kterému se má vytvořit vazba.</span><span class="sxs-lookup"><span data-stu-id="6bffb-123">If your data source contains more than one member, such as if you have created a <xref:System.Data.DataSet?displayProperty=nameWithType> that contains multiple tables, expand the data source, and then select the specific member to bind to.</span></span>

6. <span data-ttu-id="6bffb-124">Chcete-li vytvořit relaci hlavního a podrobností, v rozevíracím okně **Zvolit zdroj dat** pro druhý ovládací prvek <xref:System.Windows.Forms.DataGridView> rozbalte <xref:System.Windows.Forms.BindingSource> vytvořenou pro nadřazenou tabulku a pak v zobrazeném seznamu vyberte související podřízenou tabulku.</span><span class="sxs-lookup"><span data-stu-id="6bffb-124">To create a master/detail relationship, in the **Choose Data Source** drop-down window for a second <xref:System.Windows.Forms.DataGridView> control, expand the <xref:System.Windows.Forms.BindingSource> created for the parent table, and then select the related child table from the list shown.</span></span>

    > [!NOTE]
    > <span data-ttu-id="6bffb-125">Pokud projekt již obsahuje zdroj dat, můžete také použít okno **zdroje dat** k vytvoření datového formuláře.</span><span class="sxs-lookup"><span data-stu-id="6bffb-125">If your project already has a data source, you can also use the **Data Sources** window to create a data form.</span></span> <span data-ttu-id="6bffb-126">Další informace naleznete v tématu [okno zdroje dat](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/6ckyxa83(v=vs.120)).</span><span class="sxs-lookup"><span data-stu-id="6bffb-126">For more information, see [Data Sources Window](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-2013/6ckyxa83(v=vs.120)).</span></span>
