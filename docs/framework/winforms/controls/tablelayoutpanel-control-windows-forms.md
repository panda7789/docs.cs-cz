# <a name="tablelayoutpanel-control-windows-forms"></a>TableLayoutPanel – ovládací prvek (Windows Forms)
Ovládací prvek <xref:System.Windows.Forms.TableLayoutPanel> uspořádá svůj obsah do mřížky. Vzhledem k tomu, že rozložení se provádí v době návrhu i v době běhu, může se dynamicky měnit při změně prostředí aplikace. To dává ovládacím prvkům v panelu možnost změny velikosti, takže může reagovat na změny, jako je například změna velikosti nadřazeného ovládacího prvku nebo změna délky textu v důsledku lokalizace.
