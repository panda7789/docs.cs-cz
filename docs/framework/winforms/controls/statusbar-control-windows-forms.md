# <a name="statusbar-control-windows-forms"></a>StatusBar – ovládací prvek (Windows Forms)
> [!NOTE]
> Ovládací prvek <xref:System.Windows.Forms.ToolStripStatusLabel> nahrazuje a přidává funkce do ovládacího prvku <xref:System.Windows.Forms.StatusBar>; Nicméně ovládací prvek <xref:System.Windows.Forms.StatusBar> se zachovává pro zpětnou kompatibilitu i pro budoucí použití, pokud zvolíte.  
  
 Ovládací prvek model Windows Forms <xref:System.Windows.Forms.StatusBar> se používá ve formulářích jako oblast, obvykle se zobrazuje v dolní části okna, kde aplikace může zobrazit různé druhy informací o stavu. ovládací prvky <xref:System.Windows.Forms.StatusBar> mohou mít panely na stavovém panelu, které zobrazují ikony k označení stavu, nebo řadu ikon v animaci, která indikuje, že proces pracuje. například Microsoft Word značí, že se dokument ukládá.
