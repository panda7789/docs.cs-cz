# <a name="hscrollbar-and-vscrollbar-controls-windows-forms"></a>Ovládací prvky HScrollBar a VScrollBar (Windows Forms)
Ovládací prvky posuvníku model Windows Forms slouží k zajištění snadné navigace prostřednictvím dlouhého seznamu položek nebo velkého množství informací posouváním vodorovně nebo svisle v rámci aplikace nebo ovládacího prvku. Rolovací panely jsou běžným prvkem rozhraní systému Windows.
