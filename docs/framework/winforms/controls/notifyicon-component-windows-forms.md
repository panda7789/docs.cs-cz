# <a name="notifyicon-component-windows-forms"></a>NotifyIcon – komponenta (Windows Forms)
Komponenta model Windows Forms `NotifyIcon` zobrazí ikony v oznamovací oblasti stavu hlavního panelu pro procesy, které běží na pozadí a jinak by neměly mít uživatelská rozhraní. Příkladem může být program antivirové ochrany, ke kterému se dá dostat kliknutím na ikonu v oznamovací oblasti pro stav hlavního panelu.
