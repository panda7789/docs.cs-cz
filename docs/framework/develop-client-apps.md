# <a name="develop-client-applications-with-net-framework"></a>Vývoj klientských aplikací pomocí rozhraní .NET Framework

Existuje několik způsobů, jak vyvíjet aplikace založené na systému Windows pomocí rozhraní .NET Framework. Můžete použít některý z těchto nástrojů a rámců:

- [Univerzální platforma Windows (UPW)](/windows/uwp/)
- [Windows Presentation Foundation (WPF)](./wpf/index.md)
- [Windows Forms](./winforms/index.md)

Tato část obsahuje články, které popisují, jak vytvářet aplikace založené na systému Windows pomocí služby Windows Presentation Foundation nebo Windows Forms. Můžete však také vytvářet webové aplikace pomocí rozhraní .NET Framework a klientských aplikací pro počítače nebo zařízení, které zpřístupníte prostřednictvím obchodu Microsoft Store (aplikace UPW).
