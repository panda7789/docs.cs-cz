## <a name="iehostdll-and-ieexecexe"></a>IEHost. dll a IEExec. exe

Sestavení IEHost. dll a IEExec. exe byla odebrána z .NET Framework. Všechny jejich typy a jejich členové jsou zastaralé a nejsou podporováni v .NET Framework 4,5 a novějším. Tato sestavení byla použita pro hostování model Windows Formsch ovládacích prvků a spouštění spustitelných souborů v aplikaci Internet Explorer. Mezi alternativy k této technologii patří ClickOnce, aplikace prohlížeče XAML (XBAP) a Microsoft Silverlight.
