## <a name="iehostdll-and-ieexecexe"></a>Soubor IEHost.dll a IEExec.exe

Sestavení IEHost.dll a IEExec.exe byla odebrána z rozhraní .NET Framework. Všechny jejich typy a jejich členy jsou zastaralé a nejsou podporovány v rozhraní .NET Framework 4.5 a novější. Tato sestavení byla použita k hostování ovládacích prvků windows forms a ke spuštění spustitelných souborů v aplikaci Internet Explorer. Alternativy k této technologii patří ClickOnce, XAML browser aplikace (XBAP) a Microsoft Silverlight.
