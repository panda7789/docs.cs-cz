# <a name="extending-windows-workflow-foundation"></a>Rozšíření Windows Workflow Foundation
V následující části je popsán postup rozšiřování programovací model Windows Workflow Foundation (WF) s vlastními aktivitami a návrháři v přehostitelských prostředích mimo Visual Studio 2010.

## <a name="in-this-section"></a>V tomto oddílu
 [Přizpůsobení prostředí pro návrh pracovního postupu](customizing-the-workflow-design-experience.md) Určuje, jak se v .NET Framework 4 významně zjednodušily scénáře návrhu vlastních aktivit a pro opětovné hostování Návrhář postupu provádění Windows. Vývoj a nasazení jsou teď jednodušší a flexibilnější, protože nový programovací model návrháře aktivit je sestavený po Windows Presentation Foundation (WPF).
