# <a name="deploying-the-net-framework"></a>Nasazení rozhraní .NET Framework
Tato část dokumentace rozhraní .NET Framework obsahuje informace pro vývojáře, kteří chtějí nainstalovat rozhraní .NET Framework se svými aplikacemi, a správcům, kteří chtějí nasadit rozhraní .NET Framework v síti. Popisuje také problémy s aktivací a restartováním související s nasazením a způsob sledování průběhu instalace rozhraní .NET Framework.

 [Postupy: Ladění problémů aktivace CLR](how-to-debug-clr-activation-issues.md)
