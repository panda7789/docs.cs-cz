# <a name="net-framework-client-profile"></a>Profil klienta rozhraní .NET Framework
.NET Client Profile je podmnožinou .NET Framework, která byla poskytnuta s .NET Framework 4 a staršími verzemi a byla optimalizována pro klientské aplikace. .NET Framework je vývojová platforma pro Windows, Windows Phone a Microsoft Azure a poskytuje prostředí pro spuštění spravované aplikace a knihovnu tříd .NET Framework. .NET Framework 4 a starší verze poskytují dvě možnosti nasazení: úplný .NET Framework a profil klienta. Profil klienta podporuje rychlejší nasazení a menší instalační balíčky aplikace než plná verze rozhraní .NET Framework.  
  
 Počínaje verzí .NET Framework 4,5 byl profil klienta již ukončen a k dispozici je pouze úplný Distribuovatelný balíček. Optimalizace poskytované .NET Framework 4,5, jako je menší velikost staženého a rychlejšího nasazení, eliminují nutnost samostatného balíčku pro nasazení. Jediná verze Redistributable zjednodušuje proces instalace a možnosti nasazení aplikace.  
  
 Pokud však cílíte na .NET Framework 4 nebo 3,5 a chcete získat další informace o profilu klienta a jeho použití, přečtěte si téma [.NET Framework profil klienta](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/cc656912%28v=vs.100%29) v dokumentaci .NET Framework 4.  
  
 Při instalaci .NET Framework 4,5 se profil klienta .NET Framework 4 aktualizuje na plnou verzi .NET Framework. Informace o instalaci .NET Framework 4,5 najdete v tématu [instalace .NET Framework pro vývojáře](../install/guide-for-developers.md).
