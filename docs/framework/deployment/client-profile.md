# <a name="net-framework-client-profile"></a>Profil klienta rozhraní .NET Framework
Profil klienta .NET je podmnožinou rozhraní .NET Framework, která byla vybavena rozhraním .NET Framework 4 a staršími verzemi a byla optimalizována pro klientské aplikace. Rozhraní .NET Framework je vývojová platforma pro Windows, Windows Phone a Microsoft Azure a poskytuje prostředí pro spravování spravovaných aplikací a knihovnu tříd rozhraní .NET Framework. Rozhraní .NET Framework 4 a starší verze poskytly dvě možnosti nasazení: úplnou architekturu .NET Framework a profil klienta. Profil klienta podporuje rychlejší nasazení a menší instalační balíčky aplikace než plná verze rozhraní .NET Framework.  
  
 Počínaje rozhraním .NET Framework 4.5 byl profil klienta ukončen a je k dispozici pouze úplný redistribuovatelný balíček. Optimalizace poskytované rozhraním .NET Framework 4.5, jako je menší velikost stahování a rychlejší nasazení, eliminovaly potřebu samostatného balíčku nasazení. Jediná verze Redistributable zjednodušuje proces instalace a možnosti nasazení aplikace.  
  
 Pokud však cílíte na rozhraní .NET Framework 4 nebo 3.5 a chcete se dozvědět více o profilu klienta a o jeho použití, přečtěte si informace v dokumentaci k rozhraní [.NET Framework.](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/cc656912%28v=vs.100%29)  
  
 Při instalaci rozhraní .NET Framework 4.5 je profil klienta rozhraní .NET Framework 4 aktualizován na plnou verzi rozhraní .NET Framework. Informace o instalaci rozhraní .NET Framework 4.5 naleznete [v tématu Instalace rozhraní .NET Framework pro vývojáře](../install/guide-for-developers.md).
