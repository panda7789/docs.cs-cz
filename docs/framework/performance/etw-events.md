# <a name="etw-events-in-the-net-framework"></a>Události Trasování událostí pro Windows v rozhraní .NET Framework
Trasování událostí pro Windows (ETW) je vysoce výkonný, nízkorehatelný škálovatelný trasovací systém poskytovaný operačními systémy Windows. Doplňuje profilování a ladění podporu poskytované rozhraní .NET Framework a lze použít k řešení problémů různých scénářů.  
  
 V rozhraní .NET Framework je pro clr (CLR) k dispozici trasování událostí ETW pro běžný jazyk ový čas (CLR), [paralelní knihovnu úloh](../../standard/parallel-programming/task-parallel-library-tpl.md)a [paralelní LINQ (PLINQ).](../../standard/parallel-programming/introduction-to-plinq.md)
