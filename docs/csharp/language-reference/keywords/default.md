Klíčové `default` slovo lze použít dvěma způsoby:

- Chcete-li zadat výchozí popisek v [ `switch` příkazu](switch.md).
- Jako [výchozí operátor nebo literál](../operators/default.md) k vytvoření výchozí hodnoty typu.
