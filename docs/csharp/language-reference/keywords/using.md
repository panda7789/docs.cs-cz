# <a name="using-c-reference"></a>using (Referenční dokumentace jazyka C#)

Klíčové `using` slovo má tři hlavní použití:

- [Příkaz using](using-statement.md) definuje obor, na jehož konci bude objekt uvolněn.
- [Direktiva using](using-directive.md) vytvoří alias pro obor názvů nebo importuje typy definované v jiných oborech názvů.
- [Použití statické směrnice](using-static.md) importuje členy jedné třídy.
