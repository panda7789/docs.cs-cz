# <a name="using-c-reference"></a>using (Referenční dokumentace jazyka C#)

Klíčové slovo `using` má tři hlavní použití:

- [Příkaz using](using-statement.md) definuje obor na konci, z něhož bude objekt uvolněn.
- [Direktiva using](using-directive.md) vytvoří alias pro obor názvů nebo importuje typy definované v jiných oborech názvů.
- [Direktiva using static](using-static.md) importuje členy jedné třídy.
