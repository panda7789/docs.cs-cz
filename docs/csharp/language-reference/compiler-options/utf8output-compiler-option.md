# <a name="-utf8output-c-compiler-options"></a>-utf8output (Možnosti kompilátoru Jazyka C#)
Možnost **-utf8output** zobrazí výstup kompilátoru pomocí kódování UTF-8.  
  
## <a name="syntax"></a>Syntaxe  
  
```console  
-utf8output  
```  
  
## <a name="remarks"></a>Poznámky  
 V některých mezinárodních konfiguracích nelze správně zobrazit výstup kompilátoru v konzole. V těchto konfiguracích použijte **výstup -utf8a** přesměrovat výstup kompilátoru do souboru.  
  
 Tato možnost kompilátoru není k dispozici v sadě Visual Studio a nelze ji programově změnit.
