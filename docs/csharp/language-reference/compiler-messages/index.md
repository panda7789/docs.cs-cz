# <a name="c-compiler-errors"></a>Chyby kompilátoru jazyka C#

Některé chyby kompilátoru Jazyka C# mají odpovídající témata, která vysvětlují, proč je chyba generována, a v některých případech jak chybu opravit. Pomocí jednoho z následujících kroků zjistěte, zda je k dispozici nápověda pro konkrétní chybovou zprávu.  
  
- Najděte číslo chyby (například CS0029) v [okně Výstup](/visualstudio/ide/reference/output-window)a vyhledejte ho v dokumentech Microsoft.  
  
- Zvolte číslo chyby (například CS0029) ve [výstupním okně](/visualstudio/ide/reference/output-window)a pak zvolte klávesu F1.  
  
- V rejstříku zadejte číslo chyby do pole **Hledat.**  
  
 Pokud žádný z těchto kroků nevede k informacím o vaší chybě, přejděte na konec této stránky a odešlete zpětnou vazbu, která obsahuje číslo nebo text chyby.  
  
 Informace o konfiguraci možností chyb a upozornění v c#, naleznete v [tématu Sestavení stránky, Návrhář projektu (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
