# <a name="c-compiler-errors"></a>Chyby kompilátoru jazyka C#

Některé chyby kompilátoru jazyka C# mají odpovídající témata, která vysvětlují, proč se chyba generuje, a v některých případech, jak chybu opravit. Pomocí některého z následujících kroků zjistíte, zda je k dispozici nápovědu pro konkrétní chybovou zprávu.  
  
- Vyhledejte číslo chyby (například CS0029) v [okno výstup](/visualstudio/ide/reference/output-window)a potom ho vyhledejte v Microsoft docs.  
  
- V [okno výstup](/visualstudio/ide/reference/output-window)vyberte číslo chyby (například CS0029) a pak zvolte klávesu F1.  
  
- Do pole index zadejte číslo chyby v poli **Hledat** .  
  
 Pokud žádný z těchto kroků nevede k informacím o vaší chybě, pokračujte na konec této stránky a odešlete zpětnou vazbu, která obsahuje číslo nebo text chyby.  
  
 Informace o tom, jak nakonfigurovat možnosti chyb a upozornění v jazyce C#, naleznete v tématu [Stránka sestavení, Návrhář projektu (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
