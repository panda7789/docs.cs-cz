# <a name="compiler-error-cs0233"></a><span data-ttu-id="feccf-103">Chyba kompilátoru CS0233</span><span class="sxs-lookup"><span data-stu-id="feccf-103">Compiler Error CS0233</span></span>
<span data-ttu-id="feccf-104">identifikátor nemá předdefinovanou velikost, takže sizeof se dá použít jenom v nezabezpečeném kontextu.</span><span class="sxs-lookup"><span data-stu-id="feccf-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="feccf-105">Bez [nebezpečného](../keywords/unsafe.md) kontextu lze operátor [sizeof](../operators/sizeof.md) použít pouze pro typy, jejichž velikost je konstanta v čase kompilace.</span><span class="sxs-lookup"><span data-stu-id="feccf-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="feccf-106">Pokud se tato chyba zobrazuje, použijte nezabezpečený kontext.</span><span class="sxs-lookup"><span data-stu-id="feccf-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="feccf-107">Následující příklad generuje CS0233:</span><span class="sxs-lookup"><span data-stu-id="feccf-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
