 <span data-ttu-id="f95d0-104">Bez [nebezpečné](../keywords/unsafe.md) kontextu [sizeof](../operators/sizeof.md) operátor lze použít pouze pro typy, jejichž velikost je konstanta kompilace.</span><span class="sxs-lookup"><span data-stu-id="f95d0-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="f95d0-105">Pokud se zobrazuje tato chyba, použijte nebezpečný kontext.</span><span class="sxs-lookup"><span data-stu-id="f95d0-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="f95d0-106">Následující příklad generuje CS0233:</span><span class="sxs-lookup"><span data-stu-id="f95d0-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
