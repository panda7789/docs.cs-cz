## <a name="example"></a><span data-ttu-id="726f6-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="726f6-108">Example</span></span>

<span data-ttu-id="726f6-109">Následující ukázka generuje CS0413.</span><span class="sxs-lookup"><span data-stu-id="726f6-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
