<span data-ttu-id="1f411-104">Parametr typu typu Parameter nejde použít s operátorem as, protože nemá omezení typu třída ani omezení class.</span><span class="sxs-lookup"><span data-stu-id="1f411-104">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="1f411-105">K této chybě dochází, pokud obecný typ používá operátor [as](../operators/type-testing-and-cast.md#as-operator) , ale tento obecný typ nemá omezení typu třída.</span><span class="sxs-lookup"><span data-stu-id="1f411-105">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="1f411-106">`as`Operátor je povolen pouze s typy hodnot s možnou hodnotou null, takže parametr typu musí být omezen, aby bylo zaručeno, že se nejedná o hodnotový typ.</span><span class="sxs-lookup"><span data-stu-id="1f411-106">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="1f411-107">Chcete-li se této chybě vyhnout, použijte omezení typu třídy nebo omezení typu odkazu.</span><span class="sxs-lookup"><span data-stu-id="1f411-107">To avoid this error, use a class type constraint or a reference type constraint.</span></span>

## <a name="example"></a><span data-ttu-id="1f411-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="1f411-109">Example</span></span>

<span data-ttu-id="1f411-110">Následující ukázka generuje CS0413.</span><span class="sxs-lookup"><span data-stu-id="1f411-110">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
