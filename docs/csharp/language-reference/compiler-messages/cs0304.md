# <a name="compiler-error-cs0304"></a><span data-ttu-id="446f5-102">Chyba kompilátoru CS0304</span><span class="sxs-lookup"><span data-stu-id="446f5-102">Compiler Error CS0304</span></span>

<span data-ttu-id="446f5-103">Nelze vytvořit instanci typu proměnné typu "type", protože nemá omezení new().</span><span class="sxs-lookup"><span data-stu-id="446f5-103">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="446f5-104">Při implementaci obecné třídy a chcete `new` použít klíčové slovo k vytvoření nové instance libovolného `T`typu, který je zadán `T` pro parametr typu , musíte použít [omezení new()](../keywords/new-constraint.md) v deklaraci třídy, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="446f5-104">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="446f5-105">Omezení `new()` vynucuje bezpečnost typů tím, že `T` zaručuje, že jakýkoli konkrétní typ, který je dodáván pro má výchozí, parametrless konstruktor.</span><span class="sxs-lookup"><span data-stu-id="446f5-105">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="446f5-106">CS0304 dochází, pokud se `new` pokusíte použít operátor v těle třídy `T` k `T` vytvoření instance `new()` parametru typu, když neurčuje omezení.</span><span class="sxs-lookup"><span data-stu-id="446f5-106">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="446f5-107">Na straně klienta pokud se kód pokusí vytvořit instanci obecné třídy s typem, který nemá žádný konstruktor bez parametrů, tento kód vygeneruje [chybu kompilátoru CS0310](./cs0310.md).</span><span class="sxs-lookup"><span data-stu-id="446f5-107">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="446f5-108">Následující příklad generuje CS0304.</span><span class="sxs-lookup"><span data-stu-id="446f5-108">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="446f5-109">Operátor `new` také není povolen v metodách třídy.</span><span class="sxs-lookup"><span data-stu-id="446f5-109">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="446f5-110">Chcete-li se vyhnout chybě, `new()` deklarujte třídu pomocí omezení, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="446f5-110">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
