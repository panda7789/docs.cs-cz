# <a name="compiler-error-cs0051"></a>Chyba kompilátoru CS0051
Nekonzistentní dostupnost: typ parametru 'type' je méně dostupný než metoda "method"  
  
 Návratový typ a každý z typů odkazuje v seznamu formálních parametrů metody musí být přinejmenším stejně dostupná jako metoda sama. Ujistěte se, že typy používané v podpisy metod nejsou omylem privátním z důvodu vynechání `public` modifikátor. Další informace najdete v tématu [modifikátory přístupu](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0051:  
  
```csharp  
// CS0051.cs  
public class A  
{  
    // Try making B public since F is public  
    // B is implicitly private here  
    class B  
    {  
    }  
  
    public static void F(B b)  // CS0051  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
