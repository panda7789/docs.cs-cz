# <a name="compiler-error-cs0034"></a><span data-ttu-id="5aeb3-102">Chyba kompilátoru CS0034</span><span class="sxs-lookup"><span data-stu-id="5aeb3-102">Compiler Error CS0034</span></span>

<span data-ttu-id="5aeb3-103">Operátor 'operátor' je nejednoznačný u operandů typu "type1" a "type2"</span><span class="sxs-lookup"><span data-stu-id="5aeb3-103">Operator 'operator' is ambiguous on operands of type 'type1' and 'type2'</span></span>

 <span data-ttu-id="5aeb3-104">Operátor byl použit na dva objekty a kompilátor našel více než jeden převod.</span><span class="sxs-lookup"><span data-stu-id="5aeb3-104">An operator was used on two objects and the compiler found more than one conversion.</span></span> <span data-ttu-id="5aeb3-105">Vzhledem k tomu, že převody musí být jedinečné, musíte buď vytvořit přetypádku, nebo provést jedno z převodů explicitní.</span><span class="sxs-lookup"><span data-stu-id="5aeb3-105">Because conversions have to be unique, you either have to make a cast or to make one of the conversions explicit.</span></span> <span data-ttu-id="5aeb3-106">Další informace naleznete v [tématu User-defined operátory převodu](../operators/user-defined-conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="5aeb3-106">For more information, see [User-defined conversion operators](../operators/user-defined-conversion-operators.md).</span></span>

 <span data-ttu-id="5aeb3-107">Následující ukázka generuje CS0034:</span><span class="sxs-lookup"><span data-stu-id="5aeb3-107">The following sample generates CS0034:</span></span>

```csharp
// CS0034.cs
public class A
{
    // Allows for the conversion of A object to int.
    public static implicit operator int (A s)
    {
        return 0;
    }

    public static implicit operator string (A i)
    {
        return null;
    }
}

public class B
{
    public static implicit operator int (B s)  
    // One way to resolve this CS0034 is to make one conversion explicit.
    // public static explicit operator int (B s)
    {
        return 0;
    }

    public static implicit operator string (B i)
    {
        return null;
    }

    public static implicit operator B (string i)
    {
        return null;
    }

    public static implicit operator B (int i)
    {
        return null;
    }
}

public class C
{
    public static void Main()
    {
        A a = new A();
        B b = new B();
        b = b + a;   // CS0034
        // Another way to resolve this CS0034 is to make a cast.
        // b = b + (int)a;
    }
}
```
