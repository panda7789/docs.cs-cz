# <a name="compiler-warning-level-1-cs1598"></a>Upozornění kompilátoru (úroveň 1) CS1598
Analyzátor XML nelze načíst z následujícího důvodu: "důvod". Soubor dokumentace XML 'soubor' nebude generován.  
  
 Byla zadána možnost [-doc,](../compiler-options/doc-compiler-option.md) ale kompilátor nemohl najít a načíst soubor msxml3.dll. Zkontrolujte, zda je soubor msxml3.dll nainstalován a zaregistrován.
