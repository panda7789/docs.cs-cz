# <a name="compiler-warning-level-1-cs1598"></a>Upozornění kompilátoru (úroveň 1) CS1598
Analyzátor XML nelze načíst z následujícího důvodu: ' důvod '. Soubor dokumentace XML "File" nebude vygenerován.  
  
 Byla zadána možnost [-doc](../compiler-options/doc-compiler-option.md) , ale kompilátor nemůže najít a načíst msxml3.dll. Ujistěte se, že je soubor msxml3.dll nainstalovaný a registrovaný.
