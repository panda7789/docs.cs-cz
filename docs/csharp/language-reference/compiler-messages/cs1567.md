 <span data-ttu-id="259a0-104">Kompilace buď použila možnost kompilátoru [-win32icon,](../compiler-options/win32icon-compiler-option.md) nebo nepoužila [-win32res](../compiler-options/win32res-compiler-option.md), což způsobí, že kompilátor vygeneruje soubor, který obsahuje informace o prostředku, ale kompilátor nemohl vytvořit soubor z důvodu nedostatku místa na disku nebo jiné chyby.</span><span class="sxs-lookup"><span data-stu-id="259a0-104">Your compilation either used the [-win32icon](../compiler-options/win32icon-compiler-option.md) compiler option or did not use [-win32res](../compiler-options/win32res-compiler-option.md), which causes the compiler to generate a file that contains resource information, but the compiler was unable to create the file due to insufficient disk space or some other error.</span></span>
