# <a name="compiler-warning-level-1-cs3009"></a>Upozornění kompilátoru (úroveň 1) CS3009
'type': základní typ 'typ' není kompatibilní se standardem CLS  
  
 Základní typ byl označen jako nemusí být kompatibilní s common language specification (CLS) v sestavení, které bylo označeno jako kompatibilní se specifikací CLS. Odeberte atribut, který určuje, že sestavení je kompatibilní se standardem CLS, nebo odeberte atribut, který označuje, že typ není kompatibilní se standardem CLS. Další informace o dodržování předpisů v oblasti cls naleznete v [tématu pravidla dodržování předpisů cls](../../../standard/language-independence-and-language-independent-components.md#cls-compliance-rules) a [jazyková nezávislost a jazykově nezávislé komponenty](../../../standard/language-independence.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3009:  
  
```csharp  
// CS3009.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
[CLSCompliant(false)]  
public class B  
{  
}  
  
public class C : B   // CS3009  
{  
    public static void Main () {}  
}  
```
