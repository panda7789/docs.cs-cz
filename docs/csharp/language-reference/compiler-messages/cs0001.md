# <a name="compiler-error-cs0001"></a>Chyba kompilátoru CS0001
Vnitřní chyba kompilátoru

 Pokuste se zjistit, zda se kompilátor nezdařil z důvodu neschopnosti analyzovat neočekávanou syntaxi. Pokud se tato chyba zobrazuje opakovaně, kontaktujte prosím společnost Microsoft.
