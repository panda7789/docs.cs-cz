# <a name="compiler-error-cs1026"></a>Chyba kompilátoru CS1026
) očekávalo se  
  
 Byl nalezen neúplný příkaz.  
  
 Běžnou příčinou této chyby je umístění příkazu, nikoli výrazu, v rámci vloženého výrazu na stránce ASP.NET. Například následující je nesprávné:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 Správné je následující:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 Je interpretována následujícím způsobem:  
  
```aspx-csharp  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 Následující příklad generuje CS1026:  
  
```csharp  
// CS1026.cs  
#if (a == b   // CS1026, add closing )  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
