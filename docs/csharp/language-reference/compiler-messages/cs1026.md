# <a name="compiler-error-cs1026"></a>Chyba kompilátoru CS1026
) očekáváno  
  
 Bylo nalezeno neúplné prohlášení.  
  
 Častou příčinou této chyby je umístění příkazu, nikoli výrazu, do vložkového výrazu na stránce ASP.NET. Například následující je nesprávná:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 Následující je správné:  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 Je interpretován takto:  
  
```aspx-csharp  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 Následující příklad generuje CS1026:  
  
```csharp  
// CS1026.cs  
#if (a == b   // CS1026, add closing )  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
