# <a name="compiler-error-cs1943"></a><span data-ttu-id="eb6dd-102">Chyba kompilátoru CS1943</span><span class="sxs-lookup"><span data-stu-id="eb6dd-102">Compiler Error CS1943</span></span>
<span data-ttu-id="eb6dd-103">Výraz typu "typ" není povolen v následné klauzuli from ve výrazu dotazu se zdrojovým typem "type".</span><span class="sxs-lookup"><span data-stu-id="eb6dd-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="eb6dd-104">Odvození typu se nezdařilo ve volání "metody".</span><span class="sxs-lookup"><span data-stu-id="eb6dd-104">Type inference failed in the call to 'method'.</span></span>

## <a name="example"></a><span data-ttu-id="eb6dd-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="eb6dd-109">Example</span></span>  
 <span data-ttu-id="eb6dd-110">Následující kód generuje CS1943:</span><span class="sxs-lookup"><span data-stu-id="eb6dd-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
