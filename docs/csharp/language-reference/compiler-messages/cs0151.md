## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="40094-106">Příklad nejednoznačného převodu</span><span class="sxs-lookup"><span data-stu-id="40094-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="40094-107">K této chybě může dojít, pokud neexistuje žádný převod nebo pokud dostupné implicitní převody za následek nejednoznačnou situaci.</span><span class="sxs-lookup"><span data-stu-id="40094-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="40094-108">Následující příklad generuje CS0151:</span><span class="sxs-lookup"><span data-stu-id="40094-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="40094-109">Příklad metody void</span><span class="sxs-lookup"><span data-stu-id="40094-109">Example of void method</span></span>

```csharp
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
