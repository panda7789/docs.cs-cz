## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="7df71-107">Příklad nejednoznačného převodu</span><span class="sxs-lookup"><span data-stu-id="7df71-107">Example of ambiguous conversion</span></span>

<span data-ttu-id="7df71-108">K této chybě může dojít v případě, že není k dispozici převod nebo pokud je výsledkem dostupných implicitních převodů nejednoznačná situace.</span><span class="sxs-lookup"><span data-stu-id="7df71-108">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="7df71-109">Následující příklad generuje CS0151:</span><span class="sxs-lookup"><span data-stu-id="7df71-109">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```
