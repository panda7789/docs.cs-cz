## <a name="example"></a><span data-ttu-id="19b6f-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="19b6f-109">Example</span></span>  
 <span data-ttu-id="19b6f-110">Následující příklad ukazuje, jak můžete definovat atribut:</span><span class="sxs-lookup"><span data-stu-id="19b6f-110">The following sample shows how you might define an attribute:</span></span>  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
