# <a name="compiler-error-cs0616"></a>Chyba kompilátoru CS0616
'class' není třída atributů  
  
 Došlo k pokusu o použití třídy bez atributů v bloku atributů. Všechny typy atributů je třeba <xref:System.Attribute?displayProperty=nameWithType>zdědit z .  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0616.  
  
```csharp  
// CS0616.cs  
// compile with: /target:library  
[CMyClass(i = 5)]   // CS0616  
public class CMyClass {}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka ukazuje, jak můžete definovat atribut:  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
