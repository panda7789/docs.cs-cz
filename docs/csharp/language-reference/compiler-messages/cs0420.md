# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="90db0-102">Upozornění kompilátoru (úroveň 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="90db0-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="90db0-103">"identifikátor": odkaz na těkavé pole nebude považován za volatilní</span><span class="sxs-lookup"><span data-stu-id="90db0-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="90db0-104">Těkavé pole by `ref` obvykle nemělo být předáno pomocí parametru nebo **out,** protože nebude považováno za volatilní v rámci funkce.</span><span class="sxs-lookup"><span data-stu-id="90db0-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="90db0-105">Existují výjimky z tohoto, například při volání propojené rozhraní API.</span><span class="sxs-lookup"><span data-stu-id="90db0-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="90db0-106">Stejně jako u jakékoli upozornění, můžete použít [upozornění #pragma](../preprocessor-directives/preprocessor-pragma-warning.md) zakázat toto upozornění v těch výjimečných případech, kdy záměrně používáte těkavé pole jako referenční parametr.</span><span class="sxs-lookup"><span data-stu-id="90db0-106">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="90db0-107">Následující ukázka generuje CS0420:</span><span class="sxs-lookup"><span data-stu-id="90db0-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
