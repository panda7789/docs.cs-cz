<span data-ttu-id="baaad-104">Typ typeName musí být neabstraktního typu s veřejným konstruktorem bez parametrů, aby ho bylo možné použít jako parametr v obecném typu nebo metodě ' GENERIC '.</span><span class="sxs-lookup"><span data-stu-id="baaad-104">The type 'typename' must be a non-abstract type with a public parameterless constructor in order to use it as parameter 'parameter' in the generic type or method 'generic'</span></span>  
  
 <span data-ttu-id="baaad-105">Obecný typ nebo Metoda definuje nové omezení v klauzuli WHERE, takže jakýkoli typ musí mít veřejný konstruktor bez parametrů, aby ho bylo možné použít jako argument typu pro tento obecný typ nebo metodu.</span><span class="sxs-lookup"><span data-stu-id="baaad-105">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="baaad-106">Chcete-li se této chybě vyhnout, ujistěte se, že typ má správný konstruktor, nebo upravte klauzuli omezení obecného typu nebo metody.</span><span class="sxs-lookup"><span data-stu-id="baaad-106">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="baaad-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="baaad-107">Example</span></span>  
 <span data-ttu-id="baaad-108">Následující ukázka generuje CS0310:</span><span class="sxs-lookup"><span data-stu-id="baaad-108">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
