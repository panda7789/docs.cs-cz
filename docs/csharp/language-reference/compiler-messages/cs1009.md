# <a name="compiler-error-cs1009"></a><span data-ttu-id="252ef-103">Chyba kompilátoru CS1009</span><span class="sxs-lookup"><span data-stu-id="252ef-103">Compiler Error CS1009</span></span>
<span data-ttu-id="252ef-104">Nerozpoznaná řídicí sekvence</span><span class="sxs-lookup"><span data-stu-id="252ef-104">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="252ef-105">Neočekávaný znak následuje zpětné lomítko ( \\ ) v [řetězci](../builtin-types/reference-types.md#the-string-type).</span><span class="sxs-lookup"><span data-stu-id="252ef-105">An unexpected character follows a backslash (\\) in a [string](../builtin-types/reference-types.md#the-string-type).</span></span> <span data-ttu-id="252ef-106">Kompilátor očekává jeden z platných řídicích znaků.</span><span class="sxs-lookup"><span data-stu-id="252ef-106">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="252ef-107">Další informace naleznete v tématu [znakové řídicí znaky](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="252ef-107">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="252ef-108">Následující ukázka generuje CS1009.</span><span class="sxs-lookup"><span data-stu-id="252ef-108">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="252ef-109">Běžnou příčinou této chyby je použití znaku zpětného lomítka v názvu souboru, jak ukazuje následující příklad.</span><span class="sxs-lookup"><span data-stu-id="252ef-109">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="252ef-110">Chcete-li vyřešit tuto chybu, použijte " \\ \\ " nebo @-quoted řetězcový literál, jak ukazuje následující příklad.</span><span class="sxs-lookup"><span data-stu-id="252ef-110">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
