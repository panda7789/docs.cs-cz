# <a name="compiler-error-cs1009"></a>Chyba kompilátoru CS1009
Nerozpoznaná sekvence escape  
  
 Neočekávaný znak následuje zpětné\\lomítko ( ) v [řetězci](../builtin-types/reference-types.md#the-string-type). Kompilátor očekává jeden z platných řídicích znaků. Další informace naleznete v [tématu Znak escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).  
  
 Následující ukázka generuje CS1009.  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 Častou příčinou této chyby je použití znaku zpětného lomítka v názvu souboru, jak ukazuje následující příklad.  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 Chcete-li tuto chybu\\\\vyřešit, použijte " " nebo @-quoted literál řetězce, jak ukazuje následující příklad.  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
