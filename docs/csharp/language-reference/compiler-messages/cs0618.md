# <a name="compiler-warning-level-2-cs0618"></a>Upozornění kompilátoru (úroveň 2) CS0618
"člen" je zastaralý: "text"  
  
 Člen třídy byl `Obsolete` označen atributem, takže při odkazování na člen třídy bude vydáno upozornění. Další informace naleznete [v tématu Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).  
  
 Následující ukázka generuje CS0618:  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
