# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="84552-102">Upozornění kompilátoru (úroveň 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="84552-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="84552-103">"člen" je zastaralý: "text"</span><span class="sxs-lookup"><span data-stu-id="84552-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="84552-104">Člen třídy byl `Obsolete` označen atributem, takže při odkazování na člen třídy bude vydáno upozornění.</span><span class="sxs-lookup"><span data-stu-id="84552-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="84552-105">Další informace naleznete [v tématu Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="84552-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="84552-106">Následující ukázka generuje CS0618:</span><span class="sxs-lookup"><span data-stu-id="84552-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
