<span data-ttu-id="7db76-104">člen je zastaralý: text.</span><span class="sxs-lookup"><span data-stu-id="7db76-104">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="7db76-105">Člen třídy byl označen `Obsolete` atributem, což znamená, že upozornění bude vydáno při odkazování na člena třídy.</span><span class="sxs-lookup"><span data-stu-id="7db76-105">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="7db76-106">Další informace najdete v tématu [běžné atributy](../attributes/global.md).</span><span class="sxs-lookup"><span data-stu-id="7db76-106">For more information, see [Common Attributes](../attributes/global.md).</span></span>  
  
 <span data-ttu-id="7db76-107">Následující ukázka generuje CS0618:</span><span class="sxs-lookup"><span data-stu-id="7db76-107">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
