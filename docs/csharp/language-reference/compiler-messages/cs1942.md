## <a name="to-correct-this-error"></a><span data-ttu-id="bef61-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="bef61-106">To correct this error</span></span>

1. <span data-ttu-id="bef61-107">Pokud je proměnná rozsahu explicitně zadána, ujistěte se, že typ je buď stejný jako nebo implicitně převoditelný z typu prvků v kolekci, které iterates.</span><span class="sxs-lookup"><span data-stu-id="bef61-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="bef61-108">Pokud je proměnné rozsahu předlohou s klíčovým slovem, `var` odeberte `var`.</span><span class="sxs-lookup"><span data-stu-id="bef61-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bef61-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="bef61-109">Example</span></span>

<span data-ttu-id="bef61-110">Následující kód generuje CS1942:</span><span class="sxs-lookup"><span data-stu-id="bef61-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="bef61-111">CS1942 souvisí s CS1949, `var` protože použití s `Cast<T>` proměnnou rozsahu `var` způsobí selhání základní operace, protože není typ.</span><span class="sxs-lookup"><span data-stu-id="bef61-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
