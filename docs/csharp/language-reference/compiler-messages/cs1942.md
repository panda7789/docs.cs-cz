## <a name="example"></a><span data-ttu-id="e1130-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="e1130-110">Example</span></span>

<span data-ttu-id="e1130-111">Následující kód generuje CS1942:</span><span class="sxs-lookup"><span data-stu-id="e1130-111">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="e1130-112">CS1942 se vztahuje k CS1949, protože použití `var` s proměnnou rozsahu způsobí selhání základní operace, `Cast<T>` protože se nejedná `var` o typ.</span><span class="sxs-lookup"><span data-stu-id="e1130-112">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
