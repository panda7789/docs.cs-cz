## <a name="example"></a><span data-ttu-id="2ebf4-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="2ebf4-105">Example</span></span>  
 <span data-ttu-id="2ebf4-106">Následující příklad generuje CS1018 a navrhuje několik způsobů, jak chybu vyřešit:</span><span class="sxs-lookup"><span data-stu-id="2ebf4-106">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's parameterless constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
