# <a name="compiler-error-cs8400"></a>Chyba kompilátoru CS8400

Funkce ' Feature ' není v jazyce C# 8,0 k dispozici. Použijte prosím jazyk verze \<version> nebo vyšší.

## <a name="to-correct-this-error"></a>Oprava této chyby

Upgradujte projekt alespoň na verzi uvedenou v chybě.
