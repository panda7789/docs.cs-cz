 <span data-ttu-id="954e4-105">Pokud je proměnná [const](../keywords/const.md), je také [statická](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="954e4-105">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="954e4-106">Pokud chcete proměnnou **const** a **static** , stačí deklarovat tuto proměnnou jako **const**; Pokud je vše, co potřebujete, je **statická** proměnná, stačí ji označit jako **statickou**.</span><span class="sxs-lookup"><span data-stu-id="954e4-106">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>
