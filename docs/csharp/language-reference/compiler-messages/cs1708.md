 <span data-ttu-id="16dad-105">Nová funkce v jazyce C# 2,0 je možnost definovat vložené pole uvnitř `struct` .</span><span class="sxs-lookup"><span data-stu-id="16dad-105">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="16dad-106">K těmto polím lze přistupovat pouze prostřednictvím místních proměnných nebo polí a na levé straně výrazu nelze odkazovat jako na mezilehlé hodnoty.</span><span class="sxs-lookup"><span data-stu-id="16dad-106">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="16dad-107">K polím také nelze přicházet pomocí polí, která jsou `static` nebo `readonly` .</span><span class="sxs-lookup"><span data-stu-id="16dad-107">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>

## <a name="example"></a><span data-ttu-id="16dad-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="16dad-110">Example</span></span>  
 <span data-ttu-id="16dad-111">Následující ukázka generuje CS1708.</span><span class="sxs-lookup"><span data-stu-id="16dad-111">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
