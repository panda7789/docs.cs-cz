# <a name="compiler-error-cs1708"></a><span data-ttu-id="e99d0-102">Chyba kompilátoru CS1708</span><span class="sxs-lookup"><span data-stu-id="e99d0-102">Compiler Error CS1708</span></span>
<span data-ttu-id="e99d0-103">Vyrovnávací paměti s pevnou velikostí lze přistupovat pouze prostřednictvím místních obyvatel nebo polí.</span><span class="sxs-lookup"><span data-stu-id="e99d0-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="e99d0-104">Novou funkcí v c# 2.0 je schopnost definovat in-line `struct`pole uvnitř .</span><span class="sxs-lookup"><span data-stu-id="e99d0-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="e99d0-105">Tato pole lze přistupovat pouze prostřednictvím místních proměnných nebo polí a nelze odkazovat jako mezilehlé hodnoty na levé straně výrazu.</span><span class="sxs-lookup"><span data-stu-id="e99d0-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="e99d0-106">Pole také nelze přistupovat pole, která `static` `readonly`jsou nebo .</span><span class="sxs-lookup"><span data-stu-id="e99d0-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="e99d0-107">Chcete-li tuto chybu vyřešit, definujte proměnnou pole a přiřaďte proměnné in-line pole.</span><span class="sxs-lookup"><span data-stu-id="e99d0-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="e99d0-108">Nebo odeberte modifikátor `static` nebo `readonly` z pole představujícího in-line pole.</span><span class="sxs-lookup"><span data-stu-id="e99d0-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e99d0-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="e99d0-109">Example</span></span>  
 <span data-ttu-id="e99d0-110">Následující ukázka generuje CS1708.</span><span class="sxs-lookup"><span data-stu-id="e99d0-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
