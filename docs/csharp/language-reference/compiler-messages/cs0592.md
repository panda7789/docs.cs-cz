<span data-ttu-id="ccc91-104">Atribut Attribute není pro tento typ deklarace platný.</span><span class="sxs-lookup"><span data-stu-id="ccc91-104">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="ccc91-105">Je platný pouze v deklaracích Type.</span><span class="sxs-lookup"><span data-stu-id="ccc91-105">It is valid on 'type' declarations only.</span></span>

## <a name="example"></a><span data-ttu-id="ccc91-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="ccc91-109">Example</span></span>

 <span data-ttu-id="ccc91-110">Následující ukázka generuje CS0592:</span><span class="sxs-lookup"><span data-stu-id="ccc91-110">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
