## <a name="example"></a><span data-ttu-id="ee276-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="ee276-108">Example</span></span>

```csharp
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}

{  
    public static void Main()  
    {  
    }  
}  
```
