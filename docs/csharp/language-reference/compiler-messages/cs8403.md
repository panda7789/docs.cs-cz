# <a name="compiler-error-cs8403"></a>Chyba kompilátoru CS8403

Metoda ' Method ' s blokem iterátoru musí být ' Async ', aby vrátila ' {IAsyncEnumerable \<T> | IAsyncEnumerator \<T> }

## <a name="to-correct-this-error"></a>Oprava této chyby

Označte metodu pomocí `async` modifikátoru.
