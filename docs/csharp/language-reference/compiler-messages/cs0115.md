## <a name="example"></a><span data-ttu-id="1e95d-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="1e95d-106">Example</span></span>  
 <span data-ttu-id="1e95d-107">Následující ukázka generuje CS0115.</span><span class="sxs-lookup"><span data-stu-id="1e95d-107">The following sample generates CS0115.</span></span> <span data-ttu-id="1e95d-108">Abyste mohli vyřešit CS0115 v jednom ze dvou způsobů:</span><span class="sxs-lookup"><span data-stu-id="1e95d-108">You can resolve CS0115 in one of two ways:</span></span>

```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
