# <a name="compiler-error-cs0260"></a><span data-ttu-id="90625-103">Chyba kompilátoru CS0260</span><span class="sxs-lookup"><span data-stu-id="90625-103">Compiler Error CS0260</span></span>

<span data-ttu-id="90625-104">Chybí částečný modifikátor deklarace typu Type; Existuje další částečná deklarace tohoto typu.</span><span class="sxs-lookup"><span data-stu-id="90625-104">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="90625-105">Tato chyba označuje, že jste deklarovali více tříd, které mají stejný název.</span><span class="sxs-lookup"><span data-stu-id="90625-105">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="90625-106">Kromě toho alespoň jedna z deklarací obsahuje `partial` modifikátor.</span><span class="sxs-lookup"><span data-stu-id="90625-106">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="90625-107">Chcete-li definovat třídu v několika částech, je nutné deklarovat každou část pomocí klíčového slova `partial` .</span><span class="sxs-lookup"><span data-stu-id="90625-107">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="90625-108">K této chybě dochází také v případě, že deklarujete třídu a omylem jí dáte stejný název jako částečná třída, která je deklarována jinde ve stejném oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="90625-108">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="90625-109">Následující ukázka generuje CS0260:</span><span class="sxs-lookup"><span data-stu-id="90625-109">The following sample generates CS0260:</span></span>  

```csharp
// CS0260.cs  
// You must mark both parts of the definition of class C
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C
{  
}  
  
partial class C  
{  
}  
```
