<span data-ttu-id="c19c6-105">Při explicitní implementaci [události](../keywords/event.md) , která byla deklarována v rozhraní, je nutné ručně poskytnout `add` `remove` přístupové objekty události a, které jsou obvykle poskytovány kompilátorem.</span><span class="sxs-lookup"><span data-stu-id="c19c6-105">When explicitly implementing an [event](../keywords/event.md) that was declared in an interface, you must manually provide the `add` and `remove` event accessors that are typically provided by the compiler.</span></span> <span data-ttu-id="c19c6-106">Přístupový kód může připojit událost rozhraní k jiné události ve třídě (viz dále v tomto tématu) nebo vlastnímu typu delegáta.</span><span class="sxs-lookup"><span data-stu-id="c19c6-106">The accessor code can connect the interface event to another event in your class (shown later in this topic) or to its own delegate type.</span></span> <span data-ttu-id="c19c6-107">Další informace naleznete v tématu [jak implementovat události rozhraní](../../programming-guide/events/how-to-implement-interface-events.md).</span><span class="sxs-lookup"><span data-stu-id="c19c6-107">For more information, see [How to implement interface events](../../programming-guide/events/how-to-implement-interface-events.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="c19c6-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="c19c6-108">Example</span></span>

 <span data-ttu-id="c19c6-109">Následující ukázka generuje CS0071.</span><span class="sxs-lookup"><span data-stu-id="c19c6-109">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
