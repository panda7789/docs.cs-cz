# <a name="compiler-error-cs0071"></a>Chyba kompilátoru CS0071

Explicitní implementace rozhraní události musí používat syntaxi přistupujícího objektu události.

Při explicitní implementaci [události](../keywords/event.md) , která byla deklarována v rozhraní, je nutné ručně poskytnout `add` a `remove` přístup k událostem, které jsou obvykle poskytovány kompilátorem. Přístupový kód může připojit událost rozhraní k jiné události ve třídě (viz dále v tomto tématu) nebo vlastnímu typu delegáta. Další informace naleznete v tématu [jak implementovat události rozhraní](../../programming-guide/events/how-to-implement-interface-events.md).
  
## <a name="example"></a>Příklad

 Následující ukázka generuje CS0071.

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
