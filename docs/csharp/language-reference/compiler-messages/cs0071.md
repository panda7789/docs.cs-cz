<span data-ttu-id="09a5c-104">Při explicitní implementaci [události,](../keywords/event.md) která byla deklarována `add` v `remove` rozhraní, je nutné ručně poskytnout přístupové objekty a události, které jsou obvykle poskytovány kompilátorem.</span><span class="sxs-lookup"><span data-stu-id="09a5c-104">When explicitly implementing an [event](../keywords/event.md) that was declared in an interface, you must manually provide the `add` and `remove` event accessors that are typically provided by the compiler.</span></span> <span data-ttu-id="09a5c-105">Přistupující kód můžete připojit událost rozhraní k jiné události ve vaší třídě (dále v tomto tématu) nebo k jeho vlastní typ delegáta.</span><span class="sxs-lookup"><span data-stu-id="09a5c-105">The accessor code can connect the interface event to another event in your class (shown later in this topic) or to its own delegate type.</span></span> <span data-ttu-id="09a5c-106">Další informace naleznete v tématu [Jak implementovat události rozhraní](../../programming-guide/events/how-to-implement-interface-events.md).</span><span class="sxs-lookup"><span data-stu-id="09a5c-106">For more information, see [How to implement interface events](../../programming-guide/events/how-to-implement-interface-events.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="09a5c-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="09a5c-107">Example</span></span>

 <span data-ttu-id="09a5c-108">Následující ukázka generuje CS0071.</span><span class="sxs-lookup"><span data-stu-id="09a5c-108">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
