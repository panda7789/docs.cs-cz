# <a name="compiler-error-cs0579"></a><span data-ttu-id="75b02-102">Chyba kompilátoru CS0579</span><span class="sxs-lookup"><span data-stu-id="75b02-102">Compiler Error CS0579</span></span>
<span data-ttu-id="75b02-103">Duplikovat atribut atribut</span><span class="sxs-lookup"><span data-stu-id="75b02-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="75b02-104">Není možné zadat stejný atribut více než jednou, pokud atribut určuje **AllowMultiple=true** v jeho [AttributeUsage](../../language-reference/attributes/general.md).</span><span class="sxs-lookup"><span data-stu-id="75b02-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../language-reference/attributes/general.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="75b02-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="75b02-105">Example</span></span>  
 <span data-ttu-id="75b02-106">Následující příklad generuje CS0579.</span><span class="sxs-lookup"><span data-stu-id="75b02-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
