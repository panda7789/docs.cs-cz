# <a name="compiler-error-cs0579"></a><span data-ttu-id="2c7f5-103">Chyba kompilátoru CS0579</span><span class="sxs-lookup"><span data-stu-id="2c7f5-103">Compiler Error CS0579</span></span>
<span data-ttu-id="2c7f5-104">Duplicitní atribut Attribute</span><span class="sxs-lookup"><span data-stu-id="2c7f5-104">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="2c7f5-105">Stejný atribut není možné zadat více než jednou, pokud atribut neurčuje vlastnost **AllowMultiple = true** v jeho [AttributeUsage](../attributes/general.md).</span><span class="sxs-lookup"><span data-stu-id="2c7f5-105">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../attributes/general.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="2c7f5-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="2c7f5-106">Example</span></span>  
 <span data-ttu-id="2c7f5-107">Následující příklad generuje CS0579.</span><span class="sxs-lookup"><span data-stu-id="2c7f5-107">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
