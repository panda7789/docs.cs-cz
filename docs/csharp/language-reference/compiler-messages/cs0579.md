# <a name="compiler-error-cs0579"></a>Chyba kompilátoru CS0579
Duplicitní atribut Attribute  
  
 Stejný atribut není možné zadat více než jednou, pokud atribut neurčuje vlastnost **AllowMultiple = true** v jeho [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0579.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
