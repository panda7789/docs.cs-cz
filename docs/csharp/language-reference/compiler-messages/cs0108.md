# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="18d0c-102">Upozornění kompilátoru (úroveň 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="18d0c-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="18d0c-103">'member1' skryje zděděný člen 'member2'.</span><span class="sxs-lookup"><span data-stu-id="18d0c-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="18d0c-104">Nové klíčové slovo použijte, pokud bylo zamýšleno skrytí.</span><span class="sxs-lookup"><span data-stu-id="18d0c-104">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="18d0c-105">Proměnná byla deklarována se stejným názvem jako proměnná v základní třídě.</span><span class="sxs-lookup"><span data-stu-id="18d0c-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="18d0c-106">[Nové](../keywords/new-modifier.md) klíčové slovo však nebylo použito.</span><span class="sxs-lookup"><span data-stu-id="18d0c-106">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="18d0c-107">Toto upozornění vás informuje, že byste měli použít **nové**; proměnná je deklarována, jako by byla v prohlášení použita **nová.**</span><span class="sxs-lookup"><span data-stu-id="18d0c-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="18d0c-108">Následující ukázka generuje CS0108:</span><span class="sxs-lookup"><span data-stu-id="18d0c-108">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
