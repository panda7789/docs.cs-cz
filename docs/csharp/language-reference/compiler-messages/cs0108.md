<span data-ttu-id="6a8b4-104">' člen1 ' skrývá zděděný člen ' member2 '.</span><span class="sxs-lookup"><span data-stu-id="6a8b4-104">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="6a8b4-105">Pokud je skrytí určené, použijte klíčové slovo New.</span><span class="sxs-lookup"><span data-stu-id="6a8b4-105">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="6a8b4-106">Proměnná byla deklarována se stejným názvem jako proměnná v základní třídě.</span><span class="sxs-lookup"><span data-stu-id="6a8b4-106">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="6a8b4-107">Klíčové slovo [New](../keywords/new-modifier.md) se ale nepoužilo.</span><span class="sxs-lookup"><span data-stu-id="6a8b4-107">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="6a8b4-108">Toto upozornění vás informuje o tom, že byste měli použít **Nový**. proměnná je deklarována jako if **New** použit v deklaraci.</span><span class="sxs-lookup"><span data-stu-id="6a8b4-108">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="6a8b4-109">Následující ukázka generuje CS0108:</span><span class="sxs-lookup"><span data-stu-id="6a8b4-109">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
