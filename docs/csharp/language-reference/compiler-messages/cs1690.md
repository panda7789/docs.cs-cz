# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="d36f1-102">Upozornění kompilátoru (úroveň 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="d36f1-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="d36f1-103">Přístup k členu v "member" může způsobit výjimku za běhu, protože se jedná o pole třídy marshal-by-reference</span><span class="sxs-lookup"><span data-stu-id="d36f1-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="d36f1-104">K tomuto upozornění dochází při pokusu o volání metody, vlastnosti nebo indexeru na člena třídy, která je odvozena od <xref:System.MarshalByRefObject>, a člen je typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="d36f1-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="d36f1-105">Objekty, `MarshalByRefObject` které dědí z jsou obvykle určeny k zařazování odkazem napříč doménou aplikace.</span><span class="sxs-lookup"><span data-stu-id="d36f1-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="d36f1-106">Pokud se jakýkoli kód někdy pokusí o přímý přístup k členu typu hodnoty <xref:System.InvalidOperationException> takového objektu v doméně aplikace, dojde k běhu.</span><span class="sxs-lookup"><span data-stu-id="d36f1-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="d36f1-107">Chcete-li vyřešit upozornění, nejprve zkopírujte člen do místní proměnné a volání metody na této proměnné.</span><span class="sxs-lookup"><span data-stu-id="d36f1-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="d36f1-108">Následující ukázka generuje CS1690:</span><span class="sxs-lookup"><span data-stu-id="d36f1-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
