# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="8ec58-103">Upozornění kompilátoru (úroveň 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="8ec58-103">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="8ec58-104">Přetížená metoda "method" lišící se pouze typy nepojmenovaného pole není kompatibilní se specifikací CLS</span><span class="sxs-lookup"><span data-stu-id="8ec58-104">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="8ec58-105">K této chybě dochází, pokud existuje přetížená metoda, která přebírá nepravidelné pole, a jediným rozdílem mezi podpisy metod je typ prvku tohoto pole.</span><span class="sxs-lookup"><span data-stu-id="8ec58-105">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="8ec58-106">Chcete-li se této chybě vyhnout, zvažte použití obdélníkového pole místo vícenásobného pole; pro jednoznačné volání funkce použijte další parametr; Přejmenujte jednu nebo více přetížených metod; nebo, pokud není nutné dodržování specifikace CLS, odeberte <xref:System.CLSCompliantAttribute> atribut.</span><span class="sxs-lookup"><span data-stu-id="8ec58-106">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="8ec58-107">Další informace o kompatibilitě se specifikací CLS naleznete v tématu [nezávislost jazyků a jazykově nezávislé komponenty](../../../standard/language-independence-and-language-independent-components.md).</span><span class="sxs-lookup"><span data-stu-id="8ec58-107">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="8ec58-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="8ec58-108">Example</span></span>  
 <span data-ttu-id="8ec58-109">Následující příklad generuje CS3007:</span><span class="sxs-lookup"><span data-stu-id="8ec58-109">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
