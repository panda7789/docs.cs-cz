## <a name="example"></a><span data-ttu-id="cc002-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="cc002-109">Example</span></span>  
 <span data-ttu-id="cc002-110">Tato ukázka kódu se skládá ze souboru. cpp, který kompiluje do knihovny DLL a souboru. cs, který používá tuto knihovnu DLL.</span><span class="sxs-lookup"><span data-stu-id="cc002-110">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="cc002-111">Následující kód je pro soubor. dll a definuje vlastnost, která má být k dispozici v kódu v souboru. cs.</span><span class="sxs-lookup"><span data-stu-id="cc002-111">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
