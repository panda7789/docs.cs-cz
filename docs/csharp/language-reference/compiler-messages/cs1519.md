# <a name="compiler-error-cs1519"></a><span data-ttu-id="2868b-103">Chyba kompilátoru CS1519</span><span class="sxs-lookup"><span data-stu-id="2868b-103">Compiler Error CS1519</span></span>
<span data-ttu-id="2868b-104">Neplatný token ' token ' v deklaraci člena třídy, struktury nebo člena rozhraní</span><span class="sxs-lookup"><span data-stu-id="2868b-104">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="2868b-105">Tato chyba je vygenerována při každém výskytu tokenu v umístění, kam nepatří.</span><span class="sxs-lookup"><span data-stu-id="2868b-105">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="2868b-106">*Token* je klíčové slovo; identifikátor (název třídy, struktury, metody a tak dále); řetězec, znak nebo číselná hodnota literálu, například 108, "Hello" nebo "A"; nebo operátor nebo punctuator, například `==` nebo `;` .</span><span class="sxs-lookup"><span data-stu-id="2868b-106">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="2868b-107">Jakákoli deklarace člena [třídy](../keywords/class.md), struktury nebo členu rozhraní, která obsahuje neplatné modifikátory předtím, než typ vygeneruje tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="2868b-107">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="2868b-108">Chcete-li chybu opravit, odeberte neplatné modifikátory.</span><span class="sxs-lookup"><span data-stu-id="2868b-108">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="2868b-109">Následující ukázka vygeneruje CS1519 na pět míst, protože tokeny jsou umístěny na místech, kde nejsou platná:</span><span class="sxs-lookup"><span data-stu-id="2868b-109">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
