# <a name="compiler-error-cs1519"></a><span data-ttu-id="06773-102">Chyba kompilátoru CS1519</span><span class="sxs-lookup"><span data-stu-id="06773-102">Compiler Error CS1519</span></span>
<span data-ttu-id="06773-103">Neplatný token 'token' ve třídě, struktury nebo deklarace člena rozhraní</span><span class="sxs-lookup"><span data-stu-id="06773-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="06773-104">Tato chyba je generována vždy, když je token zjištěn v umístění, kam nepatří.</span><span class="sxs-lookup"><span data-stu-id="06773-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="06773-105">*Token* je klíčové slovo; identifikátor (název třídy, struktury, metody a tak dále); řetězec, znak nebo číselné literálové hodnoty, například 108, "Hello" nebo "A"; nebo operátor nebo interpunkční znaménko, jako `==` je nebo `;`.</span><span class="sxs-lookup"><span data-stu-id="06773-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="06773-106">Všechny [třídy](../keywords/class.md), struct nebo deklarace člena rozhraní, která obsahuje neplatné modifikátory před typem bude generovat tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="06773-106">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="06773-107">Chcete-li chybu opravit, odeberte neplatné modifikátory.</span><span class="sxs-lookup"><span data-stu-id="06773-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="06773-108">Následující ukázka generuje CS1519 na pěti místech, protože tokeny jsou umístěny v místech, kde nejsou platné:</span><span class="sxs-lookup"><span data-stu-id="06773-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
