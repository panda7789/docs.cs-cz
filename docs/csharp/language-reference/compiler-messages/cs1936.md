## <a name="example"></a><span data-ttu-id="cfd1f-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="cfd1f-110">Example</span></span>

<span data-ttu-id="cfd1f-111">Následující příklad vytváří CS1936:</span><span class="sxs-lookup"><span data-stu-id="cfd1f-111">The following example produces CS1936:</span></span>  

```csharp

class Test  
{  
    static int Main()

        object obj;

                        select x;  
        return 0;  
    }
```
