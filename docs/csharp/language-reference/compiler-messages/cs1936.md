## <a name="example"></a><span data-ttu-id="bcfec-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="bcfec-111">Example</span></span>

<span data-ttu-id="bcfec-112">Následující příklad vytvoří CS1936:</span><span class="sxs-lookup"><span data-stu-id="bcfec-112">The following example produces CS1936:</span></span>  

```csharp

class Test  
{  
    static int Main()
```
