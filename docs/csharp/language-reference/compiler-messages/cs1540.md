# <a name="compiler-error-cs1540"></a><span data-ttu-id="f00fc-102">Chyba kompilátoru CS1540</span><span class="sxs-lookup"><span data-stu-id="f00fc-102">Compiler Error CS1540</span></span>
<span data-ttu-id="f00fc-103">Nelze získat přístup k chráněnému členu "member member" prostřednictvím kvalifikátoru typu "type1"; kvalifikátor musí být typu "type2" (nebo z něj odvozen)</span><span class="sxs-lookup"><span data-stu-id="f00fc-103">Cannot access protected member 'member' via a qualifier of type 'type1'; the qualifier must be of type 'type2' (or derived from it)</span></span>  
  
 <span data-ttu-id="f00fc-104">Odvozená [třída](../keywords/class.md) nemůže získat přístup k chráněným členům své základní třídy prostřednictvím instance základní třídy.</span><span class="sxs-lookup"><span data-stu-id="f00fc-104">A derived [class](../keywords/class.md) cannot access protected members of its base class through an instance of the base class.</span></span> <span data-ttu-id="f00fc-105">Instance základní třídy deklarované v odvozené třídě může být za běhu instancí jiného typu, který je odvozen ze stejného základu, ale jinak nesouvisí s odvozenou třídou.</span><span class="sxs-lookup"><span data-stu-id="f00fc-105">An instance of the base class declared in the derived class might, at run time, be an instance of another type that is derived from the same base but is not otherwise related to the derived class.</span></span> <span data-ttu-id="f00fc-106">Vzhledem k tomu, že chráněné členy lze přistupovat pouze odvozené typy, všechny pokusy o přístup k chráněným členům, které nemusí být platné za běhu jsou označeny kompilátorem jako neplatné.</span><span class="sxs-lookup"><span data-stu-id="f00fc-106">Because protected members can be accessed only by derived types, any attempts to access protected members that might not be valid at run time are marked by the compiler as not valid.</span></span>  
  
 <span data-ttu-id="f00fc-107">Ve `Employee` třídě v následujícím `emp2` příkladu `emp3` a `Person` oba mají `emp2` typ `Manager` v době kompilace, ale má typ za běhu.</span><span class="sxs-lookup"><span data-stu-id="f00fc-107">In the `Employee` class in the following example, `emp2` and `emp3` both have type `Person` at compile time, but `emp2` has type `Manager` at run time.</span></span> <span data-ttu-id="f00fc-108">Protože `Employee` není odvozen `Manager`od , nemůže získat přístup k `Person`chráněným členům `Manager` základní třídy , prostřednictvím instance třídy.</span><span class="sxs-lookup"><span data-stu-id="f00fc-108">Because `Employee` is not derived from `Manager`, it cannot access the protected members of the base class, `Person`, through an instance of the `Manager` class.</span></span> <span data-ttu-id="f00fc-109">Kompilátor nemůže určit, jaký typ běhu `Person` dvou objektů bude.</span><span class="sxs-lookup"><span data-stu-id="f00fc-109">The compiler cannot determine what the run-time type of the two `Person` objects will be.</span></span> <span data-ttu-id="f00fc-110">Proto volání z `emp2` a volání `emp3` z příčiny chyba kompilátoru CS1540.</span><span class="sxs-lookup"><span data-stu-id="f00fc-110">Therefore, both the call from `emp2` and the call from `emp3` cause compiler error CS1540.</span></span>  
  
```csharp  
using System;  
using System.Text;  
  
namespace CS1540  
{  
    class Program1  
    {  
        static void Main()  
        {  
            Employee.PreparePayroll();  
        }  
    }  
  
    class Person  
    {  
        protected virtual void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Person class.");  
        }  
    }  
  
    class Manager : Person  
    {  
        protected override void CalculatePay()
        {  
            Console.WriteLine("CalculatePay in Manager class.");
  
        }  
    }  
  
    class Employee : Person  
    {  
        public static void PreparePayroll()  
        {  
            Employee emp1 = new Employee();  
            Person emp2 = new Manager();  
            Person emp3 = new Employee();  
            // The following line calls the method in the Employee base class,  
            // Person.  
            emp1.CalculatePay();
  
            // The following lines cause compiler error CS1540. The compiler
            // cannot determine at compile time what the run-time types of
            // emp2 and emp3 will be.  
            //emp2.CalculatePay();
            //emp3.CalculatePay();  
  
        }  
    }  
}  
```
