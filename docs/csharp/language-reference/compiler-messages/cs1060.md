## <a name="example"></a><span data-ttu-id="5650d-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="5650d-111">Example</span></span>  
 <span data-ttu-id="5650d-112">Následující kód vygeneruje CS1060, protože typ třídy `U` je členem skupiny `struct S` , ale nikdy je inicializována.</span><span class="sxs-lookup"><span data-stu-id="5650d-112">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>  
  
```csharp

namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
