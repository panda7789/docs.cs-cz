# <a name="compiler-warning-level-1-cs1060"></a>Upozornění kompilátoru (úroveň 1) CS1060
Použití pravděpodobně nepřiřazeného pole Name. Proměnné instance struktury jsou zpočátku nepřiřazené, pokud je struktura Nepřiřazená.  
  
 Členy struktury jsou inicializovány na výchozí hodnotu, pokud je neinicializujete explicitně. Výchozí hodnota pro typy třídy (a další typy odkazu) má hodnotu null. Pokud třída není inicializována před jakýmkoli pokusem o přístup k ní, bude vyvolána `NullReferenceException` za běhu. Kompilátor nemůže určit konečně, jestli se člen třídy inicializuje nebo ne, a CS1060 je upozornění a ne chyba.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Poskytněte konstruktor pro `struct`, který inicializuje všechny jeho členy.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje CS1060, protože typ třídy `U` je členem `struct S`, ale není nikdy inicializován.  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
