# <a name="compiler-error-cs0188"></a><span data-ttu-id="42599-103">Chyba kompilátoru CS0188</span><span class="sxs-lookup"><span data-stu-id="42599-103">Compiler Error CS0188</span></span>

<span data-ttu-id="42599-104">Objekt this nejde použít dřív, než se přiřadí všechna jeho pole.</span><span class="sxs-lookup"><span data-stu-id="42599-104">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="42599-105">Všechna pole v `struct` musí být přiřazena konstruktorem před tím, než konstruktor může volat metodu v `struct` .</span><span class="sxs-lookup"><span data-stu-id="42599-105">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="42599-106">Pokud se zobrazí tato chyba při pokusu o inicializaci vlastnosti v konstruktoru struktury, je řešením inicializovat pole zálohování této vlastnosti přímo namísto samotné vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="42599-106">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="42599-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="42599-107">Example</span></span>

 <span data-ttu-id="42599-108">Následující ukázka generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="42599-108">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
