# <a name="compiler-error-cs0188"></a>Chyba kompilátoru CS0188

Objekt "This" nelze použít před přiřazením všech jeho polí  
  
 Všechna pole `struct` v a musí být přiřazena konstruktorem, než `struct`může konstruktor volat metodu v .  
  
 Pokud se zobrazí tato chyba při pokusu o inicializaci vlastnosti v konstruktoru struktury, řešením je inicializovat záložní pole této vlastnosti přímo namísto samotné vlastnosti.

## <a name="example"></a>Příklad

 Následující ukázka generuje CS0188:  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
