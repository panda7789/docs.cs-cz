## <a name="example"></a><span data-ttu-id="f8d22-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="f8d22-106">Example</span></span>

// compile with: /target:library /out:cs1701_d.dll /keyfile:mykey.snk  
using System.Reflection;  
[assembly:AssemblyVersion("1.0")]  
public class A {  
   public void M1() {}  
}  
  
public class C1 {}  
```  
  
## <a name="example"></a><span data-ttu-id="f8d22-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="f8d22-109">Example</span></span>  
 <span data-ttu-id="f8d22-110">Toto je kód, který vytvoří novější verzi CS1701_d sestavení.</span><span class="sxs-lookup"><span data-stu-id="f8d22-110">This is the code that creates the newer version of assembly CS1701_d.</span></span> <span data-ttu-id="f8d22-111">Všimněte si, že se zkompiluje do jiného adresáře než starší verze, protože výstupní soubory mají stejné názvy.</span><span class="sxs-lookup"><span data-stu-id="f8d22-111">Note that it compiles into a different directory than the older version, necessary since the output files have the same names.</span></span>

   public void M2() {}  
   public void M1() {}

public class C1 {}  
```  
  
## <a name="example"></a><span data-ttu-id="f8d22-112">Příklad</span><span class="sxs-lookup"><span data-stu-id="f8d22-112">Example</span></span>  
 <span data-ttu-id="f8d22-113">Tato ukázka nastaví externí aliasy A1 a A2.</span><span class="sxs-lookup"><span data-stu-id="f8d22-113">This sample sets up the external aliases A1 and A2.</span></span>  
  
```csharp  
// CS1701_c.cs  
// compile with: /target:library /reference:A2=c:\\cs1701_d.dll /reference:A1=cs1701_d.dll  
  
extern alias A1;  
extern alias A2;  
// using System;  
using a1 = A1::A;  
using a2 = A2::A;  
  
public class Ref {  
   public static a1 A1() { return new a1(); }  
   public static a2 A2() { return new a2(); }  
  
   public static A1::C1 M1() { return new A1::C1(); }  
   public static A2::C2 M2() { return new A2::C2(); }  
}  
```
