# <a name="compiler-error-cs0019"></a>Chyba kompilátoru CS0019

Operátor ' operator ' nelze použít na operandy typu ' type ' a ' type '

 Binární operátor je použit na datové typy, které ho nepodporují. Například nemůžete použít [ &#124; ](../operators/boolean-logical-operators.md#conditional-logical-or-operator-) operátor v řetězcích, nemůžete použít operátory [+](../operators/addition-operator.md), [-](../operators/subtraction-operator.md), [\<](../operators/comparison-operators.md#less-than-operator-)nebo [>](../operators/comparison-operators.md#greater-than-operator-) u proměnných [bool](../builtin-types/bool.md) a nemůžete použít operátor [==](../operators/equality-operators.md#equality-operator-) s `struct` typem, pokud typ explicitně nepřetěžuje operátor.

 Operátor lze přetížit, aby podporoval operandy určitých typů. Další informace naleznete v tématu [přetížení operátoru](../operators/operator-overloading.md).

## <a name="example"></a>Příklad

 V následujícím příkladu je CS0019 vygenerováno na dvou místech, [](../builtin-types/bool.md) protože bool C# in není převoditelná na [int](../builtin-types/integral-numeric-types.md). CS0019 se také generuje, když se `-` operátor odčítání aplikuje na řetězec. Operátor sčítání `+` lze použít s řetězcovými operandy, protože tento operátor je přetížen třídou `String` pro provedení zřetězení řetězců.

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }
    
    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```

## <a name="example"></a>Příklad

 V následujícím příkladu je nutné zadat podmínkovou logiku mimo <xref:System.Diagnostics.ConditionalAttribute>. <xref:System.Diagnostics.ConditionalAttribute>můžete předat pouze jeden předdefinovaný symbol.

 Následující ukázka generuje CS0019:

```csharp
// CS0019_a.cs
// compile with: /target:library
using System.Diagnostics;

public class MyClass
{
   [ConditionalAttribute("DEBUG" || "TRACE")]   // CS0019
   public void TestMethod() {}

   // OK
   [ConditionalAttribute("DEBUG"), ConditionalAttribute("TRACE")]
   public void TestMethod2() {}
}
```
