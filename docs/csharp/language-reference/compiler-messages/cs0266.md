# <a name="compiler-error-cs0266"></a><span data-ttu-id="02ddb-102">Chyba kompilátoru CS0266</span><span class="sxs-lookup"><span data-stu-id="02ddb-102">Compiler Error CS0266</span></span>

<span data-ttu-id="02ddb-103">Nelze implicitně převést typ "type1" na "type2".</span><span class="sxs-lookup"><span data-stu-id="02ddb-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="02ddb-104">Existuje explicitní převod (chybí vám přetypátko?)</span><span class="sxs-lookup"><span data-stu-id="02ddb-104">An explicit conversion exists (are you missing a cast?)</span></span>

 <span data-ttu-id="02ddb-105">K této chybě dochází, když se váš kód pokusí převést mezi dvěma typy, které nelze implicitně převést, ale kde je k dispozici explicitní převod.</span><span class="sxs-lookup"><span data-stu-id="02ddb-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="02ddb-106">Další informace naleznete v [tématu Casting a Typ převody](../../programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="02ddb-106">For more information, see [Casting and Type Conversions](../../programming-guide/types/casting-and-type-conversions.md).</span></span>

 <span data-ttu-id="02ddb-107">Následující kód ukazuje příklady, které generují CS0266:</span><span class="sxs-lookup"><span data-stu-id="02ddb-107">The following code shows examples that generate CS0266:</span></span>

```csharp
// CS0266.cs
class MyClass
{
    public static void Main()
    {
        // You cannot implicitly convert a double to an integer.
        double d = 3.2;

        // The following line causes compiler error CS0266.
        int i1 = d;

        // However, you can resolve the error by using an explicit conversion.
        int i2 = (int)d;  

        // You cannot implicitly convert an object to a class type.
        object obj = new MyClass();

        // The following assignment statement causes error CS0266.
        MyClass myClass = obj;

        // You can resolve the error by using an explicit conversion.
        MyClass c = (MyClass)obj;

        // You cannot implicitly convert a base class object to a derived class type.
        MyClass mc = new MyClass();
        DerivedClass dc = new DerivedClass();

        // The following line causes compiler error CS0266.
        dc = mc;

        // You can resolve the error by using an explicit conversion.
        dc = (DerivedClass)mc;
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
