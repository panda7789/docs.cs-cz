# <a name="compiler-error-cs0052"></a><span data-ttu-id="37db2-103">Chyba kompilátoru CS0052</span><span class="sxs-lookup"><span data-stu-id="37db2-103">Compiler Error CS0052</span></span>

<span data-ttu-id="37db2-104">Nekonzistentní dostupnost: typ pole typ je méně dostupný než pole ' Field '.</span><span class="sxs-lookup"><span data-stu-id="37db2-104">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>

 <span data-ttu-id="37db2-105">Typ pole nemůže být méně dostupný než pole samotné, protože všechny veřejné konstrukce musí vracet veřejně přístupný objekt.</span><span class="sxs-lookup"><span data-stu-id="37db2-105">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>

## <a name="example"></a><span data-ttu-id="37db2-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="37db2-106">Example</span></span>

 <span data-ttu-id="37db2-107">Následující ukázka generuje CS0052:</span><span class="sxs-lookup"><span data-stu-id="37db2-107">The following sample generates CS0052:</span></span>

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```
