# <a name="compiler-error-cs1001"></a><span data-ttu-id="e72da-102">Chyba kompilátoru CS1001</span><span class="sxs-lookup"><span data-stu-id="e72da-102">Compiler Error CS1001</span></span>

<span data-ttu-id="e72da-103">Byl očekáván identifikátor.</span><span class="sxs-lookup"><span data-stu-id="e72da-103">Identifier expected</span></span>

<span data-ttu-id="e72da-104">Nezadali jste identifikátor.</span><span class="sxs-lookup"><span data-stu-id="e72da-104">You did not supply an identifier.</span></span> <span data-ttu-id="e72da-105">Identifikátor je název třídy, struktury, oboru názvů, metody, proměnné a tak dále, které zadáte.</span><span class="sxs-lookup"><span data-stu-id="e72da-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>

<span data-ttu-id="e72da-106">Následující příklad deklaruje jednoduchou třídu, ale neposkytuje název třídy:</span><span class="sxs-lookup"><span data-stu-id="e72da-106">The following example declares a simple class but does not give the class a name:</span></span>

```csharp
public class //CS1001
{
    public int Num { get; set; }
    void MethodA() {}
}
```

<span data-ttu-id="e72da-107">Následující ukázka generuje CS1001, protože při deklaraci výčtu musíte zadat členy:</span><span class="sxs-lookup"><span data-stu-id="e72da-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>

```csharp
public class Program
{
    enum Colors
    {
        'a', 'b' // CS1001, 'a' is not a valid int identifier
        // The following line shows examples of valid identifiers:
        // Blue, Red, Orange
    };

    public static void Main()
    {
    }
}
```

<span data-ttu-id="e72da-108">Názvy parametrů jsou požadovány i v případě, že je kompilátor nepoužívá, například v definici rozhraní.</span><span class="sxs-lookup"><span data-stu-id="e72da-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="e72da-109">Tyto parametry jsou povinné, aby Programátori, kteří využívají rozhraní, znali něco o tom, co tyto parametry znamenají.</span><span class="sxs-lookup"><span data-stu-id="e72da-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>

```csharp
interface IMyTest
{
    void TestFunc1(int, int);  // CS1001
    // Use the following line instead:
    // void TestFunc1(int a, int b);
}

class CMyTest : IMyTest
{
    void IMyTest.TestFunc1(int a, int b)
    {
    }
}
```
