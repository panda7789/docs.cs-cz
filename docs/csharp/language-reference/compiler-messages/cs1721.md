# <a name="compiler-error-cs1721"></a>Chyba kompilátoru CS1721

Třída ' class ' nemůže mít více základních tříd: ' class_1 ' a ' class_2 '

Nejběžnější příčinou této chybové zprávy je pokus o použití vícenásobné dědičnosti. Třída v C# může dědit pouze přímo z jedné třídy. Třída však může implementovat libovolný počet rozhraní.

## <a name="example"></a>Příklad

Následující příklad ukazuje jeden způsob, jakým je vygenerováno CS1721:

```csharp
// CS1721.cs
public class A {}
public class B {}
public class MyClass : A, B {}   // CS1721
```

## <a name="to-correct-this-error"></a>Oprava této chyby

Tuto chybu můžete odstranit různými způsoby:

- Nastavit třídu `B` dědit z `A`a `MyClass` dědit ze `B`:

    ```csharp
    public class A {}
    public class B : A {}
    public class MyClass : B {}
    ```

- Deklarace `B` jako rozhraní. Nastavit `MyClass` dědění z rozhraní `B`a `A`třídy:

    ```csharp
    public class A {}
    public interface B {}
    public class MyClass : A, B {}
    ```
