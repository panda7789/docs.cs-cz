# <a name="compiler-error-cs0507"></a><span data-ttu-id="15763-103">Chyba kompilátoru CS0507</span><span class="sxs-lookup"><span data-stu-id="15763-103">Compiler Error CS0507</span></span>
<span data-ttu-id="15763-104">' Function1 ': Při přepsání ' Access ' zděděného členu ' function2 ' nejde změnit modifikátory přístupu</span><span class="sxs-lookup"><span data-stu-id="15763-104">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="15763-105">Byl proveden pokus o změnu specifikace přístupu v přepsání metody.</span><span class="sxs-lookup"><span data-stu-id="15763-105">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="15763-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="15763-106">Example</span></span>  
 <span data-ttu-id="15763-107">Následující ukázka generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="15763-107">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="15763-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="15763-108">Example</span></span>  
 <span data-ttu-id="15763-109">K CS0507 může dojít také v případě, že se třída pokusí přepsat metodu označenou tak, jak je `protected internal` definována v odkazovaných metadatech.</span><span class="sxs-lookup"><span data-stu-id="15763-109">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="15763-110">V takovém případě by měla být přepsání metody označena jako `protected` .</span><span class="sxs-lookup"><span data-stu-id="15763-110">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp

{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="15763-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="15763-111">Example</span></span>
