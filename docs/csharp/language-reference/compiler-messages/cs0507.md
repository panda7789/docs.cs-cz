# <a name="compiler-error-cs0507"></a><span data-ttu-id="243dc-102">Chyba kompilátoru CS0507</span><span class="sxs-lookup"><span data-stu-id="243dc-102">Compiler Error CS0507</span></span>
<span data-ttu-id="243dc-103">'function1' : nelze změnit modifikátory přístupu při přepsání "přístupu" zděděného člena 'function2'</span><span class="sxs-lookup"><span data-stu-id="243dc-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="243dc-104">Došlo k pokusu o změnu specifikace přístupu v přepsání metody.</span><span class="sxs-lookup"><span data-stu-id="243dc-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="243dc-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="243dc-105">Example</span></span>  
 <span data-ttu-id="243dc-106">Následující ukázka generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="243dc-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="243dc-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="243dc-107">Example</span></span>  
 <span data-ttu-id="243dc-108">CS0507 může dojít také v případě, že třída `protected internal` se pokusí přepsat metodu označenou jako definované v odkazovaných metadat.</span><span class="sxs-lookup"><span data-stu-id="243dc-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="243dc-109">V takovém případě by měla být `protected`označena přepsání metoda jako .</span><span class="sxs-lookup"><span data-stu-id="243dc-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp

{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="243dc-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="243dc-110">Example</span></span>
