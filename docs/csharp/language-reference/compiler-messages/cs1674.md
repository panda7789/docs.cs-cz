## <a name="example"></a><span data-ttu-id="a5d6b-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="a5d6b-109">Example</span></span>

```csharp

   public void Test() {  
      using (C c = new C()) {}   // CS1674

   void IDisposable.Dispose() {}
```  
