## <a name="example"></a><span data-ttu-id="dc694-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="dc694-108">Example</span></span>  
 <span data-ttu-id="dc694-109">Následující ukázka generuje CS1674.</span><span class="sxs-lookup"><span data-stu-id="dc694-109">The following sample generates CS1674.</span></span>  
  
```csharp

using System;

   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }

class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}

      using (D d = new D()) {}  
   }

```
