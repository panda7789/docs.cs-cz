 <span data-ttu-id="c847a-105">Prvky pole musí být stejného typu nebo implicitně převoditelné na stejný typ podle pravidel odvození typu používaných kompilátorem.</span><span class="sxs-lookup"><span data-stu-id="c847a-105">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="c847a-106">Nejlepší typ musí být jeden z typů přítomných ve výrazu Array.</span><span class="sxs-lookup"><span data-stu-id="c847a-106">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="c847a-107">Prvky nebudou převedeny na nový typ, například `object` .</span><span class="sxs-lookup"><span data-stu-id="c847a-107">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="c847a-108">Pro implicitně typované pole kompilátor musí odvodit typ pole na základě typu prvků, které jsou k němu přiřazeny.</span><span class="sxs-lookup"><span data-stu-id="c847a-108">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="c847a-113">Příklad</span><span class="sxs-lookup"><span data-stu-id="c847a-113">Example</span></span>  
 <span data-ttu-id="c847a-114">Následující kód vygeneruje CS0826, protože prvky pole nejsou stejného typu a logika odvození typu kompilátoru nenajde jeden nejlepší typ:</span><span class="sxs-lookup"><span data-stu-id="c847a-114">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
