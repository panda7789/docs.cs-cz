 <span data-ttu-id="98d10-104">Všechny prvky pole musí být stejného typu nebo implicitně konvertibilní na stejný typ podle pravidel odvození typu používaných kompilátorem.</span><span class="sxs-lookup"><span data-stu-id="98d10-104">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="98d10-105">Nejlepší typ musí být jeden z typů přítomných ve výrazu pole.</span><span class="sxs-lookup"><span data-stu-id="98d10-105">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="98d10-106">Prvky nebudou převedeny na nový `object`typ, například .</span><span class="sxs-lookup"><span data-stu-id="98d10-106">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="98d10-107">Pro implicitně zadané pole musí kompilátor odvodit typ pole na základě typu prvků, které jsou mu přiřazeny.</span><span class="sxs-lookup"><span data-stu-id="98d10-107">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="98d10-112">Příklad</span><span class="sxs-lookup"><span data-stu-id="98d10-112">Example</span></span>  
 <span data-ttu-id="98d10-113">Následující kód generuje CS0826, protože prvky pole nejsou všechny stejného typu a logika odvození typu kompilátoru nenajde jeden nejlepší typ:</span><span class="sxs-lookup"><span data-stu-id="98d10-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
