# <a name="compiler-error-cs0006"></a>Chyba kompilátoru CS0006

Soubor metadat "dll_name" nebyl nalezen.
  
 Program byl zkompilován a explicitně předán název souboru, který obsahoval metadata; soubor DLL však nebyl nalezen. Další informace naleznete [v tématu -reference (C# Compiler Options)](../compiler-options/reference-compiler-option.md).
