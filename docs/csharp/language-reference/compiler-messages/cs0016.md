# <a name="compiler-error-cs0016"></a>Chyba kompilátoru CS0016

Nelze zapisovat do výstupního souboru 'soubor' - 'důvod'

 Kompilátor nemohl zapisovat do výstupního souboru. Zkontrolujte cestu k souboru a ujistěte se, že existuje. Pokud dříve sestavený soubor je již v umístění, ujistěte se, že je zapisovatelný a že žádný proces aktuálně nemá zámek na soubor. Například ujistěte se, že spustitelný soubor není spuštěn při pokusu o sestavení.
