## <a name="example"></a><span data-ttu-id="5bc88-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="5bc88-110">Example</span></span>

 <span data-ttu-id="5bc88-111">Následující ukázka generuje CS0201, protože kontrolovány sám o sobě není příkaz, i když je parametrizován operací přírůstek.</span><span class="sxs-lookup"><span data-stu-id="5bc88-111">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>

```csharp
{  
   public void Add(T x)

      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
