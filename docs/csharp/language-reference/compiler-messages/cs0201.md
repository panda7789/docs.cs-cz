   public static void Main()

## <a name="example"></a><span data-ttu-id="37437-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="37437-111">Example</span></span>

 <span data-ttu-id="37437-112">Následující ukázka vygeneruje CS0201, protože kontrolované samotný není příkaz, i když je parametrizovaný operací Increment.</span><span class="sxs-lookup"><span data-stu-id="37437-112">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>  

```csharp

{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
