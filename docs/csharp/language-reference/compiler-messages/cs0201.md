# <a name="compiler-error-cs0201"></a>Chyba kompilátoru CS0201

Jako příkaz lze použít pouze výrazy přiřazení, volání, přírůstek, snížení a nového objektu.  
  
 Kompilátor vygeneruje chybu, když narazí na neplatný příkaz. Neplatný příkaz je libovolný řádek nebo řada řádků končící středníkem, která nepředstavuje přiřazení [--](../operators/arithmetic-operators.md#decrement-operator---) [++](../operators/arithmetic-operators.md#increment-operator-) ([=](../operators/assignment-operator.md)), volání metody [()](../operators/member-access-operators.md#invocation-operator-), [nové](../operators/new-operator.md)nebo operace. Další informace naleznete v [tématu Příkazy, výrazy a operátory](../../programming-guide/statements-expressions-operators/index.md).  
  
## <a name="example"></a>Příklad

 Následující ukázka generuje CS0201, protože 2 * 3 je výraz, nikoli příkaz. Chcete-li kód zkompilovat, zkuste přiřadit hodnotu výrazu proměnné.  

```csharp
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```

## <a name="example"></a>Příklad

 Následující ukázka generuje CS0201, protože kontrolovány sám o sobě není příkaz, i když je parametrizován operací přírůstek.  

```csharp
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
