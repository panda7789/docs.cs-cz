# <a name="c-special-characters"></a>Speciální znaky v jazyce C#

Speciální znaky jsou předdefinované, kontextové znaky, které upravují prvek programu (literální řetězec, identifikátor nebo název atributu), ke kterému jsou předdefinovány. Jazyk C# podporuje následující speciální znaky:

- [@](./verbatim.md), doslovného znaku identifikátoru.

- [$](./interpolated.md), interpolovaná řetězcová znak.
