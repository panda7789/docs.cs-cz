# <a name="endregion-c-reference"></a>#endregion (referenční dokumentace jazyka C#)
`#endregion`označuje konec [#region](./preprocessor-region.md) bloku. Například:  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
