# <a name="endregion-c-reference"></a>#endregion (referenční dokumentace jazyka C#)
`#endregion` označuje konec bloku [#region](./preprocessor-region.md) . Příklad:  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
