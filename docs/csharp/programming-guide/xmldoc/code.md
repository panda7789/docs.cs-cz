# <a name="code-c-programming-guide"></a>\<code>(Průvodce programováním v C#)

## <a name="syntax"></a>Syntaxe

```xml
<code>content</code>
```

## <a name="parameters"></a>Parametry

- `content`

  Text, který chcete označit jako kód.

## <a name="remarks"></a>Poznámky

`<code>`Značka slouží k označení více řádků kódu. Slouží [\<c>](./code-inline.md) k označení, že jednořádkový text v rámci popisu by měl být označen jako kód.

Zkompilujte s [-doc](../../language-reference/compiler-options/doc-compiler-option.md) a zpracujte komentáře k dokumentaci do souboru.
