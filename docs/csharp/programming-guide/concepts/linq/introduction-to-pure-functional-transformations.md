# <a name="introduction-to-pure-functional-transformations-c"></a>Úvod do čistě funkčních transformací (C#)
Tato část zavádí funkční transformace, včetně základní koncepty a podpůrné jazykové konstrukce. Kontrastuje s objektově orientovanými a funkčními transformačními přístupy k programování, včetně poradenství ohledně přechodu na program. Přestože funkční transformace lze použít v mnoha scénářích programování, transformace XML se zde používá jako konkrétní příklad.  
  
 [Kurz: Manipulace s obsahem v dokumentu WordprocessingML (C#)](./shape-of-wordprocessingml-documents.md) poskytuje řadu příkladů, z nichž každá staví na předchozím. Tyto příklady ukazují čistě funkční transformační přístup k manipulaci s XML. Tento kurz předpokládá pracovní znalosti jazyka C#. Podrobné sémantiky konstrukce jazyka nejsou k dispozici v tomto kurzu, ale odkazy jsou k dispozici v jazykové dokumentaci podle potřeby.  
  
 Předpokládá se také pracovní znalost základních konceptů počítačové vědy a XML, včetně jmenných prostorů XML.
