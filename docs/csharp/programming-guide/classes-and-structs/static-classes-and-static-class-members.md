 <span data-ttu-id="1190e-107">Statickou třídu lze použít jako pohodlný kontejner pro sady metod, které fungují pouze na vstupních parametrech a nemusejí získávat ani nastavovat žádná interní pole instance.</span><span class="sxs-lookup"><span data-stu-id="1190e-107">A static class can be used as a convenient container for sets of methods that just operate on input parameters and do not have to get or set any internal instance fields.</span></span> <span data-ttu-id="1190e-108">Například v knihovně tříd .NET statická <xref:System.Math?displayProperty=nameWithType> Třída obsahuje metody, které provádějí matematické operace bez nutnosti ukládat nebo načítat data, která jsou jedinečná pro konkrétní instanci <xref:System.Math> třídy.</span><span class="sxs-lookup"><span data-stu-id="1190e-108">For example, in the .NET Class Library, the static <xref:System.Math?displayProperty=nameWithType> class contains methods that perform mathematical operations, without any requirement to store or retrieve data that is unique to a particular instance of the <xref:System.Math> class.</span></span> <span data-ttu-id="1190e-109">To znamená, že použijete členy třídy zadáním názvu třídy a názvu metody, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="1190e-109">That is, you apply the members of the class by specifying the class name and the method name, as shown in the following example.</span></span>

 <span data-ttu-id="1190e-110">Stejně jako u všech typů tříd jsou informace o typu statické třídy načteny modulem runtime .NET, když je načten program, který odkazuje na třídu.</span><span class="sxs-lookup"><span data-stu-id="1190e-110">As is the case with all class types, the type information for a static class is loaded by the .NET runtime when the program that references the class is loaded.</span></span> <span data-ttu-id="1190e-111">Program nemůže přesně určovat, kdy je třída načtena.</span><span class="sxs-lookup"><span data-stu-id="1190e-111">The program cannot specify exactly when the class is loaded.</span></span> <span data-ttu-id="1190e-112">Je však zaručeno, že bude načteno a že jeho pole jsou inicializována a jeho statický konstruktor byl volán před tím, než je třída poprvé odkazována v programu.</span><span class="sxs-lookup"><span data-stu-id="1190e-112">However, it is guaranteed to be loaded and to have its fields initialized and its static constructor called before the class is referenced for the first time in your program.</span></span> <span data-ttu-id="1190e-113">Statický konstruktor se volá jenom jednou a statická třída zůstane v paměti po dobu života domény aplikace, ve které se váš program nachází.</span><span class="sxs-lookup"><span data-stu-id="1190e-113">A static constructor is only called one time, and a static class remains in memory for the lifetime of the application domain in which your program resides.</span></span>
