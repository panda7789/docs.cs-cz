# <a name="compiler-error-cs1056"></a>Chyba kompilátoru CS1056
Neočekávaný znak "Character"  
  
 Kompilátor jazyka C# narazil na neočekávaný znak a nemůže identifikovat aktuálně zpracovávaný token. Například pokud kompilátor narazí na euro-Character uprostřed zpracování identifikátoru, nebude schopen klasifikovat identifikátor, protože znak euro by byl platný pouze uvnitř řetězce a kompilátor ví, že nezpracovává řetězec.
