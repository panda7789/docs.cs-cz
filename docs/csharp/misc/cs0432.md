# <a name="compiler-error-cs0432"></a><span data-ttu-id="db8bb-103">Chyba kompilátoru CS0432</span><span class="sxs-lookup"><span data-stu-id="db8bb-103">Compiler Error CS0432</span></span>
<span data-ttu-id="db8bb-104">Alias ' identifier ' nebyl nalezen.</span><span class="sxs-lookup"><span data-stu-id="db8bb-104">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="db8bb-105">K této chybě dochází, pokud použijete "::" napravo od identifikátoru, který není alias.</span><span class="sxs-lookup"><span data-stu-id="db8bb-105">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="db8bb-106">K vyřešení chyby použijte místo toho ".".</span><span class="sxs-lookup"><span data-stu-id="db8bb-106">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="db8bb-107">Následující příklad generuje CS0432:</span><span class="sxs-lookup"><span data-stu-id="db8bb-107">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
