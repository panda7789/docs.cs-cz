# <a name="compiler-error-cs0121"></a>Chyba kompilátoru CS0121

Volání je nejednoznačné mezi následujícími metodami nebo vlastnostmi: "method1" a "method2"

Z důvodu implicitní převod kompilátor nebyl schopen volat jednu formu přetížené metody. Tuto chybu můžete vyřešit jedním z následujících způsobů:

- Zadejte parametry metody tak, aby nedošlo k implicitnímu převodu.
- Odstraňte všechna přetížení pro metodu.
- Přetypování na správný typ před voláním metody.
- Použijte pojmenované argumenty.

## <a name="example"></a>Příklad

Následující příklady generují chybu kompilátoru CS0121:

```csharp
public class Program
{
    static void f(int i, double d)
    {
    }

    static void f(double d, int i)
    {
    }

    public static void Main()
    {
        f(1, 1);   // CS0121

        // Try the following code instead:
        // f(1, 1.0);
        // or
        // f(1.0, 1);
        // or
        // f(1, (double)1);   // Cast and specify which method to call.
        // or
        // f(i: 1, 1);
        // or
        // f(d: 1, 1);

        // f(i: 1, d: 1); // This still gives CS0121
    }
}
```

## <a name="example"></a>Příklad

```csharp
class Program2
{
    static int ol_invoked = 0;

    delegate int D1(int x);
    delegate T D1<T>(T x);
    delegate T D1<T, U>(U u);

    static void F(D1 d1) { ol_invoked = 1; }
    static void F<T>(D1<T> d1t) { ol_invoked = 2; }
    static void F<T, U>(D1<T, U> d1t) { ol_invoked = 3; }

    static int Test001()
    {
        F(delegate(int x) { return 1; }); // CS0121
        if (ol_invoked == 1)
            return 0;
        else
            return 1;
    }

    static int Main()
    {
        return Test001();
    }
}
```
