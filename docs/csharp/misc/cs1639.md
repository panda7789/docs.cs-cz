# <a name="compiler-error-cs1639"></a>Chyba kompilátoru CS1639
Podpis spravované třídy obálky coclass pro rozhraní není platný podpis názvu třídy.  
  
 K této chybě dochází, pokud metadata pro rozhraní nejsou platná. Tato situace by nikdy neměla nastat, pokud bylo rozhraní vygenerováno pomocí jazyka C# nebo podporovaného jazyka .NET. Zkontrolujte u dodavatele rozhraní nebo ověřte, že se sestavení rozhraní vygenerovalo úspěšně.
