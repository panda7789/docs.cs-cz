# <a name="compiler-warning-level-3-cs0665"></a>Upozornění kompilátoru (úroveň 3) CS0665
Přiřazení v podmíněném výrazu je vždycky konstantní; Měli jste v úmyslu použít = = místo =?  
  
 Podmíněný výraz použil [operátor =](../language-reference/operators/assignment-operator.md) a nikoli [operátor = =](../language-reference/operators/equality-operators.md#equality-operator-).
  
 Následující ukázka generuje CS0665:  
  
```csharp  
// CS0665.cs  
// compile with: /W:3  
class Test  
{  
   public static void Main()  
   {  
      bool i = false;  
  
      if (i = true)   // CS0665  
      // try the following line instead  
      // if (i == true)  
      {  
      }  
  
      System.Console.WriteLine(i);  
   }  
}  
```
