# <a name="compiler-error-cs0014"></a>Chyba kompilátoru CS0014
Požadovaný soubor ' file ' nebyl nalezen.  
  
 Kompilátor vyžaduje soubor, ale není v systému. Ujistěte se, že cesta je správná. Pokud je soubor systémovým souborem sady Visual Studio, bude pravděpodobně nutné opravit instalaci nebo odebrat aplikaci Visual Studio úplně znovu.
