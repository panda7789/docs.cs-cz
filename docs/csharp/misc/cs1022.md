# <a name="compiler-error-cs1022"></a>Chyba kompilátoru CS1022
Očekává se definice typu nebo oboru názvů nebo konec souboru.  
  
 Soubor se zdrojovým kódem nemá shodnou sadu složených závorek.  
  
 Následující ukázka generuje CS1022:  
  
```csharp  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
