# <a name="compiler-error-cs1555"></a>Chyba kompilátoru CS1555
Nejde najít třídu zadanou pro metodu Main.  
  
 Byla zadána třída do možnosti [-Main](../language-reference/compiler-options/main-compiler-option.md) kompilátoru, ale název třídy nebyl ve zdrojovém kódu nalezen.
