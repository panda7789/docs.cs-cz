 <span data-ttu-id="207b4-106">K této chybě dochází, pokud omezíte obecný typ na určitou třídu a pokusíte se vytvořit instanci této třídy pomocí typu hodnoty, který nelze implicitně zablokovat.</span><span class="sxs-lookup"><span data-stu-id="207b4-106">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>

## <a name="example"></a><span data-ttu-id="207b4-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="207b4-109">Example</span></span>  
 <span data-ttu-id="207b4-110">Následující příklad generuje CS0315:</span><span class="sxs-lookup"><span data-stu-id="207b4-110">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
