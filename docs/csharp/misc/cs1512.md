# <a name="compiler-error-cs1512"></a><span data-ttu-id="ce82d-103">Chyba kompilátoru CS1512</span><span class="sxs-lookup"><span data-stu-id="ce82d-103">Compiler Error CS1512</span></span>
<span data-ttu-id="ce82d-104">Klíčové slovo Base není v aktuálním kontextu k dispozici.</span><span class="sxs-lookup"><span data-stu-id="ce82d-104">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="ce82d-105">Klíčové slovo [Base](../language-reference/keywords/base.md) bylo použito mimo metodu, vlastnost nebo konstruktor.</span><span class="sxs-lookup"><span data-stu-id="ce82d-105">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="ce82d-106">Následující příklad generuje CS1512:</span><span class="sxs-lookup"><span data-stu-id="ce82d-106">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
