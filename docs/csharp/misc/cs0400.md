 <span data-ttu-id="470f8-107">Chcete-li se této chybě vyhnout, vyhledejte deklaraci identifikátoru, ověřte správný pravopis a pokud je deklarace v samostatném sestavení, ujistěte se, že máte příslušný odkaz na sestavení.</span><span class="sxs-lookup"><span data-stu-id="470f8-107">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="470f8-108">Pokud je identifikátor deklarován uvnitř jiného typu nebo oboru názvů, použijte plně kvalifikovaný název za ::.</span><span class="sxs-lookup"><span data-stu-id="470f8-108">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="470f8-109">Následující ukázka generuje CS0400:</span><span class="sxs-lookup"><span data-stu-id="470f8-109">The following sample generates CS0400:</span></span>

   }
