 <span data-ttu-id="03df4-108">Chcete-li se této chybě vyhnout, vyhledejte deklaraci identifikátoru, ověřte správné pravopis a v případě, že je deklarace v samostatném sestavení, ujistěte se, že máte odpovídající odkaz na sestavení.</span><span class="sxs-lookup"><span data-stu-id="03df4-108">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="03df4-109">Pokud je identifikátor deklarován uvnitř jiného typu nebo oboru názvů, použijte plně kvalifikovaný název za::.</span><span class="sxs-lookup"><span data-stu-id="03df4-109">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="03df4-110">Následující ukázka generuje CS0400:</span><span class="sxs-lookup"><span data-stu-id="03df4-110">The following sample generates CS0400:</span></span>
