# <a name="compiler-error-cs1604"></a>Chyba kompilátoru CS1604
Nelze přiřadit k proměnné, protože je jen pro čtení.  
  
 Přiřazení bylo provedeno na proměnnou, která je jen pro čtení. Chcete-li se této chybě vyhnout, nepřiřazujte ani neměňte tuto proměnnou v tomto kontextu.
