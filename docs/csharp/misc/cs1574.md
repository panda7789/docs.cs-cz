<span data-ttu-id="5a7e2-104">Komentář XML v konstrukci má syntakticky nesprávný atribut cref Name.</span><span class="sxs-lookup"><span data-stu-id="5a7e2-104">XML comment on 'construct' has syntactically incorrect cref attribute 'name'</span></span>  
  
 <span data-ttu-id="5a7e2-105">Řetězec předaný značce cref, například v rámci \<exception> značky, je odkazován na člen, který není dostupný v rámci aktuálního prostředí sestavení.</span><span class="sxs-lookup"><span data-stu-id="5a7e2-105">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="5a7e2-106">Řetězec, který předáte značce cref, musí být syntakticky správný název člena nebo pole.</span><span class="sxs-lookup"><span data-stu-id="5a7e2-106">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>  
  
 <span data-ttu-id="5a7e2-107">Další informace najdete v tématu [Doporučené značky pro dokumentační komentáře](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span><span class="sxs-lookup"><span data-stu-id="5a7e2-107">For more information, see [Recommended Tags for Documentation Comments](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span></span>  
  
 <span data-ttu-id="5a7e2-108">Následující ukázka generuje CS1574:</span><span class="sxs-lookup"><span data-stu-id="5a7e2-108">The following sample generates CS1574:</span></span>  
  
```csharp  
// CS1574.cs  
// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574  
// instead, uncomment and try the following line  
// /// <exception cref="System.Console.WriteLine">An exception class.</exception>  
class EClass : Exception  
{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
