# <a name="compiler-warning-level-1-cs0626"></a><span data-ttu-id="ae21a-103">Upozornění kompilátoru (úroveň 1) CS0626</span><span class="sxs-lookup"><span data-stu-id="ae21a-103">Compiler Warning (level 1) CS0626</span></span>

<span data-ttu-id="ae21a-104">Metoda, operátor nebo přistupující objekt method je označen jako External a nemá žádné atributy.</span><span class="sxs-lookup"><span data-stu-id="ae21a-104">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="ae21a-105">Zvažte přidání atributu DllImport k určení externí implementace.</span><span class="sxs-lookup"><span data-stu-id="ae21a-105">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="ae21a-106">Metoda označená jako `extern` by měla být také označena atributem, například atributem [dllimport](xref:System.Runtime.InteropServices.DllImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="ae21a-106">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="ae21a-107">Atribut určuje, kde je metoda implementována.</span><span class="sxs-lookup"><span data-stu-id="ae21a-107">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="ae21a-108">V době běhu bude program tyto informace potřebovat.</span><span class="sxs-lookup"><span data-stu-id="ae21a-108">At run time, the program will need this information.</span></span>  
  
 <span data-ttu-id="ae21a-109">Následující ukázka generuje CS0626:</span><span class="sxs-lookup"><span data-stu-id="ae21a-109">The following sample generates CS0626:</span></span>  
  
```csharp
// CS0626.cs  
// compile with: /warnaserror  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   static extern public void M(); // CS0626  
   // try the following line  
   // [DllImport("mydll.dll")] static extern public void M();  
  
   public static void Main()  
   {  
   }  
}  
```
