# <a name="compiler-error-cs0247"></a>Chyba kompilátoru CS0247
Nelze použít zápornou velikost s stackalloc  
  
 Příkazu [stackalloc](../language-reference/operators/stackalloc.md) bylo předáno záporné číslo.
  
 Následující ukázka generuje CS0247:  
  
```csharp  
// CS0247.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int[-30];   // CS0247  
   }  
}  
```
