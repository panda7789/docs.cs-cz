# <a name="compiler-error-cs1611"></a><span data-ttu-id="efc4b-103">Chyba kompilátoru CS1611</span><span class="sxs-lookup"><span data-stu-id="efc4b-103">Compiler Error CS1611</span></span>
<span data-ttu-id="efc4b-104">Parametr params nemůže být deklarovaný jako ref nebo out.</span><span class="sxs-lookup"><span data-stu-id="efc4b-104">The params parameter cannot be declared as in ref or out</span></span>  
  
 <span data-ttu-id="efc4b-105">Klíčová slova [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md) nelze použít s klíčovým slovem [params](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="efc4b-105">The keywords [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="efc4b-106">Následující ukázka generuje CS1611:</span><span class="sxs-lookup"><span data-stu-id="efc4b-106">The following sample generates CS1611:</span></span>  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
