# <a name="compiler-error-cs0013"></a>Chyba kompilátoru CS0013
Neočekávaná chyba při zápisu metadat do souboru ModelStore.dll--k vytvoření dalších uživatelských řetězců nezbývá žádné logické místo.  
  
 Modul common language runtime .NET Framework negeneroval metadata. Zkontrolujte, zda je cesta správná a zda není disk plný. Pokud potíže potrvají, bude pravděpodobně nutné opravit nebo přeinstalovat sadu Visual Studio nebo .NET Framework.
