# <a name="compiler-error-cs0555"></a><span data-ttu-id="e937f-103">Chyba kompilátoru CS0555</span><span class="sxs-lookup"><span data-stu-id="e937f-103">Compiler Error CS0555</span></span>

<span data-ttu-id="e937f-104">Uživatelem definovaný operátor nemůže převzít objekt nadřazeného typu a převést jej na objekt nadřazeného typu.</span><span class="sxs-lookup"><span data-stu-id="e937f-104">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>

<span data-ttu-id="e937f-105">Uživatelem definované převody na hodnoty ohraničující třídy nejsou povoleny; takový operátor nepotřebujete.</span><span class="sxs-lookup"><span data-stu-id="e937f-105">User-defined conversions to values of the enclosing class are not allowed; you do not need such an operator.</span></span>

```csharp
// CS0555.cs
public class MyClass
{
   // delete the following operator to resolve this CS0555
   public static implicit operator MyClass(MyClass aa)   // CS0555
   {
      return new MyClass();
   }

   public static void Main()
   {
   }
}
```
