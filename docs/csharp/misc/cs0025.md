# <a name="compiler-error-cs0025"></a>Chyba kompilátoru CS0025
Standardní soubor knihovny ' file ' nebyl nalezen.  
  
 Soubor, který je vyžadován kompilátorem, nebyl nalezen. Ověřte, zda je cesta správná a zda soubor existuje.  
  
 Pokud je soubor systémovým souborem sady Visual Studio, může být nutné opravit instalaci sady Visual Studio nebo ji úplně znovu nainstalovat.
