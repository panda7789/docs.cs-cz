# <a name="compiler-error-cs0456"></a><span data-ttu-id="bd456-103">Chyba kompilátoru CS0456</span><span class="sxs-lookup"><span data-stu-id="bd456-103">Compiler Error CS0456</span></span>
<span data-ttu-id="bd456-104">Parametr typu s parametrem Type 1 má omezení struct, takže se jako omezení pro parametr Type Name 2 nedá použít parametr Type 1.</span><span class="sxs-lookup"><span data-stu-id="bd456-104">Type parameter 'Type Parameter Name 1' has the 'struct' constraint so 'Type Parameter Name 1' cannot be used as a constraint for 'Type Parameter Name 2'</span></span>  
  
 <span data-ttu-id="bd456-105">Omezení typu hodnoty jsou implicitně zapečetěná, aby tato omezení nemohly být použita jako omezení pro druhý parametr typu.</span><span class="sxs-lookup"><span data-stu-id="bd456-105">Value type constraints are implicitly sealed so those constraints cannot be used as constraints on a second type parameter.</span></span> <span data-ttu-id="bd456-106">Důvodem je, že typy hodnot nelze přepsat.</span><span class="sxs-lookup"><span data-stu-id="bd456-106">This is because value types cannot be overridden.</span></span> <span data-ttu-id="bd456-107">Chcete-li vyřešit tuto chybu, vložte omezení typu hodnoty přímo na druhý parametr typu místo toho, aby je prováděla nepřímo prostřednictvím prvního parametru typu.</span><span class="sxs-lookup"><span data-stu-id="bd456-107">To resolve this error, put a value type constraint directly on the second type parameter, instead of doing so indirectly by means of the first type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bd456-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="bd456-108">Example</span></span>  
 <span data-ttu-id="bd456-109">Následující ukázka generuje CS0456.</span><span class="sxs-lookup"><span data-stu-id="bd456-109">The following sample generates CS0456.</span></span>  
  
```csharp  
// CS0456.cs  
// compile with: /target:library  
public class GenericsErrors  
{  
   public class G5<T> where T : struct  
   {  
      public class N<U> where U : T {}   // CS0456  
      public class N2<U> where U : struct {}   // OK  
   }  
}  
```
