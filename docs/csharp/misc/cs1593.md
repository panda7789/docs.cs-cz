# <a name="compiler-error-cs1593"></a><span data-ttu-id="19363-103">Chyba kompilátoru CS1593</span><span class="sxs-lookup"><span data-stu-id="19363-103">Compiler Error CS1593</span></span>
<span data-ttu-id="19363-104">Delegát ' del ' nepřijímá argumenty ' Number '</span><span class="sxs-lookup"><span data-stu-id="19363-104">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="19363-105">Počet argumentů předaných volání [delegáta](../language-reference/builtin-types/reference-types.md) nesouhlasí s počtem parametrů v deklaraci delegáta.</span><span class="sxs-lookup"><span data-stu-id="19363-105">The number of arguments passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="19363-106">Následující ukázka generuje CS1593:</span><span class="sxs-lookup"><span data-stu-id="19363-106">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
