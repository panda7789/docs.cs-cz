# <a name="compiler-warning-level-2-cs0164"></a>Upozornění kompilátoru (úroveň 2) CS0164
Na tento popisek se neodkazuje.

 Popisek byl deklarován, ale nikdy se nepoužil.

 Následující ukázka generuje CS0164:

```csharp
// CS0164.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        int i = 0;
    l1: // CS0164
        i++;
        // Uncomment the following lines to resolve this warning.
        // if (i < 10)
        //     goto l1;
    }
}
```
