## <a name="example"></a><span data-ttu-id="7c4fd-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="7c4fd-108">Example</span></span>  
 <span data-ttu-id="7c4fd-109">Následující ukázka generuje CS1657:</span><span class="sxs-lookup"><span data-stu-id="7c4fd-109">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
