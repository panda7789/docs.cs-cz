# <a name="compiler-error-cs0268"></a>Chyba kompilátoru CS0268
Importovaný typ Type je neplatný. Obsahuje kruhovou závislost základní třídy.  
  
 K této chybě dochází, pokud typ importovaný z jiného jazyka obsahuje kruhovou závislost základní třídy. Takový typ nelze použít v programu v jazyce C#. Chcete-li tuto chybu vyřešit, ověřte typy importované z jiných jazyků v jakémkoli odkazovaném sestavení nebo modulech.
