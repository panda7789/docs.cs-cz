# <a name="compiler-error-cs1626"></a><span data-ttu-id="72c38-103">Chyba kompilátoru CS1626</span><span class="sxs-lookup"><span data-stu-id="72c38-103">Compiler Error CS1626</span></span>
<span data-ttu-id="72c38-104">V těle bloku try s klauzulí catch nejde použít hodnotu typu yield.</span><span class="sxs-lookup"><span data-stu-id="72c38-104">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="72c38-105">Příkaz yield není povolen v bloku try, pokud je k dispozici klauzule catch spojená s blokem try.</span><span class="sxs-lookup"><span data-stu-id="72c38-105">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="72c38-106">Chcete-li se této chybě vyhnout, buď přesuňte příkaz yield z bloku try/catch/finally, nebo odeberte blok catch.</span><span class="sxs-lookup"><span data-stu-id="72c38-106">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="72c38-107">Následující ukázka generuje CS1626:</span><span class="sxs-lookup"><span data-stu-id="72c38-107">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
