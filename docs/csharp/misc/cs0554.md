 <span data-ttu-id="90140-105">Uživatelem definované převody na hodnoty odvozené třídy nejsou povoleny; takový operátor nepotřebujete.</span><span class="sxs-lookup"><span data-stu-id="90140-105">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>

 <span data-ttu-id="90140-107">Následující ukázka generuje CS0554:</span><span class="sxs-lookup"><span data-stu-id="90140-107">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
