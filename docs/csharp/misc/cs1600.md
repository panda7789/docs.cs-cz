# <a name="compiler-error-cs1600"></a>Chyba kompilátoru CS1600
Kompilace byla zrušena uživatelem.  
  
 Kompilace s kompilátorem jazyka C# byla zrušena při použití integrovaného vývojového prostředí (IDE) sady Visual Studio.
