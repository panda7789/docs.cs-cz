# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="40006-103">Upozornění kompilátoru (úroveň 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="40006-103">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="40006-104">rozhraní rozhraní označené jako CoClassAttribute není označené jako neobsahuje deklaraci ComImportAttribute.</span><span class="sxs-lookup"><span data-stu-id="40006-104">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="40006-105">Pokud zadáte **CoClassAttribute** na rozhraní, je nutné zadat také **neobsahuje deklaraci ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="40006-105">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="40006-106">Následující ukázka generuje CS0684:</span><span class="sxs-lookup"><span data-stu-id="40006-106">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
