<span data-ttu-id="3679d-104">Předání argumentu jako ref nebo out nebo převzetí jeho adresy může způsobit výjimku za běhu, protože se jedná o pole třídy Marshal-by-reference.</span><span class="sxs-lookup"><span data-stu-id="3679d-104">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="3679d-105">Libovolná třída, která je odvozena přímo nebo nepřímo, z <xref:System.MarshalByRefObject> je třída Marshal-by-reference.</span><span class="sxs-lookup"><span data-stu-id="3679d-105">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="3679d-106">Taková třída může být zařazená odkazem napříč hranicemi procesů a počítačů.</span><span class="sxs-lookup"><span data-stu-id="3679d-106">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="3679d-107">Proto instance této třídy mohou být proxy servery do vzdálených objektů.</span><span class="sxs-lookup"><span data-stu-id="3679d-107">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="3679d-108">Pole objektu proxy nelze předat jako [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3679d-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="3679d-109">Example</span></span>  
 <span data-ttu-id="3679d-110">Následující ukázka generuje CS0197.</span><span class="sxs-lookup"><span data-stu-id="3679d-110">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
