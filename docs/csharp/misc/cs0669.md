# <a name="compiler-error-cs0669"></a><span data-ttu-id="1041e-103">Chyba kompilátoru CS0669</span><span class="sxs-lookup"><span data-stu-id="1041e-103">Compiler Error CS0669</span></span>

<span data-ttu-id="1041e-104">Třída s atributem atributem ComImport nemůže mít uživatelsky definovaný konstruktor.</span><span class="sxs-lookup"><span data-stu-id="1041e-104">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="1041e-105">Vrstva interoperability COM v modulu CLR (Common Language Runtime) poskytuje konstruktor pro třídy [atributem ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="1041e-105">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="1041e-106">V důsledku toho lze objekt COM použít jako spravovaný objekt v modulu runtime.</span><span class="sxs-lookup"><span data-stu-id="1041e-106">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="1041e-107">Následující ukázka generuje CS0669:</span><span class="sxs-lookup"><span data-stu-id="1041e-107">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
