# <a name="compiler-error-cs0157"></a>Chyba kompilátoru CS0157
Ovládací prvek nemůže opustit tělo klauzule finally.  
  
 Všechny příkazy v klauzuli [finally](../language-reference/keywords/try-catch-finally.md) musí být spuštěny. Další informace naleznete v tématu [výjimky a zpracování výjimek](../programming-guide/exceptions/index.md).  
  
 Následující ukázka generuje CS0157:  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
