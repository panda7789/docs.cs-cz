# <a name="compiler-error-cs0744"></a><span data-ttu-id="8ae04-103">Chyba kompilátoru CS0744</span><span class="sxs-lookup"><span data-stu-id="8ae04-103">Compiler Error CS0744</span></span>
<span data-ttu-id="8ae04-104">Očekávalo se kontextové klíčové slovo Equals.</span><span class="sxs-lookup"><span data-stu-id="8ae04-104">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="8ae04-105">Vzor `join` klauzule je `join` ... `in` ...`on` ... `equals` , jak je znázorněno v tomto příkladu:</span><span class="sxs-lookup"><span data-stu-id="8ae04-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8ae04-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="8ae04-106">To correct this error</span></span>  
  
1. <span data-ttu-id="8ae04-107">Přidejte `equals` klíčové slovo do `join` klauzule.</span><span class="sxs-lookup"><span data-stu-id="8ae04-107">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8ae04-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="8ae04-108">Example</span></span>  
 <span data-ttu-id="8ae04-109">Následující kód generuje CS0744:</span><span class="sxs-lookup"><span data-stu-id="8ae04-109">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
