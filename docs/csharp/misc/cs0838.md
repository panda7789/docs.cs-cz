# <a name="compiler-error-cs0838"></a>Chyba kompilátoru CS0838
Strom výrazu nesmí obsahovat inicializátor multidimenzionálního pole.  
  
 Multidimenzionální pole ve stromech výrazů nelze inicializovat pomocí inicializátoru pole.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1. Před vytvořením stromu výrazu vytvořte a inicializujte pole.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0838:  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;
  
            return 1;  
        }  
    }  
}  
```
