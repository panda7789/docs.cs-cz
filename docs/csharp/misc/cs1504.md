# <a name="compiler-error-cs1504"></a>Chyba kompilátoru CS1504
Zdrojový soubor ' file ' nelze otevřít (' důvod ')  
  
 Kompilátor nemůže otevřít zdrojový soubor nebo ho přečíst. Soubor může být uzamčený jinou aplikací, nebo se může jednat o nějaký jiný problém s operačním systémem. Zpráva obsahuje důvod operačního systému pro důvody, proč kompilátor nemohl soubor otevřít nebo načíst.
