# <a name="compiler-error-cs0283"></a>Chyba kompilátoru CS0283
Typ typu nemůže být deklarovaný jako const.  
  
Typ zadaný v deklaraci [konstanty](../language-reference/keywords/const.md) musí být `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, typ [výčtu](../language-reference/builtin-types/enum.md) nebo typ odkazu, který je přiřazený k hodnotě `null`. Každý konstantní výraz musí vracet hodnotu cílového typu nebo typu, který lze implicitně převést na cílový typ.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0283.  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
