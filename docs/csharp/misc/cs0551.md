# <a name="compiler-error-cs0551"></a><span data-ttu-id="3afe0-102">Chyba kompilátoru CS0551</span><span class="sxs-lookup"><span data-stu-id="3afe0-102">Compiler Error CS0551</span></span>
<span data-ttu-id="3afe0-103">Explicitní implementace rozhraní 'implementace' chybí přistupující ho "přistupující ho"</span><span class="sxs-lookup"><span data-stu-id="3afe0-103">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="3afe0-104">Třída, která explicitně implementuje vlastnost rozhraní, musí implementovat všechny přístupové objekty, které definuje rozhraní.</span><span class="sxs-lookup"><span data-stu-id="3afe0-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="3afe0-105">Další informace naleznete v [tématu Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="3afe0-105">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="3afe0-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="3afe0-106">Example</span></span>  
 <span data-ttu-id="3afe0-107">Následující ukázka generuje CS0551.</span><span class="sxs-lookup"><span data-stu-id="3afe0-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
