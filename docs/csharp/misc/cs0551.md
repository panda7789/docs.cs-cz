# <a name="compiler-error-cs0551"></a><span data-ttu-id="49a49-103">Chyba kompilátoru CS0551</span><span class="sxs-lookup"><span data-stu-id="49a49-103">Compiler Error CS0551</span></span>
<span data-ttu-id="49a49-104">V explicitní implementaci rozhraní implementující chybí přístupová přístupová metoda.</span><span class="sxs-lookup"><span data-stu-id="49a49-104">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="49a49-105">Třída, která explicitně implementuje vlastnost rozhraní, musí implementovat všechny přistupující objekty, které definuje rozhraní.</span><span class="sxs-lookup"><span data-stu-id="49a49-105">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="49a49-106">Další informace najdete v tématu [použití vlastností](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="49a49-106">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="49a49-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="49a49-107">Example</span></span>  
 <span data-ttu-id="49a49-108">Následující ukázka generuje CS0551.</span><span class="sxs-lookup"><span data-stu-id="49a49-108">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
