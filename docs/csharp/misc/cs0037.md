# <a name="compiler-error-cs0037"></a>Chyba kompilátoru CS0037
Hodnotu null nelze převést na typ, protože se jedná o typ, který nepovoluje hodnotu null.  
  
 Kompilátor se nemůže přiřadit `null` k typu hodnoty; `null` dá se přiřadit jedině [typu odkazu](../language-reference/keywords/reference-types.md) nebo [typu hodnoty s možnou hodnotou null](../language-reference/builtin-types/nullable-value-types.md). [Struktura](../language-reference/builtin-types/struct.md) je typ hodnoty.
  
 Následující ukázka generuje CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
