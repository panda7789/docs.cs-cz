## <a name="example"></a><span data-ttu-id="261d2-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="261d2-106">Example</span></span>  
 <span data-ttu-id="261d2-107">Následující příklad generuje CS1107:</span><span class="sxs-lookup"><span data-stu-id="261d2-107">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
