<span data-ttu-id="7a117-104">Nejde volat abstraktní základní člen: Method.</span><span class="sxs-lookup"><span data-stu-id="7a117-104">Cannot call an abstract base member: 'method'</span></span>  
  
 <span data-ttu-id="7a117-105">[Abstraktní](../language-reference/keywords/abstract.md) metodu nelze volat, protože neobsahuje tělo metody.</span><span class="sxs-lookup"><span data-stu-id="7a117-105">You cannot call an [abstract](../language-reference/keywords/abstract.md) method because it does not have a method body.</span></span> <span data-ttu-id="7a117-106">Další informace naleznete v tématu [abstraktní a zapečetěné třídy a členy třídy](../programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="7a117-106">For more information, see [Abstract and Sealed Classes and Class Members](../programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span></span>  
  
 <span data-ttu-id="7a117-107">Následující ukázka generuje CS0205:</span><span class="sxs-lookup"><span data-stu-id="7a117-107">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
