# <a name="compiler-error-cs1558"></a><span data-ttu-id="b446f-102">Chyba kompilátoru CS1558</span><span class="sxs-lookup"><span data-stu-id="b446f-102">Compiler Error CS1558</span></span>
<span data-ttu-id="b446f-103">"třída" nemá vhodnou statickou Main metodu</span><span class="sxs-lookup"><span data-stu-id="b446f-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="b446f-104">Možnost [-main](../language-reference/compiler-options/main-compiler-option.md) kompilátor udal třídu, ve které chcete hledat **Main** metoda.</span><span class="sxs-lookup"><span data-stu-id="b446f-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="b446f-105">[Main](../programming-guide/main-and-command-args/index.md) Metoda však nebyla definována správně.</span><span class="sxs-lookup"><span data-stu-id="b446f-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="b446f-106">Následující příklad generuje CS1558 z důvodu neplatného návratového typu.</span><span class="sxs-lookup"><span data-stu-id="b446f-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
