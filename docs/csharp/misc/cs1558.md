# <a name="compiler-error-cs1558"></a><span data-ttu-id="2f085-103">Chyba kompilátoru CS1558</span><span class="sxs-lookup"><span data-stu-id="2f085-103">Compiler Error CS1558</span></span>
<span data-ttu-id="2f085-104">Třída nemá vhodnou statickou metodu Main.</span><span class="sxs-lookup"><span data-stu-id="2f085-104">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="2f085-105">Možnost [-Main](../language-reference/compiler-options/main-compiler-option.md) kompilátoru určila třídu, ve které chcete hledat metodu **Main** .</span><span class="sxs-lookup"><span data-stu-id="2f085-105">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="2f085-106">Metoda [Main](../programming-guide/main-and-command-args/index.md) však nebyla definována správně.</span><span class="sxs-lookup"><span data-stu-id="2f085-106">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="2f085-107">Následující příklad generuje CS1558 z důvodu neplatného návratového typu.</span><span class="sxs-lookup"><span data-stu-id="2f085-107">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
