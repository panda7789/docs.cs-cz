<span data-ttu-id="5e7c5-104">Parametr ref nebo out nelze použít uvnitř anonymní metody, výrazu lambda nebo výrazu dotazu.</span><span class="sxs-lookup"><span data-stu-id="5e7c5-104">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="5e7c5-105">K této chybě dochází, pokud použijete `in` `ref` parametr, nebo `out` uvnitř bloku anonymní metody.</span><span class="sxs-lookup"><span data-stu-id="5e7c5-105">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="5e7c5-106">Chcete-li se této chybě vyhnout, použijte místní proměnnou nebo jinou konstrukci.</span><span class="sxs-lookup"><span data-stu-id="5e7c5-106">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="5e7c5-107">Následující ukázka generuje CS1628:</span><span class="sxs-lookup"><span data-stu-id="5e7c5-107">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
