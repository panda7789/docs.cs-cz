# <a name="compiler-error-cs0043"></a>Chyba kompilátoru CS0043

Soubor PDB má nesprávný nebo neaktuální formát. Odstraňte ho a znovu sestavte.

 Odstraňte soubor PDB pro tuto kompilaci a znovu zkompilujte.

> [!NOTE]
> Tato chyba kompilátoru se už v Roslyn nepoužívá.
