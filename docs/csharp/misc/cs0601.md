# <a name="compiler-error-cs0601"></a>Chyba kompilátoru CS0601
Pro metodu s označením static a extern musí být zadaný atribut DllImport.  
  
 `DllImport`Atribut byl použit pro metodu, která nemá správná přístupová klíčová slova.  
  
 Následující ukázka generuje CS0601:  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
