# <a name="compiler-error-cs0601"></a><span data-ttu-id="b962d-103">Chyba kompilátoru CS0601</span><span class="sxs-lookup"><span data-stu-id="b962d-103">Compiler Error CS0601</span></span>
<span data-ttu-id="b962d-104">Pro metodu s označením static a extern musí být zadaný atribut DllImport.</span><span class="sxs-lookup"><span data-stu-id="b962d-104">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="b962d-105">`DllImport`Atribut byl použit pro metodu, která nemá správná přístupová klíčová slova.</span><span class="sxs-lookup"><span data-stu-id="b962d-105">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="b962d-106">Následující ukázka generuje CS0601:</span><span class="sxs-lookup"><span data-stu-id="b962d-106">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
