# <a name="compiler-error-cs0220"></a><span data-ttu-id="5ff85-103">Chyba kompilátoru CS0220</span><span class="sxs-lookup"><span data-stu-id="5ff85-103">Compiler Error CS0220</span></span>
<span data-ttu-id="5ff85-104">Operace přetečení v době kompilace v kontrolovaném režimu</span><span class="sxs-lookup"><span data-stu-id="5ff85-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="5ff85-105">Byla zjištěna operace zaškrtnutím [políčka](../language-reference/keywords/checked.md), která je výchozím nastavením a výsledkem ztráty dat.</span><span class="sxs-lookup"><span data-stu-id="5ff85-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="5ff85-106">Opravte tuto chybu tak, že opravíte vstupy na přiřazení, nebo je [nezaškrtnuto](../language-reference/keywords/unchecked.md) .</span><span class="sxs-lookup"><span data-stu-id="5ff85-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="5ff85-107">Další informace naleznete v tématu [checked a unchecked](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="5ff85-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="5ff85-108">Následující ukázka generuje CS0220:</span><span class="sxs-lookup"><span data-stu-id="5ff85-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
