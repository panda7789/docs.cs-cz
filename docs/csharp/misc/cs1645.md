# <a name="compiler-warning-level-1-cs1645"></a>Upozornění kompilátoru (úroveň 1) CS1645

Funkce ' Feature ' není součástí standardizované specifikace ISO jazyka C# a nemusí být přijímána jinými kompilátory

Funkce, kterou používáte, není součástí standardu ISO. Kód používající tuto funkci nemusí být zkompilován na jiných kompilátorech.

```csharp
// CS1645.cs
// compile with: /W:1 /t:module /langversion:ISO-1
[assembly:System.CLSCompliant(false)]
// To suppress the warning use the switch: /nowarn:1645
[module:System.CLSCompliant(false)]   // CS1645
class Test
{
}
```
