# <a name="compiler-error-cs0591"></a>Chyba kompilátoru CS0591
Hodnota argumentu atributu atributu je neplatná.  
  
 Atribut byl předán buď neplatný argument, nebo dva vzájemně se vylučující argumenty.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0591:  
  
```csharp  
// CS0591.cs  
using System;  
  
[AttributeUsage(0)]   // CS0591  
class I: Attribute  
{  
}  
  
public class a  
{  
    public static void Main()  
    {  
    }  
}  
```
