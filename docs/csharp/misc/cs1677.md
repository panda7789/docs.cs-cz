 <span data-ttu-id="7e57e-104">K této chybě dochází, když modifikátor typu parametru v anonymní metodě neodpovídá tomu, co bylo použito v deklaraci delegáta, do které metodu předáváte.</span><span class="sxs-lookup"><span data-stu-id="7e57e-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7e57e-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="7e57e-105">Example</span></span>  
 <span data-ttu-id="7e57e-106">Následující ukázka generuje CS1677:</span><span class="sxs-lookup"><span data-stu-id="7e57e-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
