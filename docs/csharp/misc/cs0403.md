## <a name="example"></a><span data-ttu-id="f38a9-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="f38a9-109">Example</span></span>  
 <span data-ttu-id="f38a9-110">Následující ukázka generuje CS0403.</span><span class="sxs-lookup"><span data-stu-id="f38a9-110">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
