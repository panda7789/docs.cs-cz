# <a name="compiler-error-cs0426"></a><span data-ttu-id="8d664-102">Chyba kompilátoru CS0426</span><span class="sxs-lookup"><span data-stu-id="8d664-102">Compiler Error CS0426</span></span>
<span data-ttu-id="8d664-103">Název typu "identifikátor" neexistuje v typu "type"</span><span class="sxs-lookup"><span data-stu-id="8d664-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="8d664-104">K této chybě dochází, když odkazujete na typ vnořený v rámci jiného typu, ale žádný takový vnořený typ neexistuje.</span><span class="sxs-lookup"><span data-stu-id="8d664-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="8d664-105">Tato situace může nastat, pokud jste nesprávně zadali název vnořeného typu.</span><span class="sxs-lookup"><span data-stu-id="8d664-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="8d664-106">Zkontrolujte pravopis použitých názvů a ověřte, zda má ohraničující typ očekávaný člen.</span><span class="sxs-lookup"><span data-stu-id="8d664-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="8d664-107">Následující ukázka generuje CS0426, protože třída C nemá vnořený typ A:</span><span class="sxs-lookup"><span data-stu-id="8d664-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
