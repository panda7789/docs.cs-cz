# <a name="compiler-error-cs0426"></a><span data-ttu-id="4db42-103">Chyba kompilátoru CS0426</span><span class="sxs-lookup"><span data-stu-id="4db42-103">Compiler Error CS0426</span></span>
<span data-ttu-id="4db42-104">Název typu ' identifikátor ' neexistuje v typu ' type '</span><span class="sxs-lookup"><span data-stu-id="4db42-104">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="4db42-105">K této chybě dochází, pokud odkazujete na typ vnořený v jiném typu, ale žádný takový vnořený typ neexistuje.</span><span class="sxs-lookup"><span data-stu-id="4db42-105">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="4db42-106">Tato situace může nastat, pokud nezadáte název vnořeného typu.</span><span class="sxs-lookup"><span data-stu-id="4db42-106">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="4db42-107">Zkontrolujte pravopis používaného názvu a ověřte, zda nadřazený typ má očekávaného člena.</span><span class="sxs-lookup"><span data-stu-id="4db42-107">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="4db42-108">Následující ukázka vygeneruje CS0426, protože třída C nemá žádný vnořený typ A:</span><span class="sxs-lookup"><span data-stu-id="4db42-108">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
