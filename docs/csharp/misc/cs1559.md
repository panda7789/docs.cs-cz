# <a name="compiler-error-cs1559"></a>Chyba kompilátoru CS1559
Klíčové slovo Object nejde použít pro metodu Main, protože je importovaný.  
  
 Byla zadána neplatná třída pro možnost [– Hlavní](../language-reference/compiler-options/main-compiler-option.md) kompilátor; třídu nelze použít jako umístění pro metodu [Main](../programming-guide/main-and-command-args/index.md) .
