# <a name="compiler-error-cs0076"></a>Chyba kompilátoru CS0076
Název čítače výčtu ' value__ ' je rezervován a nelze jej použít  
  
 Položka ve [výčtu](../language-reference/builtin-types/enum.md) nemůže mít identifikátor nazvaný **value__**.
