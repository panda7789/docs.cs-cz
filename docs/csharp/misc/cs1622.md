 <span data-ttu-id="3efa9-108">Následující ukázka generuje CS1622:</span><span class="sxs-lookup"><span data-stu-id="3efa9-108">The following sample generates CS1622:</span></span>  
  
```csharp

using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()

   }  
}  
```
