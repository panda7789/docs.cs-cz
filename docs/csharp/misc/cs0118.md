# <a name="compiler-error-cs0118"></a>Chyba kompilátoru CS0118
' construct1_name ' je ' construct1 ', ale používá se jako ' construct2 '  
  
 Kompilátor zjistil situaci, kdy se konstrukce použila v nějakém chybném způsobu nebo Nepovolená operace se pokusila o konstrukci. Mezi běžné příklady patří následující:  
  
- Pokus o vytvoření instance oboru názvů (namísto třídy)  
  
- Pokus o volání pole (místo metody)  
  
- Pokus o použití typu jako proměnné  
  
- Zkuste použít externí alias jako typ.  
  
 Chcete-li tuto chybu vyřešit, ujistěte se, že operace, kterou provádíte, je platná pro typ, na kterém provádíte operaci.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0118.  
  
```csharp  
// CS0118.cs  
// compile with: /target:library  
namespace MyNamespace  
{  
   class MyClass  
   {  
      // MyNamespace not a class  
      MyNamespace ix = new MyNamespace ();   // CS0118  
   }  
}  
```
