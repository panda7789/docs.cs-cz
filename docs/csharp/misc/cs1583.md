# <a name="compiler-error-cs1583"></a>Chyba kompilátoru CS1583
' file ' není platný soubor prostředků Win32  
  
 K této chybě dochází, pokud zadáte název souboru s parametrem [-win32res](../language-reference/compiler-options/win32res-compiler-option.md) , který není platným nebo správně formátovaným souborem prostředku Win32. V aplikaci Visual Studio je název souboru zadán v podokně aplikace v Návrháři projektu.
