# <a name="compiler-error-cs1575"></a><span data-ttu-id="f30fd-103">Chyba kompilátoru CS1575</span><span class="sxs-lookup"><span data-stu-id="f30fd-103">Compiler Error CS1575</span></span>
<span data-ttu-id="f30fd-104">Výraz stackalloc vyžaduje po typu [].</span><span class="sxs-lookup"><span data-stu-id="f30fd-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="f30fd-105">Velikost požadovaného přidělení s [stackalloc](../language-reference/operators/stackalloc.md)musí být uvedena v hranatých závorkách.</span><span class="sxs-lookup"><span data-stu-id="f30fd-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="f30fd-106">Následující ukázka generuje CS1575:</span><span class="sxs-lookup"><span data-stu-id="f30fd-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
