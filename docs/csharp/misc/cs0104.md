# <a name="compiler-error-cs0104"></a><span data-ttu-id="6362b-103">Chyba kompilátoru CS0104</span><span class="sxs-lookup"><span data-stu-id="6362b-103">Compiler Error CS0104</span></span>
<span data-ttu-id="6362b-104">odkaz je nejednoznačný odkaz mezi identifikátorem a identifikátorem.</span><span class="sxs-lookup"><span data-stu-id="6362b-104">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="6362b-105">Váš program obsahuje direktivy [using](../language-reference/keywords/using.md) pro dva obory názvů a váš kód odkazuje na název, který se zobrazí v obou oborech názvů.</span><span class="sxs-lookup"><span data-stu-id="6362b-105">Your program contains [using](../language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="6362b-106">Následující ukázka generuje CS0104:</span><span class="sxs-lookup"><span data-stu-id="6362b-106">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
