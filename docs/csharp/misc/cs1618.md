# <a name="compiler-error-cs1618"></a><span data-ttu-id="9bd90-103">Chyba kompilátoru CS1618</span><span class="sxs-lookup"><span data-stu-id="9bd90-103">Compiler Error CS1618</span></span>
<span data-ttu-id="9bd90-104">Nelze vytvořit delegáta s metodou Method, protože má atribut Conditional.</span><span class="sxs-lookup"><span data-stu-id="9bd90-104">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="9bd90-105">Nelze vytvořit delegáta s podmíněnou metodou, protože tato metoda nemusí existovat v některých sestaveních.</span><span class="sxs-lookup"><span data-stu-id="9bd90-105">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="9bd90-106">Následující ukázka generuje CS1618:</span><span class="sxs-lookup"><span data-stu-id="9bd90-106">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
