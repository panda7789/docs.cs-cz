# <a name="compiler-error-cs1618"></a><span data-ttu-id="f1b22-102">Chyba kompilátoru CS1618</span><span class="sxs-lookup"><span data-stu-id="f1b22-102">Compiler Error CS1618</span></span>
<span data-ttu-id="f1b22-103">Nelze vytvořit delegáta s metodou, protože má podmíněný atribut.</span><span class="sxs-lookup"><span data-stu-id="f1b22-103">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="f1b22-104">Delegáta s podmíněnou metodou nelze vytvořit, protože metoda nemusí v některých sestaveních existovat.</span><span class="sxs-lookup"><span data-stu-id="f1b22-104">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="f1b22-105">Následující ukázka generuje CS1618:</span><span class="sxs-lookup"><span data-stu-id="f1b22-105">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
