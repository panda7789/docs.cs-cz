# <a name="compiler-error-cs1621"></a><span data-ttu-id="44f23-103">Chyba kompilátoru CS1621</span><span class="sxs-lookup"><span data-stu-id="44f23-103">Compiler Error CS1621</span></span>
<span data-ttu-id="44f23-104">Příkaz yield se nedá použít uvnitř anonymní metody nebo výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="44f23-104">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="44f23-105">Nemůžete použít příkaz [yield](../language-reference/keywords/yield.md) v [anonymní metodě](../language-reference/operators/delegate-operator.md) nebo [výrazu lambda](../language-reference/operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="44f23-105">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../language-reference/operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="44f23-106">Následující ukázka generuje CS1621:</span><span class="sxs-lookup"><span data-stu-id="44f23-106">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
