# <a name="compiler-warning-level-1-cs0602"></a>Upozornění kompilátoru (úroveň 1) CS0602

Funkce old_feature je zastaralá. Místo toho prosím použijte new_feature.

 Funkce jazyka použitá ve vašem kódu (*old_feature*) je stále podporovaná, ale tato podpora může být v budoucí verzi odebrána. Místo toho byste měli použít doporučenou syntaxi (*new_feature*).

> [!NOTE]
> Toto upozornění kompilátoru se už v Roslyn nepoužívá.
