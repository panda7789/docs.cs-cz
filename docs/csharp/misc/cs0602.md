# <a name="compiler-warning-level-1-cs0602"></a>Upozornění kompilátoru (úroveň 1) CS0602

Funkce "old_feature" je zastaralá. Místo toho použijte "new_feature".

 Funkce jazyka používaná ve vašem kódu *(old_feature)* je stále podporována, ale tato podpora může být odebrána v budoucí verzi. Místo toho byste měli použít doporučenou syntaxi (*new_feature*).

> [!NOTE]
> Toto upozornění kompilátoru se již nepoužívá v Roslyn.
