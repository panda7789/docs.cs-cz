## <a name="example"></a><span data-ttu-id="10751-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="10751-108">Example</span></span>

<span data-ttu-id="10751-109">Následující kód generuje CS1955, protože kód se pokouší vyvolat pole a vlastnost pomocí [ `()`výrazu vyvolání ](../language-reference/operators/member-access-operators.md#invocation-expression-).</span><span class="sxs-lookup"><span data-stu-id="10751-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="10751-110">Nelze volat pole nebo vlastnost.</span><span class="sxs-lookup"><span data-stu-id="10751-110">You cannot call a field or a property.</span></span> <span data-ttu-id="10751-111">Pomocí [výrazu `.` přístupu člena](../language-reference/operators/member-access-operators.md#member-access-expression-) přístup k hodnotě, kterou ukládá.</span><span class="sxs-lookup"><span data-stu-id="10751-111">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
