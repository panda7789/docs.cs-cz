# <a name="compiler-error-cs1662"></a><span data-ttu-id="5405c-103">Chyba kompilátoru CS1662</span><span class="sxs-lookup"><span data-stu-id="5405c-103">Compiler Error CS1662</span></span>

<span data-ttu-id="5405c-104">Blok anonymní metody nejde převést na typ delegáta, protože některé z návratových typů v bloku nejsou implicitně převoditelné na návratový typ delegáta.</span><span class="sxs-lookup"><span data-stu-id="5405c-104">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="5405c-105">K této chybě dochází, pokud příkaz return bloku anonymní metody měl typ, který nebyl implicitně převeden na návratový typ delegáta.</span><span class="sxs-lookup"><span data-stu-id="5405c-105">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="5405c-106">Následující ukázka generuje CS1662:</span><span class="sxs-lookup"><span data-stu-id="5405c-106">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
