<span data-ttu-id="fc3f6-103">Před vrácením řízení volajícímu modulu musí být plně přiřazeno pole zálohování pro automaticky implementovanou vlastnost Name.</span><span class="sxs-lookup"><span data-stu-id="fc3f6-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="fc3f6-104">Zvažte volání konstruktoru bez parametrů z inicializátoru konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="fc3f6-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>

```csharp  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error

   public int i;  
}

{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
