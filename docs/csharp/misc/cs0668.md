# <a name="compiler-error-cs0668"></a><span data-ttu-id="66e9f-103">Chyba kompilátoru CS0668</span><span class="sxs-lookup"><span data-stu-id="66e9f-103">Compiler Error CS0668</span></span>
<span data-ttu-id="66e9f-104">Dva indexery mají odlišné názvy. atribut IndexerName musí být v rámci typu použit se stejným názvem pro každý indexer.</span><span class="sxs-lookup"><span data-stu-id="66e9f-104">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="66e9f-105">Hodnoty předané atributu **IndexerName** musí být stejné pro všechny Indexery v typu.</span><span class="sxs-lookup"><span data-stu-id="66e9f-105">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="66e9f-106">Další informace o atributu **IndexerName** naleznete v tématu [Třída IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="66e9f-106">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="66e9f-107">Následující ukázka generuje CS0668:</span><span class="sxs-lookup"><span data-stu-id="66e9f-107">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
