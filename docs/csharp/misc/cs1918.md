# <a name="compiler-error-cs1918"></a><span data-ttu-id="0d398-102">CS1918 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="0d398-102">Compiler Error CS1918</span></span>
<span data-ttu-id="0d398-103">Členy vlastnosti "name" typu "typ" nelze přiřadit k inicializátoru objektu protože se jedná o hodnotu typu.</span><span class="sxs-lookup"><span data-stu-id="0d398-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="0d398-104">Této chybě dochází při pokusu o použití inicializátoru objektu inicializovat vlastnosti typu Struktura, která sama o sobě vlastnost třídy, která je inicializována.</span><span class="sxs-lookup"><span data-stu-id="0d398-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0d398-105">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="0d398-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="0d398-106">Pokud musí plně inicializovat pole vlastností v inicializátoru objektu, změňte strukturu na typ třídy.</span><span class="sxs-lookup"><span data-stu-id="0d398-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="0d398-107">Po vytvoření objektu pomocí inicializátoru objektu, jinak inicializujte členy struktury v samostatné metodě volání.</span><span class="sxs-lookup"><span data-stu-id="0d398-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0d398-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="0d398-108">Example</span></span>  
 <span data-ttu-id="0d398-109">Následující příklad generuje CS1918:</span><span class="sxs-lookup"><span data-stu-id="0d398-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
