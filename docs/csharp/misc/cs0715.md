# <a name="compiler-error-cs0715"></a>Chyba kompilátoru CS0715
static Class: statické třídy nemůžou obsahovat operátory definované uživatelem.  
  
 Uživatelsky definované operátory fungují na instancích třídy. Instance statických tříd nelze vytvořit, takže není možné vytvářet instance, na jejichž základě budou fungovat. Proto uživatelsky definované operátory nejsou povoleny pro statické třídy.  
  
 Následující ukázka generuje CS0715:  
  
```csharp  
// CS0715.cs  
public static class C  
{  
   public static C operator+(C c)  // CS0715  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
