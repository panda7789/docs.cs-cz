# <a name="compiler-error-cs1055"></a><span data-ttu-id="e9eb5-103">Chyba kompilátoru CS1055</span><span class="sxs-lookup"><span data-stu-id="e9eb5-103">Compiler Error CS1055</span></span>
<span data-ttu-id="e9eb5-104">Očekával se přistupující objekt Add nebo Remove.</span><span class="sxs-lookup"><span data-stu-id="e9eb5-104">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="e9eb5-105">Pokud vaše [událost](../language-reference/keywords/event.md) není deklarována jako pole, musí definovat funkce přistupujícího objektu **Add** i **Remove** .</span><span class="sxs-lookup"><span data-stu-id="e9eb5-105">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="e9eb5-106">Následující ukázka generuje CS1055:</span><span class="sxs-lookup"><span data-stu-id="e9eb5-106">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
