# <a name="compiler-error-cs0542"></a><span data-ttu-id="fa41d-103">Chyba kompilátoru CS0542</span><span class="sxs-lookup"><span data-stu-id="fa41d-103">Compiler Error CS0542</span></span>
<span data-ttu-id="fa41d-104">' user-defined type ': názvy členů nemohou být stejné jako jejich nadřazený typ</span><span class="sxs-lookup"><span data-stu-id="fa41d-104">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="fa41d-105">Členy třídy nebo struktury nemohou mít stejný název jako třída nebo struktura, pokud člen není konstruktor.</span><span class="sxs-lookup"><span data-stu-id="fa41d-105">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="fa41d-106">Následující ukázka generuje CS0542:</span><span class="sxs-lookup"><span data-stu-id="fa41d-106">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="fa41d-107">Tato chyba může být způsobena neúmyslným vložením návratového typu do konstruktoru, který se v důsledku toho stane běžnou metodou.</span><span class="sxs-lookup"><span data-stu-id="fa41d-107">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="fa41d-108">Následující příklad generuje CS0542, protože `F` je metoda, nikoli konstruktor, protože má návratový typ:</span><span class="sxs-lookup"><span data-stu-id="fa41d-108">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="fa41d-109">Pokud má vaše třída název Item a má indexovací člen deklarovaný jako `this` , může se zobrazit tato chyba.</span><span class="sxs-lookup"><span data-stu-id="fa41d-109">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="fa41d-110">Výchozímu indexeru je uveden název Item ve vygenerovaném kódu, což vytváří konflikt.</span><span class="sxs-lookup"><span data-stu-id="fa41d-110">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>
