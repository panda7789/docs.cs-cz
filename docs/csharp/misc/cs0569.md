# <a name="compiler-error-cs0569"></a>Chyba kompilátoru CS0569
' Method2 ': nelze přepsat ' – metoda1 ', protože není jazykem podporován  
  
 K této chybě dochází, pokud odvozujete ze základní třídy, která byla napsána v jiném jazyce, a když kompilátor nepodporuje metodu, kterou se pokoušíte přepsat.
