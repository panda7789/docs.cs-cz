# <a name="compiler-warning-level-3-cs0661"></a>Upozornění kompilátoru (úroveň 3) CS0661
Class definuje operátor = = nebo! =, ale nepřepisuje Object. GetHashCode ().  
  
 Kompilátor zjistil operátor rovnosti nebo nerovnosti definovaný uživatelem, ale žádné přepsání funkce **GetHashCode** . Uživatelem definovaný operátor rovnosti nebo nerovnosti znamená, že chcete také přepsat funkci **GetHashCode** .  
  
 Následující ukázka generuje CS0661:  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
