# <a name="compiler-error-cs1613"></a><span data-ttu-id="e0434-103">Chyba kompilátoru CS1613</span><span class="sxs-lookup"><span data-stu-id="e0434-103">Compiler Error CS1613</span></span>
<span data-ttu-id="e0434-104">Spravovaná třída obálky třídy coclass pro rozhraní Interface se nenašla. (nechybí odkaz na sestavení?)</span><span class="sxs-lookup"><span data-stu-id="e0434-104">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="e0434-105">Byl proveden pokus o vytvoření instance objektu COM z rozhraní.</span><span class="sxs-lookup"><span data-stu-id="e0434-105">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="e0434-106">Rozhraní má **atributem ComImport** a `CoClass` atributy, ale kompilátor nemůže najít typ zadaný pro `CoClass` atribut.</span><span class="sxs-lookup"><span data-stu-id="e0434-106">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="e0434-107">Chcete-li vyřešit tuto chybu, můžete vyzkoušet jednu z následujících možností:</span><span class="sxs-lookup"><span data-stu-id="e0434-107">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="e0434-108">Přidejte odkaz na sestavení, které má coclass (nejvíce z času, že rozhraní a třída coclass by měly být ve stejném sestavení).</span><span class="sxs-lookup"><span data-stu-id="e0434-108">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="e0434-109">Informace najdete v tématu [reference](../language-reference/compiler-options/reference-compiler-option.md) k [dialogovému oknu odkaz nebo přidání odkazu](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) .</span><span class="sxs-lookup"><span data-stu-id="e0434-109">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="e0434-110">Opravte `CoClass` atribut rozhraní.</span><span class="sxs-lookup"><span data-stu-id="e0434-110">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="e0434-111">Následující příklad znázorňuje správné použití **CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="e0434-111">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
