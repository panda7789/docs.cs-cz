# <a name="compiler-error-cs1608"></a>Chyba kompilátoru CS1608
Povinný atribut není povolený pro typy C#.  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> není povoleno pro typy definované v kódu jazyka C#.
