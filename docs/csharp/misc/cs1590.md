# <a name="compiler-warning-level-1-cs1590"></a>Upozornění kompilátoru (úroveň 1) CS1590
Neplatný element include XML – chybějící atribut souboru  
  
 Atribut Path nebo DOC, který byl předán [\<include>](../programming-guide/xmldoc/include.md) značce, nebyl nalezen nebo je neúplný.  
  
 Následující ukázka generuje CS1590:  
  
```csharp  
// CS1590.cs  
// compile with: /W:1 /doc:x.xml  
  
/// <include path='MyDocs/MyMembers[@name="test"]/*' />   // CS1590  
// try the following line instead  
// /// <include file='x.doc' path='MyDocs/MyMembers[@name="test"]/*' />  
class Test  
{  
   public static void Main()  
   {  
   }  
}  
```
