 <span data-ttu-id="1d9a0-104">Pole (proměnná) označené klíčovým slovem [jen pro čtení](../language-reference/keywords/readonly.md) nelze předat parametru [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md) s výjimkou uvnitř konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="1d9a0-104">A field (variable) marked with the [readonly](../language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="1d9a0-105">Další informace naleznete v tématu [Fields](../programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="1d9a0-105">For more information, see [Fields](../programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="1d9a0-106">CS0192 také výsledky, `readonly` pokud pole je [statické](../language-reference/keywords/static.md) `static`a konstruktor není označen .</span><span class="sxs-lookup"><span data-stu-id="1d9a0-106">CS0192 also results if the `readonly` field is [static](../language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1d9a0-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="1d9a0-107">Example</span></span>  
 <span data-ttu-id="1d9a0-108">Následující ukázka generuje CS0192.</span><span class="sxs-lookup"><span data-stu-id="1d9a0-108">The following sample generates CS0192.</span></span>  
  
```csharp
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
