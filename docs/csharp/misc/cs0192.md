# <a name="compiler-error-cs0192"></a><span data-ttu-id="3ee6f-103">Chyba kompilátoru CS0192</span><span class="sxs-lookup"><span data-stu-id="3ee6f-103">Compiler Error CS0192</span></span>
<span data-ttu-id="3ee6f-104">Pole statického pole jen pro čtení Name nejde předat parametrem ref nebo out (kromě případu, kdy se nachází ve statickém konstruktoru).</span><span class="sxs-lookup"><span data-stu-id="3ee6f-104">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="3ee6f-105">Pole (proměnné) označené klíčovým slovem [ReadOnly](../language-reference/keywords/readonly.md) nelze předat parametru [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md) s výjimkou uvnitř konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="3ee6f-105">A field (variable) marked with the [readonly](../language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="3ee6f-106">Další informace najdete v tématu [pole](../programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="3ee6f-106">For more information, see [Fields](../programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="3ee6f-107">CS0192 také výsledky, pokud `readonly` je pole [statické](../language-reference/keywords/static.md) a konstruktor není označen `static` .</span><span class="sxs-lookup"><span data-stu-id="3ee6f-107">CS0192 also results if the `readonly` field is [static](../language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3ee6f-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="3ee6f-108">Example</span></span>  
 <span data-ttu-id="3ee6f-109">Následující ukázka generuje CS0192.</span><span class="sxs-lookup"><span data-stu-id="3ee6f-109">The following sample generates CS0192.</span></span>  
  
```csharp
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
