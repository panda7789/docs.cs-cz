 <span data-ttu-id="e9ad2-110">Následující kód vygeneruje CS0738, protože metoda třídy vrací hodnotu `void` a člen rozhraní se stejným názvem vrátí `int` :</span><span class="sxs-lookup"><span data-stu-id="e9ad2-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>
