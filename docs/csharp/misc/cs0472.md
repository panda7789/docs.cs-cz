# <a name="compiler-warning-level-2-cs0472"></a><span data-ttu-id="f046a-102">Upozornění kompilátoru (úroveň 2) CS0472</span><span class="sxs-lookup"><span data-stu-id="f046a-102">Compiler Warning (level 2) CS0472</span></span>
<span data-ttu-id="f046a-103">Výsledkem výrazu je vždy 'value1' protože hodnota typu 'value2' se nikdy nerovná 'null' typu 'value3'</span><span class="sxs-lookup"><span data-stu-id="f046a-103">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>  
  
 <span data-ttu-id="f046a-104">Kompilátor by měl upozornit, pokud používáte operátor s konstantní hodnotou null.</span><span class="sxs-lookup"><span data-stu-id="f046a-104">The compiler should warn if you use an operator with a constant null value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f046a-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="f046a-105">Example</span></span>  
 <span data-ttu-id="f046a-106">Následující ukázka generuje CS0472.</span><span class="sxs-lookup"><span data-stu-id="f046a-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
