## <a name="example"></a><span data-ttu-id="51486-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="51486-108">Example</span></span>  
 <span data-ttu-id="51486-109">Následující ukázka generuje CS1649:</span><span class="sxs-lookup"><span data-stu-id="51486-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
