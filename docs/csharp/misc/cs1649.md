 <span data-ttu-id="9237b-105">K této chybě dojde, Pokud předáte proměnnou do funkce, která je členem `readonly` pole jako `ref` `out` argument or.</span><span class="sxs-lookup"><span data-stu-id="9237b-105">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="9237b-106">Vzhledem `ref` `out` k tomu, že je možné parametry a změnit, není tato funkce povolená.</span><span class="sxs-lookup"><span data-stu-id="9237b-106">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="9237b-107">Chcete-li tuto chybu vyřešit, odeberte `readonly` klíčové slovo pro pole nebo nepředávejte členy `readonly` pole do funkce.</span><span class="sxs-lookup"><span data-stu-id="9237b-107">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="9237b-108">Můžete například zkusit vytvořit dočasnou proměnnou, která se dá upravit a předat dočasnou hodnotu jako `ref` argument, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="9237b-108">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9237b-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="9237b-109">Example</span></span>  
 <span data-ttu-id="9237b-110">Následující ukázka generuje CS1649:</span><span class="sxs-lookup"><span data-stu-id="9237b-110">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
