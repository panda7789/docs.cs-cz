# <a name="compiler-error-cs1631"></a>Chyba kompilátoru CS1631
Nelze získat hodnotu v těle klauzule catch.  
  
 Výnos prohlášení není povoleno z v rámci těla catch klauzule. Chcete-li se vyhnout této chybě, přesuňte yield prohlášení mimo tělo catch klauzule.  
  
 Následující ukázka generuje CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
