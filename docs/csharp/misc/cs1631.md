# <a name="compiler-error-cs1631"></a><span data-ttu-id="83ee2-102">Chyba kompilátoru CS1631</span><span class="sxs-lookup"><span data-stu-id="83ee2-102">Compiler Error CS1631</span></span>
<span data-ttu-id="83ee2-103">Nelze získat hodnotu v těle klauzule catch.</span><span class="sxs-lookup"><span data-stu-id="83ee2-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="83ee2-104">Výnos prohlášení není povoleno z v rámci těla catch klauzule.</span><span class="sxs-lookup"><span data-stu-id="83ee2-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="83ee2-105">Chcete-li se vyhnout této chybě, přesuňte yield prohlášení mimo tělo catch klauzule.</span><span class="sxs-lookup"><span data-stu-id="83ee2-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="83ee2-106">Následující ukázka generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="83ee2-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
