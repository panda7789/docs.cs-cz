# <a name="compiler-error-cs1631"></a><span data-ttu-id="8cda4-103">Chyba kompilátoru CS1631</span><span class="sxs-lookup"><span data-stu-id="8cda4-103">Compiler Error CS1631</span></span>
<span data-ttu-id="8cda4-104">V těle klauzule catch nejde získat hodnotu.</span><span class="sxs-lookup"><span data-stu-id="8cda4-104">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="8cda4-105">Příkaz yield není povolen v rámci těla klauzule catch.</span><span class="sxs-lookup"><span data-stu-id="8cda4-105">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="8cda4-106">Chcete-li se této chybě vyhnout, přesuňte příkaz yield mimo tělo klauzule catch.</span><span class="sxs-lookup"><span data-stu-id="8cda4-106">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="8cda4-107">Následující ukázka generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="8cda4-107">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
