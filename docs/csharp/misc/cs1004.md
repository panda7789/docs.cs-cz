# <a name="compiler-error-cs1004"></a>Chyba kompilátoru CS1004
Duplicitní modifikátor modifikátoru  
  
 Byl zjištěn duplicitní modifikátor, například modifikátor **přístupu** .  
  
 Následující ukázka generuje CS1004:  
  
```csharp  
// CS1004.cs  
public class clx  
{  
   public public static void Main()   // CS1004, two public keywords  
   {  
   }  
}  
```
