# <a name="compiler-error-cs0231"></a><span data-ttu-id="4b846-102">Chyba kompilátoru CS0231</span><span class="sxs-lookup"><span data-stu-id="4b846-102">Compiler Error CS0231</span></span>
<span data-ttu-id="4b846-103">Parametr params musí být posledním parametrem v seznamu formálních parametrů.</span><span class="sxs-lookup"><span data-stu-id="4b846-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="4b846-104">Parametr [params](../language-reference/keywords/params.md) podporuje proměnná čísla argumentů a musí být po všech ostatních parametrech.</span><span class="sxs-lookup"><span data-stu-id="4b846-104">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="4b846-105">Další informace naleznete v [tématu Metody](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="4b846-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="4b846-106">Následující ukázka generuje CS0231:</span><span class="sxs-lookup"><span data-stu-id="4b846-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
