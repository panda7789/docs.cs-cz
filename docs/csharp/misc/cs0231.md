# <a name="compiler-error-cs0231"></a><span data-ttu-id="5eb62-103">Chyba kompilátoru CS0231</span><span class="sxs-lookup"><span data-stu-id="5eb62-103">Compiler Error CS0231</span></span>
<span data-ttu-id="5eb62-104">Parametr params musí být posledním parametrem v seznamu formálních parametrů.</span><span class="sxs-lookup"><span data-stu-id="5eb62-104">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="5eb62-105">Parametr [params](../language-reference/keywords/params.md) podporuje proměnlivá čísla argumentů a musí se nacházet po všech ostatních parametrech.</span><span class="sxs-lookup"><span data-stu-id="5eb62-105">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="5eb62-106">Další informace naleznete v tématu [metody](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="5eb62-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="5eb62-107">Následující ukázka generuje CS0231:</span><span class="sxs-lookup"><span data-stu-id="5eb62-107">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
