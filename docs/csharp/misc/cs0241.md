## <a name="example"></a><span data-ttu-id="19706-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="19706-107">Example</span></span>

```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```
