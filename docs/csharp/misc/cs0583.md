# <a name="compiler-error-cs0583"></a>Chyba kompilátoru CS0583

Vnitřní chyba kompilátoru. V kompilátoru nastala vnitřní chyba. Chcete-li tento problém vyřešit, zkuste zjednodušit nebo změnit program v blízkosti umístění uvedených níže. Umístění v horní části seznamu jsou blíže k bodu, ve kterém došlo k vnitřní chybě. Chyby, jako je tato, mohou být hlášeny společnosti Microsoft pomocí možnosti /errorreport.

 Vytvořte soubor hlášení o chybách s možností "/bugreport" a odešlete zprávu přidělenému kontaktu hlášení problémů.

 K zachycení problému použijte možnost kompilátoru [-bugreport.](../language-reference/compiler-options/bugreport-compiler-option.md)

> [!NOTE]
> Tato chyba kompilátoru se již nepoužívá v Roslyn.
