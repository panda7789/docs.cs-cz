# <a name="compiler-error-cs0113"></a><span data-ttu-id="c4a19-103">Chyba kompilátoru CS0113</span><span class="sxs-lookup"><span data-stu-id="c4a19-103">Compiler Error CS0113</span></span>
<span data-ttu-id="c4a19-104">Člen funkce označený jako override nejde označit jako New nebo Virtual.</span><span class="sxs-lookup"><span data-stu-id="c4a19-104">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="c4a19-105">Vzájemně se vylučují pro označení metody pomocí klíčových slov [New](../language-reference/keywords/new-modifier.md) a [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="c4a19-105">It is mutually exclusive to mark a method with the [new](../language-reference/keywords/new-modifier.md) and [override](../language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="c4a19-106">Následující ukázka generuje CS0113:</span><span class="sxs-lookup"><span data-stu-id="c4a19-106">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
