<span data-ttu-id="2308a-104">přistupující objekt: nejde přepsat, protože vlastnost Property nemá přístupový objekt set s možnostmi overridable.</span><span class="sxs-lookup"><span data-stu-id="2308a-104">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="2308a-105">Pokus o přepsání jedné z přístupových metod pro vlastnost se nezdařil, protože přistupující objekt nelze přepsat.</span><span class="sxs-lookup"><span data-stu-id="2308a-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="2308a-106">K této chybě může dojít, pokud:</span><span class="sxs-lookup"><span data-stu-id="2308a-106">This error can occur if:</span></span>  
  
- <span data-ttu-id="2308a-107">vlastnost základní třídy není deklarována jako [Virtual](../language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="2308a-107">the base class property is not declared as [virtual](../language-reference/keywords/virtual.md).</span></span>  
  
- <span data-ttu-id="2308a-108">vlastnost základní třídy nedeklaruje přistupující objekt [Get](../language-reference/keywords/get.md) nebo [set](../language-reference/keywords/set.md) , který se pokoušíte přepsat.</span><span class="sxs-lookup"><span data-stu-id="2308a-108">the base class property does not declare the [get](../language-reference/keywords/get.md) or [set](../language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="2308a-109">Pokud nechcete přepsat vlastnost základní třídy, můžete použít klíčové slovo [New](../language-reference/keywords/new-modifier.md) před vlastností v odvozené třídě.</span><span class="sxs-lookup"><span data-stu-id="2308a-109">If you do not want to override the base class property, you can use the [new](../language-reference/keywords/new-modifier.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="2308a-110">Další informace najdete v tématu [použití vlastností](../programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="2308a-110">For more information, see [Using Properties](../programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="2308a-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="2308a-111">Example</span></span>  
 <span data-ttu-id="2308a-112">Následující ukázka vygeneruje CS0546, protože základní třída nedeklaruje přistupující objekt set pro vlastnost.</span><span class="sxs-lookup"><span data-stu-id="2308a-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
