# <a name="compiler-error-cs1561"></a>Chyba kompilátoru CS1561
Název výstupního souboru je moc dlouhý nebo neplatný.  
  
 Název výstupního souboru nesmí být delší než 256 znaků a nesmí obsahovat neplatné znaky, jako je například symbol Euro, otazník nebo zpětné lomítko, mimo jiné.
