# <a name="compiler-error-cs1511"></a><span data-ttu-id="6eee5-103">Chyba kompilátoru CS1511</span><span class="sxs-lookup"><span data-stu-id="6eee5-103">Compiler Error CS1511</span></span>
<span data-ttu-id="6eee5-104">Klíčové slovo Base není k dispozici ve statické metodě.</span><span class="sxs-lookup"><span data-stu-id="6eee5-104">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="6eee5-105">Klíčové slovo [Base](../language-reference/keywords/base.md) bylo použito ve [statické](../language-reference/keywords/static.md) metodě.</span><span class="sxs-lookup"><span data-stu-id="6eee5-105">The [base](../language-reference/keywords/base.md) keyword was used in a [static](../language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="6eee5-106">`base` dá se volat jenom v konstruktoru instance, metodě instance nebo přístupovém objektu instance.</span><span class="sxs-lookup"><span data-stu-id="6eee5-106">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6eee5-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="6eee5-107">Example</span></span>  
 <span data-ttu-id="6eee5-108">Následující ukázka generuje CS1511.</span><span class="sxs-lookup"><span data-stu-id="6eee5-108">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
