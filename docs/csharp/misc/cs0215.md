# <a name="compiler-error-cs0215"></a>Chyba kompilátoru CS0215
Návratový typ operátoru true nebo false musí být typu bool.  
  
Uživatelsky definované operátory [true a false](../language-reference/operators/true-false-operators.md) musí mít návratový typ [bool](../language-reference/builtin-types/bool.md).
  
Následující ukázka generuje CS0215:  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
