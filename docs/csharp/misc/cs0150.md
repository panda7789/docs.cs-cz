# <a name="compiler-error-cs0150"></a><span data-ttu-id="d399e-102">Chyba kompilátoru CS0150</span><span class="sxs-lookup"><span data-stu-id="d399e-102">Compiler Error CS0150</span></span>
<span data-ttu-id="d399e-103">Očekává se konstantní hodnota.</span><span class="sxs-lookup"><span data-stu-id="d399e-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="d399e-104">Proměnná byla nalezena tam, kde byla očekávána konstanta.</span><span class="sxs-lookup"><span data-stu-id="d399e-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="d399e-105">Další informace naleznete v tématu [switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="d399e-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="d399e-106">Následující ukázka generuje CS0150:</span><span class="sxs-lookup"><span data-stu-id="d399e-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="d399e-107">Tato chyba je také vytvořena, když je zadána velikost pole s hodnotou proměnné a inicializována inicializátorem pole.</span><span class="sxs-lookup"><span data-stu-id="d399e-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="d399e-108">Chcete-li chybu odebrat, inicializujte pole v samostatném příkazu nebo příkazech.</span><span class="sxs-lookup"><span data-stu-id="d399e-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
