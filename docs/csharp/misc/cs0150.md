# <a name="compiler-error-cs0150"></a><span data-ttu-id="d3189-103">Chyba kompilátoru CS0150</span><span class="sxs-lookup"><span data-stu-id="d3189-103">Compiler Error CS0150</span></span>
<span data-ttu-id="d3189-104">Očekává se konstantní hodnota.</span><span class="sxs-lookup"><span data-stu-id="d3189-104">A constant value is expected</span></span>  
  
 <span data-ttu-id="d3189-105">Byla nalezena proměnná, kde byla očekávána konstanta.</span><span class="sxs-lookup"><span data-stu-id="d3189-105">A variable was found where a constant was expected.</span></span> <span data-ttu-id="d3189-106">Další informace naleznete v tématu [Switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="d3189-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="d3189-107">Následující ukázka generuje CS0150:</span><span class="sxs-lookup"><span data-stu-id="d3189-107">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="d3189-108">Tato chyba je vytvořena také v případě, že je zadána velikost pole s proměnnou hodnotou a inicializována s inicializátorem pole.</span><span class="sxs-lookup"><span data-stu-id="d3189-108">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="d3189-109">Chcete-li odstranit chybu, inicializujte pole v samostatném příkazu nebo příkazech.</span><span class="sxs-lookup"><span data-stu-id="d3189-109">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
