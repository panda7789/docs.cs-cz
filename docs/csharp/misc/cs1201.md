# <a name="compiler-warning-level-1-cs1201"></a>Upozornění kompilátoru (úroveň 1) CS1201
Funkce ' invalid Feature ' je zastaralá. Místo toho použijte možnost ' platná funkce '.  
  
 Funkce, kterou se pokoušíte použít, je nyní zastaralá. Aktualizujte kód tak, aby místo něj používal platnou funkci.
