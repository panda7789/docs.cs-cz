# <a name="compiler-warning-level-1-cs1203"></a>Upozornění kompilátoru (úroveň 1) CS1203
Funkce ' Feature ' je zastaralá. Místo toho použijte funkci ' Feature '.  
  
 Funkce ' invalid Feature ' je zastaralá. Místo toho použijte ' platnou funkci '.  
  
 Funkce, kterou se pokoušíte použít, je nyní zastaralá. Aktualizujte kód tak, aby místo něj používal platnou funkci.
