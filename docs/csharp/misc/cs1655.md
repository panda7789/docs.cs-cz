# <a name="compiler-error-cs1655"></a><span data-ttu-id="dddc3-103">Chyba kompilátoru CS1655</span><span class="sxs-lookup"><span data-stu-id="dddc3-103">Compiler Error CS1655</span></span>
<span data-ttu-id="dddc3-104">Pole proměnné nelze předat jako argument ref nebo out, protože se jedná o typ proměnné jen pro čtení.</span><span class="sxs-lookup"><span data-stu-id="dddc3-104">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="dddc3-105">K této chybě dochází, pokud se pokoušíte předat člen proměnné [foreach](../language-reference/keywords/foreach-in.md) , [using](../language-reference/keywords/using-statement.md) nebo [pevnou](../language-reference/keywords/fixed-statement.md) proměnnou funkce jako argument ref nebo out.</span><span class="sxs-lookup"><span data-stu-id="dddc3-105">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="dddc3-106">Vzhledem k tomu, že tyto proměnné jsou v těchto kontextech považovány za jen pro čtení, to není povoleno.</span><span class="sxs-lookup"><span data-stu-id="dddc3-106">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="dddc3-107">Následující ukázka generuje CS1655:</span><span class="sxs-lookup"><span data-stu-id="dddc3-107">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
