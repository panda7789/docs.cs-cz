# <a name="compiler-error-cs1655"></a><span data-ttu-id="5ffd4-102">Chyba kompilátoru CS1655</span><span class="sxs-lookup"><span data-stu-id="5ffd4-102">Compiler Error CS1655</span></span>
<span data-ttu-id="5ffd4-103">Nelze předat pole 'proměnné' jako ref nebo out argument, protože se jedná o "typ proměnné jen pro čtení"</span><span class="sxs-lookup"><span data-stu-id="5ffd4-103">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="5ffd4-104">K této chybě dochází, pokud se pokoušíte předat člen [foreach](../language-reference/keywords/foreach-in.md) proměnné, [using](../language-reference/keywords/using-statement.md) proměnné nebo [pevné](../language-reference/keywords/fixed-statement.md) proměnné funkce jako ref nebo out argument.</span><span class="sxs-lookup"><span data-stu-id="5ffd4-104">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="5ffd4-105">Vzhledem k tomu, že tyto proměnné jsou považovány za jen pro čtení v těchto kontextech, to není povoleno.</span><span class="sxs-lookup"><span data-stu-id="5ffd4-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="5ffd4-106">Následující ukázka generuje CS1655:</span><span class="sxs-lookup"><span data-stu-id="5ffd4-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
