 <span data-ttu-id="e830a-105">Pokud má částečná metoda implementaci, omezení obecného typu musí být stejné jako omezení definované v signatuře metody.</span><span class="sxs-lookup"><span data-stu-id="e830a-105">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
