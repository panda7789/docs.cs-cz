 <span data-ttu-id="2ddc3-104">Toto upozornění může dojít v několika situacích, ve kterých můžete ověřit kompilátor nikdy odkazován proměnné:</span><span class="sxs-lookup"><span data-stu-id="2ddc3-104">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>
