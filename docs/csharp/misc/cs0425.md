## <a name="example"></a><span data-ttu-id="4a3fb-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="4a3fb-108">Example</span></span>

```csharp

class C1  
{  
}  
  
class C2  
{  
}

{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}

    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425

    {  
    }  
}  
```
