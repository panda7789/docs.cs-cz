# <a name="compiler-error-cs0202"></a><span data-ttu-id="5be79-103">Chyba kompilátoru CS0202</span><span class="sxs-lookup"><span data-stu-id="5be79-103">Compiler Error CS0202</span></span>
<span data-ttu-id="5be79-104">příkaz foreach vyžaduje, aby typ vrácené typ byl typu. GetEnumerator () musí mít vhodnou veřejnou metodu MoveNext a veřejnou vlastnost Current.</span><span class="sxs-lookup"><span data-stu-id="5be79-104">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="5be79-105"><xref:System.Collections.IEnumerable.GetEnumerator%2A>Funkce, která slouží k povolení použití příkazů foreach, nemůže vracet ukazatel nebo pole; musí vracet instanci třídy, která může fungovat jako enumerátor.</span><span class="sxs-lookup"><span data-stu-id="5be79-105">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="5be79-106">Správné požadavky, které slouží jako enumerátor, zahrnují veřejnou vlastnost Public a veřejnou metodu MoveNext.</span><span class="sxs-lookup"><span data-stu-id="5be79-106">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>

```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
