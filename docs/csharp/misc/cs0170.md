# <a name="compiler-error-cs0170"></a><span data-ttu-id="6a600-102">Chyba kompilátoru CS0170</span><span class="sxs-lookup"><span data-stu-id="6a600-102">Compiler Error CS0170</span></span>
<span data-ttu-id="6a600-103">Použití pravděpodobně nepřiřazeného pole ' Field '</span><span class="sxs-lookup"><span data-stu-id="6a600-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="6a600-104">Pole ve struktuře bylo použito bez prvotní inicializace.</span><span class="sxs-lookup"><span data-stu-id="6a600-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="6a600-105">Chcete-li tento problém vyřešit, nejprve určete, které pole nebylo inicializováno, a poté jej inicializujte předtím, než se pokusíte o přístup k němu.</span><span class="sxs-lookup"><span data-stu-id="6a600-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="6a600-106">Další informace o inicializaci struktur naleznete v tématu [typy struktury](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="6a600-106">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="6a600-107">Následující ukázka generuje CS0170:</span><span class="sxs-lookup"><span data-stu-id="6a600-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
