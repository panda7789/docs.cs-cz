## <a name="example"></a><span data-ttu-id="74ba8-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="74ba8-107">Example</span></span>  
 <span data-ttu-id="74ba8-108">Tento příklad znázorňuje počet různých operací s typy hodnot s možnou hodnotou null, které způsobí tuto chybu.</span><span class="sxs-lookup"><span data-stu-id="74ba8-108">This example illustrates a number of the different operations with nullable value types that will cause this error.</span></span>  
  
```csharp

using System;  
public  class Test
{  
    public static void Main()  
    {  
        int a = 5;  
        int? b = a + null;    // CS0458  
        int? qa = 15;  
        b = qa + null;        // CS0458  
        b -= null;            // CS0458  
        int? qa2 = null;  
        b = qa2 + null;       // CS0458  
        qa2 -= null;          // CS0458  
    }  
}  
```
