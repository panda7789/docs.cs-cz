# <a name="compiler-warning-level-2-cs0458"></a>Upozornění kompilátoru (úroveň 2) CS0458
Výsledkem výrazu je vždy 'null' typu 'název typu'  
  
 Toto upozornění je `nullable` způsobeno výrazem, který vždy vede v aplikaci `null`.  
  
 Následující kód generuje upozornění CS0458.  
  
## <a name="example"></a>Příklad  
 Tento příklad ilustruje počet různých operací `nullable` s typy, které způsobí tuto chybu.  
  
```csharp  
// CS0458.cs  
using System;  
public  class Test
{  
    public static void Main()  
    {  
        int a = 5;  
        int? b = a + null;    // CS0458  
        int? qa = 15;  
        b = qa + null;        // CS0458  
        b -= null;            // CS0458  
        int? qa2 = null;  
        b = qa2 + null;       // CS0458  
        qa2 -= null;          // CS0458  
    }  
}  
```
