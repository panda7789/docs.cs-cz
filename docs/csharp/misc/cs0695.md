# <a name="compiler-error-cs0695"></a><span data-ttu-id="f1bec-103">Chyba kompilátoru CS0695</span><span class="sxs-lookup"><span data-stu-id="f1bec-103">Compiler Error CS0695</span></span>
<span data-ttu-id="f1bec-104">obecný typ nemůže implementovat generické rozhraní i obecné rozhraní, protože u některých náhrad parametrů typu můžou být sjednocené.</span><span class="sxs-lookup"><span data-stu-id="f1bec-104">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="f1bec-105">K této chybě dochází, pokud obecná třída implementuje více než jeden Parametrizace stejného obecného rozhraní a existuje nahrazení parametru typu, které by měla být totožná se dvěma rozhraními.</span><span class="sxs-lookup"><span data-stu-id="f1bec-105">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="f1bec-106">Chcete-li se této chybě vyhnout, implementujte pouze jedno rozhraní nebo změňte parametry typu, aby se zabránilo konfliktu.</span><span class="sxs-lookup"><span data-stu-id="f1bec-106">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="f1bec-107">Následující ukázka generuje CS0695:</span><span class="sxs-lookup"><span data-stu-id="f1bec-107">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```
