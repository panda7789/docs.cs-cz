# <a name="compiler-error-cs1527"></a><span data-ttu-id="9916e-103">Chyba kompilátoru CS1527</span><span class="sxs-lookup"><span data-stu-id="9916e-103">Compiler Error CS1527</span></span>

<span data-ttu-id="9916e-104">Elementy definované v oboru názvů nejde explicitně deklarovat jako Private, Protected, Protected Internal nebo Private Protected.</span><span class="sxs-lookup"><span data-stu-id="9916e-104">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="9916e-105">Deklarace typů v oboru názvů můžou mít buď [veřejný](../language-reference/keywords/public.md) , nebo [interní](../language-reference/keywords/internal.md) přístup.</span><span class="sxs-lookup"><span data-stu-id="9916e-105">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="9916e-106">Pokud není zadaná žádná přístupnost, výchozí hodnota je **interní** .</span><span class="sxs-lookup"><span data-stu-id="9916e-106">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="9916e-107">Následující ukázka generuje CS1527:</span><span class="sxs-lookup"><span data-stu-id="9916e-107">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="9916e-108">Následující příklad generuje CS1527, protože pokud není v kódu programu explicitně deklarován žádný obor názvů, jsou všechny deklarace typů umístěny implicitně v rámci globálního oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="9916e-108">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
