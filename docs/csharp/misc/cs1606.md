# <a name="compiler-error-cs1606"></a>Chyba kompilátoru CS1606
Podpis sestavení se nezdařil; výstup nesmí být podepsán – důvod  
  
 Sestavení bylo vytvořeno, ale v případě, že se kompilátor pokusil dokončit podepisování, došlo k chybě.
