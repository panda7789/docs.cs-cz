 <span data-ttu-id="97461-105">Příkaz [foreach](../language-reference/keywords/foreach-in.md) byl špatně vytvořen.</span><span class="sxs-lookup"><span data-stu-id="97461-105">A [foreach](../language-reference/keywords/foreach-in.md) statement was poorly formed.</span></span>

{  
   public static void Main()  
   {  
      int[] myarray = new int[3] {1,2,3};  
  
      foreach (int in myarray)   // CS0230

      }  
   }
