 <span data-ttu-id="74212-105">K této chybě dochází, pokud typ ve zdrojovém souboru je v konfliktu s oborem názvů v jiném zdrojovém souboru.</span><span class="sxs-lookup"><span data-stu-id="74212-105">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="74212-106">K tomu obvykle dochází v případě, že jeden nebo oba přicházejí z přidaného modulu.</span><span class="sxs-lookup"><span data-stu-id="74212-106">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="74212-107">Chcete-li tento problém vyřešit, přejmenujte typ nebo obor názvů, který způsobil konflikt.</span><span class="sxs-lookup"><span data-stu-id="74212-107">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="74212-108">Následující příklad generuje CS0438:</span><span class="sxs-lookup"><span data-stu-id="74212-108">The following example generates CS0438:</span></span>

 <span data-ttu-id="74212-110">Potom zkompilujte tento soubor:</span><span class="sxs-lookup"><span data-stu-id="74212-110">Then compile this file:</span></span>

```csharp  
   public class A { }  
}  
```  
  
 <span data-ttu-id="74212-111">A potom tento soubor zkompilujte:</span><span class="sxs-lookup"><span data-stu-id="74212-111">And then compile this file:</span></span>
