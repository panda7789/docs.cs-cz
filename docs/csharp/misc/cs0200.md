<span data-ttu-id="5fd55-104">Došlo k pokusu o přiřazení hodnoty [vlastnosti](../programming-guide/classes-and-structs/using-properties.md), ale vlastnost nemá přistupující objekt set nebo přiřazení bylo mimo konstruktor.</span><span class="sxs-lookup"><span data-stu-id="5fd55-104">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="5fd55-105">Vyřešte chybu přidáním přistupujícího přístupového počitadla sady.</span><span class="sxs-lookup"><span data-stu-id="5fd55-105">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="5fd55-106">Další informace naleznete v tématu [Jak deklarovat a používat vlastnosti zápisu pro čtení](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span><span class="sxs-lookup"><span data-stu-id="5fd55-106">For more information, see [How to declare and use read write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="5fd55-107">Příklady</span><span class="sxs-lookup"><span data-stu-id="5fd55-107">Examples</span></span>
<span data-ttu-id="5fd55-108">Následující ukázka generuje CS0200:</span><span class="sxs-lookup"><span data-stu-id="5fd55-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="5fd55-109">Následující ukázka používá [automaticky implementované vlastnosti](../programming-guide/classes-and-structs/auto-implemented-properties.md), [inicializační objekty objektu](../programming-guide/classes-and-structs/object-and-collection-initializers.md)a stále generuje CS0200:</span><span class="sxs-lookup"><span data-stu-id="5fd55-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="5fd55-110">Přiřazení vlastnosti nebo indexeru 'vlastnost', která je jen pro čtení, lze dosáhnout přidáním set přistupujícího objektu nebo přiřazením vlastnostv konstruktoru objektu.</span><span class="sxs-lookup"><span data-stu-id="5fd55-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
