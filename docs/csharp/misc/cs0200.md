# <a name="compiler-error-cs0200"></a>Chyba kompilátoru CS0200
Vlastnost nebo indexer 'vlastnost' nelze přiřadit -- je jen pro čtení

Došlo k pokusu o přiřazení hodnoty [vlastnosti](../programming-guide/classes-and-structs/using-properties.md), ale vlastnost nemá přistupující objekt set nebo přiřazení bylo mimo konstruktor. Vyřešte chybu přidáním přistupujícího přístupového počitadla sady. Další informace naleznete v tématu [Jak deklarovat a používat vlastnosti zápisu pro čtení](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).

## <a name="examples"></a>Příklady
Následující ukázka generuje CS0200:

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

Následující ukázka používá [automaticky implementované vlastnosti](../programming-guide/classes-and-structs/auto-implemented-properties.md), [inicializační objekty objektu](../programming-guide/classes-and-structs/object-and-collection-initializers.md)a stále generuje CS0200:

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

Přiřazení vlastnosti nebo indexeru 'vlastnost', která je jen pro čtení, lze dosáhnout přidáním set přistupujícího objektu nebo přiřazením vlastnostv konstruktoru objektu.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
