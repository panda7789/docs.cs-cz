## <a name="examples"></a><span data-ttu-id="f7cbe-108">Příklady</span><span class="sxs-lookup"><span data-stu-id="f7cbe-108">Examples</span></span>

<span data-ttu-id="f7cbe-109">Následující ukázka generuje CS0200:</span><span class="sxs-lookup"><span data-stu-id="f7cbe-109">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="f7cbe-110">Následující příklad používá [automaticky implementované vlastnosti](../programming-guide/classes-and-structs/auto-implemented-properties.md) a [Inicializátory objektů](../programming-guide/classes-and-structs/object-and-collection-initializers.md) a stále generuje CS0200:</span><span class="sxs-lookup"><span data-stu-id="f7cbe-110">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="f7cbe-111">Chcete-li přiřadit vlastnost nebo indexeru vlastnost, která je jen pro čtení, přidejte přístupový objekt set nebo přiřaďte hodnotu v konstruktoru objektu.</span><span class="sxs-lookup"><span data-stu-id="f7cbe-111">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
