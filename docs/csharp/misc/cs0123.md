# <a name="compiler-error-cs0123"></a><span data-ttu-id="6d93f-103">Chyba kompilátoru CS0123</span><span class="sxs-lookup"><span data-stu-id="6d93f-103">Compiler Error CS0123</span></span>
<span data-ttu-id="6d93f-104">Žádné přetížení pro metodu neodpovídá delegátovi Delegate.</span><span class="sxs-lookup"><span data-stu-id="6d93f-104">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="6d93f-105">Pokus o vytvoření delegáta se nezdařil, protože se nepoužil správný podpis.</span><span class="sxs-lookup"><span data-stu-id="6d93f-105">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="6d93f-106">Instance delegáta musí být deklarovány se stejnou signaturou jako deklarace delegáta.</span><span class="sxs-lookup"><span data-stu-id="6d93f-106">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="6d93f-107">Tuto chybu můžete vyřešit tak, že upravíte buď metodu, nebo podpis delegáta.</span><span class="sxs-lookup"><span data-stu-id="6d93f-107">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="6d93f-108">Další informace najdete v tématu [Delegáti](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="6d93f-108">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="6d93f-109">Následující ukázka generuje CS0123.</span><span class="sxs-lookup"><span data-stu-id="6d93f-109">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
