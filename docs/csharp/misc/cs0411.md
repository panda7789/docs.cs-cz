## <a name="example"></a><span data-ttu-id="045a1-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="045a1-108">Example</span></span>  
 <span data-ttu-id="045a1-109">Následující ukázka generuje CS0411:</span><span class="sxs-lookup"><span data-stu-id="045a1-109">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs

{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
