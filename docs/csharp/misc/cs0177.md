# <a name="compiler-error-cs0177"></a><span data-ttu-id="f38d1-103">Chyba kompilátoru CS0177</span><span class="sxs-lookup"><span data-stu-id="f38d1-103">Compiler Error CS0177</span></span>
<span data-ttu-id="f38d1-104">Výstupní parametr ' Parameter ' musí být přiřazen před tím, než ovládací prvek opustí aktuální metodu</span><span class="sxs-lookup"><span data-stu-id="f38d1-104">The out parameter 'parameter' must be assigned to before control leaves the current method</span></span>  
  
 <span data-ttu-id="f38d1-105">Na parametr označený klíčovým slovem [out](../language-reference/keywords/out-parameter-modifier.md) se nepřiřadila hodnota v těle metody.</span><span class="sxs-lookup"><span data-stu-id="f38d1-105">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="f38d1-106">Další informace najdete v tématu [předávání parametrů](../programming-guide/classes-and-structs/passing-parameters.md) .</span><span class="sxs-lookup"><span data-stu-id="f38d1-106">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>  
  
 <span data-ttu-id="f38d1-107">Následující ukázka generuje CS0177:</span><span class="sxs-lookup"><span data-stu-id="f38d1-107">The following sample generates CS0177:</span></span>  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
