## <a name="example"></a><span data-ttu-id="2efc8-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="2efc8-108">Example</span></span>  
 <span data-ttu-id="2efc8-109">Následující příklad generuje CS1930, `num` protože identifikátor se používá `from` pro proměnnou rozsahu v klauzuli a pro proměnnou rozsahu zavedenou `let` klauzulí.</span><span class="sxs-lookup"><span data-stu-id="2efc8-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;
    }  
}  
```
