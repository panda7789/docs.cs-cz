# <a name="compiler-error-cs0449"></a><span data-ttu-id="2993b-103">Chyba kompilátoru CS0449</span><span class="sxs-lookup"><span data-stu-id="2993b-103">Compiler Error CS0449</span></span>
<span data-ttu-id="2993b-104">Omezení class nebo struct musí být zadané před všemi ostatními omezeními.</span><span class="sxs-lookup"><span data-stu-id="2993b-104">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="2993b-105">Omezení pro parametr typu obecného typu nebo metody se musí nacházet v určitém pořadí: `class` nebo `struct` musí být první, pokud je k dispozici, jakákoli omezení rozhraní a nakonec omezení konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="2993b-105">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="2993b-106">Tato chyba je způsobená `class` omezením nebo, které se `struct` nezobrazuje jako první.</span><span class="sxs-lookup"><span data-stu-id="2993b-106">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="2993b-107">Chcete-li tuto chybu vyřešit, přeuspořádání klauzulí omezení.</span><span class="sxs-lookup"><span data-stu-id="2993b-107">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2993b-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="2993b-108">Example</span></span>  
 <span data-ttu-id="2993b-109">Následující ukázka generuje CS0449.</span><span class="sxs-lookup"><span data-stu-id="2993b-109">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
