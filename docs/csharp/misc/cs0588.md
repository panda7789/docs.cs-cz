# <a name="compiler-error-cs0588"></a>Chyba kompilátoru CS0588

Vnitřní chyba kompilátoru: fáze ' LEX '

 Pokuste se zjistit, zda se kompilátor nezdařil z důvodu neschopnosti analyzovat neočekávanou syntaxi. Pokud se nejedná o tento případ, zkuste [nás kontaktovat](/visualstudio/ide/feedback-options).

> [!NOTE]
> Tato chyba kompilátoru se už v Roslyn nepoužívá.
