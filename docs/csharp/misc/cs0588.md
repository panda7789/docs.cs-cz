# <a name="compiler-error-cs0588"></a>Chyba kompilátoru CS0588

Vnitřní chyba kompilátoru: fáze LEX

 Pokuste se zjistit, zda kompilátor selhává z důvodu jeho neschopnost analyzovat neočekávanou syntaxi. Pokud tomu tak není, zkuste [s námi mluvit](/visualstudio/ide/feedback-options).

> [!NOTE]
> Tato chyba kompilátoru se již nepoužívá v Roslyn.
