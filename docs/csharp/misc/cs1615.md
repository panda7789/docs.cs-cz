# <a name="compiler-error-cs1615"></a><span data-ttu-id="09821-103">Chyba kompilátoru CS1615</span><span class="sxs-lookup"><span data-stu-id="09821-103">Compiler Error CS1615</span></span>
<span data-ttu-id="09821-104">Argument Number by neměl být předaný s klíčovým slovem klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="09821-104">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="09821-105">Jedno z klíčových slov `ref` nebo **out** bylo použito, pokud funkce nepřijala `ref` parametr nebo **out** pro tento argument.</span><span class="sxs-lookup"><span data-stu-id="09821-105">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="09821-106">Chcete-li tuto chybu vyřešit, odeberte nesprávné klíčové slovo a použijte příslušné klíčové slovo, které odpovídá deklaraci funkce, pokud existuje.</span><span class="sxs-lookup"><span data-stu-id="09821-106">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="09821-107">Následující ukázka generuje CS1615:</span><span class="sxs-lookup"><span data-stu-id="09821-107">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
