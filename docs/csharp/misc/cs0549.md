# <a name="compiler-error-cs0549"></a><span data-ttu-id="a8cb5-103">Chyba kompilátoru CS0549</span><span class="sxs-lookup"><span data-stu-id="a8cb5-103">Compiler Error CS0549</span></span>
<span data-ttu-id="a8cb5-104">' function ' je nový virtuální člen v zapečetěné třídě ' class '</span><span class="sxs-lookup"><span data-stu-id="a8cb5-104">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="a8cb5-105">[Zapečetěná](../language-reference/keywords/sealed.md)[Třída](../language-reference/keywords/class.md) se nedá použít jako základní třída.</span><span class="sxs-lookup"><span data-stu-id="a8cb5-105">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="a8cb5-106">Proto je nevhodné mít virtuální metodu v zapečetěné třídě.</span><span class="sxs-lookup"><span data-stu-id="a8cb5-106">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="a8cb5-107">Následující ukázka generuje CS0549:</span><span class="sxs-lookup"><span data-stu-id="a8cb5-107">The following sample generates CS0549:</span></span>

```csharp  
{  
   virtual public void TestMethod() {}   // CS0549

}  
```
