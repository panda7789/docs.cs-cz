# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="2de92-103">Upozornění kompilátoru (úroveň 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="2de92-103">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="2de92-104">Hodnotu goto Case nejde implicitně převést na typ Type.</span><span class="sxs-lookup"><span data-stu-id="2de92-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="2de92-105">Použijete `goto case` -li, musí existovat implicitní převod z hodnoty Case pro příkaz Přejít na typ přepínače.</span><span class="sxs-lookup"><span data-stu-id="2de92-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2de92-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="2de92-106">Example</span></span>  
 <span data-ttu-id="2de92-107">Následující ukázka generuje CS0469.</span><span class="sxs-lookup"><span data-stu-id="2de92-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
