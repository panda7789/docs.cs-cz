# <a name="compiler-error-cs0568"></a><span data-ttu-id="d54cd-102">Chyba kompilátoru CS0568</span><span class="sxs-lookup"><span data-stu-id="d54cd-102">Compiler Error CS0568</span></span>
<span data-ttu-id="d54cd-103">Struktury nemůžou obsahovat explicitní konstruktory bez parametrů.</span><span class="sxs-lookup"><span data-stu-id="d54cd-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="d54cd-104">Každá [Struktura](../language-reference/builtin-types/struct.md) již má konstruktor bez parametrů, který inicializuje objekt na nulu.</span><span class="sxs-lookup"><span data-stu-id="d54cd-104">Each [struct](../language-reference/builtin-types/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="d54cd-105">Proto konstruktory, které lze vytvořit pro strukturu, musí mít jeden nebo více parametrů.</span><span class="sxs-lookup"><span data-stu-id="d54cd-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="d54cd-106">Následující ukázka generuje CS0568:</span><span class="sxs-lookup"><span data-stu-id="d54cd-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
