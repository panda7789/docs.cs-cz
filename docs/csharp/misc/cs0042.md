# <a name="compiler-error-cs0042"></a>Chyba kompilátoru CS0042
Neočekávaná chyba při vytváření souboru s ladicími informacemi ' file ' – důvod '  
  
 Kompilátor nemohl vytvořit ladicí informace; *důvod* obsahuje další informace o chybovém stavu.
