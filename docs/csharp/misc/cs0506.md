# <a name="compiler-error-cs0506"></a><span data-ttu-id="41257-103">Chyba kompilátoru CS0506</span><span class="sxs-lookup"><span data-stu-id="41257-103">Compiler Error CS0506</span></span>
<span data-ttu-id="41257-104">' Function1 ': zděděný člen ' function2 ' nelze přepsat, protože není označen jako "Virtual", "Abstract" nebo "override"</span><span class="sxs-lookup"><span data-stu-id="41257-104">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="41257-105">Metoda byla přepsána, která nebyla explicitně označena jako **virtuální**, **abstraktní**nebo `override` .</span><span class="sxs-lookup"><span data-stu-id="41257-105">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="41257-106">Následující ukázka generuje CS0506:</span><span class="sxs-lookup"><span data-stu-id="41257-106">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
