<span data-ttu-id="bb471-104">Přistupující objekt Add nebo Remove musí mít tělo.</span><span class="sxs-lookup"><span data-stu-id="bb471-104">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="bb471-105">Klíčové slovo **Add** nebo **Remove** v definici [události](../language-reference/keywords/event.md) musí mít tělo.</span><span class="sxs-lookup"><span data-stu-id="bb471-105">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="bb471-106">Další informace najdete v tématu [události](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="bb471-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="bb471-107">Následující ukázka generuje CS0073:</span><span class="sxs-lookup"><span data-stu-id="bb471-107">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
