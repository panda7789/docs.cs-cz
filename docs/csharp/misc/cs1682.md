# <a name="compiler-warning-level-1-cs1682"></a>Upozornění kompilátoru (úroveň 1) CS1682
Odkaz na typ 'typ' tvrdí, že je vnořený do "vnořeného typu", ale nebyl nalezen  
  
 K této chybě dochází při importu odkazů, které nesouhlasí s jinými odkazy nebo s kódem, který jste napsali. Běžným způsobem, jak získat tuto chybu, je napsat kód, který odkazuje na třídu v metadatech, a potom tuto třídu odstraníte nebo změníte její definici.  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS1682.cs  
// compile with: /target:library /keyfile:mykey.snk  
public class A {  
   public class N1 {}  
}  
```  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS1682_b.cs  
// compile with: /target:library /reference:CS1682.dll  
using System;  
public class Ref {  
  
   public static A A1() {  
      return new A();  
   }  
  
   public static A.N1 N1() {
      return new A.N1();  
   }  
}  
```  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS1682_c.cs  
// compile with: /target:library /keyfile:mykey.snk /out:CS1682.dll  
public class A {  
   public void M1() {}  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1682.  
  
```csharp  
// CS1682_d.cs  
// compile with: /reference:CS1682.dll /reference:CS1682_b.dll /W:1  
// CS1682 expected  
class Tester {  
   static void Main()  
   {  
      Ref.A1().M1();  
   }  
}  
```
