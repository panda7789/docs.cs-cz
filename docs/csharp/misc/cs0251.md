# <a name="compiler-warning-level-2-cs0251"></a>Upozornění kompilátoru (úroveň 2) CS0251
Indexování pole záporným indexem (indexy polí vždy začínají nulou)  
  
 Nepoužívejte ke indexování pole záporné číslo.  
  
 Následující ukázka generuje CS0251:  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
