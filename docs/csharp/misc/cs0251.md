# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="da91c-103">Upozornění kompilátoru (úroveň 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="da91c-103">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="da91c-104">Indexování pole záporným indexem (indexy polí vždy začínají nulou)</span><span class="sxs-lookup"><span data-stu-id="da91c-104">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="da91c-105">Nepoužívejte ke indexování pole záporné číslo.</span><span class="sxs-lookup"><span data-stu-id="da91c-105">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="da91c-106">Následující ukázka generuje CS0251:</span><span class="sxs-lookup"><span data-stu-id="da91c-106">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
