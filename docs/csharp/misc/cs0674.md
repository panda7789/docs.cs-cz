# <a name="compiler-error-cs0674"></a><span data-ttu-id="406b8-102">Chyba kompilátoru CS0674</span><span class="sxs-lookup"><span data-stu-id="406b8-102">Compiler Error CS0674</span></span>
<span data-ttu-id="406b8-103">Nepoužívejte atribut System.ParamArrayAttribute.</span><span class="sxs-lookup"><span data-stu-id="406b8-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="406b8-104">Místo toho použijte klíčové slovo params.</span><span class="sxs-lookup"><span data-stu-id="406b8-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="406b8-105">Kompilátor Jazyka C# neumožňuje <xref:System.ParamArrayAttribute?displayProperty=nameWithType>použití ; místo toho použijte [params.](../language-reference/keywords/params.md)</span><span class="sxs-lookup"><span data-stu-id="406b8-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="406b8-106">Následující ukázka generuje CS0674:</span><span class="sxs-lookup"><span data-stu-id="406b8-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
