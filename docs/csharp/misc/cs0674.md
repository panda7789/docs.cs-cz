ParamArrayAttribute.</span><span class="sxs-lookup"><span data-stu-id="2dac7-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="2dac7-105">Místo toho použijte klíčové slovo params.</span><span class="sxs-lookup"><span data-stu-id="2dac7-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="2dac7-106">Kompilátor jazyka C# nepovoluje použití <xref:System.ParamArrayAttribute?displayProperty=nameWithType> [parametrů](../language-reference/keywords/params.md) ; použijte místo nich.</span><span class="sxs-lookup"><span data-stu-id="2dac7-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="2dac7-107">Následující ukázka generuje CS0674:</span><span class="sxs-lookup"><span data-stu-id="2dac7-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
