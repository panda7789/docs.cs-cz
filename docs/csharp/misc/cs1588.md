# <a name="compiler-error-cs1588"></a>Chyba kompilátoru CS1588
Nejde určit adresář společného jazykového modulu runtime--důvod.  
  
 Kompilátor nemůže načíst adresář, ve kterém je nainstalován modul CLR .NET Framework Common Language Runtime. Tato chyba označuje, že modul CLR (Common Language Runtime) není správně nainstalován.
