<span data-ttu-id="96d40-104">Částečná metoda nemůže mít modifikátory přístupu ani modifikátory Virtual, abstract, override, New, Sealed nebo extern.</span><span class="sxs-lookup"><span data-stu-id="96d40-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="96d40-105">Z důvodu jejich zvláštního chování podléhají částečné metody omezením pro modifikátory, které mohou přijmout.</span><span class="sxs-lookup"><span data-stu-id="96d40-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>

1. <span data-ttu-id="96d40-107">Odeberte z deklarace metody neautorizovaný modifikátor.</span><span class="sxs-lookup"><span data-stu-id="96d40-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="96d40-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="96d40-108">Example</span></span>  
 <span data-ttu-id="96d40-109">Následující příklad generuje CS0750:</span><span class="sxs-lookup"><span data-stu-id="96d40-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
