# <a name="compiler-error-cs1577"></a>Chyba kompilátoru CS1577
Vytváření sestavení nebylo úspěšné – důvod  
  
 Část sestavení-Generation kompilace selhala poté, co kompilátor jazyka C# úspěšně zkompiluje zdrojový kód. Další informace najdete v dokumentaci k chybám pro nástroj ALink ([Al.exe chyby a upozornění nástroje](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)).
