# <a name="compiler-error-cs0520"></a>Chyba kompilátoru CS0520
Předdefinovaný typ Name je deklarovaný nesprávně.  
  
 Jeden nebo více souborů vyžadovaných kompilátorem se nenašlo. Přeinstalujte .NET Framework modul CLR (Common Language Runtime).
