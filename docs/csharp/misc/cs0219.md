# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="934d7-103">Upozornění kompilátoru (úroveň 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="934d7-103">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="934d7-104">Proměnná ' Variable ' je přiřazena, ale její hodnota se nikdy nepoužívá.</span><span class="sxs-lookup"><span data-stu-id="934d7-104">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="934d7-105">Kompilátor vydá upozornění úrovně tři, pokud deklarujete a přiřadíte proměnnou, ale nepoužijete ji.</span><span class="sxs-lookup"><span data-stu-id="934d7-105">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="934d7-106">Kompilátor vygeneruje toto upozornění pouze v případě, že hodnota proměnné je konstanta v čase kompilace.</span><span class="sxs-lookup"><span data-stu-id="934d7-106">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="934d7-107">Přiřazení nekonstantního výrazu nebo výsledku metody místní proměnné usnadňuje sledování těchto výrazů v ladicím programu.</span><span class="sxs-lookup"><span data-stu-id="934d7-107">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="934d7-108">Dojde také k dosažení výsledku, zabránění uvolňování paměti, když je tato proměnná dostupná.</span><span class="sxs-lookup"><span data-stu-id="934d7-108">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="934d7-109">Následující příklad ukazuje případy, kdy a kdy není vygenerováno upozornění:</span><span class="sxs-lookup"><span data-stu-id="934d7-109">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
