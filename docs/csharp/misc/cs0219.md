# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="bb2c7-102">Upozornění kompilátoru (úroveň 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="bb2c7-102">Compiler Warning (level 3) CS0219</span></span>

 <span data-ttu-id="bb2c7-104">Kompilátor vydá upozornění úrovně tři, když deklarujete a přiřadíte proměnnou, ale nepoužívejte ji.</span><span class="sxs-lookup"><span data-stu-id="bb2c7-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="bb2c7-105">Kompilátor generuje toto upozornění pouze v případě, že hodnota proměnné je konstanta času kompilace.</span><span class="sxs-lookup"><span data-stu-id="bb2c7-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="bb2c7-106">Přiřazení nekonstantního výrazu nebo výsledku metody k místní proměnné usnadňuje sledování těchto výrazů v ladicím programu.</span><span class="sxs-lookup"><span data-stu-id="bb2c7-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="bb2c7-107">Také umožňuje výsledek dosažitelný, brání uvolňování paměti, zatímco tato proměnná je dosažitelná.</span><span class="sxs-lookup"><span data-stu-id="bb2c7-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="bb2c7-108">Následující ukázka ukazuje případy, kdy a kdy není generováno upozornění:</span><span class="sxs-lookup"><span data-stu-id="bb2c7-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
