<span data-ttu-id="9ae98-104">Vlastnost nebo indexer s názvem nejde přiřadit – je jen pro čtení.</span><span class="sxs-lookup"><span data-stu-id="9ae98-104">Property or indexer 'name' cannot be assigned to -- it is read only</span></span>  
  
 <span data-ttu-id="9ae98-105">Pole [jen pro čtení](../language-reference/keywords/readonly.md) může převzít pouze přiřazení v konstruktoru nebo v deklaraci.</span><span class="sxs-lookup"><span data-stu-id="9ae98-105">A [readonly](../language-reference/keywords/readonly.md) field can only take an assignment in a constructor or at declaration.</span></span> <span data-ttu-id="9ae98-106">Další informace naleznete v tématu [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="9ae98-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>

## <a name="example"></a><span data-ttu-id="9ae98-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="9ae98-108">Example</span></span>  
 <span data-ttu-id="9ae98-109">Následující ukázka generuje CS0191.</span><span class="sxs-lookup"><span data-stu-id="9ae98-109">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
