# <a name="compiler-error-cs0174"></a>Chyba kompilátoru CS0174
Pro odkaz Base se vyžaduje základní třída.  
  
 K této chybě dochází pouze v případě, že .NET Framework modul CLR (Common Language Runtime) zkompiluje zdrojový kód `System.Object` třídy, což je jediná třída, která nemá žádnou základní třídu.
