 <span data-ttu-id="a261e-105">Implementace vlastnosti v odvozené třídě obsahuje přistupující objekt, který nebyl zadán v základním rozhraní.</span><span class="sxs-lookup"><span data-stu-id="a261e-105">The implementation of a property in a derived class contains an accessor that was not specified in the base interface.</span></span>

## <a name="example"></a><span data-ttu-id="a261e-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="a261e-107">Example</span></span>  
 <span data-ttu-id="a261e-108">Následující ukázka generuje CS0550.</span><span class="sxs-lookup"><span data-stu-id="a261e-108">The following sample generates CS0550.</span></span>  
  
```csharp  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
