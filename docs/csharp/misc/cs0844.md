## <a name="example"></a><span data-ttu-id="bca3d-112">Příklad</span><span class="sxs-lookup"><span data-stu-id="bca3d-112">Example</span></span>

            num = 5; // CS0844

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
