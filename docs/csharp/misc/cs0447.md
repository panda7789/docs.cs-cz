# <a name="compiler-error-cs0447"></a><span data-ttu-id="a1cb9-103">Chyba kompilátoru CS0447</span><span class="sxs-lookup"><span data-stu-id="a1cb9-103">Compiler Error CS0447</span></span>

<span data-ttu-id="a1cb9-104">Atributy nejde použít pro argumenty typu, jenom u parametrů typu.</span><span class="sxs-lookup"><span data-stu-id="a1cb9-104">Attributes cannot be used on type arguments, only on type parameters</span></span>

<span data-ttu-id="a1cb9-105">K této chybě dochází, pokud použijete atribut na argument typu, který se nachází v příkazu vyvolání.</span><span class="sxs-lookup"><span data-stu-id="a1cb9-105">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="a1cb9-106">Je přijatelné použít atribut pro parametr typu v příkazu třídy nebo deklarace metody, jako je například následující:</span><span class="sxs-lookup"><span data-stu-id="a1cb9-106">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```csharp
class C<[some attribute] T> {…}
```

<span data-ttu-id="a1cb9-107">Tato chyba se vygeneruje v následujícím řádku kódu.</span><span class="sxs-lookup"><span data-stu-id="a1cb9-107">The following line of code will generate this error.</span></span> <span data-ttu-id="a1cb9-108">Předpokládá se, že třída `C` definovaná v předchozím řádku kódu má zavolanou statickou metodu `MyStaticMethod` .</span><span class="sxs-lookup"><span data-stu-id="a1cb9-108">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```csharp

```

## <a name="example"></a><span data-ttu-id="a1cb9-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="a1cb9-109">Example</span></span>

```csharp
// CS0447.cs
using System;

namespace Test41
{
    public interface I<A>
    {
        void Meth<B>();
    }
    public class B : I<int>
    {
        void I<[Test] int>.Meth<X>() { }  // CS0447
    }
}
```
