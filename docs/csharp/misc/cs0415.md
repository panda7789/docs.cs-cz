# <a name="compiler-error-cs0415"></a><span data-ttu-id="ed3aa-103">Chyba kompilátoru CS0415</span><span class="sxs-lookup"><span data-stu-id="ed3aa-103">Compiler Error CS0415</span></span>
<span data-ttu-id="ed3aa-104">Atribut IndexerName je platný jenom pro indexer, který nepředstavuje explicitní deklaraci člena rozhraní.</span><span class="sxs-lookup"><span data-stu-id="ed3aa-104">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="ed3aa-105">K této chybě dochází, pokud použijete atribut IndexerName v indexeru, který byl explicitní implementací rozhraní.</span><span class="sxs-lookup"><span data-stu-id="ed3aa-105">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="ed3aa-106">Tato chyba se může vyhnout odebráním názvu rozhraní z deklarace indexeru, pokud je to možné.</span><span class="sxs-lookup"><span data-stu-id="ed3aa-106">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="ed3aa-107">Další informace naleznete v tématu [Třída IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="ed3aa-107">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="ed3aa-108">Následující ukázka generuje CS0415:</span><span class="sxs-lookup"><span data-stu-id="ed3aa-108">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
