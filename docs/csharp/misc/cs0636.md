# <a name="compiler-error-cs0636"></a><span data-ttu-id="223eb-102">Chyba kompilátoru CS0636</span><span class="sxs-lookup"><span data-stu-id="223eb-102">Compiler Error CS0636</span></span>
<span data-ttu-id="223eb-103">Atribut FieldOffset lze umístit pouze na členy typů označených strukturním rozložením(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="223eb-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="223eb-104">Atribut **StructLayout(LayoutKind.Explicit)** musíte použít v deklaraci struktury, pokud obsahuje všechny členy označené atributem **FieldOffset.**</span><span class="sxs-lookup"><span data-stu-id="223eb-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="223eb-105">Další informace naleznete v tématu [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="223eb-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="223eb-106">Následující ukázka generuje CS0636:</span><span class="sxs-lookup"><span data-stu-id="223eb-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
