# <a name="compiler-error-cs1651"></a>Chyba kompilátoru CS1651
Pole statického pole jen pro čtení Identifier nejde předat parametrem ref nebo out (kromě případu, kdy se nachází ve statickém konstruktoru).  
  
 K této chybě dojde, Pokud předáte proměnnou do funkce, která je členem statického pole jen pro čtení jako argument REF. Vzhledem k tomu, že je možné změnit parametry ref, není tato funkce povolená. Chcete-li tuto chybu vyřešit, odeberte klíčové slovo **ReadOnly** pro pole nebo nepředávejte do funkce členy pole jen pro čtení. Můžete například zkusit vytvořit dočasnou proměnnou, která se dá upravit a předat dočasnou hodnotu jako argument REF, jak je znázorněno v následujícím příkladu.  
  
 Následující ukázka generuje CS1651:  
  
```csharp  
// CS1651.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{
  public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void f(ref int iref)  
   {  
   }  
  
   static void Main()  
   {  
      f(ref Outer.inner.i);  // CS1651  
      // Try this instead:  
      // int tmp = Outer.inner.i;  
      // f(ref tmp);  
   }  
}  
```
