# <a name="compiler-error-cs1651"></a><span data-ttu-id="5ec26-103">Chyba kompilátoru CS1651</span><span class="sxs-lookup"><span data-stu-id="5ec26-103">Compiler Error CS1651</span></span>
<span data-ttu-id="5ec26-104">Pole statického pole jen pro čtení Identifier nejde předat parametrem ref nebo out (kromě případu, kdy se nachází ve statickém konstruktoru).</span><span class="sxs-lookup"><span data-stu-id="5ec26-104">Fields of static readonly field 'identifier' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="5ec26-105">K této chybě dojde, Pokud předáte proměnnou do funkce, která je členem statického pole jen pro čtení jako argument REF.</span><span class="sxs-lookup"><span data-stu-id="5ec26-105">This error occurs if you pass a variable to a function that is a member of a static readonly field as a ref argument.</span></span> <span data-ttu-id="5ec26-106">Vzhledem k tomu, že je možné změnit parametry ref, není tato funkce povolená.</span><span class="sxs-lookup"><span data-stu-id="5ec26-106">Since ref parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="5ec26-107">Chcete-li tuto chybu vyřešit, odeberte klíčové slovo **ReadOnly** pro pole nebo nepředávejte do funkce členy pole jen pro čtení.</span><span class="sxs-lookup"><span data-stu-id="5ec26-107">To resolve this error, remove the **readonly** keyword on the field, or do not pass the members of the readonly field to the function.</span></span> <span data-ttu-id="5ec26-108">Můžete například zkusit vytvořit dočasnou proměnnou, která se dá upravit a předat dočasnou hodnotu jako argument REF, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="5ec26-108">For example, you might try creating a temporary variable which can be modified and passing the temporary as a ref argument, as shown in the following example.</span></span>  
  
 <span data-ttu-id="5ec26-109">Následující ukázka generuje CS1651:</span><span class="sxs-lookup"><span data-stu-id="5ec26-109">The following sample generates CS1651:</span></span>  
  
```csharp  
// CS1651.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{
  public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void f(ref int iref)  
   {  
   }  
  
   static void Main()  
   {  
      f(ref Outer.inner.i);  // CS1651  
      // Try this instead:  
      // int tmp = Outer.inner.i;  
      // f(ref tmp);  
   }  
}  
```
