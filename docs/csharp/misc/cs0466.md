# <a name="compiler-error-cs0466"></a>Chyba kompilátoru CS0466
' – metoda1 ' by neměl mít parametr params, protože ' Method2 '  
  
 Nelze použít `params` parametr u člena třídy, pokud jej implementované rozhraní nepoužívá.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0466.  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
