# <a name="compiler-error-cs0466"></a><span data-ttu-id="df075-103">Chyba kompilátoru CS0466</span><span class="sxs-lookup"><span data-stu-id="df075-103">Compiler Error CS0466</span></span>
<span data-ttu-id="df075-104">' – metoda1 ' by neměl mít parametr params, protože ' Method2 '</span><span class="sxs-lookup"><span data-stu-id="df075-104">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="df075-105">Nelze použít `params` parametr u člena třídy, pokud jej implementované rozhraní nepoužívá.</span><span class="sxs-lookup"><span data-stu-id="df075-105">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="df075-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="df075-106">Example</span></span>  
 <span data-ttu-id="df075-107">Následující ukázka generuje CS0466.</span><span class="sxs-lookup"><span data-stu-id="df075-107">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
