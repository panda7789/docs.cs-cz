 <span data-ttu-id="bc7eb-104">K této chybě dochází při pokusu `get` o přístup k nepřístupnému přistupujícímu objektu.</span><span class="sxs-lookup"><span data-stu-id="bc7eb-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="bc7eb-105">Chcete-li tuto chybu vyřešit, zvyšte přístupnost přistupujícího ortu nebo změňte umístění volání.</span><span class="sxs-lookup"><span data-stu-id="bc7eb-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="bc7eb-106">Další informace naleznete [v tématu Přístupový objekt přístupkový přístup](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) a [vlastnosti](../programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="bc7eb-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="bc7eb-107">Následující příklad generuje CS0271:</span><span class="sxs-lookup"><span data-stu-id="bc7eb-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
