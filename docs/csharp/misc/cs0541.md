 <span data-ttu-id="634e4-105">Explicitní deklarace [rozhraní](../language-reference/keywords/interface.md) se našla mimo [třídu](../language-reference/keywords/class.md) nebo [strukturu](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="634e4-105">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="634e4-106">Následující ukázka generuje CS0541:</span><span class="sxs-lookup"><span data-stu-id="634e4-106">The following sample generates CS0541:</span></span>  
  
```csharp

namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace

      void IFace.F();   // CS0541  
   }  
}  
```
