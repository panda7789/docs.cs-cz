# <a name="compiler-error-cs0160"></a><span data-ttu-id="fdc74-103">Chyba kompilátoru CS0160</span><span class="sxs-lookup"><span data-stu-id="fdc74-103">Compiler Error CS0160</span></span>
<span data-ttu-id="fdc74-104">Předchozí klauzule catch už zachytává všechny výjimky tohoto nebo nadtypu (typ).</span><span class="sxs-lookup"><span data-stu-id="fdc74-104">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="fdc74-105">Řada příkazů [catch](../language-reference/keywords/try-catch.md) musí být v sestupném pořadí odvození.</span><span class="sxs-lookup"><span data-stu-id="fdc74-105">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="fdc74-106">Například nejvíce odvozené objekty musí být uvedeny jako první.</span><span class="sxs-lookup"><span data-stu-id="fdc74-106">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="fdc74-107">Další informace naleznete v tématu [výjimky a zpracování výjimek](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="fdc74-107">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="fdc74-108">Následující ukázka generuje CS0160:</span><span class="sxs-lookup"><span data-stu-id="fdc74-108">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
