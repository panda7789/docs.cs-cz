# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="79bbe-103">Upozornění kompilátoru (úroveň 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="79bbe-103">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="79bbe-104">Nejednoznačný odkaz v atributu cref: Method název1</span><span class="sxs-lookup"><span data-stu-id="79bbe-104">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="79bbe-105">Předpokládá se metoda "Method název2", ale mohla by také odpovídat jiným přetížením, včetně ' Method Název3 '.</span><span class="sxs-lookup"><span data-stu-id="79bbe-105">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="79bbe-106">V komentáři k dokumentaci XML v kódu nelze odkaz přeložit.</span><span class="sxs-lookup"><span data-stu-id="79bbe-106">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="79bbe-107">K tomu může dojít, pokud je metoda přetížena nebo pokud jsou nalezeny dva různé identifikátory se stejným názvem.</span><span class="sxs-lookup"><span data-stu-id="79bbe-107">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="79bbe-108">Chcete-li vyřešit upozornění, použijte kvalifikovaný název pro jednoznačné použití odkazu nebo Zahrňte konkrétní přetížení do závorek.</span><span class="sxs-lookup"><span data-stu-id="79bbe-108">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="79bbe-109">Následující ukázka generuje CS0419.</span><span class="sxs-lookup"><span data-stu-id="79bbe-109">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
