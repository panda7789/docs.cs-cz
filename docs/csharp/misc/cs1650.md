# <a name="compiler-error-cs1650"></a><span data-ttu-id="f395b-103">Chyba kompilátoru CS1650</span><span class="sxs-lookup"><span data-stu-id="f395b-103">Compiler Error CS1650</span></span>
<span data-ttu-id="f395b-104">Pole statického pole jen pro čtení Identifier nelze přiřadit (kromě případu, kdy se nacházejí v rámci statického konstruktoru nebo inicializátoru proměnné).</span><span class="sxs-lookup"><span data-stu-id="f395b-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="f395b-105">K této chybě dochází, pokud se pokusíte změnit člena pole, který je jen pro čtení a statický, kde není povoleno ho upravovat.</span><span class="sxs-lookup"><span data-stu-id="f395b-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="f395b-106">Chcete-li tuto chybu vyřešit, omezte přiřazení na pole typu ReadOnly na konstruktor nebo inicializátor proměnné nebo odeberte `readonly` klíčové slovo z deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="f395b-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
