# <a name="compiler-error-cs1666"></a><span data-ttu-id="30bfa-102">Chyba kompilátoru CS1666</span><span class="sxs-lookup"><span data-stu-id="30bfa-102">Compiler Error CS1666</span></span>
<span data-ttu-id="30bfa-103">Vyrovnávací paměti s pevnou velikostí obsažené v neopravených výrazech nelze použít.</span><span class="sxs-lookup"><span data-stu-id="30bfa-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="30bfa-104">Zkuste použít příkaz fixed.</span><span class="sxs-lookup"><span data-stu-id="30bfa-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="30bfa-105">K této chybě dochází, pokud použijete vyrovnávací paměť s pevnou velikostí ve výrazu zahrnujícím třídu, která není sama opravena.</span><span class="sxs-lookup"><span data-stu-id="30bfa-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="30bfa-106">Runtime je zdarma přesunout unfixed třídy kolem optimalizovat přístup k paměti, což by mohlo vést k chybám při použití vyrovnávací paměti s pevnou velikostí.</span><span class="sxs-lookup"><span data-stu-id="30bfa-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="30bfa-107">Chcete-li se této `fixed` chybě vyhnout, použijte klíčové slovo na příkazu.</span><span class="sxs-lookup"><span data-stu-id="30bfa-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="30bfa-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="30bfa-108">Example</span></span>  
 <span data-ttu-id="30bfa-109">Následující ukázka generuje CS1666.</span><span class="sxs-lookup"><span data-stu-id="30bfa-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);
   }
}  
```
