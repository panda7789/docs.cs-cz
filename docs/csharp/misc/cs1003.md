# <a name="compiler-error-cs1003"></a>Chyba kompilátoru CS1003
Chyba syntaxe, očekával se Char.  
  
 Kompilátor vygeneruje tuto chybu pro některou z několika chybových podmínek. Zkontrolujte kód a najděte chybu syntaxe.  
  
 Následující ukázka generuje CS1003:  
  
```csharp  
// CS1003.cs  
public class b  
{  
   public static void Main()  
   {  
      int[] a;  
      a[);   // CS1003  
   }  
}  
```
