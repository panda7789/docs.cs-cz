# <a name="compiler-error-cs0543"></a>Chyba kompilátoru CS0543
' enumeration ': hodnota výčtu je příliš velká, aby se vešla do svého typu  
  
 Hodnota, která byla přiřazena prvku ve [výčtu](../language-reference/builtin-types/enum.md) , je mimo rozsah datového typu.  
  
 Následující ukázka generuje CS0543:  
  
```csharp  
// CS0543.cs  
namespace x  
{  
   enum I : byte  
   {a = 255, b, c}   // CS0543  
   public class clx  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
