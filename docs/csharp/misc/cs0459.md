# <a name="compiler-error-cs0459"></a><span data-ttu-id="74acd-103">Chyba kompilátoru CS0459</span><span class="sxs-lookup"><span data-stu-id="74acd-103">Compiler Error CS0459</span></span>

<span data-ttu-id="74acd-104">Nejde převzít adresu lokální proměnné určené jen pro čtení.</span><span class="sxs-lookup"><span data-stu-id="74acd-104">Cannot take the address of a read-only local variable</span></span>

 <span data-ttu-id="74acd-105">V jazyce C# existují tři běžné scénáře, které generují místní proměnné jen pro čtení: `foreach` , `using` a `fixed` .</span><span class="sxs-lookup"><span data-stu-id="74acd-105">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="74acd-106">V každém z těchto případů nemůžete zapisovat do místní proměnné určené jen pro čtení nebo přebírat její adresu.</span><span class="sxs-lookup"><span data-stu-id="74acd-106">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="74acd-107">Tato chyba se vygeneruje, když se kompilátor pokusí převzít adresu lokální proměnné určené jen pro čtení.</span><span class="sxs-lookup"><span data-stu-id="74acd-107">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>

## <a name="example"></a><span data-ttu-id="74acd-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="74acd-108">Example</span></span>

 <span data-ttu-id="74acd-109">Následující příklad generuje CS0459 při pokusu o převzetí adresy místní proměnné jen pro čtení ve `foreach` smyčce a v `fixed` bloku příkazu:</span><span class="sxs-lookup"><span data-stu-id="74acd-109">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
