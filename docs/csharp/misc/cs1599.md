<span data-ttu-id="c1a68-104">Metoda nebo delegát nemůže vracet typ typu.</span><span class="sxs-lookup"><span data-stu-id="c1a68-104">Method or delegate cannot return type 'type'</span></span>  
  
 <span data-ttu-id="c1a68-105">Některé typy v knihovně tříd .NET Framework, například <xref:System.TypedReference> , <xref:System.RuntimeArgumentHandle> a <xref:System.ArgIterator> nelze je použít jako návratové typy, protože mohou být použity k provádění nebezpečných operací.</span><span class="sxs-lookup"><span data-stu-id="c1a68-105">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as return types because they can potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="c1a68-106">Následující ukázka generuje CS1599:</span><span class="sxs-lookup"><span data-stu-id="c1a68-106">The following sample generates CS1599:</span></span>

```csharp  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
