# <a name="compiler-error-cs1599"></a>Chyba kompilátoru CS1599
Metoda nebo delegát nemůže vracet typ typu.  
  
 Některé typy v knihovně tříd .NET Framework, například <xref:System.TypedReference> , <xref:System.RuntimeArgumentHandle> a <xref:System.ArgIterator> nelze je použít jako návratové typy, protože mohou být použity k provádění nebezpečných operací.  
  
 Následující ukázka generuje CS1599:  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
