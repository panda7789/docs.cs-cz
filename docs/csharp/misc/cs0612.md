# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="a9286-102">Upozornění kompilátoru (úroveň 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="a9286-102">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="a9286-103">"člen" je zastaralý</span><span class="sxs-lookup"><span data-stu-id="a9286-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="a9286-104">Návrhář třídy označil člena atributem [Zastaralé](../language-reference/attributes/general.md#obsolete-attribute).</span><span class="sxs-lookup"><span data-stu-id="a9286-104">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="a9286-105">To znamená, že člen nemusí být podporovány v budoucí verzi třídy.</span><span class="sxs-lookup"><span data-stu-id="a9286-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="a9286-106">Následující ukázka ukazuje, jak přístup k zastaralému členu generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="a9286-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
