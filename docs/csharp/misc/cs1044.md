<span data-ttu-id="633f1-104">V příkazu for, using, fixed nebo Declaration nelze použít více než jeden typ.</span><span class="sxs-lookup"><span data-stu-id="633f1-104">Cannot use more than one type in a for, using, fixed, or declaration statement</span></span>  
  
 <span data-ttu-id="633f1-105">Kompilátor nalezl neplatný příkaz.</span><span class="sxs-lookup"><span data-stu-id="633f1-105">The compiler found an invalid statement.</span></span>  
  
 <span data-ttu-id="633f1-106">Následující ukázka generuje CS1044:</span><span class="sxs-lookup"><span data-stu-id="633f1-106">The following sample generates CS1044:</span></span>  
  
```csharp  
// CS1044.cs  
using System;  
  
public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),  
             MyClass mc2 = new MyClass())   // CS1044, remove an instantiation  
      {  
      }  
   }  
}  
```
