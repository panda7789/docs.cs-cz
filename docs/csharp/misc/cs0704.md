# <a name="compiler-error-cs0704"></a>Chyba kompilátoru CS0704
Hledání členů nelze provést v typu Type, protože se jedná o parametr typu.  
  
 Vnitřní typ nelze zadat pomocí parametru typu. Zkuste použít požadovaný typ explicitně.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0704:  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
