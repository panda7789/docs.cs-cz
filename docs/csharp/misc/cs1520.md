# <a name="compiler-error-cs1520"></a>Chyba kompilátoru CS1520
Metoda musí mít návratový typ.  
  
 Metoda, která je deklarována ve třídě, struktuře nebo rozhraní, musí mít explicitní návratový typ. V následujícím příkladu má metoda Square vrácenou hodnotu [řetězce](../language-reference/builtin-types/reference-types.md):  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 Následující ukázka generuje CS1520:  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 Alternativně může dojít k této chybě, pokud se případ názvu konstruktoru liší od názvu třídy nebo struktury, jako v následujícím příkladu. Vzhledem k tomu, že název není přesně stejný jako název třídy, kompilátor interpretuje jako regulární metodu, nikoli konstruktor a vytváří chybu:  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
