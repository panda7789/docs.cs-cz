# <a name="compiler-warning-level-1-cs1723"></a>Upozornění kompilátoru (úroveň 1) CS1723
Xml komentář k 'param' má cref atribut 'atribut', který odkazuje na parametr typu  
  
 Tato chyba je generována komentářem XML, který odkazuje na parametr typu.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1723.  
  
```csharp  
// CS1723.cs  
// compile with: /t:library /doc:filename.XML  
///<summary>A generic list class.</summary>  
///<see cref="T" />   // CS1723  
// To resolve comment the previous line.  
public class List<T>
{  
}  
```
