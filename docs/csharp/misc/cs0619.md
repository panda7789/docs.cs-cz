<span data-ttu-id="10af3-105">Člen třídy byl označen jako [zastaralý](../language-reference/attributes/general.md#obsolete-attribute) atribut, takže při odkazování na člena třídy bude vyvolána chyba.</span><span class="sxs-lookup"><span data-stu-id="10af3-105">A class member was marked with the [Obsolete](../language-reference/attributes/general.md#obsolete-attribute) attribute, such that an error will be issued when the class member is referenced.</span></span>

## <a name="example"></a><span data-ttu-id="10af3-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="10af3-106">Example</span></span>

<span data-ttu-id="10af3-107">Následující ukázka generuje CS0619:</span><span class="sxs-lookup"><span data-stu-id="10af3-107">The following sample generates CS0619:</span></span>  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```
