# <a name="compiler-warning-level-4-cs0422"></a>Upozornění kompilátoru (úroveň 4) CS0422

Možnost /incremental již není podporována.

 Přírůstková kompilace (**/incr** nebo **/incremental**) není v c# 2.0 podporována.

> [!NOTE]
> Toto upozornění kompilátoru se již nepoužívá v Roslyn.
