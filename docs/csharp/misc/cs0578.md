<span data-ttu-id="f1073-105"><xref:System.Diagnostics.ConditionalAttribute> nelze použít na metodu, která má návratový typ jiný než `void` .</span><span class="sxs-lookup"><span data-stu-id="f1073-105"><xref:System.Diagnostics.ConditionalAttribute> cannot be applied to a method that has a return type other than `void`.</span></span> <span data-ttu-id="f1073-106">Důvodem je, že jakýkoli jiný návratový typ pro metodu může být vyžadován jinou částí programu.</span><span class="sxs-lookup"><span data-stu-id="f1073-106">The reason for this is that any other return type for a method may be needed by another part of your program.</span></span>

## <a name="example"></a><span data-ttu-id="f1073-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="f1073-107">Example</span></span>

<span data-ttu-id="f1073-108">Následující ukázka generuje CS0578.</span><span class="sxs-lookup"><span data-stu-id="f1073-108">The following sample generates CS0578.</span></span> <span data-ttu-id="f1073-109">Chcete-li tuto chybu vyřešit, je nutné buď odstranit <xref:System.Diagnostics.ConditionalAttribute> , nebo změnit návratovou hodnotu metody na `void` .</span><span class="sxs-lookup"><span data-stu-id="f1073-109">To resolve this error, you must either delete <xref:System.Diagnostics.ConditionalAttribute>, or you must change the return value of the method to `void`.</span></span>

```csharp
// CS0578.cs
// compile with: /target:library
public class MyClass
{
    [System.Diagnostics.ConditionalAttribute("a")]   // CS0578
    public int TestMethod()
    {
        return 0;
    }
}
```
