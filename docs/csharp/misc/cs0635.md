# <a name="compiler-error-cs0635"></a>Chyba kompilátoru CS0635
' Attribute ': System. Interop. UnmanagedType. CustomMarshaller vyžaduje pojmenované argumenty ComType a Marshal  
  
 Argumenty **ComType** a **Marshal** musí být zadány, pokud má formát zařazení hodnotu **System. Interop. UnmanagedType. CustomMarshaller**.
