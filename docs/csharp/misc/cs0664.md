# <a name="compiler-error-cs0664"></a><span data-ttu-id="5d128-103">Chyba kompilátoru CS0664</span><span class="sxs-lookup"><span data-stu-id="5d128-103">Compiler Error CS0664</span></span>
<span data-ttu-id="5d128-104">Literály typu Double nelze implicitně převést na typ Type; Chcete-li vytvořit literál tohoto typu, použijte příponu přípony.</span><span class="sxs-lookup"><span data-stu-id="5d128-104">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="5d128-105">Přiřazení nelze dokončit; k opravě instrukce použijte příponu.</span><span class="sxs-lookup"><span data-stu-id="5d128-105">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="5d128-106">Dokumentace pro každý typ identifikuje odpovídající příponu pro typ.</span><span class="sxs-lookup"><span data-stu-id="5d128-106">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="5d128-107">Další informace o převodech naleznete v tématu [přetypování a převody typu](../programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="5d128-107">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="5d128-108">Následující ukázka generuje CS0664:</span><span class="sxs-lookup"><span data-stu-id="5d128-108">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
