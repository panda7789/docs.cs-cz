# <a name="compiler-error-cs0721"></a>Chyba kompilátoru CS0721
Typ: statické typy nejde používat jako parametry.  
  
 Statický typ není smysluplný jako parametr. Vzhledem k tomu, že nelze vytvořit žádné instance statických typů, žádná instance by nebyla nikdy předána jako parametr.  
  
 Následující ukázka generuje CS0721:  
  
```csharp  
// CS0721.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
