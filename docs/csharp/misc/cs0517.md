# <a name="compiler-error-cs0517"></a>Chyba kompilátoru CS0517
Třída Class nemá žádnou základní třídu a nemůže volat konstruktor Base.  
  
 K CS0517 může dojít pouze v případě, že .NET Framework modul CLR (Common Language Runtime) zkompiluje zdrojový kód třídy objektu, což je jediná třída, která nemá žádnou základní třídu.
