# <a name="compiler-error-cs0210"></a><span data-ttu-id="9fb51-103">Chyba kompilátoru CS0210</span><span class="sxs-lookup"><span data-stu-id="9fb51-103">Compiler Error CS0210</span></span>
<span data-ttu-id="9fb51-104">Je nutné zadat inicializátor v deklaracích příkazu fixed nebo using.</span><span class="sxs-lookup"><span data-stu-id="9fb51-104">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="9fb51-105">Je nutné deklarovat a inicializovat proměnnou v [příkazu fixed](../language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="9fb51-105">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="9fb51-106">Další informace naleznete v tématu [nezabezpečený kód a ukazatele](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="9fb51-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="9fb51-107">Následující ukázka generuje CS0210:</span><span class="sxs-lookup"><span data-stu-id="9fb51-107">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="9fb51-108">Následující ukázka také generuje CS0210, protože [příkaz using](../language-reference/keywords/using-statement.md) nemá žádný inicializátor.</span><span class="sxs-lookup"><span data-stu-id="9fb51-108">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test
{  
   static void Main()
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
