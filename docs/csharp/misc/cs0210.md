# <a name="compiler-error-cs0210"></a>Chyba kompilátoru CS0210
Je nutné zadat inicializátor v pevné nebo pomocí prohlášení  
  
 Proměnnou je nutné deklarovat a inicializovat v [pevném příkazu](../language-reference/keywords/fixed-statement.md). Další informace naleznete [v tématu Nebezpečný kód a ukazatele](../programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0210:  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 Následující ukázka také generuje CS0210, protože [using prohlášení](../language-reference/keywords/using-statement.md) nemá inicializační prostředek.  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test
{  
   static void Main()
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
