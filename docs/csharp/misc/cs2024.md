# <a name="compiler-error-cs2024"></a>Chyba kompilátoru CS2024

Neplatné číslo zarovnání oddílu souboru #

Možnosti kompilátoru [-zarovnávání](../language-reference/compiler-options/filealign-compiler-option.md) hodnoty byla předána neplatná hodnota.

## <a name="example"></a>Příklad

Následující ukázka generuje CS2024:

```csharp
// CS2024.cs
// compile with: /filealign:ex
// CS2024 expected
class MyClass
{
    public static void Main()
    {
    }
}
```
