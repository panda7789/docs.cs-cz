# <a name="compiler-error-cs0214"></a><span data-ttu-id="232d8-103">Chyba kompilátoru CS0214</span><span class="sxs-lookup"><span data-stu-id="232d8-103">Compiler Error CS0214</span></span>
<span data-ttu-id="232d8-104">Ukazatele a vyrovnávací paměti pevné velikosti se můžou používat jenom v nezabezpečeném kontextu.</span><span class="sxs-lookup"><span data-stu-id="232d8-104">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="232d8-105">Ukazatele lze použít pouze s klíčovým slovem [unsafe](../language-reference/keywords/unsafe.md) .</span><span class="sxs-lookup"><span data-stu-id="232d8-105">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="232d8-106">Další informace naleznete v tématu [nezabezpečený kód a ukazatele](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="232d8-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="232d8-107">Následující ukázka generuje CS0214:</span><span class="sxs-lookup"><span data-stu-id="232d8-107">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
