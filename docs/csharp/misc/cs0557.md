# <a name="compiler-error-cs0557"></a><span data-ttu-id="bd191-103">Chyba kompilátoru CS0557</span><span class="sxs-lookup"><span data-stu-id="bd191-103">Compiler Error CS0557</span></span>
<span data-ttu-id="bd191-104">Duplicitní uživatelem definovaný převod v typu Class</span><span class="sxs-lookup"><span data-stu-id="bd191-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="bd191-105">Ve třídě nejsou povoleny duplicitní rutiny převodu.</span><span class="sxs-lookup"><span data-stu-id="bd191-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="bd191-106">Následující příklad generuje CS0557:</span><span class="sxs-lookup"><span data-stu-id="bd191-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
