# <a name="compiler-error-cs1660"></a><span data-ttu-id="107da-103">Chyba kompilátoru CS1660</span><span class="sxs-lookup"><span data-stu-id="107da-103">Compiler Error CS1660</span></span>
<span data-ttu-id="107da-104">Blok anonymní metody nejde převést na typ Type, protože to není typ delegáta.</span><span class="sxs-lookup"><span data-stu-id="107da-104">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="107da-105">K této chybě dochází, pokud se pokusíte přiřadit nebo jinak převést blok anonymní metody na typ, který není typem delegáta.</span><span class="sxs-lookup"><span data-stu-id="107da-105">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="107da-106">Následující ukázka generuje CS1660:</span><span class="sxs-lookup"><span data-stu-id="107da-106">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
