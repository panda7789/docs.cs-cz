# <a name="compiler-error-cs0448"></a>Chyba kompilátoru CS0448
Návratový typ operátorů + + a--musí být nadřazeného typu nebo musí být odvozený z nadřazeného typu.  
  
 Pokud přepíšete `++` `--` operátory nebo, musí vracet stejný typ jako nadřazený typ nebo vracet typ, který je odvozen z nadřazeného typu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0448.  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0448.  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
