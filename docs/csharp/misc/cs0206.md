## <a name="example"></a><span data-ttu-id="f0269-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="f0269-107">Example</span></span>  
 <span data-ttu-id="f0269-108">Následující ukázka generuje CS0206:</span><span class="sxs-lookup"><span data-stu-id="f0269-108">The following sample generates CS0206:</span></span>  
  
```csharp  
// CS0206.cs  
public class MyClass  
{  
    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set  
        {  
        }  
    }  
  
    public static void MyMeth(ref int i)  
    // public static void MyMeth(int i)  
    {  
    }  
  
    public static void Main()  
    {  
        MyMeth(ref P);   // CS0206  
        // try the following line instead  
        // MyMeth(P);   // CS0206  
    }  
}  
```
