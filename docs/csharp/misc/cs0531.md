# <a name="compiler-error-cs0531"></a><span data-ttu-id="63792-103">Chyba kompilátoru CS0531</span><span class="sxs-lookup"><span data-stu-id="63792-103">Compiler Error CS0531</span></span>
<span data-ttu-id="63792-104">' member ': členy rozhraní nemohou mít definici</span><span class="sxs-lookup"><span data-stu-id="63792-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="63792-105">Metody deklarované v [rozhraní](../language-reference/keywords/interface.md) musí být implementovány ve třídě, která je z ní odvozena a nikoli v samotném rozhraní.</span><span class="sxs-lookup"><span data-stu-id="63792-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="63792-106">Následující ukázka generuje CS0531:</span><span class="sxs-lookup"><span data-stu-id="63792-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
