# <a name="compiler-error-cs2022"></a>CS2022 chyby kompilátoru
Možnosti ' / out' a '/ cíl' musí být před názvy zdrojových souborů  
  
 [/Out (nastavit název výstupního souboru)](../../csharp/language-reference/compiler-options/out-compiler-option.md) a [/Target (určení formátu výstupního souboru)](../../csharp/language-reference/compiler-options/target-compiler-option.md) – možnosti kompilátoru, pokud zadaný v příkazovém řádku musí předcházet soubory zdrojového kódu.
