# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="b7ce3-102">Upozornění kompilátoru (úroveň 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="b7ce3-102">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="b7ce3-103">Možné nezamýšlené porovnání odkazů; chcete-li získat porovnání hodnot, přetypovat pravou stranu na typ 'typ'</span><span class="sxs-lookup"><span data-stu-id="b7ce3-103">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="b7ce3-104">Kompilátor provádí porovnání odkazů.</span><span class="sxs-lookup"><span data-stu-id="b7ce3-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="b7ce3-105">Chcete-li porovnat hodnotu řetězců, přetypovat pravé straně `type`výrazu .</span><span class="sxs-lookup"><span data-stu-id="b7ce3-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="b7ce3-106">Následující ukázka generuje CS0253:</span><span class="sxs-lookup"><span data-stu-id="b7ce3-106">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
