# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="869c2-103">Upozornění kompilátoru (úroveň 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="869c2-103">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="869c2-104">Možná nezamýšlené porovnání odkazů; Chcete-li porovnat hodnoty, přetypujte pravou stranu na typ Type.</span><span class="sxs-lookup"><span data-stu-id="869c2-104">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="869c2-105">Kompilátor provádí porovnání odkazů.</span><span class="sxs-lookup"><span data-stu-id="869c2-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="869c2-106">Pokud chcete porovnat hodnotu řetězců, přetypujte pravou stranu výrazu na `type` .</span><span class="sxs-lookup"><span data-stu-id="869c2-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="869c2-107">Následující ukázka generuje CS0253:</span><span class="sxs-lookup"><span data-stu-id="869c2-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
