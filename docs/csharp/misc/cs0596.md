# <a name="compiler-error-cs0596"></a>Chyba kompilátoru CS0596
Atribut GUID musí být zadaný s atributem atributem ComImport.  
  
 Atribut **GUID** musí být přítomen při použití atributu **atributem ComImport** .  
  
 Následující ukázka generuje CS0596:  
  
```csharp  
// CS0596.cs  
using System.Runtime.InteropServices;  
  
namespace x  
{  
   [ComImport]   // CS0596  
   // try the following line to resolve this CS0596  
   // [ComImport, Guid("00000000-0000-0000-0000-000000000001")]  
   public class a  
   {  
   }  
  
   public class b  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
