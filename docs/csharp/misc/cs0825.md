## <a name="example"></a><span data-ttu-id="ea8e1-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="ea8e1-107">Example</span></span>

```csharp

class Test  
{  
    // Both of these declarations trigger CS0825
    private var genreName;
    private var bookTitles = new List<string>();
  
    static int Main()  
    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```
