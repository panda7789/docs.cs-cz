## <a name="example"></a><span data-ttu-id="fee84-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="fee84-108">Example</span></span>

```csharp

{  
    // Both of these declarations trigger CS0825
    private var genreName;
    private var bookTitles = new List<string>();
  
    static int Main()  
    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```
