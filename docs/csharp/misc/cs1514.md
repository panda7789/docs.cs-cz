# <a name="compiler-error-cs1514"></a>Chyba kompilátoru CS1514
{očekáváno  
  
 Kompilátor očekával levou složenou závorku ( `{` ), která se nenašla.  
  
 Následující ukázka generuje CS1514:  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
