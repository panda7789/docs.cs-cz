# <a name="compiler-error-cs0226"></a>Chyba kompilátoru CS0226
Výraz __arglist se může vyskytovat jenom uvnitř volání nebo výrazu New.  
  
 Nepodporované klíčové slovo `__arglist` se může vyskytovat pouze ve volání metody nebo v novém výrazu.  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS0226:  
  
```csharp  
// cs0226.cs  
using System;  
  
public class C  
    {  
    public static int Main ()  
        {  
        __arglist(1,"This is a string"); // CS0226  
        return 0;  
        }  
    }  
```
