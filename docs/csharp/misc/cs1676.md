# <a name="compiler-error-cs1676"></a><span data-ttu-id="f0d4c-102">Chyba kompilátoru CS1676</span><span class="sxs-lookup"><span data-stu-id="f0d4c-102">Compiler Error CS1676</span></span>
<span data-ttu-id="f0d4c-103">Parametr 'number' musí být deklarován pomocí klíčového slova "klíčové slovo"</span><span class="sxs-lookup"><span data-stu-id="f0d4c-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="f0d4c-104">K této chybě dochází, když se modifikátor typu parametru v anonymní metodě liší od modifikátoru použitého v deklaraci delegáta, kterému metodu předáváte.</span><span class="sxs-lookup"><span data-stu-id="f0d4c-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="f0d4c-105">Následující ukázka generuje CS1676:</span><span class="sxs-lookup"><span data-stu-id="f0d4c-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
