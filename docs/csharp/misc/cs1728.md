# <a name="compiler-error-cs1728"></a>Chyba kompilátoru CS1728
Nelze svázat delegáta na 'člen', protože je členem 'typu'  
  
 Delegáty nelze svázat `Nullable` s členy typů hodnot.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1728:  
  
```csharp  
// CS1728.cs  
class Test  
{  
   delegate T GetT<T>();  
   delegate T GetT1<T>(T t);  
  
   delegate bool E(object o);  
   delegate int I();  
   delegate string S();  
  
   static void Main()  
   {  
      int? x = null;  
      int? y = 5;  
  
      GetT<int> d1 = x.GetValueOrDefault;   // CS1728  
      GetT<int> d2 = y.GetValueOrDefault;   // CS1728  
      GetT1<int> d3 = x.GetValueOrDefault;   // CS1728  
      GetT1<int> d4 = y.GetValueOrDefault;   // CS1728  
   }  
}  
```
