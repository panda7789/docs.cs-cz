# <a name="compiler-error-cs0243"></a><span data-ttu-id="7a000-102">Chyba kompilátoru CS0243</span><span class="sxs-lookup"><span data-stu-id="7a000-102">Compiler error CS0243</span></span>

<span data-ttu-id="7a000-103">Podmíněný atribut není platný pro metodu , protože se jedná o metodu přepsání.</span><span class="sxs-lookup"><span data-stu-id="7a000-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="7a000-104">Atribut <xref:System.Diagnostics.ConditionalAttribute> není povolen u metody, která je označena klíčovým slovem [přepsání.](../language-reference/keywords/override.md)</span><span class="sxs-lookup"><span data-stu-id="7a000-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="7a000-105">Další informace naleznete [v tématu Vědět, kdy použít přepsání a nová klíčová slova](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="7a000-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="7a000-106">Kompilátor se nikdy neváže na metody přepsání.</span><span class="sxs-lookup"><span data-stu-id="7a000-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="7a000-107">Pouze váže na základní metody a běžný jazyk runtime volá přepsání, podle potřeby.</span><span class="sxs-lookup"><span data-stu-id="7a000-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="7a000-108">Následující kód generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="7a000-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
