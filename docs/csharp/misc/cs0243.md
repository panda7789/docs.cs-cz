# <a name="compiler-error-cs0243"></a><span data-ttu-id="61eea-103">Chyba kompilátoru CS0243</span><span class="sxs-lookup"><span data-stu-id="61eea-103">Compiler error CS0243</span></span>

<span data-ttu-id="61eea-104">Atribut Conditional není pro metodu platný, protože je to metoda override.</span><span class="sxs-lookup"><span data-stu-id="61eea-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="61eea-105"><xref:System.Diagnostics.ConditionalAttribute>Atribut není povolen pro metodu, která je označena klíčovým slovem [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="61eea-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="61eea-106">Další informace najdete v tématu o tom, [Jak používat klíčová slova override a New](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="61eea-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="61eea-107">Kompilátor se nikdy neváže k metodám přepsání.</span><span class="sxs-lookup"><span data-stu-id="61eea-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="61eea-108">Váže se pouze k základní metodě a modul CLR (Common Language Runtime) zavolá přepsání podle potřeby.</span><span class="sxs-lookup"><span data-stu-id="61eea-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="61eea-109">Následující kód generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="61eea-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
