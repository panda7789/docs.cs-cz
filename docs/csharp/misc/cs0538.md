# <a name="compiler-error-cs0538"></a><span data-ttu-id="0f7bb-103">Chyba kompilátoru CS0538</span><span class="sxs-lookup"><span data-stu-id="0f7bb-103">Compiler Error CS0538</span></span>
<span data-ttu-id="0f7bb-104">název v explicitní deklaraci rozhraní není rozhraní.</span><span class="sxs-lookup"><span data-stu-id="0f7bb-104">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="0f7bb-105">Byl proveden pokus o explicitní deklaraci [rozhraní](../language-reference/keywords/interface.md), ale rozhraní nebylo zadáno.</span><span class="sxs-lookup"><span data-stu-id="0f7bb-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="0f7bb-106">Následující ukázka generuje CS0538:</span><span class="sxs-lookup"><span data-stu-id="0f7bb-106">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
