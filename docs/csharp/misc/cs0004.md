# <a name="compiler-error-cs0004"></a>Chyba kompilátoru CS0004
Upozornění se považuje za chybu.  
  
 Zkompilujete s `/warnaserror` možností kompilátoru, což způsobí, že kompilátor generuje upozornění jako chyby.
