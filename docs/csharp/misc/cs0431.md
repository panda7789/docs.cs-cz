 <span data-ttu-id="2919b-105">Použili jste "::" s aliasem, který odkazuje na typ.</span><span class="sxs-lookup"><span data-stu-id="2919b-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="2919b-106">Chcete-li tuto chybu vyřešit, použijte operátor "." .</span><span class="sxs-lookup"><span data-stu-id="2919b-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="2919b-107">Následující ukázka generuje CS0431:</span><span class="sxs-lookup"><span data-stu-id="2919b-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
