# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="91d73-103">Upozornění kompilátoru (úroveň 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="91d73-103">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="91d73-104">Přípona l je snadno zaměnitelná s číslicí 1 – pro přehlednost použijte L.</span><span class="sxs-lookup"><span data-stu-id="91d73-104">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="91d73-105">Kompilátor se upozorní, když detekuje přetypování na dlouhou dobu pomocí malých písmen l namísto velkých písmen L.</span><span class="sxs-lookup"><span data-stu-id="91d73-105">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="91d73-106">Následující ukázka generuje CS0078:</span><span class="sxs-lookup"><span data-stu-id="91d73-106">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
