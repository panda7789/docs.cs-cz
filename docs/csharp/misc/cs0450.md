# <a name="compiler-error-cs0450"></a><span data-ttu-id="39860-102">Chyba kompilátoru CS0450</span><span class="sxs-lookup"><span data-stu-id="39860-102">Compiler Error CS0450</span></span>
<span data-ttu-id="39860-103">"Název parametru typu": nelze zadat třídu omezení i omezení "třída" nebo "struktura".</span><span class="sxs-lookup"><span data-stu-id="39860-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="39860-104">Pokud je parametr typu omezen omezením typu struktury, je logicky protichůdné, aby byl také omezen určitým typem třídy, protože struktura a třída jsou vzájemně se vylučující kategorie.</span><span class="sxs-lookup"><span data-stu-id="39860-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="39860-105">Pokud je parametr typu omezen určitým omezením typu třídy, je podle definice omezen omezením typu třídy, a proto je určení omezení typu třídy nadbytečné.</span><span class="sxs-lookup"><span data-stu-id="39860-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="39860-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="39860-106">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
