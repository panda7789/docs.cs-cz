# <a name="compiler-error-cs0131"></a><span data-ttu-id="ae940-103">Chyba kompilátoru CS0131</span><span class="sxs-lookup"><span data-stu-id="ae940-103">Compiler Error CS0131</span></span>
<span data-ttu-id="ae940-104">Levou stranou přiřazení musí být proměnná, vlastnost nebo indexer.</span><span class="sxs-lookup"><span data-stu-id="ae940-104">The left-hand side of an assignment must be a variable, property or indexer</span></span>  
  
 <span data-ttu-id="ae940-105">V příkazu přiřazení se k levé straně přiřadí hodnota pravé strany.</span><span class="sxs-lookup"><span data-stu-id="ae940-105">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="ae940-106">Levá strana musí být proměnná, vlastnost nebo indexer.</span><span class="sxs-lookup"><span data-stu-id="ae940-106">The left-hand side must be a variable, property, or indexer.</span></span>  
  
 <span data-ttu-id="ae940-107">Chcete-li tuto chybu opravit, ujistěte se, že jsou všechny operátory na pravé straně a že levá strana je proměnná, vlastnost nebo indexer.</span><span class="sxs-lookup"><span data-stu-id="ae940-107">To fix this error, make sure that all operators are on the right-hand side and that the left-hand side is a variable, property, or indexer.</span></span> <span data-ttu-id="ae940-108">Další informace naleznete v tématu [příkazy, výrazy a operátory](../programming-guide/statements-expressions-operators/index.md).</span><span class="sxs-lookup"><span data-stu-id="ae940-108">For more information, see [Statements, Expressions, and Operators](../programming-guide/statements-expressions-operators/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="ae940-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="ae940-109">Example</span></span>  
 <span data-ttu-id="ae940-110">Následující ukázka generuje CS0131.</span><span class="sxs-lookup"><span data-stu-id="ae940-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ae940-111">Příklad</span><span class="sxs-lookup"><span data-stu-id="ae940-111">Example</span></span>  
 <span data-ttu-id="ae940-112">K této chybě může dojít také v případě, že se pokusíte provést aritmetické operace na levé straně operátoru přiřazení, jak je uvedeno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="ae940-112">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
