## <a name="example"></a><span data-ttu-id="ecdf6-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="ecdf6-107">Example</span></span>  
 <span data-ttu-id="ecdf6-108">Následující ukázka generuje CS0412:</span><span class="sxs-lookup"><span data-stu-id="ecdf6-108">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
