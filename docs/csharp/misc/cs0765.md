# <a name="compiler-error-cs0765"></a><span data-ttu-id="b60a1-103">Chyba kompilátoru CS0765</span><span class="sxs-lookup"><span data-stu-id="b60a1-103">Compiler Error CS0765</span></span>
<span data-ttu-id="b60a1-104">V stromech výrazů nejde používat částečné metody, které mají jenom definici deklarace nebo odebrané podmíněné metody.</span><span class="sxs-lookup"><span data-stu-id="b60a1-104">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="b60a1-105">I když volání odebrané částečné metody je výraz, není přijatelný výraz ve stromu výrazu.</span><span class="sxs-lookup"><span data-stu-id="b60a1-105">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b60a1-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="b60a1-106">To correct this error</span></span>  
  
1. <span data-ttu-id="b60a1-107">Přidejte implementující deklaraci částečné metody nebo odeberte kód, který způsobuje, že podmíněná metoda bude vyloučena z kompilace.</span><span class="sxs-lookup"><span data-stu-id="b60a1-107">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b60a1-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="b60a1-108">Example</span></span>  
 <span data-ttu-id="b60a1-109">Následující kód generuje CS0765 ve dvou umístěních:</span><span class="sxs-lookup"><span data-stu-id="b60a1-109">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
