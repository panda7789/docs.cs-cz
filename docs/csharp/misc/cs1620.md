# <a name="compiler-error-cs1620"></a><span data-ttu-id="16d9a-103">Chyba kompilátoru CS1620</span><span class="sxs-lookup"><span data-stu-id="16d9a-103">Compiler Error CS1620</span></span>
<span data-ttu-id="16d9a-104">Argument Number se musí předávat s klíčovým slovem klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="16d9a-104">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="16d9a-105">K této chybě dochází, Pokud předáváte argument funkci, která přebírá parametr [ref](../language-reference/keywords/ref.md) nebo [out](../language-reference/keywords/out-parameter-modifier.md) a nezahrnuje `ref` `out` klíčové slovo or v bodě volání, nebo obsahuje nesprávné klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="16d9a-105">This error occurs if you are passing an argument to a function that takes a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="16d9a-106">Text chyby indikuje vhodné klíčové slovo, které se má použít, a který argument způsobil chybu.</span><span class="sxs-lookup"><span data-stu-id="16d9a-106">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="16d9a-107">Následující ukázka generuje CS1620:</span><span class="sxs-lookup"><span data-stu-id="16d9a-107">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
