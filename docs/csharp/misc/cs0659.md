# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="9fd64-103">Upozornění kompilátoru (úroveň 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="9fd64-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="9fd64-104">Class Přepisuje Object. Equals (Object o), ale nepřepisuje Object. GetHashCode ().</span><span class="sxs-lookup"><span data-stu-id="9fd64-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="9fd64-105">Kompilátor zjistil přepsání <xref:System.Object.Equals%2A?displayProperty=nameWithType> metody, ale nepřepisuje <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> metodu.</span><span class="sxs-lookup"><span data-stu-id="9fd64-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="9fd64-106">Přepsání <xref:System.Object.Equals%2A> implikuje, které je také vhodné přepsat <xref:System.Object.GetHashCode%2A> .</span><span class="sxs-lookup"><span data-stu-id="9fd64-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="9fd64-107">Následující kód generuje CS0659:</span><span class="sxs-lookup"><span data-stu-id="9fd64-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
