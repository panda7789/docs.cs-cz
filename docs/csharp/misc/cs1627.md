# <a name="compiler-error-cs1627"></a><span data-ttu-id="8c410-103">Chyba kompilátoru CS1627</span><span class="sxs-lookup"><span data-stu-id="8c410-103">Compiler Error CS1627</span></span>
<span data-ttu-id="8c410-104">Po příkazu yield return se očekával výraz.</span><span class="sxs-lookup"><span data-stu-id="8c410-104">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="8c410-105">K této chybě dochází `yield` , pokud se používá bez výrazu.</span><span class="sxs-lookup"><span data-stu-id="8c410-105">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="8c410-106">Chcete-li se této chybě vyhnout, vložte do příkazu příslušný výraz.</span><span class="sxs-lookup"><span data-stu-id="8c410-106">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="8c410-107">Následující ukázka generuje CS1627:</span><span class="sxs-lookup"><span data-stu-id="8c410-107">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
