# <a name="compiler-error-cs0629"></a><span data-ttu-id="e8fce-103">Chyba kompilátoru CS0629</span><span class="sxs-lookup"><span data-stu-id="e8fce-103">Compiler error CS0629</span></span>

<span data-ttu-id="e8fce-104">Člen podmíněného členu nemůže implementovat člen rozhraní Base Class v typu Type Name.</span><span class="sxs-lookup"><span data-stu-id="e8fce-104">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="e8fce-105">[Podmíněný atribut](../language-reference/attributes/general.md#conditional-attribute) nelze použít při implementaci rozhraní.</span><span class="sxs-lookup"><span data-stu-id="e8fce-105">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="e8fce-106">Následující ukázka generuje CS0629:</span><span class="sxs-lookup"><span data-stu-id="e8fce-106">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
