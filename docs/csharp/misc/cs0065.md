# <a name="compiler-error-cs0065"></a><span data-ttu-id="90df4-103">Chyba kompilátoru CS0065</span><span class="sxs-lookup"><span data-stu-id="90df4-103">Compiler Error CS0065</span></span>

<span data-ttu-id="90df4-104">' Event ': vlastnost události musí obsahovat přistupující objekty Add i Remove.</span><span class="sxs-lookup"><span data-stu-id="90df4-104">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="90df4-105">Událost, která není pole, musí mít obě metody přístupu.</span><span class="sxs-lookup"><span data-stu-id="90df4-105">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="90df4-106">Následující ukázka generuje CS0065:</span><span class="sxs-lookup"><span data-stu-id="90df4-106">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
