# <a name="compiler-error-cs0719"></a>Chyba kompilátoru CS0719
' type ': prvky pole nemohou být statického typu  
  
 Pole statického typu nedává smysl, protože prvky pole jsou instance, ale není možné vytvářet instance statických typů.  
  
 Následující ukázka generuje CS0719:  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
