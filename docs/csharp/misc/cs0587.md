# <a name="compiler-error-cs0587"></a>Chyba kompilátoru CS0587

Vnitřní chyba kompilátoru: fáze ' Stage '

 Zkuste zjistit, jestli se kompilátor nedaří kvůli neschopnosti analyzovat neočekávanou syntaxi. Pokud se nejedná o tento případ, zkuste [nás kontaktovat](/visualstudio/ide/feedback-options).

> [!NOTE]
> Tato chyba kompilátoru se už v Roslyn nepoužívá.
