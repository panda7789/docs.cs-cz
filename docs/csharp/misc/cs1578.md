 <span data-ttu-id="41c68-105">Po direktivě [#line](../language-reference/preprocessor-directives/preprocessor-line.md) je povolen pouze název souboru (v uvozovkách) nebo Jednořádkový komentář.</span><span class="sxs-lookup"><span data-stu-id="41c68-105">After a [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>

```csharp

{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
