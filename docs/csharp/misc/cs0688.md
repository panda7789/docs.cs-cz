## <a name="example"></a><span data-ttu-id="9261f-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="9261f-108">Example</span></span>  
 <span data-ttu-id="9261f-109">Následující ukázka generuje CS0688.</span><span class="sxs-lookup"><span data-stu-id="9261f-109">The following sample generates CS0688.</span></span> <span data-ttu-id="9261f-110">Chcete-li vyřešit upozornění bez úpravy základní třídy, odeberte atribut zabezpečení z přepsání metody.</span><span class="sxs-lookup"><span data-stu-id="9261f-110">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="9261f-111">Tím se problém se zabezpečením nevyřeší.</span><span class="sxs-lookup"><span data-stu-id="9261f-111">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
