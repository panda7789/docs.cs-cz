 <span data-ttu-id="b8681-105">K této chybě dochází, Pokud importovaný typ a importovaný vnořený obor názvů mají stejný plně kvalifikovaný název.</span><span class="sxs-lookup"><span data-stu-id="b8681-105">This error occurs when an imported type and an imported nested namespace have the same fully qualified name.</span></span> <span data-ttu-id="b8681-106">Pokud je odkazováno na tento název, kompilátor nemůže rozlišovat mezi nimi.</span><span class="sxs-lookup"><span data-stu-id="b8681-106">When that name is referenced, the compiler is unable to distinguish between the two.</span></span> <span data-ttu-id="b8681-107">Pokud můžete změnit importovaný zdrojový kód, můžete chybu vyřešit tak, že změníte název buď typ, nebo obor názvů tak, aby obě byla v rámci sestavení jedinečná.</span><span class="sxs-lookup"><span data-stu-id="b8681-107">If you can change the imported source code, you can resolve the error by changing the name of either the type or the namespace so that both are unique within the assembly.</span></span>  
  
 <span data-ttu-id="b8681-108">Následující kód generuje chybu CS0434.</span><span class="sxs-lookup"><span data-stu-id="b8681-108">The following code generates error CS0434.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b8681-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="b8681-109">Example</span></span>  
 <span data-ttu-id="b8681-110">Tento kód vytvoří první kopii typu s totožným plně kvalifikovaným názvem.</span><span class="sxs-lookup"><span data-stu-id="b8681-110">This code creates the first copy of the type with the identical fully qualified name.</span></span>

```csharp  
namespace TypeBindConflicts
{  
    namespace NsImpAggPubImp
    {  
        public class X { }  
    }  
}  
```
