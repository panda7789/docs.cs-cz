## <a name="example"></a><span data-ttu-id="9a40b-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="9a40b-106">Example</span></span>  
 <span data-ttu-id="9a40b-107">Následující ukázka generuje CS0730.</span><span class="sxs-lookup"><span data-stu-id="9a40b-107">The following sample generates CS0730.</span></span> <span data-ttu-id="9a40b-108">Skládá se ze dvou zdrojových souborů.</span><span class="sxs-lookup"><span data-stu-id="9a40b-108">It consists of two source files.</span></span> <span data-ttu-id="9a40b-109">Nejprve zkompilujte soubor knihovny `CS0730a.cs` a zkompilujte soubor `CS0730.cs` odkazující na soubor knihovny.</span><span class="sxs-lookup"><span data-stu-id="9a40b-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
