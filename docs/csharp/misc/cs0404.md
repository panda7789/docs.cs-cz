# <a name="compiler-error-cs0404"></a>Chyba kompilátoru CS0404
Neočekávaná hodnota <: atributy nemůžou být obecné.  
  
 Parametry obecného typu nejsou v atributech povoleny. Odeberte parametr typu a lomené závorky.  
  
 Následující ukázka generuje CS0404:  
  
```csharp  
// CS0404.cs  
[MyAttrib<int>]  // CS0404  
class C  
{  
   public static void Main()  
   {  
  
   }  
}  
```
