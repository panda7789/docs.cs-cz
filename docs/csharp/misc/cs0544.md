 <span data-ttu-id="60a19-105">Došlo k pokusu o přepsání datového typu, který není vlastností, jako [vlastnost](../programming-guide/classes-and-structs/properties.md), která není povolena.</span><span class="sxs-lookup"><span data-stu-id="60a19-105">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

```csharp

{  
   public int i;  
}

{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {

      {  
         return 0;  
      }  
   }  
}  
```
