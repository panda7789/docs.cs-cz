# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="c8f7d-103">Upozornění kompilátoru (úroveň 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="c8f7d-103">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="c8f7d-104">Možná nezamýšlené porovnání odkazů; Chcete-li porovnat hodnoty, přetypujte levou stranu na typ "typ".</span><span class="sxs-lookup"><span data-stu-id="c8f7d-104">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="c8f7d-105">Kompilátor provádí porovnání odkazů.</span><span class="sxs-lookup"><span data-stu-id="c8f7d-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="c8f7d-106">Pokud chcete porovnat hodnotu řetězců, přetypujte levou stranu výrazu na `type` .</span><span class="sxs-lookup"><span data-stu-id="c8f7d-106">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="c8f7d-107">Následující ukázka generuje CS0252:</span><span class="sxs-lookup"><span data-stu-id="c8f7d-107">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
