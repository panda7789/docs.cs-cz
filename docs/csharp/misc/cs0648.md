# <a name="compiler-error-cs0648"></a>Chyba kompilátoru CS0648
Type je typ, který jazyk nepodporuje.  
  
 Metadata vygenerovaná z jiného jazyka, případně C++, obsahovala typ, který nebyl označen jako spravovaný. Informace o tomto typu jsou zahrnuty v metadatech, ale tento typ není k dispozici pro programy napsané v jazyce C#.
