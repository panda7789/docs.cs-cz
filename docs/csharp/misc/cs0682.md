# <a name="compiler-error-cs0682"></a>Chyba kompilátoru CS0682
' typ1 ' nemůže implementovat ' typ2 ', protože není jazykem podporován  
  
 K této chybě dochází při pokusu o implementaci rozhraní napsaného v jiném jazyce a kompilátor nepodporuje rozhraní.
