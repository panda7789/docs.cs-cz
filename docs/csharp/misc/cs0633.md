## <a name="example"></a><span data-ttu-id="e4795-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="e4795-106">Example</span></span>  
 <span data-ttu-id="e4795-107">Tento příklad ilustruje CS0633 <xref:System.Diagnostics.ConditionalAttribute>pomocí .</span><span class="sxs-lookup"><span data-stu-id="e4795-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="e4795-108">Následující ukázka generuje CS0633.</span><span class="sxs-lookup"><span data-stu-id="e4795-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="e4795-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="e4795-109">Example</span></span>  
 <span data-ttu-id="e4795-110">Tento příklad ilustruje CS0633 <xref:System.Runtime.CompilerServices.IndexerNameAttribute>pomocí .</span><span class="sxs-lookup"><span data-stu-id="e4795-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]
   {
      get { return i; }
   }  
}  
```
