# <a name="compiler-error-cs1021"></a><span data-ttu-id="884e9-103">Chyba kompilátoru CS1021</span><span class="sxs-lookup"><span data-stu-id="884e9-103">Compiler Error CS1021</span></span>

<span data-ttu-id="884e9-104">Integrální konstanta je moc velká.</span><span class="sxs-lookup"><span data-stu-id="884e9-104">Integral constant is too large</span></span>  
  
<span data-ttu-id="884e9-105">Hodnota reprezentovaná celočíselným literálem je větší než <xref:System.UInt64.MaxValue?displayProperty=nameWithType> .</span><span class="sxs-lookup"><span data-stu-id="884e9-105">A value represented by an integer literal is greater than <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span></span>  
  
<span data-ttu-id="884e9-106">Následující ukázka generuje CS1021:</span><span class="sxs-lookup"><span data-stu-id="884e9-106">The following sample generates CS1021:</span></span>  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

<span data-ttu-id="884e9-107">Následující kód generuje také CS1021:</span><span class="sxs-lookup"><span data-stu-id="884e9-107">The following code also generates CS1021:</span></span>

```csharp

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```
