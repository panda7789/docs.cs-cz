# <a name="compiler-error-cs0175"></a><span data-ttu-id="b03e5-103">Chyba kompilátoru CS0175</span><span class="sxs-lookup"><span data-stu-id="b03e5-103">Compiler Error CS0175</span></span>
<span data-ttu-id="b03e5-104">Použití klíčového slova Base není v tomto kontextu platné.</span><span class="sxs-lookup"><span data-stu-id="b03e5-104">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="b03e5-105">Klíčové slovo [Base](../language-reference/keywords/base.md) se musí použít k určení konkrétního člena základní třídy.</span><span class="sxs-lookup"><span data-stu-id="b03e5-105">The [base](../language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="b03e5-106">Další informace naleznete v tématu [konstruktory](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="b03e5-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="b03e5-107">Následující ukázka generuje CS0175:</span><span class="sxs-lookup"><span data-stu-id="b03e5-107">The following sample generates CS0175:</span></span>  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
