# <a name="compiler-error-cs1664"></a>Chyba kompilátoru CS1664
Vyrovnávací paměť pevné velikosti s délkou Length a typem Type je moc velká.  
  
 Maximální velikost vyrovnávací paměti pevné velikosti (jak je určena délkou vynásobené velikostí elementu) je 2 ^ 31 = 268435455.
