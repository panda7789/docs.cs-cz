# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="a628e-103">Upozornění kompilátoru (úroveň 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="a628e-103">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="a628e-104">člen: v zapečetěné třídě je deklarovaný nový chráněný člen.</span><span class="sxs-lookup"><span data-stu-id="a628e-104">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="a628e-105">[Zapečetěná](../language-reference/keywords/sealed.md) třída nemůže zavést [chráněného](../language-reference/keywords/protected.md) člena, protože žádná jiná třída nebude moci dědit ze `sealed` třídy a použít `protected` člena.</span><span class="sxs-lookup"><span data-stu-id="a628e-105">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="a628e-106">Následující ukázka generuje CS0628:</span><span class="sxs-lookup"><span data-stu-id="a628e-106">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
