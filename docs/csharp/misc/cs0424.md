<span data-ttu-id="4d727-104">' class ': třída s atributem atributem ComImport nemůže specifikovat základní třídu</span><span class="sxs-lookup"><span data-stu-id="4d727-104">'class': a class with the ComImport attribute cannot specify a base class</span></span>

 <span data-ttu-id="4d727-107">Následující ukázka generuje CS0424:</span><span class="sxs-lookup"><span data-stu-id="4d727-107">The following sample generates CS0424:</span></span>
