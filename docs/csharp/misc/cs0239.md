 <span data-ttu-id="80578-105">Člen nemůže [přepsat](../language-reference/keywords/override.md) [zapečetěný](../language-reference/keywords/sealed.md) zděděný člen.</span><span class="sxs-lookup"><span data-stu-id="80578-105">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="80578-106">Další informace naleznete v tématu [checked a unchecked](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="80578-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>

```csharp  
// CS0239.cs

{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}

{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
