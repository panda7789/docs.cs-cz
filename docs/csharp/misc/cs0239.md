# <a name="compiler-error-cs0239"></a>Chyba kompilátoru CS0239
člen: nejde přepsat zděděný člen zděděný člen, protože je zapečetěný.  
  
 Člen nemůže [přepsat](../language-reference/keywords/override.md) [zapečetěný](../language-reference/keywords/sealed.md) zděděný člen. Další informace naleznete v tématu [checked a unchecked](../language-reference/keywords/checked-and-unchecked.md).  
  
 Následující ukázka generuje CS0239:  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
