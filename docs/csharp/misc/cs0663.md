# <a name="compiler-error-cs0663"></a>Chyba kompilátoru CS0663
Nelze definovat přetížené metody, které se liší pouze parametrem ref a out.  
  
 Metody, které se liší pouze při jejich použití [v](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) a [out](../language-reference/keywords/out-parameter-modifier.md) u parametru nejsou povoleny.  
  
 Následující ukázka generuje CS0663:  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
