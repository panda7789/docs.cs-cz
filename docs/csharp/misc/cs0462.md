# <a name="compiler-error-cs0462"></a><span data-ttu-id="6b00b-102">Chyba kompilátoru CS0462</span><span class="sxs-lookup"><span data-stu-id="6b00b-102">Compiler Error CS0462</span></span>
<span data-ttu-id="6b00b-103">Zděděné členy 'member1' a 'member2' mají stejný podpis v typu 'type', takže nemohou být přepsány</span><span class="sxs-lookup"><span data-stu-id="6b00b-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="6b00b-104">Tato chyba vzniká zavedením generik.</span><span class="sxs-lookup"><span data-stu-id="6b00b-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="6b00b-105">Za normálních okolností nelze mít dvě verze metody ve třídě se stejným podpisem.</span><span class="sxs-lookup"><span data-stu-id="6b00b-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="6b00b-106">Ale s obecnými typy můžete zadat obecnou metodu, která může duplikovat jinou metodu, pokud je vytvořena instance s určitým typem.</span><span class="sxs-lookup"><span data-stu-id="6b00b-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6b00b-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="6b00b-107">Example</span></span>  
 <span data-ttu-id="6b00b-108">Při `C<int>` vytvoření instance jsou vytvořeny dvě verze `F` metody se stejným podpisem, takže `D` přepsání ve třídě nemůže rozhodnout, na kterou z nich se má přepsání použít.</span><span class="sxs-lookup"><span data-stu-id="6b00b-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="6b00b-109">Následující ukázka generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="6b00b-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
