# <a name="compiler-error-cs0462"></a><span data-ttu-id="55c88-103">Chyba kompilátoru CS0462</span><span class="sxs-lookup"><span data-stu-id="55c88-103">Compiler Error CS0462</span></span>
<span data-ttu-id="55c88-104">Zděděné členy ' člen1 ' a ' member2 ' mají stejný podpis v typu ' type ', takže je nelze přepsat</span><span class="sxs-lookup"><span data-stu-id="55c88-104">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="55c88-105">Tato chyba se vyskytne při zavádění obecných typů.</span><span class="sxs-lookup"><span data-stu-id="55c88-105">This error arises with the introduction of generics.</span></span> <span data-ttu-id="55c88-106">Za normálních okolností nemůžete mít ve třídě se stejnou signaturou dvě verze metody.</span><span class="sxs-lookup"><span data-stu-id="55c88-106">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="55c88-107">Ale s obecnými typy můžete určit obecnou metodu, která by mohla duplikovat jinou metodu, pokud je vytvořena instance konkrétního typu.</span><span class="sxs-lookup"><span data-stu-id="55c88-107">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="55c88-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="55c88-108">Example</span></span>  
 <span data-ttu-id="55c88-109">Při `C<int>` vytvoření instance se vytvoří dvě verze metody `F` se stejnou signaturou, takže přepsání ve třídě `D` nemůže rozhodnout, která z nich se má použít pro přepsání.</span><span class="sxs-lookup"><span data-stu-id="55c88-109">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="55c88-110">Následující ukázka generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="55c88-110">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
