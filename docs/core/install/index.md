# <a name="download-and-install-net-core"></a>Stažení a instalace rozhraní .NET Core

Tento článek obsahuje informace o stažení a instalaci jádra .NET Core. Existují dvě části .NET Core, **runtime,** který se používá ke spuštění aplikací a **Sada SDK,** která se používá k vytváření aplikací. Sada SDK obsahuje dobu runtime.

- Pokud jste uživatel, který potřebuje .NET Core ke spuštění aplikací, přečtěte si informace [o instalaci rozhraní .NET Core Runtime](runtime.md).

- Pokud jste vývojář, který k vytváření aplikací potřebuje rozhraní .NET Core, [přečtěte si informace o instalaci sady .NET Core SDK](sdk.md).

> [!NOTE]
> Sada .NET Core SDK obsahuje dobu runtime.

## <a name="dependencies"></a>Závislosti

.NET Core je podporován ve Windows, Linuxu a macOS. Úplný seznam požadavků naleznete v tématu [Podporované operační systémy](dependencies.md).
