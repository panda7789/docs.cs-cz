Pokud chcete vyvíjet aplikace .NET, nainstalujte sadu SDK (která obsahuje modul runtime). Nebo, pokud potřebujete pouze spustit aplikace, nainstalujte modul runtime. Pokud instalujete modul runtime, doporučujeme nainstalovat modul **runtime ASP.NET Core** , protože obsahuje modul runtime .NET Core i ASP.NET Core.

Pokud jste již nainstalovali sadu SDK nebo modul runtime, použijte `dotnet --list-sdks` `dotnet --list-runtimes` k zobrazení nainstalovaných verzí příkazy a. Další informace najdete v tématu [Jak ověřit, zda je rozhraní .NET Core již nainstalováno](../how-to-detect-installed-versions.md).
