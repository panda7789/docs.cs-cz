# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Přerušující změny v model Windows Forms (.NET Framework do .NET Core)

Podpora WPF a model Windows Forms se přidala do .NET Core ve verzi 3,0. Pokud migrujete model Windows Forms nebo Windows Presentation Foundation aplikaci z .NET Framework do .NET Core, můžou se na vaši aplikaci projevit zásadní změny uvedené v tomto článku.

Zásadní změny jsou seskupené podle verze .NET Core, ve které byly zavedeny.
