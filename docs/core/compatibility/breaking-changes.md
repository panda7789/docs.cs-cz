## <a name="github-issues-and-announcements"></a>Problémy a oznámení githubu

Můžete také zobrazit jednotlivé problémy, které podrobně popisují nejnovější změny zavedené v .NET Core v následujících úložištích GitHub:

- Pro rozhraní .NET Core je úložiště [dotnet/docs.](https://github.com/dotnet/docs/issues?q=is%3Aissue+label%3Abreaking-change)
- Pro ASP.NET Core, úložiště [aspnet/Oznámení.](https://github.com/aspnet/Announcements/issues?q=is%3Aissue+is%3Aopen+label%3A%22Breaking+change%22+label%3A3.0.0)
- Pro core entity frameworku je úložiště [dotnet/efcore.](https://github.com/dotnet/efcore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change)
