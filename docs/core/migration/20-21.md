# <a name="migrate-from-net-core-20-to-21"></a>Migrace z .NET Core 2,0 na 2,1

Tento článek popisuje základní kroky pro migraci vaší aplikace .NET Core 2,0 na 2,1. Pokud chcete aplikaci ASP.NET Core migrovat na 2,1, přečtěte si téma [migrace z ASP.NET Core 2,0 na 2,1](/aspnet/core/migration/20_21).

Přehled nových funkcí v rozhraní .NET Core 2,1 najdete v tématu [co je nového v rozhraní .NET core 2,1](../whats-new/dotnet-core-2-1.md).

## <a name="update-the-project-file-to-use-21-versions"></a>Aktualizovat soubor projektu tak, aby používal verze 2,1

- Otevřete soubor projektu (soubor \*. csproj, \*. vbproj nebo \*. fsproj).

- Změňte [cílovou hodnotu rozhraní .NET Framework](../../standard/frameworks.md) z `netcoreapp2.0` na `netcoreapp2.1`. Cílová architektura je definována prvkem `<TargetFramework>` nebo `<TargetFrameworks>`.

  Změňte například `<TargetFramework>netcoreapp2.0</TargetFramework>` na `<TargetFramework>netcoreapp2.1</TargetFramework>`.

- Odeberte `<DotNetCliToolReference>` odkazy na nástroje, které jsou součástí sady SDK .NET Core 2,1 (v 2.1.300 nebo novější). Mezi tyto odkazy patří:

  - [dotnet-Watch](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-watch/README.md) (Microsoft. dotnet. sledovací proces. Tools)
  - [dotnet-User-tajná](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-user-secrets/README.md) (Microsoft. Extensions. SecretManager. Tools)
  - [dotnet-sql-cache](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-sql-cache/README.md) (Microsoft.Extensions.Caching.SqlConfig.Tools)
  - [dotnet-ef](/ef/core/miscellaneous/cli/dotnet) (Microsoft.EntityFrameworkCore.Tools.DotNet)
  
  V předchozích verzích .NET Core SDK odkaz na jeden z těchto nástrojů v souboru projektu vypadá podobně jako v následujícím příkladu:

  ```xml
  <DotNetCliToolReference Include="Microsoft.EntityFrameworkCore.Tools.DotNet" Version="2.0.0" />
  ```

  Vzhledem k tomu, že .NET Core SDK už tuto položku nepoužívá, zobrazí se upozornění podobné následujícímu, pokud stále máte odkazy na jeden z těchto nástrojů v projektu:
  
  `The tool 'Microsoft.EntityFrameworkCore.Tools.DotNet' is now included in the .NET Core SDK. Here is information on resolving this warning.`
  
  Odebráním odkazů na `<DotNetCliToolReference>` pro tyto nástroje ze souboru projektu se tento problém vyřeší.
