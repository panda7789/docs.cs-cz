# <a name="f-compiler-messages"></a>F#zprávy kompilátoru

Tato část podrobně popisuje chyby kompilátoru a upozornění, F# které kompilátor vygeneruje pro určité konstrukce. Výchozí sady chyb lze změnit pomocí:

- Zpracování specifických upozornění, jako kdyby došlo k chybám pomocí možnosti kompilátoru `-warnaserror+`,

- Ignorování konkrétních upozornění pomocí možnosti kompilátoru `-nowarn`

Pokud v této části ještě není zaznamenáno určité upozornění nebo chyba, přejdete na konec této stránky a odešlete zpětnou vazbu, která obsahuje číslo nebo text chyby.
