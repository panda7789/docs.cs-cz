```fsharp
// Defines a function that takes a name and produces a greeting.
```

## <a name="rich-data-types"></a><span data-ttu-id="ab20a-117">Formátované datové typy</span><span class="sxs-lookup"><span data-stu-id="ab20a-117">Rich data types</span></span>

<span data-ttu-id="ab20a-118">Datové typy, jako jsou [záznamy](./language-reference/records.md) a [rozlišené sjednocení](./language-reference/discriminated-unions.md) , umožňují reprezentovat složitá data a domény.</span><span class="sxs-lookup"><span data-stu-id="ab20a-118">Data types such as [Records](./language-reference/records.md) and [Discriminated Unions](./language-reference/discriminated-unions.md) let you represent complex data and domains.</span></span>

```fsharp
// Group data with Records
type SuccessfulWithdrawal = {
    Amount: decimal
    Balance: decimal
}

type FailedWithdrawal = {
    Amount: decimal
    Balance: decimal
    IsOverdraft: bool
}

// Use discriminated unions to represent data of 1 or more forms
type WithdrawalResult =
    | Success of SuccessfulWithdrawal
    | InsufficientFunds of FailedWithdrawal
    | CardExpired of System.DateTime
    | UndisclosedFailure
```

<span data-ttu-id="ab20a-119">F#záznamy a rozlišené sjednocení jsou ve výchozím nastavení nenulové, neměnné a srovnatelné, takže je velmi snadné je používat.</span><span class="sxs-lookup"><span data-stu-id="ab20a-119">F# records and discriminated unions are non-null, immutable, and comparable by default, making them very easy to use.</span></span>

```fsharp
// Returns a WithdrawalResult
let withdrawMoney amount = // Implementation elided

let handleWithdrawal amount =
    let w = withdrawMoney amount

    // The F# compiler enforces accounting for each case!
    match w with
    | Success s -> printfn "Successfully withdrew %f" s.Amount
```
