# <a name="road-to-modern-applications-based-on-containers"></a>Cesta k moderním aplikacím založeným na kontejnerech

Pravděpodobně čtete tuto knihu, protože plánujete vývoj nových aplikací nebo vyhodnocujete dopad používání Dockeru, Kontejnery a nové přístupy, jako jsou mikroslužby ve vaší společnosti.

Přijetí nových paradigmat vývoje musí být před zahájením projektu přijato s opatrností, aby bylo možné posoudit dopad na vývojové týmy, rozpočet nebo infrastrukturu.

Společnost Microsoft pracuje na bohatých pokynech, ukázkových aplikacích a sadě e-knih, které vám pomohou učinit informované rozhodnutí a vést váš tým úspěšným vývojem, nasazením a provozem nových aplikací.

Tato kniha patří do sady průvodců společnosti Microsoft, které pokrývají mnoho potřeb a výzev, kterým budete čelit během procesu vývoje nových moderních aplikací založených na kontejnerech.

Další e-knihy Microsoftu související s kontejnery Dockeru najdete v seznamu níže:

- **Modernizace stávajících aplikací .NET pomocí cloudu Azure a kontejnerů s Windows** \
