<span data-ttu-id="28679-175">Pojďme začít s některým implementačním kódem pro rozhraní sběrnice událostí a možnými implementacemi pro účely průzkumu.</span><span class="sxs-lookup"><span data-stu-id="28679-175">Let’s start with some implementation code for the event bus interface and possible implementations for exploration purposes.</span></span> <span data-ttu-id="28679-176">Rozhraní by mělo být obecné a jednoduché, jako v následujícím rozhraní.</span><span class="sxs-lookup"><span data-stu-id="28679-176">The interface should be generic and straightforward, as in the following interface.</span></span>

```csharp
public interface IEventBus
{
    void Publish(IntegrationEvent @event);

    void Subscribe<T, TH>()

    void SubscribeDynamic<TH>(string eventName)
        where TH : IDynamicIntegrationEventHandler;

    void Unsubscribe<T, TH>()
        where TH : IIntegrationEventHandler<T>

}
```
