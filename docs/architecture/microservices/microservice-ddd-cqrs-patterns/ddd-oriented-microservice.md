### <a name="the-application-layer"></a><span data-ttu-id="8466d-179">Aplikační vrstva</span><span class="sxs-lookup"><span data-stu-id="8466d-179">The application layer</span></span>

<span data-ttu-id="8466d-180">Pohybující se na aplikační vrstvy, můžeme opět citovat Eric Evans kniha [Domény Driven Design:](https://domainlanguage.com/ddd/)</span><span class="sxs-lookup"><span data-stu-id="8466d-180">Moving on to the application layer, we can again cite Eric Evans's book [Domain Driven Design](https://domainlanguage.com/ddd/):</span></span>
