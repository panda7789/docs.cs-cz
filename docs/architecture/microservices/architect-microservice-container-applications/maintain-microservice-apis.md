<span data-ttu-id="e2497-116">Někdy ale budete muset v rozhraní API služby udělat významné a nekompatibilní změny.</span><span class="sxs-lookup"><span data-stu-id="e2497-116">However, sometimes you need to make major and incompatible changes to a service API.</span></span> <span data-ttu-id="e2497-117">Vzhledem k tomu, že pravděpodobně nebudete moci vynutit, aby klientské aplikace nebo služby class="sxs-lookup"><span data-stu-id="e2497-118">If you're using an HTTP-based mechanism such as REST, one approach is to embed the API version number in the URL or into an HTTP header.</span></span> <span data-ttu-id="e2497-119">Pak se můžete rozhodnout mezi implementací obou verzí služby současně v rámci stejné instance služby nebo nasazením různých instancí, které jednotlivé verze rozhraní API
