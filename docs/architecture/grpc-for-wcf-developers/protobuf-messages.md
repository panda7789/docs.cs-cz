## <a name="declaring-a-message"></a><span data-ttu-id="855fc-106">Deklarace zprávy</span><span class="sxs-lookup"><span data-stu-id="855fc-106">Declaring a message</span></span>

<span data-ttu-id="855fc-107">Ve službě WCF může být `Stock` třída pro obchodní obchodní aplikace, jak je uvedeno v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="855fc-107">In WCF, a `Stock` class for a stock market trading application might be defined like the following example:</span></span>

```csharp
namespace TraderSys

    [DataContract]
    public class Stock
    {
        [DataMember]
        public int Id { get; set;}
        [DataMember]
        public string Symbol { get; set;}
        [DataMember]
        public string DisplayName { get; set;}
        [DataMember]
        public int MarketId { get; set; }
    }
}
```

<span data-ttu-id="855fc-108">Chcete-li implementovat ekvivalentní třídu v Protobuf, musí být deklarována v souboru `.proto`.</span><span class="sxs-lookup"><span data-stu-id="855fc-108">To implement the equivalent class in Protobuf, it must be declared in the `.proto` file.</span></span> <span data-ttu-id="855fc-109">Kompilátor `protoc` pak vygeneruje třídu .NET jako součást procesu sestavení.</span><span class="sxs-lookup"><span data-stu-id="855fc-109">The `protoc` compiler will then generate the .NET class as part of the build process.</span></span>

```protobuf
syntax "proto3";

option csharp_namespace = "TraderSys";

    int32 id = 1;
    string symbol = 2;
    string display_name = 3;
    int32 market_id = 4;

}  
```

<span data-ttu-id="855fc-110">První řádek deklaruje použitou verzi syntaxe.</span><span class="sxs-lookup"><span data-stu-id="855fc-110">The first line declares the syntax version being used.</span></span> <span data-ttu-id="855fc-111">Verze 3 jazyka byla vydána v 2016 a jedná se o doporučenou verzi pro služby gRPC Services.</span><span class="sxs-lookup"><span data-stu-id="855fc-111">Version 3 of the language was released in 2016 and is the recommended version for gRPC services.</span></span>

<span data-ttu-id="855fc-112">`option csharp_namespace` řádek určuje obor názvů, který se má použít pro generované C# typy.</span><span class="sxs-lookup"><span data-stu-id="855fc-112">The `option csharp_namespace` line specifies the namespace to be used for the generated C# types.</span></span> <span data-ttu-id="855fc-113">Tato možnost bude ignorována, pokud je soubor `.proto` kompilován pro jiné jazyky.</span><span class="sxs-lookup"><span data-stu-id="855fc-113">This option will be ignored when the `.proto` file is compiled for other languages.</span></span> <span data-ttu-id="855fc-114">Pro soubory Protobuf je běžné, že obsahují možnosti specifické pro jazyk pro několik jazyků.</span><span class="sxs-lookup"><span data-stu-id="855fc-114">It is common for Protobuf files to contain language-specific options for several languages.</span></span>

<span data-ttu-id="855fc-115">Definice `Stock` zprávy určuje čtyři pole, každý s typem, název a číslo pole.</span><span class="sxs-lookup"><span data-stu-id="855fc-115">The `Stock` message definition specifies four fields, each with a type, a name, and a field number.</span></span>
