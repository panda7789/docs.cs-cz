### <a name="exception-message-has-changed-for-failed-datacontract-serialization-in-case-of-an-unknown-type"></a>V případě neznámého typu se změnila zpráva výjimky pro neúspěšnou serializaci kontraktu DataContract.

#### <a name="details"></a>Podrobnosti

Počínaje .NET Framework 4,6 byla vyjasnění zpráva o výjimce, <xref:System.Runtime.Serialization.DataContractSerializer?displayProperty=fullName> Pokud <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer?displayProperty=fullName> došlo k serializaci nebo deserializaci kvůli chybějícím známým typům.

#### <a name="suggestion"></a>Návrh

Aplikace by neměly záviset na konkrétních zprávách o výjimkách. Pokud aplikace na této zprávě závisí, aktualizujte ji tak, aby očekávala novou zprávu, nebo (nejlépe) ji změňte tak, aby byla závislá jenom na typu výjimky.

| Name    | Hodnota       |
|:--------|:------------|
| Rozsah   |Edge|
|Verze|4.6|
|Typ|Modul runtime
