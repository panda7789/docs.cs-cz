> [!IMPORTANT]
> Všechny verze rozhraní .NET Framework od rozhraní .NET Framework 4 jsou aktuální aktualizace, takže v systému může být k dispozici pouze jedna verze 4.x. Kromě toho jsou v některých verzích operačního systému Windows předinstalovány konkrétní verze rozhraní .NET Framework. To znamená, že:
>
> - Pokud je v počítači již nainstalována novější verze 4.x, nelze nainstalovat předchozí verzi 4.x.
> - Pokud je operační systém dodáván s předinstalovaným s konkrétní verzí rozhraní .NET Framework, nelze do stejného počítače nainstalovat předchozí verzi 4.x.
> - Pokud nainstalujete novější verzi, není nutné nejprve odinstalovat předchozí verzi.
