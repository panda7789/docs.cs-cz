### <a name="types-in-microsoftvisualbasicapplicationservices-namespace-not-available"></a>Typy v oboru názvů Microsoft. VisualBasic. ApplicationServices nejsou k dispozici.

Typy v oboru názvů <xref:Microsoft.VisualBasic.ApplicationServices?displayProperty=fullName> nejsou k dispozici.

#### <a name="version-introduced"></a>Představená verze

.NET Core 3,0 Preview 8

#### <a name="change-description"></a>Změnit popis

Typy v oboru názvů <xref:Microsoft.VisualBasic.ApplicationServices?displayProperty=fullName> byly k dispozici v některých verzích .NET Core 3,0 Preview. Již nejsou od verze .NET Core 3,0 Preview 9 k dispozici.

Typy byly odebrány, aby nedocházelo k zbytečným závislostem sestavení nebo k zásadním změnám v následujících verzích.

#### <a name="recommended-action"></a>Doporučená akce

Pokud váš kód závisí na použití <xref:Microsoft.VisualBasic.ApplicationServices> typy a jejich členů, můžete použít odpovídající typ nebo člen v knihovně třídy .NET. Například někteří <xref:System.Environment?displayProperty=nameWithType> a <xref:System.Security.Principal.WindowsIdentity?displayProperty=nameWithType> členové poskytují ekvivalentní funkce pro vlastnosti třídy <xref:Microsoft.VisualBasic.ApplicationServices.User?displayProperty=nameWithType>.

#### <a name="category"></a>Kategorie

Visual Basic
